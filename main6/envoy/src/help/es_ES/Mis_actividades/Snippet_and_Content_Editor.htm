Editor de fragmentos y contenido

Resumen:

Permite añadir y eliminar fragmentos en las páginas traducidas.  

Nombre de la página

Nombre con la ruta de acceso completa a la página que está corrigiendo.

Modo de visualización

Permite ver la página con el código HTML interpretado (Vista previa) o como código HTML puro (texto).

Icono Añadir fragmento: Púlselo dos veces para añadir un fragmento en este lugar de la página. Se abrirá el cuadro de diálogo Seleccionar fragmento, desde el que puede seleccionar el fragmento que desea añadir en este punto.
