Seleccionar fragmento

Resumen:

Enumera todos los fragmentos que pueden copiarse y modificarse.

Cuadro de texto Seleccionar fragmento

Aquí se muestran todos los fragmentos que pueden copiarse o modificarse. Si el nombre del fragmento no va seguido de un número, se trata de un fragmento genérico que, como es de sólo lectura, sólo puede copiarse. Una vez que haya seleccionado y copiado el fragmento genérico con el botón Copiar, podrá modificar la copia según su contexto actual.

Descripción

Descripción de sólo lectura del fragmento seleccionado. Pulse Modificar para cambiar la descripción.

Valor

Texto HTML de sólo lectura del fragmento. Pulse Modificar para cambiar el texto.
