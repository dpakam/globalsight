Editor de segmentos (imágenes)

Resumen:

Permite cambiar la ruta de acceso a una imagen y enviar una imagen que la sustituya.

Origen

La dirección web original y una vista previa de la imagen.

Destino

Dirección web: Aquí puede cambiar la ruta de acceso a la imagen.

Examinar, Enviar:  O busque una imagen en una unidad de disco local y envíela al servidor para reemplazar la imagen existente.

Resultados de la comparación

Presenta las coincidencia parciales que se encontraron en la memoria de traducción para este segmento.  Junto a cada coincidencia se muestra un porcentaje de semejanza entre el texto encontrado y el original.
