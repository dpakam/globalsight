Editor de segmentos (texto)

Resumen:

Permite traducir un segmento.

Origen

El texto original que se debe traducir.

Destino

Introduzca la traducción o las correcciones en este cuadro.

Coincidencias terminológicas

Pulse una coincidencia para copiarla al final del área de destino; pulse con el botón derecho del ratón para examinar la base de datos terminológica de este proyecto (vea Examinar la base de datos terminológica).

Resultados de la comparación

Presenta las coincidencia parciales que se encontraron en la memoria de traducción para este segmento.  Junto a cada coincidencia se muestra un porcentaje de semejanza entre el texto encontrado y el original.

Versiones del segmento

El segmento tal como apareció en cada una de las actividades anteriores.

Guardar

Guarda el trabajo hecho en este segmento.

Es posible que aparezcan errores relacionados con los marcadores de posición (vea Errores de marcadores de posición).

 

Atajo de teclado: Ctrl+S

Segmento anterior

Pasa al segmento traducible anterior.

Se le pedirá que guarde el segmento en el que está trabajando.

 

Atajo de teclado: Ctrl+Re Pág

Siguiente segmento

Pasa al próximo segmento traducible.

Se le pedirá que guarde el segmento en el que está trabajando.

 

Atajo de teclado: Ctrl+Av Pág

Revertir

Permite volver a la última versión traducida del segmento.

Detalles

Contiene información sobre una segmento que el servicio de asistencia al cliente de GlobalSight puede emplear para resolver posibles problemas en él.

Etiquetas abreviadas/detalladas

Permite alternar entre las etiquetas abreviadas y las etiquetas detalladas (vea Marcadores de posición)

Cerrar

Cierra esta ventana.

Se le pedirá que guarde el segmento en el que está trabajando.

Atajos de teclado

 

Utilice los siguientes atajos de teclado para navegar más rápidamente por el Editor de segmentos:

Ctrl+S: Guardar segmento

Ctrl+Re Pág: Segmento anterior

Ctrl+Av Pág: Próximo segmento

Ctrl+B: Negrita

Ctrl+I: Cursiva

Ctrl+U: Subrayado

Ctrl+Flecha izquierda: Palabra a la izquierda

Ctrl+Flecha derecha: Palabra a la derecha

Mayús+Ctrl+Flecha izquierda: Seleccionar palabra a la izquierda

Mayús+Ctrl+Flecha derecha: Seleccionar palabra a la derecha
