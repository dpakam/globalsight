Editor principal

Resumen:

Permite traducir el contenido con el editor del sistema.

{segmento}

  • Pulse cualquier segmento para modificarlo (vea Editor de segmentos).

Debe aceptar una actividad para poder editar algo.  Si el editor de segmentos no se abre tras haber seleccionado un segmento, cierre el editor principal, acepte la actividad y luego pulse el nombre de una página para volver a abrir el editor principal.

Lista

Muestra los segmentos en una lista ordenada

Texto

Muestra los segmentos en una lista con códigos que no pueden traducirse, como HTML.

Vista previa

(Sólo HTML) muestra una vista previa de la página, los segmentos se pueden traducir todavía.

Modos de presentación

Permite ver el texto original y el traducido dispuestos verticalmente.

Permite ver el texto original y el traducido dispuestos horizontalmente.

Permite ver sólo el texto original o el traducido (pulse el botón circular Origen o Destino).

 

Navegación por las páginas

Pasa a la página anterior de la actividad.

Pasa a la página siguiente de la actividad.

Desbloquear

Desbloquea todas la coincidencias exactas para que puedan modificarse.

Si el jefe de proyectos no ha habilitado la opción Desbloquear, este botón no aparecerá.

Archivos de referencia

Muestra los archivos de referencia que están disponibles para la traducción.

Bases de datos terminológicas

Abre las bases de datos terminológicas asociadas con este proyecto.

Información sobre la página

Contiene información sobre una página que el servicio de asistencia al cliente de GlobalSight puede emplear para resolver posibles problemas en ella.

Editor de fragmentos

Abre el Editor de fragmentos y contenido, que le permite añadir y eliminar fragmentos de la página que hay que traducir.

Vista previa (el enlace aparece junto a los registros de la base de datos)

  • Pulse este enlace para abrir una vista previa de sólo lectura con los registros de origen y de destino de la base de datos.

Este enlace sólo aparece si se están traduciendo registros de una base de datos.

Cerrar

Cierra el editor principal.

El trabajo que haga se guardará a medida que usted modifique los segmentos en el editor de segmentos, sin necesidad de guardar.
