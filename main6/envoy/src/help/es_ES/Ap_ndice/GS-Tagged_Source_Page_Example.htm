Ejemplo de una página de origen con etiquetas GS

Ejemplo

 

<HTML>

<HEAD>

<TITLE>Página de prueba para etiquetas GS</TITLE>

</HEAD>

<BODY>

<H1>Página de prueba para etiquetas GS</H1>

<H2>Mi lista</H2>

<OL>

<LI>Texto de elemento de lista.

<LI>Texto de elemento de lista.

<LI>Texto de elemento de lista.

<GS ADD="LIST_ITEM"/> <!-- Aquí se puede añadir un elemento a la lista. -->

<P>Texto del párrafo 1.

<GS ADD="LIST_ITEM"/> <!-- Aquí se puede añadir un párrafo. -->

<P>

<TABLE>

<TR>

<TD>Texto de la tabla</TD> <!-- Aquí se puede añadir una tabla. -->

</TR>

</TABLE>

<GS ADD="TABLE"/>

</BODY>

</HTML>
