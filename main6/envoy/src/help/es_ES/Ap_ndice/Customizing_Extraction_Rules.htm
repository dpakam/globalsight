Personalizar la configuración de extracción

En System4, la extracción se puede personalizar modificando los archivos de propiedades que se mencionan más adelante. Personalizar la extracción permite controlar la cantidad de datos que se pasan a traducción. Por ejemplo, muchos archivos HTML contienen información acerca del tamaño de las fuentes, los colores del fondo, las direcciones web de las imágenes y otros elementos que no deben traducirse. De hecho, es mejor que el traductor ni siquiera vea estos datos, ya que complican la visualización e impiden identificar fácilmente los datos importantes que hay que traducir.

 

Si modifica los archivos de propiedades de extracción a continuación, puede indicar a System4 que ignore mucha de esta información que no es relevante para la traducción. Esto también agiliza el proceso de importación, reduce la cantidad de material superfluo que se incluye en las memorias de traducción y, lo que es más importante, contribuye a facilitar el trabajo de traducción.
