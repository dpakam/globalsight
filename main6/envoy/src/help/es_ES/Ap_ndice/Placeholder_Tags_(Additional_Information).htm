Marcadores de posición (información adicional)

Definición

Los marcadores de posición son representaciones de las etiquetas html.  Dichos marcadores sólo se utilizan para representar las etiquetas html que posiblemente deban moverse de lugar durante el proceso de traducción, como los enlaces.

Ejemplo

Necesito ir a la tienda.

Si todas las etiquetas html se mostraran de esta forma:

Necesito ir a la <a href="www.a-store.com">tienda</a>.

existiría la posibilidad de dañar accidentalmente el código HTML borrando una comilla o un ángulo (" >).  Los marcadores de posición permiten proteger la integridad del código:

Necesito ir a la [l1]tienda[/l1].

 

Observe que la etiqueta HTML tiene un comienzo y un final.  En el ejemplo anterior, esto indica dónde comienza y termina el enlace.  Para asegurar que los participantes en la traducción han introducido las etiquetas inicial y final, y que no han borrado las etiquetas por completo, el sistema presenta mensajes de error.

Marcadores de posición optativos

Negrita: [b][/b]

Cursiva: [i][/i]

Subrayado: [u][/u]

Espacio duro: [nbsp]

 

Estas etiquetas pueden añadirse o quitarse según convenga para la correcta traducción del texto; esto resulta útil en los idiomas en los que no es posible utilizar la letra cursiva sin dificultar la lectura o en los que se debe intercalar un espacio duro como parte de la puntuación.

Etiquetas sin elemento de cierre

Observe que la etiqueta [nbsp] no tiene etiqueta de cierre.  De forma análoga, cuando una etiqueta HTML abarca más de un segmento en el material original, se representa con una etiqueta [x].  Las etiquetas [x] no pueden eliminarse, incluso si representan una etiqueta <b> que simplemente abarca más de un segmento.
