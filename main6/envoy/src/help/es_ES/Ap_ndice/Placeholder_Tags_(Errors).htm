Marcadores de posición (errores)

Principios generales

  • Todos las etiquetas que aparecen en el original también deben aparecer en el texto traducido.

Excepciones: negrita [b] y [/b], cursiva [I] y [/I], subrayado [u] y [/u] y espacio duro [nbsp].

  • En todos los casos, debe haber una etiqueta de apertura y una de cierre (/etiqueta).  

Ejemplo: Ésta es la etiqueta de apertura de una fuente: [f1] y ésta es la correspondiente de cierre: [/f1].

Excepciones: Las etiquetas [x] y [nbsp] no tienen etiquetas de cierre.  [x] representa una etiqueta que no puedese borrarse.  Normalmente, esto se debe a que la primera parte aparece en un segmento y la última parte en otro.  Ambas etiquetas se representan mediante [x].

Mensajes de error

Faltan estas etiquetas:

  • Debe comprobar que todos las etiquetas del texto original aparezcan también en el texto traducido.

Ejemplo de texto original: Here is a pretty [f1] cat [/f1].

Ejemplo de la traducción: ¡Qué lindo gatito!

Mensaje de error: Faltan estas etiquetas: [f1] [/f1].

Solución: Añada las etiquetas que faltan para continuar: ¡Qué lindo [f1]gatito[/f1]!

  • Este mensaje también aparece cuando es preciso comprobar que cada marcador de posición tiene los ángulos de apertura [ y de cierre ]

Ejemplo de texto original: Here is a pretty [f1] cat [/f1].

Ejemplo de la traducción: ¡Qué lindo [f1gatito[/f1]!

Mensaje de error: Faltan estas eiquetas: [f1].

Solución: Añada el ángulo que falta para continuar: ¡Qué lindo [f1]gatito[/f1]!

Estas etiquetas no están emparejadas:

  • Debe cerciorarse de añadir las etiquetas por pares.  Aunque puede añadir o eliminar estas etiquetas según resulte necesario, debe añadirlas o eliminarlas por pares.  

Ejemplo de texto original:  Here is a missing [b] cat [/b].

Ejemplo de la traducción: Aquí está el [b]gatito perdido.

Mensaje de error: Estas etiquetas no están emparejadas: [b].

Solución: Añada las etiquetas que faltan para continuar: Aquí está el [b]gatito perdido[/b].

 

No pueden añadirse estas etiquetas:

  • Este mensaje aparece cuando no puede añadirse a la traducción una etiqueta que no figura en el original (salvo negrita, cursiva y subrayado en HTML solamente).

Ejemplo de texto original:  Here is a missing cat.

Ejemplo de la traducción: Aquí está el [f1]gatito[/f1] perdido.

Mensaje de error: No pueden añadirse estas etiquetas: [f1]  [/f1].

Solución: Aquí está el gatito perdido.

  • Este mensaje de error también aparece cuando es necesario introducir ángulos en el texto de un segmento. Como el primer ángulo está reservado para las etiquetas de System4, debe introducir dos ángulos, que el editor principal interpreta como uno solo.

Ejemplo de texto original:  Here is a missing cat.

Ejemplo de la traducción: Aquí está el [gatito] perdido.

Mensaje de error: No pueden añadirse estas etiquetas: [gatito].

Solución: Aquí está el [gatito] perdido.
