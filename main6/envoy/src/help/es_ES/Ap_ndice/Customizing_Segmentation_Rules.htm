Personalizar la configuración de segmentación

Para personalizar System4 de modo que cada segmento se cree por oración o párrafo, puede modificar el archivo <inicio_weblogic>/system4/properties/Diplomat.properties.

 

Para segmentar los datos por oración (predeterminado), establezca la propiedad "segment" como "sentence". Por ejemplo:

segmentation=sentence

 

Para segmentar los datos al nivel de párrafo (predeterminado), establezca la propiedad "segment" en "paragraph". Por ejemplo:

segmentation=sentence

 

Debe reiniciar Weblogic para que los cambios surtan efecto.
