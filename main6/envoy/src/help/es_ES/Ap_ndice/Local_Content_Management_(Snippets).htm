Gestión del contenido local (fragmentos)

La función Gestión del contenido local permite añadir y eliminar secciones de HTML (oraciones y formato) en un idioma de destino en particular. Esto permite crear páginas traducidas que contienen secciones enteras que no aparecen en las páginas originales. Esta capacidad de adaptación posibilita el no tener que mantener una correspondencia exacta en cada oración entre las páginas originales y las páginas traducidas.

 

Las secciones o fragmentos de HTML que pueden añadirse o eliminarse en las páginas traducidas se denominan fragmentos. En esta sección se describe cómo crear, modificar, eliminar y gestionar los fragmentos.

Habilitar la función de gestión del contenido local

Para habilitar la función de gestión del contenido local, debe asignar el valor true a la "Gestión del contenido local" durante la instalación de System4.

Crear fragmentos

Para crear fragmentos, importe los archivos de importación de fragmentosdesde la pantalla Importación de fragmentos , a la que se accede mediante el menú Configuración. Esto introducirá automáticamente los datos de la Biblioteca de fragmentos a la que accederá más tarde desde el editor principal para añadir los fragmentos a las páginas de destino.

 

Añadir fragmentos a las páginas de destino

Para añadir fragmentos a las páginas de destino:

  1. Cree una Biblioteca de fragmentos importando los archivos de importación de fragmentos desde la pantalla Importación de fragmentos.

  2. Importe una página de origen con etiquetas GS. La página de origen con etiquetas GS debe contener las que correspondan a los fragmentos de la Biblioteca de fragmentos. Encontrará una referencia en el Ejemplo de una página de origen con etiquetas GS.

  3. Abra la página en el editor principal y pulse el enlace "Editor de fragmentos" para abrir Editor de fragmentos y contenido.

  4. Utilice el Editor de fragmentos y contenido para añadir y eliminar contenido de la página en el idioma de destino de la actividad en la que está.

 

Eliminar fragmentos de las páginas de destino

Para eliminar fragmentos de las páginas de destino:

  1. Abra la página de destino en el Editor principal.

  2. Pulse el enlace Editor de fragmentos.

  3. En el Editor de fragmentos y contenido, elija las secciones del archivo que están marcadas como eliminables.

  4. Pulse Eliminar.

 

Exportar páginas traducidas con fragmentos

Para exportar las páginas traducidas en las que ha añadido o eliminado fragmentos, siga el proceso normal de exportación de archivos.

Exportar páginas originales con fragmentos

Si desea modificar o volver a importar una página original sin perder el trabajo hecho con los fragmentos en System4, puede exportarla con todos los fragmentos (etiquetas GS) añadidos o eliminados. Para hacerlo, pulse el botón "Exportar páginas de origen" de la pantalla Mis trabajos.

Grupo Gerente de idiomas

Hemos creado un grupo de usuarios especial para System4 llamado Gerente de idiomas cuyo fin es gestionar las páginas traducibles que tienen contenido específico en ese idioma (fragmentos). Los usuarios que pertenecen al grupo Gerente de idiomas cuentan con un nivel intermedio de privilegios del sistema, entre Participante en la traducción y Jefe de proyectos. Al igual que los jefes de proyectos, un gerente de idiomas puede importar, traducir y exportar archivos, pero sólo en los idiomas definidos en los cargos asignados. Usted definió sus cargos como parte del proceso Crear usuario.
