Tipos de elementos excluidos

Los tipos de elementos excluidos son segmentos que pueden excluirse del proceso de traducción. La exclusión de ciertos tipos de elementos puede facilitar y agilizar el trabajo de traducción, ya que elimina los datos no pertinentes que de otra manera estorbarían el trabajo del traductor. Cuando se excluye un elemento, no se puede modificar con el editor del sistema y tampoco se incluye en el archivo de traducción modificable con el editor del usuario.

 

Introduzca los tipos de elementos para excluir en el cuadro "Tipos de elementos excluidos" de la pantalla Perfil de traducción. Debe delimitarse cada elemento con una pleca " | ".

 

La "Lista completa" que encontrará más adelante contiene todos los elementos que pueden excluirse. Para facilitarle esta tarea, también hemos incluido dos listas de tipos de elementos excluidos de uso común, una larga y otra corta. Simplemente copie y pegue estas listas en el cuadro "Tipos de elementos excluidos" de la pantalla Perfil de traducción (quite los saltos de línea) y luego importe los archivos.
