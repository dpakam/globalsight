Reglas de vista previa de la base de datos

La siguiente definición de tipo de documento (DTD) contiene comentarios que dan información sobre cómo escribir un archivo XML que System4 puede utilizar para poner en vista previa dinámica el contenido de la base de datos.

 

<!-- Modificado por GlobalSight con XML Spy v3.0.7 NT (http://www.xmlspy.com) -->

<!-- PreviewUrlXml.dtd define el archivo PreviewUrlXml para describir las direcciones web empleadas para la vista previa -->

<!ELEMENT previewUrlXml (idioma*)>

<!-- Se debe indicar cada idioma empleado en el sistema. El idioma comprende un componente sourceUrls y/o targetUrl -->

<!ELEMENT locale (sourceUrls?, targetUrls?)>

<!-- El identificador del idioma es el nombre del idioma, como en_US -->

<!ATTLIST locale  name CDATA #REQUIRED>

<!-- El elemento sourceUrls contiene uno o varios elementos url. La codificación especifica la codificación de caracteres IANA en

la cual  deben verse las direcciones web. -->

<!ELEMENT sourceUrls (url+)>

<!ATTLIST sourceUrls  encoding CDATA "UTF-8">

<!-- El elemento sourceUrls contiene uno o varios elementos url. La codificación es la de caracteres IANA

en la que  deben verse las direcciones web. -->

<!ELEMENT targetUrls (url+)>

<!ATTLIST targetUrls  encoding CDATA "UTF-8">

<!-- El elemento url contiene toda la información necesaria para

    activar las conexiones HTTP GET y POST con las direcciones web (sin cookies) -->

<!ELEMENT url (label, href, arg*)>

<!ATTLIST url type (get | post) #REQUIRED>

<!--label es una pequeña etiqueta que se puede utilizar al presentar la dirección web -->

<!ELEMENT label (#PCDATA)>

<!-- href el la href de la dirección web -->

<!ELEMENT href (#PCDATA)>

<!-- Un arg comprende un parámetro y posiblemente un valor. Ambos se pueden sustituir.

   La sustitución puede provenir de los datos de origen o de los datos de destino. Si el parámetro o el valor no se va a sustituir,

   el valor es "none".-->

<!ELEMENT arg (parameter, value?)>

<!-- parameter es el componente de parámetro de un argumento. -->

<!ELEMENT parameter (#PCDATA)>

<!ATTLIST parameter substitution_source (source | target | none) "none">

<!-- value es el valor del parámetro. -->

<!ELEMENT value (#PCDATA)>

<!ATTLIST value  substitution_source (source | target | none) "none">
