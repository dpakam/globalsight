Base de datos terminológica

Conjunto de términos y sus correspondientes traducciones en los idiomas de destino.   Puede facilitarse información adicional acerca del término, como una definición, uso gramatical y contexto.

El sistema informa al usuario cuando un segmento contiene un término que forma parte de la base de datos terminológica.   Se proporciona la traducción y se puede ver información adicional acerca de ese término.
