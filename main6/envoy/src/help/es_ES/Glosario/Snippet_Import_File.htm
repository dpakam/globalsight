Archivo de importación de fragmentos

Archivo HTML que contiene fragmentos genéricos y específicos que van a importarse a la biblioteca de fragmentos. Un archivo de importación de fragmentos contiene segmentos de HTML entre etiquetas GS Encontrará instrucciones sobre cómo escribir las etiquetas GS en el tema Sintaxis de las etiquetas GS.

 

También puede ver un ejemplo de un archivo de importación de fragmentos en Ejemplo de archivo de importación de fragmentos. Este Ejemplo de archivo de importación de fragmentos se utiliza con el Ejemplo de una página de origen con etiquetas GS.
