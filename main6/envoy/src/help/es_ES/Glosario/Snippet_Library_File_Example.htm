Ejemplo de archivo de importación de fragmentos

Ejemplo

 

<!-- Archivo de importación de fragmentos -->

<!-- Este archivo contiene fragmentos genéricos que  -->

<!-- se utilizarán para rellenar su biblioteca de fragmentos.  -->

 

 

<GS EXTRACT ="LIST_ITEM">

<LI>Texto de elemento de lista.

</GS>

 

<GS EXTRACT ="PARAGRAPH">

Texto de un párrafo

</GS>

 

<GS EXTRACT ="TABLE">

<TABLE>

    <TR>

        <TD>Texto de una celda de una tabla.</TD>

    </TR>

</TABLE>

</GS>
