Primeros pasos

Guías de configuración para:

archivos estándar, archivos XML, archivos de TeamSite y datos de la base de datos

Cada una de estas guías ofrece instrucciones que lo guían paso a paso por el proceso de configuración de System4 para el tipo de contenido indicado.

 

Configuración

Combinaciones de idiomas

Permite determinar los idiomas de origen y destino para uso general del sistema

Tipos de actividades

Permite crear y modificar las actividades que los usuarios deben realizar como parte del proceso de traducción.

Usuarios

Permite añadir y modificar la información sobre el usuario y el acceso al sistema.

Memoria de traducción

Crear las memorias de traducción vacías en las que se van a almacenar las traducciones.

Proyectos

Permite crear y modificar proyectos para los grupos de traducción del contenido.

Perfiles de traducción

Permite crear y modificar perfiles para realizar el trabajo de traducción.

Terminología

Permite mantener las bases de datos terminológicas que se utilizarán como referencia durante el proceso de traducción.

Moneda

Permite especificar la tasa de conversión a la moneda del sistema desde varias monedas distintas.

Tarifa

Determinar el coste por palabra de las actividades en las combinaciones de idiomas.

Archivos de referencia

Permite enviar los glosarios específicos para la combinación de idiomas para que estén a disposición de los participantes en el proceso de traducción.

Parámetros del sistema

Permite cambiar varios parámetros globales del sistema (incluidos el límite de coincidencias parciales y el correo electrónico del administrador del servidor de web).

Registros

Registros de System4 que pueden ayudar a determinar la causa de los problemas relacionados con System4.

 

Fuentes de datos

Perfil de archivos

Seleccionar la codificación, el formato de archivo y el perfil de traducción a utilizar para importar el contenido.

Extensiones de archivo

Crear nuevas extensiones para los formatos de archivo compatibles.

Reglas XML

Crear conjuntos de reglas para determinar el contenido XML que se adaptará dentro de los esquemas XML.

Importar

Importar los archivos a System4 manualmente para su traducción.

Ramas de TeamSite*

Indicar las ramas de TeamSite de destino para las ramas de origen de TeamSite y los idiomas

Conexiones a la base de datos*

Especificar las cadenas de conexión y los controladores para leer y grabar en las bases de datos.

Perfiles de bases de datos*

Especificar las instrucciones SQL de lectura, vista previa y grabación de la información traducible de las tablas de la base de datos.

Configuración de importación de la base de datos*

Definir las tablas de registros pendientes que se deben supervisar y establecer los parámetros de tamaño para los trabajos siguientes (vea también Tabla de registros pendientes).

Vista previa de base de datos*

Crear instrucciones escritas en XML que detallan la ruta a la plantilla de página que se debe utilizar para presentar dinámicamente los datos de origen y de destino en una vista previa.

* Estos elementos sólo aparecerán si cuenta con integración de TeamSite o de bases de datos.

 

Común

Mis trabajos

Presenta en detalle el estado de todos los trabajos en el sistema (administrador) o de todos los trabajos creados para su proyecto (jefe de proyectos)

Mis actividades

Muestra todas las actividades que le han sido asignadas.

Informes

Muestra los informes acerca de la utilización del sistema.
