Exportar páginas

Pueden exportarse las páginas de cualquier proceso de producción (idioma de destino) en el que se hayan terminado todos los pasos de traducción (actividades).

 

  • Las páginas del proceso de producción se pueden exportar una vez terminadas todas las actividades.

  • Los trabajos que contienen páginas que pueden exportarse pueden verse en las fichas "Traducidos" y "Archivados".

  • Pulse el enlace Exportar (ficha Traducidos) o Volver a exportar (ficha Archivados) correspondiente a un trabajo o un proceso de producción.

  • En la pantalla de exportación, elija un proceso de producción para exportar todas sus páginas.

  • Pulse el triángulo para expandir el proceso de producción y seleccionar las páginas que desea exportar.
