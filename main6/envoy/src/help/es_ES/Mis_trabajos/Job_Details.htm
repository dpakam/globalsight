Detalles del trabajo

Resumen:

Permite ver información detallada acerca de un trabajo.

Identificador del trabajo:

El identificador del trabajo asignado por el sistema.

Nombre del trabajo:

Nombre dado al trabajo por la persona que lo asignó o generado por el sistema.

Iniciador:

Nombre del usuario que creó el trabajo o, si el trabajo se creó automáticamente, del jefe de proyectos.

Idioma de origen:

El idioma de origen del contenido de esta actividad.

Proyecto:

El nombre del proyecto.

Ejemplo: Investigación y desarrollo de servicios mundiales

Perfil de traducción:

Perfil de traducción empleado para crear este trabajo.

Determina los procesos de producción y sus correspondientes actividades para este trabajo.

Fecha de creación:

Fecha y hora de creación del proyecto.

Fuente de datos:

Perfil de archivos utilizado para importar el contenido en estas páginas.

Recuento total de palabras originales:

Recuento total de palabras de todas las páginas de origen de un trabajo.

Prioridad:

La prioridad asignada a este trabajo.

Páginas

Pulse el nombre de una página para ver el contenido a traducir en modo de sólo lectura.

Procesos de producción

 

Idioma de destino

El idioma de destino de este proceso de producción.

Recuento de palabras de destino

Desglosa la cantidad de palabras con coincidencia exacta o parcial, o sin coincidencia , para cada página de esta actividad.

% terminado

Porcentaje del proceso de producción que está terminado, calculado a base del estado de las actividades. Por ejemplo, si se ha terminado una de las tres actividades de un proceso de producción, la columna "% terminado" mostrará "33%".

Estado

El estado de este proceso de producción.Cada proceso de producción de un trabajo puede encontrarse en un estado distinto. Los estados se describen a continuación:

ARCHIVADO: El proceso de producción ha sido archivado.

RESERVADO_PROCESO_POR_LOTE: El proceso de producción está a la espera de que se importen todas las páginas que se habían seleccionado en la pantalla Importar. Durante este período de espera, el trabajo aparecerá en la ficha Pendientes. Una vez que se hayan importado todas las páginas, el proceso de producción se traslada a Preparados, en el caso de un trabajo Manual, o a En curso, si se trata de un trabajo Automático.

CANCELADO: Se ha descartado el proceso de producción.

DESPACHADO: El proceso de producción está En curso.

EXPORTADO: Se ha exportado el proceso de producción. El trabajo aparecerá en la ficha Exportados.

FALLÓ_LA_EXPORTACIÓN: Indica que hay al menos una página del proceso de producción que no se pudo exportar correctamente.

FALLÓ_LA_IMPORTACIÓN:  Indica que hay al menos una página del proceso de producción que no se pudo importar correctamente.

TRADUCIDO: Se han terminado todas las actividades del proceso de producción. El proceso de producción está preparado para la exportación.

PENDIENTE: El proceso de producción está a la espera de que se alcancen las condiciones de recuento de palabras o de temporizador especificadas. Cuando se satisfacen dichas condiciones, el proceso de producción pasa a la ficha Preparados, en el caso de un trabajo Manual, o a En curso, si se trata de un trabajo Automático.

PREPARADO_PARA_DESPACHO: El proceso de producción está esperando en la ficha Preparados para ser despachado.

Descartar

Descarta los procesos de producción seleccionados.

Ver error

Muestra el error asociado al proceso de producción seleccionado.

Comentarios

Muestra los comentarios que han publicado los participantes en la traducción sobre este proceso de producción.

Archivar

Pasa el proceso de producción seleccionado al estado Archivado.

Detalles

Muestra una representación gráfica de sólo lectura del proceso de producción. También muestra un historial de todas las actividades que forman parte del proceso de producción.

Modificar

Abre el proceso de producción en el editor de gráficos para que se lo pueda modificar.

Despachar

Despacha el proceso de producción seleccionado. Esta acción inicia el proceso de producción y lo traslada a la ficha En curso.
