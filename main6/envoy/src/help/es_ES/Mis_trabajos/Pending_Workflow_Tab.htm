Ficha de procesos de producción Pendientes

Resumen:

En esta ficha se muestran los trabajos que no están preparados para ser despachados.

  • Utilice Preparar para pasar un trabajo pendiente a la ficha Preparados.

  • Utilice Descartar para quitar un trabajo por completo de System4.

  • Si se produce un error durante la importación, "Examinar error" muestra los errores (vea Error de importación).

(encabezado)

Pulse cualquier encabezado para ordenar la lista de trabajos por esa categoría.

Ejemplo: Pulse "Nombre del trabajo" para ordenar las actividades alfabéticamente por nombre o pulse "Proyecto" para ordenarlas alfabéticamente por proyecto.

Ver error

Muestra los errores que se produjeron durante la importación.

Descartar

Permite descartar los trabajos seleccionados.Todo el trabajo realizado se guardará en la memoria de traducción.

Preparar

Traslada los trabajos seleccionados a la ficha Preparados.
