Pantalla principal Ubicaciones de exportación

Resumen:

Permite crear ubicaciones de exportación que pueden elegirse al exportar archivos.

Nuevo

Permite añadir una ubicación de exportación al sistema.

Modificar

Permite modificar las propiedades de una ubicación de exportación.

Definir como predeterminado

Permite establecer la ubicación de exportación seleccionada como la predeterminada del sistema para exportación.

Quitar

Quita la ubicación seleccionada.
