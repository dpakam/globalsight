Crear/modificar perfiles de archivos

Resumen:

Permite seleccionar la codificación, el formato de archivo y el perfil de traducción que se utilizará para importar el contenido.

Nombre:

Para visualización en System4.

Descripción:

Optativa: los administradores y los jefes de proyectos pueden ver esta descripción en System4.

Perfiles de traducción:

Seleccione el perfil de traducción que debe utilizarse al importar este tipo de contenido.

Codificación del archivo de origen:

Seleccione la codificación del contenido de origen.

Formato del archivo de origen:

Seleccione el formato (vea Formatos de datos compatibles) del contenido de origen.  Esto indica a System4 cómo debe leer el contenido.

Nota: Si selecciona el formato XML, aparecerá un cuadro de diálogo adicional (Reglas XML).

Reglas XML:

Seleccione las reglas XML que deben utilizarse para determinar lo que puede traducirse dentro del contenido XML  (vea también Reglas XML y Escribir reglas XML).

Extensiones de archivo:

Cuando se importan los archivos manualmente, sólo se muestran para exportación los que tienen la extensión seleccionada.
