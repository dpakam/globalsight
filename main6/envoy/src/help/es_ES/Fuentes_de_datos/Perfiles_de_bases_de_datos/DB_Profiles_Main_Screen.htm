Pantalla principal Perfiles de bases de datos

Sólo podrá ver estas pantallas si cuenta con integración de bases de datos.

 

Resumen:

Especifique las instrucciones SQL de lectura, vista previa y grabación para la información que puede traducirse de las tablas de la base de datos.

Nuevo

Permite crear un nuevo perfil de base de datos (vea Información básica).

Modificar

Permite modificar el perfil de base de datos seleccionado* (vea Información básica).

Información

Información de resumen acerca del perfil de base de datos seleccionado*.

* Para seleccionar una conexión a la base de datos, pulse en cualquier parte de la fila correspondiente.
