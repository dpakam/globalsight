Instrucción SQL de lectura de la base de datos

Sólo podrá ver estas pantallas si cuenta con integración de bases de datos.

 

Resumen:

Escriba las instrucciones SQL necesarias para extraer los datos de las columnas indicadas de una fila traducible.

Conexión de lectura:

Seleccione la conexión a la base de datos que va a utilizar.

Instrucción SQL de lectura:

La instrucción SQL de lectura lee todas las columnas traducibles, contextuales e invisibles de un mismo registro. Las columnas que se leen deben coincidir con las especificadas en la información sobre columnas de la pantalla anterior. Cada registro traducible se identifica en la tabla de registros pendientes por medio de su clave primaria. Utilice %%1%% para hacer referencia a la clave primaria. Si el registro emplea una clave compuesta, utilice %%1%% para hacer referencia al primer elemento, %%2%% para hacer referencia al segundo, %%3%% para hacer referencia al tercero, y así sucesivamente (vea también Tabla de registros pendientes).

 

Ejemplo:

SELECT short_desc, long_desc, dept, prod_id, properties

FROM items

WHERE prop_id= %%1%% AND dept='%%2%%'
