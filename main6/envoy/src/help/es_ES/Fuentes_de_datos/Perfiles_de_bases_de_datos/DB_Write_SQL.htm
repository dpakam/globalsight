Instrucción SQL de grabación en la base de datos

Sólo podrá ver estas pantallas si cuenta con integración de bases de datos.

 

Resumen:

Escriba las instrucciones SQL necesarias para volver a colocar los datos traducidos finales para las columnas indicadas.

Conexión de grabación:

Seleccione la conexión a la base de datos que se debe utilizar para grabar los datos traducidos en la base de datos.

Instrucción SQL de grabación de inserción:

Si no existe ningún registro con esa clave primaria, Instrucción SQL de grabación inserta un nuevo registro en la base de datos especificada. Si el registro ya existe, se utiliza Instrucción SQL de grabación de actualización. Esta instrucción SQL debe incluir todas las columnas necesarias para definir el registro. Utilice %%1%% para hacer referencia al primer elemento de la clave primaria y %%2%%, %%3%%, etcétera para hacer referencia a los elementos adicionales de una clave compuesta  (vea también Tabla de registros pendientes). Utilice %%tablename.column%% para hacer referencia a los datos necesarios para cada columna del registro. Los nombres de la tabla y la columna se deben adquirir como parte de la instrucción SQL de lectura y se deben definir en la información de la columna.

 

Ejemplo:

INSERT INTO items(short_desc, long_desc, dept, prod_id, properties) VALUES ('%%items.short_desc%%','%%items.long_desc%%',

'%%2%%','%%1%%', '%%items.properties%%')

Instrucción SQL de grabación de actualización:

Si ya existe un registro con esa clave primaria, Instrucción SQL de grabación de actualización actualiza un registro ya existente en la base de datos especificada para generar la vista previa. Si el registro no existe, se utiliza Instrucción SQL de grabación de inserción. Esta instrucción SQL se crea de la misma forma que la Instrucción SQL de inserción de vista previa antes descrita utilizando el comando Update en lugar de Insert, pero además se debe especificar una cláusula Where para asegurar que se actualice el registro correcto.

 

Ejemplo:

UPDATE items

SET short_desc='items.short_desc',

long_desc='items.long_desc',

properties='%%items.properties%%'

WHERE dept='%%2%%' AND prod_id='%%1%%'
