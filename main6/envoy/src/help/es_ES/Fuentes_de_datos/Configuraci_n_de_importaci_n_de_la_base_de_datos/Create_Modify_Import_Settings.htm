Crear/modificar configuraciones de importación

Sólo podrá ver estas pantallas si cuenta con integración de bases de datos.

 

Resumen:

Permite definir una tabla de registros pendientes que deben supervisarse y establecer los parámetros de tamaño del trabajo (vea también Tabla de registros pendientes).

Nombre:

Para visualización en System4.

Descripción:

Optativa, los administradores pueden ver esta descripción en System4.

Tabla de registros pendientes:

Nombre de la tabla de registros pendientes que desea supervisar.  Si tiene más de una tabla de registros pendientes, temdrá que crear más de una configuración de importación de la base de datos.

Registros por página:

Enumera la cantidad de registros que deben aparecer en una página.  En System4, la traducción se lleva a cabo dentro del contexto de las páginas, que suelen ser archivos sueltos.  En el caso de los registros de la base de datos, una página es simplemente un conjunto de registros.  

Páginas por lote:

Cantidad de páginas que constituyen un lote.  Si el perfil de traducción está establecido en por lote, este valor determina la cantidad de páginas (vea la información antes proporcionada) necesaria para que el trabajo alcance el límite de tamaño y salga del estado Pendientes.

Forzar página parcial después de:

Una página parcial no contiene la cantidad completa de registros por página (vea la información antes proporcionada).  Después de la cantidad de días indicada, la página se considerará completa independientemente de la cantidad de registros.

Conexión a la base de datos:

Seleccione la conexión a la base de datos que debe utilizarse al leer la tabla de registros pendientes antes indicada.
