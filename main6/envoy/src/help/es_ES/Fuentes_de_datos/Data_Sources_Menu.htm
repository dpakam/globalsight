Fuentes de datos

Elemento de menú

Descripción

Perfil de archivos

Seleccionar la codificación, el formato de archivo y el perfil de traducción a utilizar para importar el contenido.

Extensiones de archivo

Crear nuevas extensiones para los formatos de archivo compatibles.

Reglas XML

Crear conjuntos de reglas para determinar el contenido XML que se adaptará dentro de los esquemas XML.

Importar

Importar los archivos a System4 manualmente para su traducción.

Ramas de TeamSite*

Indicar las ramas de TeamSite de destino para las ramas de origen de TeamSite y los idiomasde destino.

Conexiones a la base de datos*

Especificar las cadenas de conexión y los controladores para leer y grabar en las bases de datos.

Perfiles de bases de datos*

Especificar las instrucciones SQL de lectura, vista previa y grabación de la información traducible de las tablas de la base de datos.

Configuración de importación de la base de datos*

Definir las tablas de registros pendientes que se deben supervisar y establecer los parámetros de tamaño para los trabajos siguientes (vea también Tabla de registros pendientes).

Vista previa de base de datos*

Crear instrucciones escritas en XML que detallan la ruta a la plantilla de página que se debe utilizar para presentar dinámicamente los datos de origen y de destino en una vista previa.

* Estos elementos sólo aparecerán si cuenta con integración de TeamSite o de bases de datos.
