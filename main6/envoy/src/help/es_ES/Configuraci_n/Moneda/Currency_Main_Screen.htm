Pantalla principal Moneda

Resumen:

Permite especificar la tasa de conversión a la moneda del sistema desde varias monedas distintas.

Nuevo

Permite añadir al sistema una nueva tasa de conversión de moneda.

Si utiliza la función de conversión automática de monedas, ya se han incorporado al sistema casi todas las conversiones necesarias.

Modificar

Permite modificar la tasa de conversión de moneda seleccionada* (vea Crear/modificar moneda).

* Para seleccionar una moneda, pulse en cualquier parte de la fila correspondiente.
