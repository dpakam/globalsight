Importar

Resumen:

Importa los términos nuevos a una base de datos terminológica enviando un archivo de texto con formato.

Examinar

Permite localizar en su ordenador el archivo de texto de la base de datos terminológica que quiere importar.

Formato del archivo de importación

Seleccione el formato del archivo de la base de datos terminológica:

XML: La base de datos terminológica debe tener una definición específica de tipo de documento (dtd) de XML.

CSV: Cualquier archivo de texto delimitado con determinado carácter.

Codificación de archivo

Sólo CSV: Seleccione la codificación de archivo.

La primera línea es el encabezado de la columna.

Sólo CSV: Si la primera línea de un archivo de texto delimitado es un encabezado, marque la casilla.

Delimitador de columna

Sólo CSV: Identifique el delimitador empleado en el archivo de texto.
