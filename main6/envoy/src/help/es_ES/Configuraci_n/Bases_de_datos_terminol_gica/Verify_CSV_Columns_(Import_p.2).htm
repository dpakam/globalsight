Verificar columnas CSV (Importar, pág. 2)

Resumen:

Permite definir cada columna en un archivo de texto delimitado.

Volver a analizar

Optativo: Fíjese en los detalles de las columnas y, si le parece que se ha leído el archivo incorrectamente, cambie la configuración y pulse Volver a analizar.

Comprobar la importación

Optativo: Una vez que haya definido las columnas, pulse Comprobar la importación para detectar cualquier error que exista en el archivo de texto.

Configurar

Permite definir cada columna del archivo de texto (vea Definir columna de base de datos terminológica).
