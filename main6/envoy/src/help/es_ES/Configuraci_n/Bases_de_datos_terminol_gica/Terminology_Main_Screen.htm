Pantalla principal Terminología

Resumen:

Permite mantener las bases de datos terminológicas que se utilizarán como referencia durante el proceso de traducción.

Nueva

Permite crear una nueva base de datos terminológica vacía (vea Crear/modificar/clonar bases de datos terminológicas).

Modificar

Permite modificar la base de datos terminológica seleccionada*  (vea Crear/modificar/clonar bases de datos terminológicas).

Clonar

Permite crear una copia de la base de datos terminológica seleccionada* (vea Crear/modificar/clonar bases de datos terminológicas).

Quitar

Permite quitar la base de datos terminológica seleccionada*.

Importar

Permite importar nuevos términos a la base de datos terminológica seleccionada*.

Examinar

Muestra los términos en la base de datos terminológica seleccionada*.

* Para seleccionar una base de datos terminológica, pulse en cualquier parte de la fila correspondiente
