Examinar base de datos terminológica

Resumen:

Permite consultar la base de datos terminológica para encontrar un término en particular o un grupo de términos.

Origen

Elija el idioma en el que debe buscarse el texto que ha escrito en el cuadro Consulta.

Destino

Actualmente, esta opción no está activa.

Consulta

Introduzca el texto que desea buscar:

Utilice el signo de admiración de cierre ( ! ) delante del texto para buscar el texto exacto que ha escrito.

Utilice el signo numérico ( # ) delante del texto para buscar una coincidencia parcial con el texto que ha introducido.

Utilice el asterisco ( * ) como comodín para indicar varios caracteres.

Utilice el signo de interrogación de cierre ( ? ) para indicar cualquier carácter.

Ejecutar

Lleva a cabo la consulta y presenta los resultados bajo Términos encontrados.

Términos encontrados

Presenta los términos hallados en la consulta.

(término)

Pulse un término para ver los detalles correspondientes.

Detalles del término

Muestra toda la información acerca del término.
