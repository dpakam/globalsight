Crear/modificar/clonar bases de datos terminológicas

Resumen:

Permite asignar un nombre y una descripción a bases de datos terminológicas específicas.

Nombre

Para visualización en System4.

Descripción

Optativa: sólo los administradores pueden verla.

 

Ficha Idiomas

Resumen:

Permite definir los idiomas empleados en la base de datos terminológica.

Añadir

Indique un nuevo idioma para utilizar en la base de datos terminológica.

Quitar

Quita el idioma seleccionado* de la base de datos terminológica.

* Para seleccionar un idioma, pulse en cualquier parte de la fila correspondiente.
