Definir columna de base de datos terminológica

Resumen:

Permite definir las características de una columna en un archivo de texto delimitado.

Nombre de la columna:

Identifica la columna de pantalla anterior que se mostrará, con los datos del encabezado de la columna.

Asignar columna al tipo:

Determina el tipo de información que contiene esta columna.

 

Estado del concepto, Área del concepto, Definición del concepto y Origen del concepto, se refieren todos a la totalidad de las entradas de la base de datos terminológica.

 

Término, Uso, Tipo de término, Categoría gramatical, Género, Número, Definición del término, Ejemplo del término y Origen se aplican a una de las columnas (términos) de una entrada.

Idioma del término:

Se utiliza sólo con términos: especifica el idioma del término.

Asignar a la columna:

Especifica la columna con la que está relacionada la información (normalmente, un término).
