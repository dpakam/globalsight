Adjuntar procesos de producción a idiomas de destino

Debe haber un proceso de producción asociado a cada idioma de destino del perfil de traducción. Utilice esta pantalla para asociar un proceso de producción a cada idioma de destino de este perfil de traducción.

 

  1. Elija un idioma de destino.

  2. Elija un proceso de producción de la lista.

  3. Pulse Adjuntar proceso de producción.

  4. Repita esta acción para cada idioma de destino que forme parte del perfil de traducción.

  5. Pulse Guardar.
