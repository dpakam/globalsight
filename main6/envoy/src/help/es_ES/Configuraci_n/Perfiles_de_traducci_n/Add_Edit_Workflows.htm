Añadir/Modificar procesos de producción

Resumen:

Para añadir procesos de producción a este perfil de traducción o modificar los procesos de producción ya asociados a este perfil de traducción.

Cancelar

Permite cancelar la creación o modificación del perfil de traducción.

Anterior

Pasa a la pantalla anterior.

Modificar

Permite modificar el proceso de producción seleccionado.

Añadir

Permite añadir un proceso de producción a este perfil de traducción.

Guardar

Permite modificar el perfil de traducción.
