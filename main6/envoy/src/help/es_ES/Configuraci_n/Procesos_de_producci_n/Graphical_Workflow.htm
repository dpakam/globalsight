Representación gráfica del proceso de producción

Para crear un proceso de producción debe haber un nodo inicial, un nodo final, uno o más nodos de actividad y flechas que conecten todos los nodos. Pulse los botones que aparecen en la parte superior izquierda del subprograma "Representación gráfica del proceso de producción" para crear y conectar los nodos.

 

Para utilizar los botones, pulse el que desea utilizar y luego pulse en el espacio de trabajo. El elemento seleccionado del proceso de producción aparecerá en el espacio de trabajo.

 

Encontrará algunos ejemplos en Ejemplos de procesos de producción.

 

Descripción de los botones

 

Botón

Nombre

Descripción

Nodo de actividad, botón

Crea un nodo de actividad en el espacio de trabajo.

Nodo inicial/final, botón

Crea un nodo inicial y un nodo final en el espacio de trabajo. El nodo inicial (verde) es el nodo predeterminado en el espacio de trabajo, de modo que no necesitará crearlo. Para crear un nodo final (rojo), pulse una vez en el espacio de trabajo.

Nodo condicional, botón

Crea un nodo condicional en el espacio de trabajo.

Flecha de conexión, botón

Conecta dos nodos en el espacio de trabajo. Para utilizarlo, pulse un nodo, luego arrastre a otro nodo y luego suelte el botón del ratón.

Puntero, botón

Transforma el cursor en un puntero que permite seleccionar el elemento del proceso de producción que se desea eliminar o mover en la pantalla.

Guardar, botón

Guarda el proceso de producción.

 

 

Descripciones de los elementos del proceso de producción

Elemento

Nombre

Descripción

Nodo inicial

El nodo inicial es el punto en el que comienza el proceso de producción. Todos los procesos de producción deben contar con un nodo inicial.

Nodo final

El nodo final es el punto en el que termina el proceso de producción. Todos los procesos de producción deben contar con un nodo final.

Nodo de actividad

El nodo de actividad es el punto en el que se asignan uno o varios usuarios a una actividad (etapa) del proceso de producción.

 

Pulse con el botón derecho en el nodo de actividad y elija la ventana emergente Propiedades para asignar los usuarios a la actividad y definir otras configuraciones afines a esta. Para conectar los nodos de actividad a otros elementos del proceso de producción, utilice la herramienta Flechas de conexión.

 

No hay límite a la cantidad de nodos de actividad que puede haber en un proceso de producción.

Nodo condicional

Los nodos condicionales permiten crear varias rutas para un proceso de producción. Cuando se extienden varias flechas de conexión desde un mismo nodo condicional, el usuario del proceso de producción activo puede elegir hacia dónde debe encaminarse este. Encontrará algunos ejemplos en  Ejemplos de procesos de producción.

Flechas de conexión

Las flechas de conexión conectan los elementos de un proceso de producción. El cuadro Acción de la flecha tiene como fin la claridad visual, pero los usuarios del proceso de producción no pueden verlo.
