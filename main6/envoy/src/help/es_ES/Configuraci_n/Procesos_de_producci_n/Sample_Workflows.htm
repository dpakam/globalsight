Proceso de producción de una etapa

Éste es un proceso de producción sencillo de una etapa con un nodo inicial, una actividad y un nodo final. El proceso de producción termina cuando el traductor termina la actividad de traducción.

 

Proceso de producción de dos etapas

Éste es un proceso de producción sencillo de dos etapas. Desde traducción, se pasa directamente a corrección y luego a terminado.

 

Proceso de producción con bucle

Este proceso de producción con bucle permite volver a la etapa de traducción, o a cualquier otra, una y otra vez hasta que la traducción resulte satisfactoria. Luego se puede continuar con las demás etapas del proceso de producción.

 

Después de la etapa de traducción, Juan Pérez, el corrector, corrige la traducción en la etapa de "corrección" y luego decide si el proceso de producción debe volver a traducción o si puede pasar a la revisión final.

 

 

Proceso de producción con bucle y opción de terminar

El proceso de producción  que se muestra a continuación es igual al proceso de producción con bucle antes descrito, salvo que también da a Juan Pérez la opción adicional de reenviar el proceso de producción a una etapa de control de calidad cuando él concluya la etapa de corrección. De esta forma, cuando Juan Pérez termine la corrección contará con las siguientes opciones de envío para el proceso de producción:

  1. De vuelta a traducción

  2. A control de calidad, desde donde pasará a revisión final

  3. Directamente a revisión final
