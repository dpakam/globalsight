Crear/modificar proyectos

Resumen:

Permite crear un proyecto y asignarle un jefe de proyectos.

Nombre*:

Para visualización en System4.

Jefe de proyectos*:

Seleccione el usuario responsable de los trabajos asignados a este proyecto.  El usuario debe contar con permisos de jefe de proyectos (no de administrador o participante en la traducción)

Memoria de traducción*:

Seleccione la memoria de traducción en la que deben introducirse las nuevas traducciones.

Base de datos terminológica:

Optativo: seleccione una base de datos terminológica.

Descripción:

Optativa: sólo los administradores pueden verla.

* Los elementos marcados con un asterisco son obligatorios.
