Parámetros del sistema

Resumen:

Permite cambiar los parámetros que afectan a todo el sistema.

Exportar estilo de nombre de directorio:

Idioma: Los archivos se exportan en carpetas según la abreviatura propia del idioma de destino, por ejemplo, en_GB para indicar Inglés (Reino Unido).

Idioma: Los archivos se exportan en carpetas según el idioma indicado por la abreviatura propia del idioma de destino, por ejemplo, en_GB para indicar Inglés (Reino Unido).  No se recomienda utilizar esta opción cuando hay varias regiones que tienen el mismo idioma como, por ejemplo, Alemán (Alemania) y Alemán (Austria).

Dirección electrónica del administrador del servidor web:

Algunos errores del sistema generan mensajes específicos para el administrador del equipo en el que se ejecuta System4.

Límite de coincidencia parcial:

Las coincidencias que superen este porcentaje se consideran coincidencias parciales.

Sustituir términos que coincidan:

Configura el sistema para sustituir automáticamente las coincidencias terminológicas del segmento, para que el traductor no tenga que seleccionarlas manualmente.

Cambiar el color de los hiperenlaces:

 

Se utiliza con la vista previa de HTML.  Los segmentos traducibles están enlazados y, si el color de fondo de las páginas es igual que el color predeterminado de los enlaces, es posible que prefiera sustituir la configuración predeterminada.

Color de los hiperenlaces:

El color que se sustituye para los hiperenlaces no visitados.

Color del hiperenlace activo:

El color que se sustituye para los hiperenlaces activos.

Color de los hiperenlaces visitados:

El color que se sustituye para los hiperenlaces visitados.

Habilitar autenticación de correo electrónico:

Habilita la función de autenticación de correo electrónico de System4. Esta función sólo se debe habilitar si el servidor de correo electrónico empleado por System4 exige la autenticación.

Nombre de usuario para autenticación de correo electrónico:

El nombre de usuario del servidor de correo electrónico que se empleará para autenticar todos los mensajes electrónicos enviados desde System4. Se trata de un usuario especial del servidor de correo electrónico que se crea específicamente para fines de autenticación del correo electrónico en System4. Debe añadir este usuario al servidor de correo electrónico que especificó antes. Este parámetro se ignorará si la función Habilitar autenticación de correo electrónico está inhabilitada.

 

Nota: El nombre de usuario no es el nombre de usuario del remitente del correo electrónico de System4, ya que los mensajes de System4 pueden provenir de distintos usuarios.

Contraseña de autenticación de correo electrónico:

La contraseña del usuario antes especificado.Este parámetro se ignorará si la función Habilitar autenticación de correo electrónico está inhabilitada.
