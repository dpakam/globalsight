Pantalla principal Memoria de traducción

Resumen:

Crear las memorias de traducción vacías en las que se van a almacenar las traducciones.

Nuevo

Permite añadir una nueva memoria de traducción al sistema (vea Crear/modificar memorias de traducción).

Modificar

Permite modificar la descripción de la memoria de traducción seleccionada* (vea Crear/modificar memoria de traducción).

* Para seleccionar una memoria de traducción, pulse en cualquier parte de la fila correspondiente
