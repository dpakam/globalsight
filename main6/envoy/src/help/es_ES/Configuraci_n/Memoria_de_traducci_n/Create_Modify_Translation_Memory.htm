Crear/modificar memoria de traducción

Resumen:

Permite crear una memoria de traducción vacía en la que pueden almacenarse las traducciones.

Nombre*:

Para visualización en System4.

Área:

Descripción optativa: El ámbito de uso de los términos de esta memoria de traducción.

Ejemplos: Informática, medicina, enseñanza.

Empresa:

Descripción optativa: La empresa para la que se crean los términos de esta memoria de traducción.

Ejemplos: Márketing, ventas, ingeniería.

Descripción:

Descripción optativa de esta memoria de traducción.
