Create/Edit TeamSite Server

These screens are only viewable if you have a TeamSite integration.

 

Summary:

Create or edit a TeamSite server configuration.  This server will be used to send and receive localized content to GlobalSight.

Server Name

The IP address or domain name of the Teamsite Server.

Description

Optional description of the server.

Operating System

Operating system of the Teamsite server.

Export Port

The TeamSite webserver port used by GlobalSight to receive data exported from TeamSite. For TeamSite 4.5, TeamSite export port and TeamSite import port should be the same. For TeamSite 5.0.x and greater, TeamSite export port should have the same value as http_port in the /etc/iw.cfg file on the TeamSite server.

Import Port

The TeamSite webserver port used by GlobalSight when TeamSite imports data from GlobalSight. For TeamSite 4.5, TeamSite export port and TeamSite import port should be the same. For TeamSite 5.0.x and greater, TeamSite import port should have the same value as customer_webserver_port in the /etc/iw.cfg file on the TeamSite server.

Proxy Port

The TeamSite proxy port.

Home Diretory

The directory on the TeamSite server where TeamSite is installed.

Teamsite User

A TeamSite username with access privileges for creating Branches, Workareas and directories. This is used during the export of content back to TeamSite.  If the branch or workarea already exists this user is only required to have permissions to create directories.

Mount

The directory on the Teamsite server that is shared between TeamSite and GlobalSight. This usually on NFS or Samba mount.

Teamsite User Type

The user type of the TeamSite User (above).

Allow Locale Specific Reimport

Checked: this will allow to re-import the same file from TeamSite for different file profiles that have different locales. For example: FP_french is a File Profile that has a Localization Profile that allows en_US to fr_FR translation only. FP_german is a different File Profiile that has a Localization Profile that allows en_US to de_DE translation only. If the TeamSite re-import parameter is true, you can create a job, Job1, for a file HelloWorld.html using FP_French  and another  job, Job2, for HelloWorld.html using FP_german.

Unchecked: GlobalSight issues an error while creating a new job for the same file, if a job in progress has not been completed yet.

Buttons

 

Cancel

Cancels the TeamSite server configuration creation/edit.

Next.

Continues to the next step in configuration: Add Content Store.
