Import TeamSite File into GlobalSight

 

1.       The Task window opens.

· If your user level is Author, the Task window displays the tasks that are assigned to you.

· If your user level is Master, Administrator, or Editor, the Task window will be empty.

2.       Press the Workarea Button.

3.       Navigate in the Directory Panel to display the contents of the file directory that contains the files you want to import into GlobalSight. 

4.       Select the files you want to import into GlobalSight by checking them.  You can select single files, or you can select full directories.

5.       Click on the File drop-down menu, and select the Translate Files menu item.

6.       In the TeamSite/GlobalSight Import Screen, select a GlobalSight File Profile from the drop-down menu of existing GlobalSight File Profiles.

The selected File Profile needs to support the file format and character encoding of the files in the directory you are selecting for import into GlobalSight.

There are two possible behaviors that result from your choice of a GlobalSight File Profile:

·         Word Count Process

·         Batch Process

 

For a definition of these two concepts within GlobalSight, go to the GlobalSight Glossary.
 

7.       WORD COUNT PROCESSING: When you select a File Profile, you invoke a GlobalSight Localization Profile (LP).  If the LP calls for Word Count processing, you are ready to go.

8.       Press the OK Button.  This begins the import process for Word Count Jobs.

9.       BATCH PROCESSING: When you select a File Profile, you invoke a GlobalSight Localization Profile (LP).  If the LP calls for Batch processing, a text box is provided where you enter in a GlobalSight Job Name for this Batch Job. 

NOTE:  This name must be restricted to alphanumeric characters; any symbol characters like & or $ or % are not supported in GlobalSight.

10.   Once you have entered in the Job Name, press the OK button.  This begins the import process for Batch Jobs.


 

11.   An import that is able to transfer a file into GlobalSight has been routed to GlobalSight.

The GlobalSight TeamSite GUI Interface Module shows you the message shown in the Status Message Screen to the left.

NOTE: Even though a file imports into GlobalSight, there may still be an import error condition that needs to be resolved.  Your GlobalSight Project Manager will be advised of any import errors.

12.   Press the OK button to dismiss the message.

13.   Continue to import files until all the files you need to translate are in GlobalSight.

14.   Continue other work in TeamSite.

15.   During import, there is a GlobalSight check to ensure that the file formats and character encoding of the imported TeamSite files match the File Profile selected in the GlobalSight TeamSite GUI Interface Module. 

If the GlobalSight File Profile does not support the file format or character encoding of the files selected for import into GlobalSight, the files are not imported into GlobalSight.

The GlobalSight TeamSite GUI Interface Module shows you the message shown in the Status Message Screen to the left.

16.   Press the OK button.

17.   Select the appropriate File Profile that supports the file types for the TeamSite files you are importing into GlobalSight, and import the files again.
