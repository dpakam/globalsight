Introduction

GlobalSight integrates the content localization process into Interwoven’s TeamSite Content Management System (CMS) environment. Using the GlobalSight web-based application and the TeamSite web-based application together, you can create localization projects from within TeamSite, calling GlobalSight just like any other TeamSite menu item function. Once set up, the two-application localization process of TeamSite content is very simple for all users to perform.
