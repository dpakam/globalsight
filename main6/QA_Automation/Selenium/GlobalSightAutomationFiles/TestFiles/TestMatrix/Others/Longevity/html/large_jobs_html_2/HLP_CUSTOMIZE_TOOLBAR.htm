Customize Toolbars dialog box

Use this dialog box to customize buttons on the Review and Review Main toolbars. You can add, remove, and/or rearrange buttons in a toolbar. You can create a Custom toolbar that contains buttons that you use frequently, or combine all Review or Review Main buttons in one toolbar. The toolbars are saved when you exit Review and invoked when you restart Review.

Note: This feature is not available in QuickStart.

Available buttons list and Toolbar buttons list

Identifies buttons in the selected toolbar in Review or the Review Main window. The Available Buttons List identifies buttons not displayed in the selected toolbar. The Toolbar Buttons List identifies buttons displayed in the selected toolbar. Click a button, then double-click (or drag) it to add or remove it from the toolbar.

Add and Remove buttons:

  • Add (>) – Moves the button selected in the Available Buttons List to the Toolbar Buttons List (and displays it in the toolbar in Review).
  • Remove (>) – Moves the button selected in the Toolbar Buttons List to the Available Buttons List (and does not display it in the toolbar in Review).

Close

Closes the Customize Toolbars dialog box, accepting changes made to the toolbar.

Reset

Restores the default set of buttons to the selected toolbar in Review.

Move up

Moves the selected button up one line in the list. On the toolbar, the button shifts left one space.

Move down

Moves the selected button down one line in the list. On the toolbar, the button shifts right one space.
