Overlay tab

Use this tab to overlay multiple chromatograms, pump gradients, calibration curves, spectra, molecular weight distributions, Viscosity Law plots, or dissolution plots in Review (or Compare), or when printing a report.

Use this tab to overlay multiple chromatograms, pump gradients, calibration curves, spectra, molecular weight distributions, or Viscosity Law plots in Review (or Compare).

Overlay type

Do not overlay

Disables the overlay of chromatograms, pump gradients, calibration curves, spectra, molecular weight distributions, or Viscosity Law plots in Review (or Compare) or in a report.

Disables the overlay of chromatograms, pump gradients, calibration curves, spectra, molecular weight distributions, dissolution plots, or Viscosity Law plots in Review (or Compare) or in a report.

Make stack plot

Selected data in individual (horizontal) stack plots, one stack plot for each displayed data item. This selection opens multiple plots to fit next to each other in a horizontal array. Depending on the number of data items, this selection can display many small, independent plots.

Overlay in single plot

Overlays all chromatograms, pump gradients, calibration curves, spectra, molecular weight distributions, or Viscosity Law plots in a single plot.

Overlays all chromatograms, pump gradients, calibration curves, spectra, molecular weight distributions, dissolution plots, or Viscosity Law plots in a single plot.

Chromatogram annotation (Chromatogram plots only)

Note: Select Make Stack Plot or Overlay in Single Plot to assign chromatogram annotation to the overlaid chromatograms.

All Chroms in overlay

Selected chromatogram information (from the Chromatogram tab) for all overlaid chromatograms.

Only one Chrom in overlay

Selected chromatogram information (from the Chromatogram tab) for the selected chromatogram. If you specify this selection in a Chromatogram plot from Review, this is the selected chromatogram in the 2D Channels table. If you specify this selection in a Chromatogram plot in a report group (Report Publisher), this is the first chromatogram in the overlay.

Z-Axis

Note: Select Overlay in Single Plot to assign z-axis plotting.

Use Z-Axis plotting

Applies z-axis spacing to the overlaid data in the plot or in the report (which can be chromatograms, calibration curves, spectra, GPC distributions, and so on).

Grid lines

Grid lines for the z-axis in the background of the plot.

Note: Enable the Use Z-Axis Plotting check box to turn on and define grid lines.

Angle

Counterclockwise angle (in degrees) of rotation for the z-axis.

Entries: 0 to 90 (0 equals horizontal rotation and 90 equals vertical rotation)

Default: 15

Spacing

Distance between overlaid data along the z-axis. Entries: 0 to 100. Default: 10.

Pump gradients (Chromatogram plots only)

Includes the associated pump gradient in the Chromatogram plot. If more than one chromatogram is overlaid, the gradient from only the first chromatogram prints in the report. These pump plots can be included above the chromatogram (using check boxes):

  • Overlay %A Flow
  • Overlay %B Flow
  • Overlay %C Flow
  • Overlay %D Flow
  • Overlay Total Flow

Dissolution plot (Dissolution plots only)

Includes overlaid dissolution plot information (one plot per component overlaid as specified) from a Waters or non-Waters dissolution system along with chromatographic data from one or more detector channels. When used with the Waters Alliance Dissolution System, displays dissolution plot information for up to six or eight dissolution vessels per bath; when used with a non-Waters dissolution system, displays dissolution plot data for up to 99 vessels.

These dissolution plots can be overlaid:

  • Overlay Baths – Overlays plots from two dissolution test baths (assuming a two-bath system).

Note: If you overlay only the bath and not the vessel and/or channels, the software displays a plot for each vessel and channel in use, with baths A and B overlaid. For example, if you are using an eight-vessel system and chose not to overlay the vessel plots, the software displays eight separate plots (with each plot containing the overlaid dissolution bath plot).

  • Overlay Vessels – Overlays plot information from each vessel (up to six or eight) in each dissolution bath.

Note: If you overlay the bath and vessel data, the software displays an overlaid plot of both baths and all vessels in use.

  • Overlay Channels – Overlays plot information from each data channel used with the dissolution system.

Note: If you overlay the bath, vessel, and channel data, the software displays an overlaid plot of both baths, all vessels in use, and the data channels.
