Validation test data table properties - Test Data tab

Use the Test Data tab options to define the type of validation results whose chromatographic data you want to include in a report group.

Option Usage

Selected Test

(Default) Reports the chromatographic data used to calculate the validation results for the selected test type.

System Precision Data for All tests

Reports the chromatographic data used to calculate the system precision results for all test types. When selected, all other options are unavailable.

Test Data check box

(Default) Reports the chromatographic data used to calculate the validation results for the selected validation test type.

System Precision Data check box

Reports the chromatographic data used to calculate the system precision results for the selected validation test type.

Test Selection

Select the test type whose chromatographic data you want to include in a report group.

Tips:

  • The Linearity Comparison test reports chromatographic data from the two linearity tests.
  • The LOD/LOQ test reports chromatographic data from the referenced linearity test if you do not use the Signal-to-Noise method. If you use the Signal-to-Noise method, it reports the data acquired for LOD/LOQ.
