General tab (2410)

Use this tab to define general operating and data channel parameters for the 2410 RI Detector.

Channel name

Software-designated name for the channel, which appears in the Active Channels table in the Instrument Method Editor.

Description

Optional description for the detector data channel, up to 250 alphanumeric characters. The description appears in the Active Channels table in the Instrument Method Editor and in the Real-Time Plot pane in Run Samples.

Sampling rate

Rate used to collect data for the channel. Set a sampling rate to achieve a minimum of 15 data points across the narrowest peak of interest.

Rate used to collect data for the channel. Typically, for LC, IC, and GC, set a sampling rate to achieve a minimum of 15 data points across the narrowest peak of interest.

Choices: 1, 2, 5, 10

Default: 1

Unit label

Specifies whether the analog output is to read in mV or DelRIU.

Choices: mV or DelRIU

Default: mV

Filter time

Time constant of the noise filter to achieve the optimum signal-to-noise ratio. This parameter corresponds to TC SEC on the Waters RI Detector. See Auto Zero at t0 below for more details about setting Filter Time.

Choices: 0.2, 1, 3, 10

Default: 1.0

Sensitivity

Sensitivity of both the integrator and recorder signals. The higher the setting, the larger a response for a given RI change, and the higher the baseline noise. For easiest operation, choose the lowest sensitivity that yields satisfactory peaks.

Choices: 1, 2, 4, 8, 16, 32, 64, 128, 256, 512, 1024

Default: 4

Polarity

Polarity of the peaks. Samples detected with the 2410 refractometer can yield positive or negative peaks, depending on whether their RIs are greater than or less than the RIs of the solvent.

Choices: + or – (positive or negative)

Default: + (positive)

Note: Reversing polarity has no effect on peak integration or resolution.

Auto zero at t0

Adjusts the zero offset of the analog output to compensate for baseline changes.

Choices: Enabled or Disabled

Default: Enabled

Note: A shorter Filter Time setting decreases the time it takes for the detector to auto zero. A longer Filter Time setting increases the time it takes for the detector to auto zero.
