Degas tab (2790/2795)

Use this tab to select the vacuum degas mode.

Degas mode

Specifies the vacuum degas operating mode for the 2790/2795 separations module. Depending on the type of degasser you have installed, select one of these options:

  • Off – Turns the degasser off.
  • On – Turns the degasser on.
  • Normal – Cycles the vacuum degasser on and off.
  • Continuous – Continuously runs the vacuum degasser.
