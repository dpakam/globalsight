Derived Channels - First (only) channel tab

Use this tab to define the left side of the derivation equation for the first data channel. A description of the data channel appears at the bottom of the tab.

Constant

Specifies the constant by which the first data channel is to be multiplied (left side of the equation).

Entries: –9999.999 to 9999.999.

Default: 1.000.

Channel

Specifies the type of data channel to create. These channels relate to the instrument method specified in the Method Set Editor (from the Instrument Method list). Depending on the type of data channel you select, the software prompts you to enter parameters to define the channel, as indicated in this table.

Tip: The allowable entries for wavelength or mass will depend upon the chosen instrument method.

Timed wavelength or timed mass

Allows you to create a channel that specifies a wavelength or a mass at specified starting times. When you select Timed Wavelength or Timed Mass from the First (Only) Channels tab Channel list, a table appears with two columns. In the first column, enter the time to start the wavelength or mass value you specify in the second column. The software uses the information from the table to extract the derived channel. The software does not normalize the values between changes in wavelength or mass.

Peak separation

Allows you to specify a tolerance range or "window" that is used when extracting a chromatogram from a 3D MS data set. The values of the window are defined as:

  • Upper range: all masses < [mass + (0.5 x peak separation value)]
  • Lower range: all masses ³ [mass – (0.5 x peak separation value)]

The intensity value used in the chromatogram is the sum of all peaks in the window (greater than or equal to the lower value and less than the upper value). If no peaks fall in the range, the intensity value is 0.

The peak separation value is expressed in daltons. The default is 1.0 daltons.
