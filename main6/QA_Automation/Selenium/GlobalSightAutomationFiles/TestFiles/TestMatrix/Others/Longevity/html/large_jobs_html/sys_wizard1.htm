New Chromatographic System Wizard – Select Server Page

Use this page to select the Empower Node connected to the instruments that you need to build your chromatographic system. Select the appropriate Empower Node, then click Next.

If you do not see the desired Empower Node in the list, either you have not defined it or you do not have access to it. To create an Empower Node, select File > New > Empower Node.

A message informs you that Empower software is attempting to connect to the Empower Node that you selected.
