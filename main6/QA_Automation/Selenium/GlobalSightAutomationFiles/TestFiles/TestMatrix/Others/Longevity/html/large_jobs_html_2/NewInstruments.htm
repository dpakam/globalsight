Add Instruments dialog box

Use this dialog box to select instruments to add to your instrument method. Adding instruments in this manner lets you construct an instrument method with no configured system displayed in the Instruments and Active Channels tree, or lets you add instruments to an existing configured chromatographic system.

Use this dialog box to select instruments to add to your instrument method. Adding instruments in this manner lets you construct an instrument method with no configured system or lets you add instruments to an existing configured chromatographic system.

Note: Adding an instrument to an existing chromatographic system does not change the originally configured system; it only adds an instrument to the Instruments and Active Channels tree.
