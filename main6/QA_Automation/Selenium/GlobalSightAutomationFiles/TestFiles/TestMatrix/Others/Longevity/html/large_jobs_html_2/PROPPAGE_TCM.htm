Waters TCM properties

Use the TCM tab to define parameters for controlling the temperatures of up to three column heaters regulated by a Waters Temperature Control Module. Column heaters are attached to the External Device Connections (Units 1, 2, and 3) on the rear panel of the TCM.

Note: Set the column heater operating parameters for a Waters 600 Series Controller (600E, 616, 625, or 626) in the Temperature tab for the Waters 600 Series Controller.

Zone 1 enable to zone 3 enable

Enables the specific column heater (1, 2, or 3).

Choices: Enabled or Disabled

Default: Disabled

Zone 1 set point to zone 3 set point

Operating temperature in °C for column heater 1, 2, or 3. This value must be less than or equal to the corresponding High Limit value and greater than or equal to the corresponding Low Limit value.

Default: 0.0 °C

Zone 1 high limit to zone 3 high limit

High-temperature limit for column heater 1, 2, or 3. This value must be greater than or equal to the corresponding Set Point value.

If the heater temperature exceeds this limit, the TCM shuts off the heater and sounds the alarm. The "OVER" LED on the TCM illuminates.

Caution: To avoid damaging your column, set the High Limit parameter to a value no greater than the maximum limit for the corresponding column.

Default: 110.0

Recommended setting: Within 5 °C of the Set Point setting.

Zone 1 low limit to zone 3 low limit

Low-temperature limit for column heater 1, 2, or 3. This value must be less than or equal to the corresponding Set Point value.

If the heater temperature falls below this limit, the TCM sounds an alarm, but does not turn off the heater. The alarm continues to sound until the condition is corrected.

Default: 0.0 °C
