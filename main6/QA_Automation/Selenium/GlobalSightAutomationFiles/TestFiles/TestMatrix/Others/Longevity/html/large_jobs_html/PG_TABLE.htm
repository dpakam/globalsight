Table tab

Use this tab to define the content (table columns, column order) and general characteristics of a custom report group table.

Note: Units for the columns in a table appear in the table column headings.

Available fields list

Fields in the Empower database available to include in the table. Click an item to view the list of available fields, then double-click (or drag) the desired field in the Available Fields tree. The field is added to the Table Template as a column.

Note: The Available Fields list also includes the Result Sign Off database fields (Sign Off Date, Sign Off Full Name, and Sign Off Reason). To view the result sign-off information, create a report table that contains the Result Signoff database fields. This table can also contain sample- and channel-level information to further identify results.

Table title

Title directly above the table. The table title can have up to 250 alphanumeric characters without restrictions. If multiple lines are required, press Ctrl+Enter to force the title onto multiple lines.

Note: When you specify a table title, ensure that the title is not wider than the potential table-width. If the title is longer than the generated table, the title is truncated. Correct this problem by shortening the title or forcing it onto multiple lines.

Alignment

Position of the title above the table.

Choices: Left Alignment, Center Alignment, Right Alignment

Default: Center Alignment

Table template

Headings and order of the table columns as they are moved from the Available Fields list. To move a column, drag it to a different location in the Table Template.

You can also right-click a column to access shortcut menu options. The options that appear depend on the report group you select and the condition of the table.

Attention: Do not select Ascend or Descend for a column if it is not available on the Order By tab. Doing so might result in data not appearing in the table.

Choices:

  • Ascend – Sorts values in the column in ascending order.
  • Descend – Sorts values in the column in descending order.
  • No Order – Removes sorting from the column. This option is available if Ascend or Descend is used.
  • Hide Column – Hides the selected column.
  • Show All Columns – Displays all columns in the table.
  • Delete Column – Deletes the selected column.
  • Column Properties – Accesses the Column Properties dialog box, where you change the name, alignment, data precision, and summary function of the column.
  • Table Properties – Accesses the Table Properties dialog box.

To delete a column from the Table Template, right-click the column name, then select Delete Column (or highlight the columns and press Delete). You cannot leave blanks between columns.

Note: You can also specify sorting order in the individual columns of the Table Template.

Column properties

Accesses the Column Properties dialog box to modify the attributes of a specified column in the Table tab. To modify column attributes, select the corresponding column in the Table Template, then click Column Properties. After redefining column attributes, click OK to return to the Table tab.

Print blank columns

Prints table columns that do not contain values.

Default: Enabled

Include Duplicate Results

Reported data includes duplicate results that have the same Channel ID. Duplicate results for standards occur when calibrating using bracketing with overlap. When calculating summary information, such as the % RSD, for bracketed standards throughout a sample set, to include each bracketed standard only once in the calculation, clear this checkbox to exclude the duplicate results (those with the smaller result ID).

Default: Enabled
