New Project wizard – Name Entry page

Use this page to enter a name for the new project and optional comments.

Project name

Project in the Empower database and in the Empower Configuration tree in Configuration Manager. Enter up to 30 alphanumeric characters for the project name (You can include spaces and underscores). Use a meaningful project name to share the project with other users.

Enter up to 30 alphanumeric characters for the project name (You can include spaces and underscores). Use a meaningful project name to share the project with other users.

Comment

Enter a description about the nature of the project, creation date, instrumentation involved, and so on.

Raw Data Selection

Select the location where you want to store the project's raw data files.

Tip: This selection appears only when you configure more than one raw data path (View > Manage Raw Data Files in Configuration Manager.)
