Password Required dialog box

Use this dialog box to enter a password to access resources that are password-protected. This dialog box appears when you first attempt to access the following password-protected resources:

  • Acquisition server when creating a new chromatographic system
  • Acquisition Server Properties dialog box
  • Chromatographic System Properties dialog box
  • Message Center Properties dialog box

Once you enter the correct password, it is stored with your user name in the database and you do not have to enter it unless you change it.

Enter the required password in the text box, then click OK to proceed. The resource becomes available to you. If you click Cancel, you can view the resource properties, but you cannot change them.
