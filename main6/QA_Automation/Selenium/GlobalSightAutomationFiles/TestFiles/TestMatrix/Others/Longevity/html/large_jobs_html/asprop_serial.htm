Serial ports tab

Use this tab to configure serial instruments connected to serial ports of the busLAC/E card, PC COM ports, or LAC/E³² Empower Node. When you configure an instrument in this tab, the instrument appears in the Instruments tab, where you can add information about the instrument.

Use this tab to configure serial instruments connected to serial ports of the busLAC/E card or the PC COM ports. When you configure an instrument in this tab, the instrument appears in the Instruments tab, where you can add information about the instrument.

BusLAC/E serial ports

Specifies serial instruments, if any, that are currently configured on the busLAC/E card serial ports (1 to 4). Clicking the list displays instruments that can be added.

Default: None

PC serial ports

Specifies serial instruments, if any, that are currently configured on the COM ports (COM1 to COM4) of the serial card in your Empower Personal workstation. Clicking the list displays instruments that can be added.

Specifies serial instruments, if any, that are currently configured on the COM ports (COM1 to COM4) of the serial card in your Breeze 2 workstation. Clicking the list displays instruments that can be added.

Default: None

8-port serial card

Specifies serial instruments, if any, that are currently configured on the 8-port serial card ports (1 to 8). Clicking the list displays instruments that can be added.

Default choices: None, 6890, 5890S, 7673S, SAT/IN 2, Agilent 1100, or W474

Default: None

Considerations

Be aware of the following considerations:

  • If a field displays "Busy," the port is in use and cannot be configured.
  • If a field displays "Not Available," the busLAC/E or LAC/E³² Empower Node is not available and cannot be configured.
  • If a field displays "Not Available," the busLAC/E or LAC/E³² Breeze 2 Node is not available and cannot be configured.
  • The software identifies serial ports (other than busLAC/E ports) by their COM port number. The current port numbers were increased by two. For example, Port 1 on the 8-port serial card in the current configuration corresponds to COM 3. Port 2 is Port 4.
