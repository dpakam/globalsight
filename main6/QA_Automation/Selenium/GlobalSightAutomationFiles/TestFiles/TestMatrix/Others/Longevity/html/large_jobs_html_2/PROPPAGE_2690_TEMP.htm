Temperature tab (2690/2695, 2690D/2695D, and 2790/2795)

Use this tab to define parameters that control temperature for the optional column heater and optional sample compartment heater/cooler.

Column temperature enable

Enables temperature control of the optional column heater.

Choices: Enabled or Disabled

Default: Disabled

Column temperature target

Target operating temperature for the optional column heater. This value must be at least 5 °C above ambient.

Range: 4.0 to 65.0 °C

Minimum setting: 5 °C above ambient

Default: 25.0 °C.

Column temperature range

Maximum allowable temperature deviation from the value set for the Column Temperature Target. If the column heater temperature deviates beyond the range, an error message appears and the run stops.

Default:±5.0 °C

Sample temperature enable

Enables temperature control of the optional sample compartment heater/cooler.

Choices: Enabled or Disabled

Default: Disabled

Sample temperature target

Target operating temperature for the optional sample compartment heater/cooler.

Default: 25.0 °C

Sample temperature range

Maximum allowable temperature deviation from the value set for the Sample Temperature Target. If the sample compartment temperature deviates beyond the range, an error message appears (and the run stops if the default method in the separations module has a Stop Flow alert response for sample temperature errors).

Default: ±5.0 °C
