New Project wizard – Copy Selection page

Use this page to specify if you want to copy view filters, custom fields, methods, or preferences from a selected project to the new project.

Copy

You can perform the following tasks:

  • View Filters – Copies all view filters from a specified project into the new project.
  • Custom Fields – Copies all custom fields from a specified project into the new project.
  • Methods – Copies all methods from a specified project into the new project.
  • Preferences – Copies preferences (for example, table and plot properties) from a specified project into the new project.
  • From Project – Specifies the project from which Empower software copies view filters, custom fields, and methods. Default: Defaults.

Tips:

  • If project Defaults is unavailable, another project appears at the top of the list. If no project is available, the check boxes are not selectable. In this case, you can create an empty project only. You can also create an empty project when another project is available and you clear the Copy check boxes.
  • If the Method Validation option is installed and you are creating a validation working project, you must select a validation template project. This validation template project can contain validation protocol methods and other associated methods to use in your validation studies. Any custom fields in the validation template project are copied to the validation working project. When you generate validation results, Empower uses these custom fields to track method validation information.

Use this page to specify if you want to copy all methods from an existing project to the new project.

To copy methods from another project to the new project:

  1. Select the Seed new project with methods from project check box.
  2. Select a project from the list and then click Next.
