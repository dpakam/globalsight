MS centroid tab

Use this tab to convert MS continuum data to centroid and identify how peaks are found and how they are centroided.

Center

Select one of the following to specify how Empower software calculates the mass (m/z) of a peak (the position of the centered spectrum stick):

  • Top – Places the spectrum stick at the peak maximum.
  • Centroid Top (%) – Places the spectrum stick at the centroid of the peak. This is the default. Enter the percentage of the resolved portion used to calculate the centroid in the percentage field.

Percentage Field

The percentage you specify dictates where the summation runs over the top of the peak.

Default: 80%

Range: 0 to 100 (good values to use are 60 to 95%)

The centroid of the peak is calculated at:

Sum of (intensity*(m/z)) / sum of (intensity)

Median: Represents the mass of the peak. The spectrum stick divides the peak area in half.

Intensity

Select either parameter to specify how Empower software calculates the intensity (height) of the spectrum stick:

  • Heights – Reads the intensity value from the continuum spectrum at the mass (m/z) of the spectrum stick.
  • Areas – Reads the peak area as the intensity of the spectrum stick.

Peak detection

Use the following to specify how the peaks will be detected:

  • Minimum Peak Width at Half-height (Dalton) – Identifies the minimum range a peak should be visible in the expected shape across the peak top. Default: 0.500 Dalton.

Note: If you set the value too low, too many peaks appear in the results. If you set the value too high, many continuum peaks may be misread as a single centroid peak.

  • Smooth – Smooths over a noise or oversampled spectrum before performing peak detection.

Interaction with Spectrum Review Table

The software performs smoothing based on the value in the Smoothing Window column in the Spectrum Review table. Choices:

  • No value in the Smoothing Window column and the Smooth check box is selected – Performs smoothing with the window size specified by the Minimum Peak Width at Half-height field on this tab.
  • Value in the Smoothing Window column – Performs smoothing with the value in this column, overriding the Smooth check box.
