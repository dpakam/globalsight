GPC Processing Method wizard

Use this page to instruct the software how to proceed with creating or modifying a GPC processing method in Review. Choices:

  • Create New Processing Method – Use the wizard to create a new processing method and start a new calibration (which activates the New Processing Method dialog box where you select the processing type for the new processing method).
  • Edit Existing GPC Processing Method and Keep the Calibration – Use wizard to edit the current GPC processing method in Review, keeping the calibration curve associated with the processing method.
  • Edit Existing GPC Processing Method and Clear the Calibration – Use wizard to edit the current GPC processing method in Review and start a new calibration for the modified processing method.
