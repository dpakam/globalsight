Channel 1 tab (2996 PDA)

Use this tab to define characteristics of analog output channel 1 for the 2996 PDA Detector.

Note: Enable the Analog Ch 1 Enable parameter in the 2996 PDA General tab.

Output mode

How data is to be read (as it is sent to the analog output channels). Choices:

  • Off – No output (Default).
  • Absorbance – Output represents absorbance at the wavelength specified by the Output Wavelength parameter (see below).

Note: Ratio Wavelength, Ratio Threshold, Ratio Minimum, and Ratio Maximum parameters are not accessible when you select Absorbance mode.

  • Ratio – Output represents the ratio of absorbances at two wavelengths. The numerator wavelength is specified by the Output Wavelength parameter, and the denominator wavelength is specified by the Ratio Wavelength parameter (see below).

Bandwidth (absorbance and ratio modes)

Spectral bandwidth of the analog output channel.

Default: 4.8 nm

Output wavelength (absorbance and ratio modes)

Output wavelength. In Ratio mode, the absorbance at the Output Wavelength is used to calculate ratio in the formula:

Ratio = Absorbance at Output Wavelength/Absorbance at Ratio Wavelength

Output Wavelength must be within the wavelength range specified by the Start Wavelength and End Wavelength parameters in the 2996 PDA General tab.

Range when Bandwidth is set to 1.2: Start Wavelength to End Wavelength

Range at all other Bandwidth settings: Start Wavelength + (Bandwidth/2) to End Wavelength – (Bandwidth/2)

Default: 254.0 nm

Offset (absorbance and ratio modes)

Adds an optional constant offset in AU to the analog output signal.

Default: 0.0 AU

Ratio wavelength (ratio mode)

Denominator wavelength (in nanometers) for the analog output channel. Ratio Wavelength must be within the wavelength range specified by the Start Wavelength and End Wavelength parameters in the 2996 General tab.

Range when Bandwidth is set to 1.2: Start Wavelength to End Wavelength

Range at all other Bandwidth settings: Start Wavelength + (Bandwidth/2) to End Wavelength – (Bandwidth/2)

Default: 254.0 nm

Ratio threshold (ratio mode)

Absorbance threshold for the numerator and denominator wavelength channels. A ratio is plotted only when the absorbance in both channels is above this value. Otherwise, the output is 0.0 V plus the Offset value.

Default: 0.001 AU

Ratio minimum (ratio mode)

Lower limit of the ratio output. This setting is equal to 0.0 V on the output scale.

Default: 0.001 ratio units/V

Ratio maximum (ratio mode)

Upper (full-scale) limit of the ratio output.

Default: 100.00 ratio units/V

Filter type (absorbance and ratio modes)

Filter type used on the analog output channel. The Hamming filter is designed to create the same degree of peak-height degradation as the RC filter for the same response time, but enhances filtering of high-frequency noise.

Choices: Hamming or RC

Default: Hamming

Filter response (absorbance and ratio modes)

Response time in seconds for the Filter Type specified above.

Default: 0
