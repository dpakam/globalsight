Calibration tab (GPC)

Use this tab to define how to calibrate GPC/V standards using relative or universal calibration.

Use this tab to define how to calibrate GPC standards using relative or universal calibration.

Field

Description

GPC technique

Calibration technique used to generate a calibration curve. The choices are:

  • Relative Calibration – Generates a calibration curve that plots Log Mol Wt as a function of Retention Time or Elution Volume.
  • Universal Calibration – Generates a calibration curve that plots Log Hydrodynamic Volume as a function of Retention Time or Elution Volume. The hydrodynamic volumes are calculated using the standard’s molecular weights and its K and alpha values (GPC modified universal) or intrinsic viscosity(s) from the viscosity channel data (GPCV Benoit’s protocol).
  • Universal Calibration – Generates a calibration curve that plots Log Hydrodynamic Volume as a function of Retention Time or Elution Volume. The hydrodynamic volumes are calculated using the standard’s molecular weights and its K and alpha values (GPC modified universal) or intrinsic viscosity(s) from the viscosity channel data (GPC Benoit’s protocol).

Tip: When you change the GPC technique for a GPC processing method, a message indicates that this action forces creation of a new calibration curve for the processing method. Either click OK to keep the modified GPC technique (and force creation of the calibration curve) or click Cancel to keep the existing calibration curve.

Fit type

The curve-fitting algorithm to be applied to the calibration curve.

V0, Vt

Exclusion volume (V0) and total retention volume (Vt) for the column set. If you do not enter values for V0 and Vt:

  • For fit types other than Bounded, the software uses the retention times (or elution volumes) of the first and last points in the Points table. V0 and Vt are set as follows:
  • Narrow standards use the point value.
  • Broad standards use the start and end of the distribution to set V0 and Vt.
  • Cumulative match broad standard uses the point value.
  • For the Bounded fit type curve, the software calculates V0 and Vt based on the calibration data.

Note: Peak slices that fall outside V0 and Vt are identified as such in the Molecular Weight Distribution table and no molecular weight or log (MolWt) is reported. You can extend the molecular weight range by entering values for V0 and Vt. However, if you extend V0 and Vt too far, the slope of the calibration curve may change sign, making the curve invalid.

X-Axis

Calculation of the calibration curve is performed using one of these as the x-axis:

  • Time (in minutes)
  • Volume (in milliliters)

Flow rate

The flow rate used to calculate elution volumes when the x-axis of the calibration curve is elution volume. Enter the flow rate accurately, because it is used to calculate your calibration curves. An accurate flow rate is necessary to accurately calculate the intrinsic viscosities of your GPCV standards and unknowns.

Reference peak, standard/unknown

A reference peak and its retention time window. If the software finds a peak within the window, it adjusts the retention times for the chromatogram based on the reference peak retention time. The software can use the reference peak to adjust retention times in only standards, in only unknowns, or in both standards and unknowns.

Retention Time button and text box

Retention time for the reference peak (standard or unknown). If you click the Retention Time button, the software selects the retention time of the selected peak within the Chromatogram plot of the Review Main window.

Note: The Retention Time button and text box are available only when you select the Standard and/or Unknown check box.

RT window (%)

Retention time window for the reference peak as a percentage of the peak’s retention time.

Entries: 0.001 to 10.000.

Default: 1.000.

Guidelines:

  • A peak that falls within this window is identified as the reference peak.
  • If more than one peak falls within the window, the peak closest to the retention time is identified as the reference peak.
  • If no peak is found within the window, the software adds to the result, a missing peak that has the same retention time as the reference peak.

Rule: If the software does not find the reference peak within the RT window, it stops processing the standard or unknown.

Calibration order of molecular weights

The way to order the narrow standard molecular weights when matching the molecular weights with peaks to produce the calibration points:

Option

Purpose
High to Low Used for traditional GPC and GPCV separations.
Low to High Used for Capillary Electrophoresis (CE) and Field Flow Fractionation (FFF), but not for bounded fit-type calibration curves or for distribution calculations.

Rule: Do not use broad standards or broad unknowns with a GPC processing method that uses the Calibration Order of Molecular Weights set Low to High.

When you change the calibration order of molecular weights for an existing GPC processing method, a message indicates that this action forces creation of a new calibration curve for the processing method. Either click OK to keep the modified GPC technique (and force creation of the calibration curve) or click Cancel to keep the existing calibration curve.
