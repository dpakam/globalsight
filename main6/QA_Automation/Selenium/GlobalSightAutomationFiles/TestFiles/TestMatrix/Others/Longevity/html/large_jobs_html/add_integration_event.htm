Add Integration Event

  • QuickStart  or Pro > Browse Project > click the tab for the data type you want > right-click the data you want to use > Review Data Main window
  • Find Data > click the tab for the data type you want > right-click the data you want to use > Review Data Main window

Complete the information in this dialog box to add a peak detection or integration timed event. Most events cannot overlap themselves, and some events cannot overlap others.

Field

Description

Time (mins)

Enter a start time for the event. All events require a start time. The default value is 0.00, so the event will start at the beginning of the chromatogram.

Type

Select the type of timed event you want to add from the list. The list contains only the timed events supported by the selected integration algorithm.

Value

This field is active if the timed event you selected requires a value. Otherwise, the field is inactive.

Stop (mins)

Enter a stop time for the event. If you leave the field blank, the event will stop at the end of the chromatogram.

Show Integration Events on Chromatogram Plot

Select this check box if you want the time and type of timed event to be marked on the chromatogram.
