Annotation Object Properties - Line tab

Use this dialog box to change properties of the line or arrow in Report Preview. Access this dialog box by right-clicking the line or arrow and selecting Properties.

Line attributes

Style

Select a line style: solid, dashed, dotted, dash dot, or dash dot dot.

Thickness

Click the up arrow to increase the thickness of the line, and click the down arrow to decrease it. The range for this value is 0 to 1000.

Notes:

  • Waters recommends that you specify a line thickness of 0 when using line styles other than Solid. Line thicknesses vary with monitor and printer resolution, which can result in the various line style selections being displayed as Solid. A selection of 0 ensures that the line styles appear properly.
  • Using a thickness greater than 10 can cause a line to appear distorted.

Line color

Accesses the Color dialog box and changes the color of the line. You can use all colors supported by the Windows operating system, which includes the basic and custom colors.

Line background

Transparent

Makes the background for the line transparent. Clearing this check box activates the Color button.

Color

Accesses the Color dialog box and change the background color of the line.

Note: The background color does not appear for a solid line.

Arrow style

Change the currently selected line by selecting a line, a line with an arrow on the front, or a line with an arrow on both ends from the list.
