New Chromatographic System wizard – Type Entry page

Use this page to select a new system type.

Create New System

Specifies that you create a new chromatographic system using available instruments connected to an Empower Node. The system might not be directly connected to your Empower Personal workstation.

Connect to Existing System

Connect to an existing Empower chromatographic system on this workstation or if you have an Enterprise configuration, another Node. The current user must have access to the selected chromatographic system. If the system is password-protected, you need the password to access the system. Then you can use the chromatographic system to acquire and process data, and to generate reports.
