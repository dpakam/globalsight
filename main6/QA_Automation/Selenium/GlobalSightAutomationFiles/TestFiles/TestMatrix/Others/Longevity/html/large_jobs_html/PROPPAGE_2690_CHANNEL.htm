Channel tab (2690/2695, 2690D/2695D, and 2790/2795)

Use this tab to specify the Waters 2690/2695, 2690D/2695D, or 2790/2795 parameter to be transmitted over the IEEE-488 interface and monitored by Empower software. Specify one of these parameters:

  • Flow Rate (Default)
  • System Pressure
  • %A, %B, %C, %D (each is a separate selection)
  • Loop Pressure (2690/2695 and 2690D/2695D only)
  • Primary Pressure (2690/2695 and 2690D/2695D only)

Channel enabled

Activates the data channel for the separations module. This parameter must be enabled to monitor one of the above parameters.

Choices: Enabled or Disabled

Default: Disabled

Channel name

Software-designated name for the channel, which appears in the Active Channels table in the Instrument Method Editor.

Description

Optional description for the channel, up to 250 alphanumeric characters. The description appears in the Active Channels table in the Instrument Method Editor and in the Real-Time Plot pane in Run Samples.

Parameter to monitor

Parameter (see above) that is to be transmitted over the IEEE-488 interface as a data channel and monitored by Empower software. During a run, you can view the selected parameter from Run Samples. From the Channel view in the Project window, you can view the selected parameter as data in the Channel column.
