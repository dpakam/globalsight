Valves Tab (6890)

Use this page to specify the parameters associated with optional valves 1 through 8.

Valve Settings:

Parameter

Function

Type

Specifies the type for Valve 1 through 8.

Choices: Not Installed, Multiposition, Gas Sampling, Switching, Other
 

Initial Position

Indicates the initial valve position.

Choices:

  • Multiposition = 1 through 32
  • All other valves are either on or off 

Default: 1 (multiposition), or Off for all other valves.

Multiposition

Enables Invert BCD and sets the pulse delay.

Choices: 1.0 through 100.0 seconds

Default: 1.0 sec

Multiposition:

Parameter

Function

Multiposition

Enables Invert BCD and sets the pulse delay.

Choices: 1.0 through 100.0 seconds

Default: 1.0 sec

Only one multiposition valve can be configured.

Gas Sampling Valve Settings:

Parameter

Function

Loop Volume

Specifies the volume for the loop.

Range: 0.001 through 10.000 ml

Default: 1.000 ml

Load Time

Indicates the amount of time to wait in the load position.

Range: 0.00 through 999.99 minutes

Default: 0.50 min

Inject Time

Specifies the time to wait in the inject position.

Range: 0.00 through 999.99 minutes

Default: 0.50 min

Inlet

Indicates the inlet to use for sampling.

Choices: Front, Back, and None

Default: None

Only two gas sampling valves can be configured in Empower.
