Waters GPCV Instrument Properties tab

Note: The GPCV instrument is available only if the Empower GPCV option is installed on your system.

Use this tab to specify instrument method parameters for creating a GPCV channel set using the "virtual" instrument, which does not physically exist in a system. After specifying parameters here (and parameters for other instruments in your GPCV system), save them as an instrument method. If you later define a GPCV channel set in the Method Set wizard, these GPCV parameters transfer to corresponding fields in the Define GPCV Channel Set wizard page.

Notes:

  • These GPCV parameters are used only for processing GPCV data, not acquirinig GPCV data.
  • You can add the GPCV instrument using the Add Instrument command or tool in the Instrument Method Editor. If the instrument method contains the GPCV instrument, click the GPCV instrument icon to display this tab.

Concentration detector parameters

Channel

Concentration channel for your system. Select the channel from the list. You can select a PDA channel for this field.

Flow rate (ml/min)

Flow rate used to calculate retention times. This is the flow rate you must enter in your processing method when the x-axis of the calibration is set to elution volume. This parameter is used only for processing; not defining the flow rate for data acquisition.

Default: None

Note: The value entered here is used for reference. It is transferred only to the Flow Rate field in your processing method when you create a new processing method. To create a new processing method, select Process > Calibrate Light Scattering System or Process > GPCV or Process > GPCV-LS Data or Process > GPC-LS Data in Review. Otherwise, you must enter the correct flow rate value in your processing method. Click for details on defining the flow rate in a GPC/V processing method.

Connected to RI detector

Specifies that the concentration channel is connected to an RI detector.

RI calibration constant

Calibration constant for the RI detector in units of 1/µV. This field is active only if Connected to RI Detector is enabled. Default: None.

Viscometry detector parameters

Channel

Viscometer channel for your system. Select the channel from the list.

Interdetector time (min)

Time offset for data points in the viscometer channel.

Default: 0.000 min

Zero offset

Offset that compensates for the difference between the baseline zero of the RI detector and the baseline zero of the viscometer (at zero flow rate). When a Viscotek dual capillary viscometer is connected through a Waters busSAT/IN Module to the system, this parameter is used with the 2nd Channel Constant parameter in the Empower calculation of the derived channel.

Default: 0.000 mV

2nd channel

Optional second viscometer channel for your system (required when a Viscotek dual capillary viscometer is connected through a Waters busSAT/IN Module to the system). Select the appropriate SAT/IN channel from the list.

2nd channel constant

Compensates for the pressure differential between the two channels. This parameter is used with the Zero Offset parameter in the Empower software calculation of the derived channel (for a Viscotek dual capillary viscometer), and is active only if a 2nd Channel is selected.

Default: 1.000000

Description

Optional description for the GPCV instrument parameters that define the GPCV channel set.
