Degas tab (2690/2695, 2690D/2695D, and 2790/2795)

Use this tab to set the helium sparge rate for selected reservoirs or to select the vacuum degas mode.

Reservoirs to sparge (mL/min) (2690/2695 and 2690D/2695D Only)

Sparge rate in mL/min for reservoirs A, B, C, and D to be helium sparged. Select reservoirs for sparging by entering a value greater than zero in the A, B, C, and D fields.

Default: 0.0 mL/min (no sparge)

Degas mode

Vacuum degas operating mode for the separations module. Depending on the type of degasser you installed, select one option:

  • Off – Turns the degasser off.
  • On – Turns the degasser on.
  • Normal – Cycles the vacuum degasser on and off.
  • Continuous – Continuously runs the vacuum degasser.
