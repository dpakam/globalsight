New Sample Set Method wizard – Describe Run Time Options page

Use this page to select your desired run mode.

Use this page to select your desired run mode and interactive system suitability operating mode.

Note: This page only appears if you launched the New Sample Set Method Wizard from Run Samples.

Run mode

Choose from the following run modes:

  • Run Only – Acquires data without processing the data or printing a report. Interactive system suitability operating mode is effectively disabled.
  • Run and Process – Acquires and processes data (using the processing method in the method set) without printing a report.
  • Run and Report – Acquires and processes data (using the processing method in the method set), and prints a report, or you can print a report from the Samples table – it does not have to be in the Method Set. The software prints any reports of the processed data, depending on the type of report method included in the associated method set.

Interactive system suitability

Empower software allows you to select an interactive system suitability operating mode during acquisition and processing (this mode does not require installation of the System Suitability option). This selection provides you with fault detection functionality similar to that provided in the System Suitability option (for example, to stop acquisition if a specified "must" peak in the processing method is not detected during processing). The following selections are available from the drop-down list:

  • Continue on Fault – Continues to make injections and/or acquire data even if a Fault condition occurs during processing (default).
  • Reinject on Fault – Stops acquiring data and repeats the current injection if a Fault condition occurs during processing. Injections are repeated as many times as you specify in Customize Defaults. If the reinjection also results in a Fault, acquisition is stopped.
  • Stop on Fault – Stops acquisition when a Fault condition occurs during processing.
  • Next Sample on Fault – Skips to next sample when a Fault condition occurs during processing.
  • Next Sample Set on Fault – Skips to next sample set when a Fault condition occurs during processing.

Note: Both Stop on Fault and Reinject on Fault selections add extra time to the injection process. These selections instruct Empower software to wait until the current injection is processed and checked for faults before performing the next injection. Fault checking occurs for each Run Mode selection.

After making your selections, click Next to advance to the next wizard page.
