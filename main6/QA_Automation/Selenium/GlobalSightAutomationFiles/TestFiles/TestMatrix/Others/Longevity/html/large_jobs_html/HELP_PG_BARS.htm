Bars tab

Use this tab to define characteristics of the System Suitability RSD plot or the method validation plots when printed in a report.

Bars to edit

Type of data to display in the System Suitability RSD or validation plots.

System Suitability RSD plot choices: Normal Data, Faulted Data. Default: Normal Data.

Validation plot choices: Data Points

Line attributes

Line style

Line style to use for the System Suitability bar plot lines.

Choices: Solid, Dashed, Dotted, Dash Dot, Dash Dot Dot.

Default: Solid.

Thickness

Line weight of the System Suitability bar plot lines.

Entries: 0 to 1000.

Default: 0 (thinnest line of 1 pixel).

Note: Waters recommends that you specify a line thickness of 0 when using line styles other than Solid. Line thicknesses vary with monitor and printer resolution, which can result in the various line style selections being displayed as Solid. A selection of 0 ensures that the line styles are displayed properly.

Line color

Color of the lines used to create the System Suitability bar plot. Click Line Color to access the Color dialog box. You can use all colors supported by the Windows operating system, which includes basic colors and custom colors.

Fill attributes

Transparent

Transparent background for the System Suitability bar plot lines. When this property is selected, objects behind the System Suitability bar plot line are visible.

Solid

Solid fill pattern for the System Suitability bar plot lines. When this property is selected, objects behind the System Suitability bar plot line are not visible. You can choose a solid fill color for the bar plot by clicking the Fill Color button (see below).

Note: You must clear the Transparent check box to access the Solid option button.

Hatch

Hatch fill pattern for the System Suitability bar plot lines.

Choices: Horizontal, Vertical, Forward Diagonal, Back Diagonal, Cross, Diagonal Cross.

Default: Forward Diagonal.

Note: To use a fill color for a points plot, select Solid for the Fill Attributes parameter.

You can choose a fill color for the hatch fill pattern in the bar plot by clicking the Fill Color button (see below).

Note: Clear the Transparent check box to access the Hatch option button.

Fill color

Color of the System Suitability bar plot lines (for the Solid or Hatch fill pattern). Click Fill Color to access the Color dialog box. You can use all colors supported by the Windows operating system, which includes colors and custom colors.
