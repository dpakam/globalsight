Component Editor

If your samples are LC samples, the Components view tab appears with its associated view table. If your samples are GPC/V samples, the following view tabs (and their associated view tables) appear: Mol Weights, K-Alphas, Moments, Distribution, Cumulative, and Components.

If your samples are LC samples, the Components view tab appears with its associated view table.

Component Editor user interface elements:

UI element

Usage

Menu Bar

Displays menus for performing the commands used in the Component Editor.

Toolbar

Displays shortcut buttons that provide quick access to many of the primary commands used in the Component Editor.

Sample Type Dialog Bar

Displays the Standards Only and Standard & Unknowns selections that let you choose which component information is to be available in the table.

Current Vial Section

Displays information about the current vial, including its row in the Samples table, vial #, and level.

In All samples mode, the display of these fields change to reflect the currently selected vial: Sample Name, Sample ID, and Sample type.

View Tables and Pages

Display the view tables or pages that correspond to the selected view tab and mode tab (see below). You enter component information for each sample in the columns of the view tables, or in the fields of the pages. Click for more information on the different Component Editor views.

View and Mode Tabs

The view tabs at the top of the view table:

  • Components, Mol Weights (GPC/V samples only)
  • K-Alphas (GPC/V samples only)
  • Moments (GPC/V samples only)
  • Distribution (GPC/V samples only)
  • Cumulative (GPC/V samples only).

The Current and All Samples mode tabs appear at the bottom of the view table.

The combination of selected view and mode tabs determines which view appears.

For example, if you select the Mol Weights view tab and the All Samples mode tab, the view table displays a read-only view for all samples (only one row of information for each sample).

In All Samples mode, the Purity tab appears after the Value tab, and displays purity columns for each sample line.

If you select the Mol Weights view tab and Current Samples mode tab, the view table displays a table for only the currently selected sample (all rows of information for the sample).

Previous and Next

Display the component data in the current view table for the previous or next row in the Samples table (in Run Samples, Alter Sample window, or Sample Set Method Editor). The Row, Vial, Level, and Type values in the Current Vial section (see above) change accordingly.
