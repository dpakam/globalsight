Method Set wizard – Define GPCV Channel Set page

Use this page to select channels to use as the GPCV channel set.

Note: If the instrument method you select for this method set includes parameters for a GPCV instrument, parameters on this page are loaded from the instrument method. Edit the parameters as required.

To define the GPCV channel set:

  1. Select the concentration channel from the Concentration Channel list.
  2. For a single-channel viscometer, select the viscometer channel from the Viscometry Channel list. (This is the differential pressure. If you are using a non-Waters viscometer, click for details.)
  3. For a dual-channel viscometer (such as a Viscotek):
  4. Select the inlet pressure (IP) channel from the Viscometry Channel list.
  5. Select the differential pressure (DP) channel from the Second Viscometry Channel list.
  1. Enter an Interdetector Time to add an offset (in minutes) to the data points in the derived channel. Specify a positive or negative time offset. Entries: –500.000 to 500.000 minutes. Default: 0.000 minutes.
  1. Enter a Viscometer Zero Offset to compensate for the difference between the baseline zero of the viscometer at the specified flow rate and the baseline zero of the viscometer at zero flow rate. Entries: –9999.999 to 9999.999 mV. Default: 0.000 mV.

Note: For a two-channel viscometer, enter the first viscometer constant in the Viscometer Zero Offset field. Enter the second viscometer constant in the second Channel Viscometer Offset field.

  1. If appropriate, enter a second channel viscometer offset to compensate for the pressure differential between the channels.
  2. If appropriate, select the Concentration Channel Connected to RI Detector check box.
  3. After creating a GPCV channel set, click Next.
