Annotation Object Properties - Rectangle tab

Use this dialog box to change properties of the rectangle or ellipse in Report Preview. Access this dialog box by right-clicking the rectangle or ellipse and selecting Properties.

Line attributes

Line style

Select one outline style for the rectangle or ellipse: solid, dashed, dotted, dash dot, or dash dot dot.

Thickness

Click the up arrow to increase the thickness of the outline, and click the down arrow to decrease it.

Range: 0 to 1000

Notes:

  • Waters recommends that you specify a line thickness of 0 when using line styles other than Solid. Line thicknesses vary with monitor and printer resolution, which can result in the various line style selections being displayed as Solid. A selection of 0 ensures that the line styles appear properly.
  • Using a thickness greater than 10 can cause a shape to appear as a solid object.

Line color

Accesses the Color dialog box and changes the color of the outline. You can use all colors supported by the Windows operating system, which includes the basic and custom colors.

Fill attributes

Transparent

Makes the fill transparent. Clearing this check box activates the Solid and Hatch option buttons.

Solid

Creates a solid fill for the rectangle or ellipse. To change the fill color, click Fill Color.

Hatch

Creates a hatch fill for the rectangle or ellipse. To change the color and the pattern, click Hatch Bkgnd and the pattern list.

Fill Color

Accesses the Color dialog box and changes the solid fill color. You can use all colors supported by the Windows operating system, which includes the basic and custom colors.

Hatch bkgnd

Accesses the Color dialog box and changes the hatch fill color. You can use all colors supported by the Windows operating system, which includes basic and custom colors.

Pattern

Select the hatch pattern: horizontal, vertical, forward diagonal, back diagonal, cross, or diagonal cross.

Rectangle/ellipse

You can change the selected object to a rectangle or ellipse by selecting the shape from the list.
