Select user from the LDAP tree dialog box

In this dialog box select an LDAP user from the available LDAP tree nodes and then modify his or her Empower user account.  You can select and then modify only one Empower user account at a time.

Prerequisite: LDAP Authentication enabled (View > System Policies > User Account tab > Authentication > LDAP)

You can access this dialog box by clicking Search LDAP from the User Properties dialog box.

Requirements: Alter Users privilege or Administrator privilege

To modify an Empower user account, navigate to the appropriate LDAP node, select the user's name from the LDAP Directory tree, and then click Associate with User.

Tips:

  • You can search for an LDAP user by entering their name in the LDAP directory list field and then clicking Search .
  • When you switch the authentication mode to LDAP, Empower software automatically disables existing local user accounts. If you are associating a user account for the first time, set the user account status to Active so that the user can log in to Empower.

Result: The User Properties dialog reappears with the LDAP information populating the LDAP User DN field. You cannot edit this field.
