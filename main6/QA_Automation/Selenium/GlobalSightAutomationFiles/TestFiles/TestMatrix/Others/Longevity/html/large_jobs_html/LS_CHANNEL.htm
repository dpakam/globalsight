Edit Channel Set Definition dialog box

Use this dialog box to do the following things:

  • Combine data from concentration channels, up to eighteen LS channels, and one or two viscometer channels.
  • Save the multiple channels as a channel set.

A channel set allows you to process multiple channels of an injection for one GPC-LS or GPCV-LS result. To set up a GPC-LS channel set click or to set up a GPCV-LS channel set click.

Name of channel set

Name for the channel set.

Concentration channel

Concentration channel (selected from the list).

Concentration channel connected to RI detector

Specifies the selected concentration channel is connected to an RI detector.

Viscometry channel (GPCV-LS channel set)

Derived viscometer channel (selected from the list).

Light Scattering channels table

Identifies the channel name for each LS channel. The Process check box indicates the channel data that will be processed. If the check box is clear, the channel will not be processed.
