Inlets tab (5890)

Use this property page to set injector A and B inlet zone temperatures.

Injector A enable/injector B enable

Enables or disables temperature inlet zone A and/or B for the corresponding injector.

Choices: Any combination of enabled or disabled for injectors A and B

Default: Disabled

Injector A temperature/injector B temperature

Inlet zone temperatures for injectors A and B. Enable corresponding Injector Enable check box before the value takes effect.

Default: 50 °C
