Purity tab

Use this tab to specify Spectral Contrast parameters and Peak Purity testing parameters.

Spectral contrast parameters

These parameters control how spectra are compared for Peak Purity and spectrum Library Matching. You set them using the top pane of the Purity tab.

Wavelength limit (nm) start, stop

Start and stop wavelengths (in nanometers) for Spectral Contrast processing. Use the Wavelength Limit parameters to exclude from Spectral Contrast calculations regions of a spectrum where there is no component absorbance or where the solvent strongly absorbs. These regions do not contribute to spectrum Library Matching or Peak Purity testing. To optimize Peak Purity testing using wavelength ranges, click.

Note: If you leave the wavelength limits at the default range (190 to 800 nm), the software uses the starting and ending wavelengths of the acquired wavelength range as the Wavelength Limit.

Entries: 190.0 to 800.0 nm

Defaults: Start, 190.0; Stop, 800.0

Noise interval time (min) start, stop

Start and stop times (in minutes) for the noise interval, which is used for spectrum Library Matching or Peak Purity testing.

Entries: 0.00 to 655.90

Defaults: Start, 0.25; Stop, 0.75

Note: Empower software requires a noise interval to calculate the Threshold Angle.

Purity parameters

Purity parameters control the Peak Purity measurement. Peak Purity testing refers to a computation that uses a single pass of all spectra in a peak to detect a coelution within a peak. Multicomponent Peak Purity testing refers to a computation that uses multiple passes to detect one or more coelutions within a peak.

Purity enabled

Peak Purity testing is performed on all peaks identified during integration.

Default: Disabled

Active peak region (%)

Region of each peak over which you want to perform Peak Purity testing. Entries: 0.0 to 100.0%, where 100% is the width of the peak from liftoff to peak center and from peak center to touchdown.

Default: 100%

Note: Leave the active peak region parameter set to the default value of 100% to enable Peak Purity testing throughout the whole elution range of the peak.

Threshold criteria

Estimates the noise and/or solvent contributions to the Purity Angle of a spectrum during Peak Purity testing, and flags purity results that exceed this threshold. Choices:

  • AutoThreshold – Empower software calculates the Noise Angle (based on the Noise Interval you specify) and the Solvent Angle (based on the absorbance of the peak), then adds the two angles to compute the Threshold Angle for each peak.
  • Noise + Solvent – Adds the Solvent Angle to the Noise Angle. You must fill in the Solvent Angle field.
  • Noise – Reports and uses only the Noise Angle, which is calculated for each peak based on total system performance and the peak’s height, ignoring solvent contribution.
  • Solvent – Reports and uses only the Solvent Angle. You must fill in a value in the Solvent Angle field.

Solvent angle (degrees)

Estimates the solvent contribution to Spectral Contrast.

Entries: 0.00 to 90.00

Default: 1.0

Purity passes

Number of times the software performs Peak Purity testing on a peak, basing the next purity pass on the apex and maximum impurity spectra from each pass. For each purity pass, the software calculates a new Purity Angle and Purity Threshold, as well as a new Purity and Noise plot.

Choices: 1 to 4

Default: 1
