Sample Set Method Editor

The Sample Set Method Editor lets you manually create a new sample set method or modify an existing sample set method.

Parts of the window

The Sample Set Method Editor includes the following parts:

  • Menu Bar – Displays menus for performing the commands used in the Sample Set Method Editor.
  • Toolbar – Displays shortcut buttons that provide quick access to the primary commands used in the Sample Set Method Editor.
  • Samples table – Lists all identifiers for each sample within a sample set.
  • Apply Table Preferences list – Lists the table preferences available in the project. Use table preferences to select the fields that appear in the samples table. Select the table preference you want from the Apply Table Preferences toolbar, or click File > Apply Table Preferences. If the Apply Table Preferences toolbar is not visible, click View > Toolbars > Apply Table Preferences.
