Validation result table properties - Validation Results tab

Use the Validation Results tab parameters to define the type of results you want to include in a validation report group.

Parameters Usage

Selected Test

(Default) Reports the results for the validation test type that you select within the Test Selection group box.

System Precision for Selected Test

Reports system precision results for the validation test type that you select within the Test Selection group box.

System Precision Data for All Tests

Report system precision results for all validation test types.

Test Selection options

Select the test type whose results you want to include in a report. (Linearity is the default.)

Tip: When a study contains more than one test of the same type, you can use these Empower features to report the results individually for each test:

  • Use Data Filtering with the Test Description field when you want to report the results from one of multiple tests of the same type
  • Use the Order By and Group By features to report the results for each test in their own table (typically for the Assessed field, Test Description field, and/or Component fields).
