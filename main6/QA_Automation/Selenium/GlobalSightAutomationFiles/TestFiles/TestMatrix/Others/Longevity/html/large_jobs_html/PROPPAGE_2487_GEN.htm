General tab (W2487)

Use this tab to define general operating parameters for the Waters 2487 Dual l Absorbance Detector.

Wavelength mode

Detector wavelength mode.

Choices: Single or Dual

Default: Single

Channel 1 and channel 2 parameters

Enable channel 1 and/or channel 2 data channels, and specify channel descriptions in these two sections of the tab.

Enable

Specifies the data channel as an active channel for the 2487 Detector. Clear this check box if the channel is unused. If dual wavelength is needed, select this check box for Channel 1 and Channel 2.

Choices: Enabled or Disabled

Default: Channel 1 and Channel 2, Disabled

Name

Software-designated name for the channel, which appears in the Active Channels table in the Instrument Method Editor.

Description

Optional description for the data channel, up to 250 alphanumeric characters. The description appears in the Active Channels table in the Instrument Method Editor and in the Real-Time Plot pane in Run Samples.
