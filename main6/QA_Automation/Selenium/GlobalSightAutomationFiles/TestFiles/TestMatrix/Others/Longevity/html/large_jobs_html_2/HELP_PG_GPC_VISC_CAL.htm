Viscosity Law Calibration tab

Use this tab to define characteristics of a GPCV Viscosity Law calibration curve plot when the plot is displayed in Review or Compare, or printed in a report.

Use this tab to define characteristics of a GPC Viscosity Law calibration curve plot when the plot is displayed in Review or Compare, or printed in a report.

Information to report

Specifies which Viscosity Law calibration curve data is included in the report. The choices are:

  • All Points – All points used to create the Viscosity Law calibration curve.
  • Ignored Points – Points that were specified to be ignored when calculating the Viscosity Law calibration curve.
