View Filter Editor

Use this window to create and edit view filters that define how data appears in the Project or Configuration Manager window view table.

To open the View Filter Editor in the Project or Configuration Manager window, select Edit > View in the dialog bar. The last view filter used for the selected view appears.

The View Filter Editor includes the following components:

  • Menu bar – Displays commands for using the View Filter Editor.
  • Type field – Lists all views, allowing you to switch to different views directly from the View Filter Editor. The Field Names change according to the selected view.
  • Toolbar – Displays shortcut buttons that provide quick access to primary commands used in the View Filter Editor.
  • View Filter table – Displays column headings included in the filter and any condition operators you enter into table cells to further limit the displayed information.
  • Field Names – Lists all available names you can add as column headers in the View Filter table.
