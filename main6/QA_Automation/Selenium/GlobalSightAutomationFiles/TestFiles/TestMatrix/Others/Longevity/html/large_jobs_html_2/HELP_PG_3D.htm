3D tab

Use this tab to define characteristics of the PDA or MS 3D plot displayed in Review or printed in a report.

Use this tab to define characteristics of the PDA or 3D plot displayed in Review.

Levels

Number

Number of absorbance levels (PDA data) or intensity levels (MS data) into which the PDA or MS data set is divided for contour display.

Number of absorbance levels (PDA data) into which the PDA data set is divided for contour display.

Range for Linear distribution: 2 to 200.

Range for Exponential distribution: 2 to 15.

Default: Linear Distribution, 10 levels.

Linear distribution/exponential distribution

Algorithm for assigning contours in the 3D plot.

Choices: Linear, Exponential.

Default: Linear. To accentuate low-level features, use an exponential contour distribution and increase the number of contours.

Legend (Report publisher only)

Draw contour/3D legend

Display of the plot legend to the right of the 3D plot.

Textual legend/Graphical legend

How the legend appears in the 3D plot. Choices: Textual, Graphical.

Default: Graphical.

Low contour

Low contour

Absorbance value (PDA data) or intensity value (MS data) of the lowest contour line. If you have altered the contour setting, click the AutoScale check box to reset the contour bands to the default settings.

Absorbance value (PDA data) of the lowest contour line. If you have altered the contour setting, click the AutoScale check box to reset the contour bands to the default settings.

Note: The Low Contour text box is grayed out when the AutoScale check box is selected.

AutoScale

Scales the 3D plot to the lowest contour line. When you enable this function, you cannot manually enter a low contour setting.

Default: Enabled.

High contour

High contour

Absorbance value (PDA data) or intensity value (MS data) of the highest contour line. If you altered the contour setting, the AutoScale check box to reset the contour bands to the default settings.

Absorbance value (PDA data) of the highest contour line. If you altered the contour setting, the AutoScale check box to reset the contour bands to the default settings.

Note: The High Contour text box is disabled when the AutoScale check box is selected.

AutoScale

Scales the 3D plot to the highest contour line. When you enable this function, you cannot manually enter a high contour setting.

Default: Enabled.

3D view

Low wavelength

Low wavelength/Mass in front

Reverses the direction of the wavelength axis in the 3D plot in your report, allowing you to view the 3D plot from the front or the back.

Default: Disabled.

Chromatogram mode (MS)

Changes the plot from the characteristic MS stick plot (a discrete spike for each mass-intensity pair) to a chromatographic plot (where each mass is a separate time-resolved chromatogram).

Default: Disabled.

Angle

Alters the viewing angle of the 3D display. Increasingly positive numbers cause the cube to rotate clockwise; increasingly negative numbers cause the cube to rotate counterclockwise.

Entries: –90.0 to +90.0 degrees.

Default: +15.0 degrees.

Height

Varies the difference in height between the top line of the front face and the top line of the rear face of the view cube in which the 3D plot is drawn. For example, a value of 0 (no difference in height) displays the front face overlaid on the rear face. A value of 50 superimposes the top line of the front face on the bottom line of the rear face of the cube. A value of 100 (maximum difference in height) displays an overhead view (similar to the Contour plot) of the view cube.

Entries: 0 to 100%.

Default: 20%.
