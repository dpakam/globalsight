New Sample Set Method wizard – Describe Samples page

Use this page to define information about your samples. The Describe Samples page contains the fields and controls described below.

Sample information section

Use this section to define information about your sample injections.

Number of samples

Specifies the total number of sample vials in the tray. The software determines the maximum number of samples you can enter in this field according to:

  • Previously entered New Sample Set Method Wizard parameters (for example, parameters that define standard vial locations, number of standard vials in a group, and so on).
  • The capacity of your autoinjector or autosampler tray or carousel.

Number of injections per vial

Specifies the number of injections to be taken from each sample vial.

Valid entries: 1 to 99

Note: If you are creating a sample set method for dual tower GC operation, you must select the same value for Number of Injections per Vial for both the Front and Back tower to allow for simultaneous injections. To create a dual tower sample set method, you use the wizard twice, once for each tower.

Injection volume

Specifies the volume in microliters of the injected sample, valid entries are 0.0 to 29999.9. Note the following considerations:

  • For Waters autosamplers, small injector loops may omit the maximum volume. Ensure hardware configuration is compatible with your desired injector size.
  • For Agilent 1100 autosamplers, the maximum injection volume is the seat capillary volume, plus the syringe size. Ensure that the hardware configuration is specified correctly in your method.

Specifies the volume in microliters of the injected sample, valid entries are 0.0 to 29999.9.

Tip: Small injector loops may omit the maximum volume. Ensure hardware configuration is compatible with your desired injector size.

For Agilent GCs, valid entries are 1 to 5 units. Each unit corresponds to 10 percent of syringe capacity. For example, 2 units of a 25 microliter syringe is 10 microliters. So entering 2 will cause a 10 microliter injection. However, if you are using the optional nanoliter adaptor, these percentages change. Please consult the manufacturer’s documentation to determine the correct setting for injection volume.

Note: If you are using a 2690D/2695D, creating a Dissolution Sample Set Method, and pooling injections, the pool volume and injection volume must be equal.

Run time

Specifies the length of time in minutes to collect data for a sample. Valid entries are 0 to 650.0.

Note: If you are creating a sample set method for dual tower GC operation, you must select the same value for Run Time for both the Front and Back tower to allow for simultaneous injections. To create a dual tower sample set method, you use the wizard twice, once for each tower.

Maximum # of samples per group

Specifies the maximum number of samples in a sample group. This field appears only if you chose to inject standards before or around sample groups (in the Select Location of Standards page). The software determines the maximum number you can enter in this field according to your Number of Sample selection and the capacity of your autoinjector or autosampler tray or carousel. The value you enter for Number of Samples is divided by the value for Maximum # of Samples Per Group to determine the number of sample groups for your sample set.

For example, if you entered a value of 25 for Number of Samples, and 6 for Maximum # of Samples Per Group, your sample set would contain a total of five sample groups—four groups containing six samples, and one group containing one sample.

Options button

Displays the Enter Optional Information page, where you can specify a Sample Weight factor, a Dilution factor, a Data Start time and a Next Injection Delay time.

Method set section

Use this section to define or create the method set used to acquire, process, and/or report data for your samples.

Select

Specifies the method set used to acquire, process, and/or report data for the samples (this could be the same or a different method set from the one you selected in the previous wizard page for acquiring, processing, and/or reporting standards). Select the appropriate method set from the drop-down list. If you are using the Pro interface, click Create New if you need to create a new method set.

Select the appropriate method set from the drop-down list.

Note: If you are creating a sample set method for dual tower GC operation, you must select the same method set for both the Front and Back tower. To create a dual tower sample set method, you use the wizard twice, once for each tower.

Create new button (Pro interface only)

Displays the first page of the New Method Set Wizard, where Empower software interactively leads you through the steps to create a new method set.

After making your selections, click Next to advance to the next wizard page.
