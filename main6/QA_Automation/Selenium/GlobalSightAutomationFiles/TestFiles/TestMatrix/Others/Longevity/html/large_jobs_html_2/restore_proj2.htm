Restore Project wizard – Project Details page

If you restore one project, this page appears. Use this page to enter the project name and the tablespace quota, and to view statistics related to the last backup. Choices:

  • Project Name – Name of the project that you are restoring. Allows for entry of a new name (up to 30 alphanumeric characters).
  • Backup Details – Type of the last backup that was performed on this project, the user who performed the backup, and a backup date/time stamp.
  • Backup Comments – Entered when the project was last backed up.
