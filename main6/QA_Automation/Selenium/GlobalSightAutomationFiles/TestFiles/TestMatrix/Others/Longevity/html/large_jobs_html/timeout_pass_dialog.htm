Timeout Password Required dialog box

Based on a system policy, the software closes when inactive after a specified number of minutes. You must enter your password to log back on to the software and unlock your work session.

Tip: If you unlock the Pro Login window first, it unlocks all other Empower applications, such as the Browse Project and Run Samples windows. If you are logged on to the Pro interface, you may need to enter your password twice, once to unlock Empower applications, such as Browse Project and Run Samples windows, and once to unlock the Pro Login window. If you are logged on to QuickStart, you only need to enter your password once.

Other users can log on to the software, but to unlock your work session, your must enter your password.
