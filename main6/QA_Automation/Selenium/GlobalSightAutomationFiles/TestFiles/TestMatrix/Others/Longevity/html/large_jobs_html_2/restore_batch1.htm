Restore Project wizard - Set Project Parameters page

If you restore multiple projects, this page appears. To restore multiple projects, all projects must reside in the same directory\path and must include an .EXP and .INF file. Project parameters you can set include:

  • Selecting which project you want to restore
  • Editing the project name
  • Selecting a new raw data path for the location of the project files
  • Setting a new tablespace quota

Name

Name of the project you want to restore. It is taken from the directory you specified in the Restore Project Wizard – Start Software page.

Restore?

If you want to restore this project, leave this check box selected. If you do not want to restore this project, clear the check box.

Proposed Name

If the project name being restored exists in the database, a new name is proposed. You can change this project name.

Path

Location of the data files for this project. To select a different raw data path, click the down arrow to select from a list of configured paths.

Tablespace Quota

Amount of tablespace (in megabytes) allocated to the selected project. Nonspecific tables like systems, users, and audit trails are included in project tablespace.

Full Audit Trail

If the system policy has Full Audit Trail enabled, this check box is enabled.

Audit Trail Method Changes

If the system policy has Full Audit Trail enabled and Require User Comments on Method changes enabled, this check box is enabled.

Audit Result Changes

If the system policy has Full Audit Trail enabled and Require User Comments on Result changes enabled, this check box is enabled.

Audit Sample Changes

If the system policy has Full Audit Trail and Require User Comments on Sample Information changes enabled, this check box is enabled.

Deletion Changes

If the system policy has Full Audit Trail and Require User Comments on Deletion changes enabled, this check box is enabled.
