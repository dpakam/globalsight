Restore Project(s) wizard – Start Software page

This page outlines the steps required to restore a project. Follow the instructions on the page to select from among these parameters.

Note: If you have not configured your backup program to run from Empower software, you can start your backup program from the Windows Start menu, restore the project, then return to this page to finish the restoration.

Choices:

  • Browse – Select the directory that contains your backed up project files. If the directory contains single project backup files, the project will be restored. If the directory contains multiple backed up project files, Empower will batch restore the backed up project files.
  • Start Backup Software – Starts your backup software for you automatically if you have configured Empower software to run your backup program.

Tips:

  • When restoring single projects, the software displays a warning box whenever an error occurs. You must respond to the warning before the restore operation continues.
  • For batch restore operations, the software completes the process without operator entry, generating a log file, which is found in the Empower\Logs directory, that lists any integrity errors found during the restore operation. This facilitates unattended restore operations by ensuring that warning messages do not stop the batch process. The batch process is halted only when it encounters errors that cause the project to be deleted or cause the restore operation to fail.
