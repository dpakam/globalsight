Triple Plot tab

Use this tab to define characteristics of a Triple Library Match plot displayed in Review or printed in a report.

Plot characteristics

Label spectra

Spectra in the Triple Library Match plot.

Autoscale difference plot

Normalizes the difference plot to the largest peak in the difference plot in the Triple Library Match plot.

Show Wiley structures

Associated mass spectra and compound structure in the upper-left corner of the plot (as defined in the Wiley Registry of Mass Spectral Data and NIST/EPA/NIH Mass Spectral Library, available only if the Wiley Library option or Wiley NIST Library option is installed).

Lib/difference plots (Report Publisher only)

Use match 1/Use match 2/Use match 3

Specifies which match plot spectra you want to view (match 1, match 2, or match 3). If there is only one match, and you specify Spectra 2 (for example), the second library match plot is empty and the No library match message appears within the plot.

Default: Enabled

Z-axis (Report Publisher only)

Use Z-axis plotting

Enables the z-axis for the Triple Library Match plot.

Grid lines

Grid lines for the z-axis in the background of the Triple Library Match plot.

Note: Select Use Z-Axis Plotting to enable and define grid lines.

Angle

Counterclockwise angle (in degrees) of rotation for the z-axis.

Entries: 0 to 90 (where 0 equals horizontal rotation and 90 equals vertical rotation)

Default: 15

Spacing

Distance (in pixels) between spectra along the z-axis.

Entries: 0 to 100

Default: 10
