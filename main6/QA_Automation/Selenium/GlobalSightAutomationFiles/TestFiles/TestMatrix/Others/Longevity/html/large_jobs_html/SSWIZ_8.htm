New Sample Set Method wizard – Identification pag

Use this page to enter information that identifies standards and samples in your sample set. This page contains the fields described below.

How should your standards be identified?

Your standards are identified through the following selections:

  • SampleName – Specifies the identifier for your standards. Once you finish creating the sample set method, this identifier appears in the SampleName column of the Samples table. The default identifier, Std, is automatically assigned by the software as an identifier for all standards in the SampleName field. The SampleName custom field is a default custom field that is provided with the software. If you make an entry in the Default field for the SampleName custom field (in Configuration Manager), the entry replaces Std as the SampleName identifier.
  • SampleName – Specifies the identifier for your standards. Once you finish creating the sample set method, this identifier appears in the SampleName column of the Samples table. The default identifier, Std, is automatically assigned by the software as an identifier for all standards in the SampleName field.

Note: This page of the wizard appears once for each sample identifier custom field (default and user-created). For example, if you create a sample identifier custom field called LotNumber, this page appears and displays LotNumber as the name of the field.

  • Incrementing Prefix – Specifies an incrementing prefix (letter or number) for the standard identifier specified in the SampleName field. Letters increment alphabetically; numbers increment numerically.

Note: If you mix letters and numbers in a prefix, be aware that the software increments only the last group of like characters. For example, if the prefix is AB12, the software increments the numbers from 12 to 13 and leaves the letters alone.

  • Incrementing Suffix – Specifies an incrementing suffix (letter or number) for the standard identifier specified in the SampleName field. Letters increment alphabetically; numbers increment numerically.

Note: If you mix letters and numbers in a suffix, be aware that the software increments only the last group of like characters. The software will automatically alphabetize custom sample identifiers.

How should your samples be identified?

Your samples are identified through the following selections:

  • SampleName – Specifies the identifier for your samples. Once you finish creating the sample set method, this identifier appears in the SampleName column of the Samples table. The default identifier, Unk, is automatically assigned by Empower software as an identifier for all samples in the SampleName field. The SampleName custom field is a default custom field that is provided with the Empower software. If you make an entry in the Default field for the SampleName custom field (in Configuration Manager), the entry replaces Unk as the SampleName identifier.
  • SampleName – Specifies the identifier for your samples. Once you finish creating the sample set method, this identifier appears in the SampleName column of the Samples table. The default identifier, Unk, is automatically assigned by Breeze software as an identifier for all samples in the SampleName field.  

Note: This page of the wizard appears once for each sample identifier custom field (default and user-created). For example, if you create a sample identifier custom field called LotNumber, this page appears and displays LotNumber as the name of the field.

  • Incrementing Prefix – Specifies an incrementing prefix (letter or number) for the sample identifier specified in the SampleName field. Letters increment alphabetically; numbers increment numerically.

Note: If you mix letters and numbers in a prefix, be aware that the software increments only the last group of like characters. For example, if the prefix is AB12, the software increments the numbers from 12 to 13 and leaves the letters alone.

  • Incrementing Suffix – Specifies an incrementing suffix (letter or number) for the sample identifier specified in the SampleName field. Letters increment alphabetically; numbers increment numerically.

Note: If you mix letters and numbers in a suffix, be aware that the software increments only the last group of like characters. For example, if the suffix is AB12, the software increments the numbers from 12 to 13 and leaves the letters alone.

After making your selections, click Next to advance to the next wizard page.
