Text tab

Use this tab to specify a line of text for the Text drawing object.

Text alignment

Position of the text line within the Text drawing object.

Choices: Left, Center, Right

Default: Left

Text entry box

Text to appear in the Text drawing object (up to 250 alphanumeric characters without restrictions). To force the text onto multiple lines, press Ctrl+Enter. You can format the text line using the Fonts tab.

If the text line overlaps the Text drawing object (as sized in the Report Page), a section of the text line is not printed.
