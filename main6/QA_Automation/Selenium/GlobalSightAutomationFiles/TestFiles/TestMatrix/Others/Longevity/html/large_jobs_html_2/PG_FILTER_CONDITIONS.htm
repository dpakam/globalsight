Filtering Data in the Data Filter Conditions dialog box

Use this dialog box to filter your report data. To filter data in a report group, select a field to filter data for, and then set the conditions to filter on.

To select the fields on which to filter data:

  1. Right-click the report group to filter data for, and then select the group’s properties.
  2. Click the Order By tab.
  3. Click Data Filtering Conditions to open the Data Filter Conditions dialog box. The Available Fields list displays field name categories for the selected report group.
  4. To display the field names within a category, double-click the category.
  5. Double-click the field name to filter on.

Result: The name appears in the Column Name cell in the Data Filter Conditions table.

To define filter conditions for the field name:

  1. Click the Condition 1 cell to open the Create a Condition dialog box.

Rule: The capabilities available in this dialog box depend on the report group you are working with. A condition typically consists of an operator (for example, Equals) and a reference value (for example, 1218) that, in combination, limit data that is displayed.

These wildcard symbols are available when entering a reference value:

  • ? – Represents one character. For example, if you enter Peak? and the Peak Name field is being filtered, the report contains the results for Peaks 1 – 9. However, if there are results for Peaks 10 and 11, they are not reported because the peak number has an extra character.
  • * – Represents multiple characters. For example, Equals Acet* adds the results for the Acetone and Acetaminophen peaks to the report. If the peak name does not begin with "acet" it is not included in the report.
  1. Click OK to enter the specified condition in the Data Filter Conditions table.
  2. To add additional conditions for the column name, click the Condition 2 – Condition 5 cells to open the Create a Condition dialog box.

Note: You cannot modify the Or column, which indicates the Or relationship between conditions.

Data filter conditions table properties

The conditions for each row in the table, the field name, are separated by Or conditions. For example, for each field name, either condition 1 or condition 2 is reported.

Reading the table from top to bottom, an "and" condition is applied to the Condition columns when reporting data. The report includes all field names in the table for the conditions reported. If Data Filter Conditions were set as in this example, the report contains only standards that have peak names starting with "acet."
