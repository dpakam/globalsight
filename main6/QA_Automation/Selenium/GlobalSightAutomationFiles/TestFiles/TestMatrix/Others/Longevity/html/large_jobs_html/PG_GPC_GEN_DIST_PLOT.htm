General GPC Distribution Plots ta

Use this tab to define general characteristics of the General GPC Distribution plot when printed in a report.

Axis data

Specifies data to include in axes of the General GPC Distribution plot.

Axis data:

Data type

Usage

Slice MW

Plots the molecular weight of the polymer at each slice.

Slice Log MW

Plots the log (base 10) of the molecular weight of each slice.

Slice RT

Plots the retention time at each slice.

Slice Adj. RT

Plots the retention time of each slice adjusted for variations in retention time as observed relative to a reference peak.

Slice EV

Plots the elution volume (in mL) of each slice adjusted for the flow rate you entered in the Calibration tab of the Processing Method window. Elution Volume is equal to Adjusted Retention Time if you did not enter a flow rate (assumes a flow rate of 1.0 mL/min).

Slice Area

Plots the area of each slice, which is the height of the integrated peak at the reported retention time multiplied by the duration of the slice.

Slice Height

Plots the height of each slice of the integrated peak at the reported retention time.

dwt/d(log M)

Plots the weight-normalized area of the slice. The absolute value of the area of the slice is divided by the slope of the calibration curve at the adjusted retention time or elution volume of the slice. The ratios of area to slope for all slices are summed, then each ratio is divided by the sum. This is the default left axis.

Cumulative %

Plots the amount (as percent area) of the polymer accumulated up to and including the slice. This is the default right axis.

Slice Fitted [h] (Universal Calibration Only)

Plots the log (base 10) of the intrinsic viscosity from the fitted viscosity curve (if a GPCV peak). It is calculated from the molecular weight, K, and alpha values when this is a GPC peak.

Number Density

Plots the normalized distribution based on the number of molecules in a slice.

Slice lambda

Plots lambda, the probability for long-chain branching factor, for the slice.

Slice Specific Viscosity

Plots the height of the viscometer peak at the reported retention time.

Slice Observed MW

Plots the molecular weight calculated for the slice by extrapolating the light scattering data to the zero-angle.

Slice R0 (GPC/V-LS Option)

Plots the Rayleigh ratio at scattering angle 0.

Slice Observed Rg (GPC/V-LS Option)

Plots the log observed radius of gyration calculated for the slice by extrapolating the LS data to the zero-angle using the Rg Fit specified in the Slicing tab in the Processing Method window. Use this for comparison with the theorectical linear and fitted curve. This is the default selection for Plot3 Data.

Slice Fitted Rg (GPC/V-LS Option)

Plots the log fitted radius of gyration calculated for the slice by fitting the LS data according to the Slicing tab field Rg Fit parameter. This is the default selection for Left X-Axis Data.

Slice Linear Rg (GPC/V-LS Option)

Plots the log linear radius of gyration calculated from the Slice Fitted Rg value by fitting a line to the low molecular weight end of the Fitted Rg when the Fitting Model is set to Structural and the Rg is set to one of the higher-order polynomial fits or either of two step fits. The default selection for Plot4 Data.

Slice R1, R2, R3 (GPC/V-LS Option)

Plots the slice height of the first to third LS peaks.

Slice Observed HV (GPC/V-LS Option)

Plots the log observed hydrodynamic volume.

Slice Fitted HV (GPC/V-LS Option)

Plots the log fitted hydrodynamic volume calculated for the peak according to the Fitting Model, Intrinsic Viscosity Fit, and Rg Fit parameters.

Slice P (GPC/V-LS Option)

Plots the local polydispersity of the slice.

Slice Epsilon (GPC/V-LS Option)

Plots the overall shape factor, epsilon.

Slice K(LS) (GPC/V-LS Option)

Plots Slice K(LS), where log Slice K(LS) is the intercept of the line tangent to the curve that produced the Slice Fitted Rg values.

Slice alpha(LS) (GPC/V-LS Option)

Plots the slope of the line tangent to the curve that produced the Slice Fitted Rg values.

Slice Unbranched Rg (GPC/V-LS Option)

Plots the log unbranched radius of gyration of the slice calculated only if you entered unbranched values of K(LS) and alpha(LS).

Left-Axis Data, Right-Axis Data

Data displayed along the left and/or right y-axes of the General GPC Distribution plot. The available left and right y-axis data choices are identical to those described for the X-Axis Data parameter above. If desired, you can select Blank as the left- and/or right-axis data (no data is displayed).

Additional plots

Displays of up to six additional data plots in the General GPC Distribution plot (Plot3 to Plot8). See the X-Axis Data parameter for a complete listing of plot choices. If desired, select Blank as a data plot choice (no data is displayed).

Plot data

Specifies data to include and plotting direction for the values on the x-axis:

  • High to Low – x-axis values decrease from left to right (default).
  • Low to High – x-axis values increase from left to right.
  • MW Averages – Molecular weight average labels (Mn, Mw, Mv, Mz, and Mz+1) on the Distribution plot. This parameter is enabled only when X-Axis Data is set to Slice MW or Slice Log MW.
  • MP - Peak molecular weight (at the peak apex) labels on the Distribution plot.
  • MW Markers – Mol Wt Marker 1 to Mol Wt Marker 6, which indicate the molecular weights you entered in the Slicing tab of the Processing Method window. This parameter is enabled only when X-Axis Data is set to Slice MW or Slice Log MW.
