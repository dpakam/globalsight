Flow tab (PCM/15XX)

Use this tab to define operating parameters that control up to three Waters 500 Series Pumps or 1500 Series Pumps (1515 and 1525), attached to a PCM.

Pump enable

Read-only status box that displays the on/off status of the pump (checked equals on).

Pump pressure limits

Sets high-pressure and low-pressure limits for the pumps attached to the PCM. These limits apply to the total pressure for all pumps in the system, not to pressure for an individual pump.

High pressure limit

If system pressure exceeds this limit for 1.2 seconds, the system shuts down.

Default: 4000 psi

Note: When determining the high-pressure limit for your system, remember the pressure limitations of your column; see the column manual. When using extended-flow pump heads, adjust the high-pressure limit accordingly.

Low pressure limit

If system pressure falls below this limit for 60 seconds, the system shuts down.

Range: 0 to 6000 psi (for 100-µL heads), 0 to 3000 psi (for 225-µL heads)

Default: 0.0 psi

Pump mode

Specifies the pump operating mode, which can be one of the following:

  • Isocratic (displays Flow and %A–%C text boxes)
  • Gradient (displays the gradient table)

Flow

Percentage of solvent flow from each reservoir. The total of all solvents in each gradient segment must equal 100%. The system calculates individual flow rates when you enter the percentage value for each solvent. The software automatically fills in the solvent column in the next column to the right so that the total percentage of the current row yields 100%. For examples of how the autofill feature works, see the related topic link.
