Plot tab

Use this tab in Review, Compare, or Report Publisher to change axis units and labels of a plot. Access this tab by right-clicking the plot or report and selecting Properties.

Use this tab in Review or Compare to change axis units and labels of a plot. Access this tab by right-clicking the plot and selecting Properties.

Axis units and alternate units label

Select the units to display on the plot. Options depend on the type of plot you are working with and which axis that plot uses. Options are X-Units, Y-Units, Right Units, and Top Units.

Note: If you select a unit, you can change the unit label displayed on the plot by entering a new label name in the Alternate Units Label field.

Other characteristics

X-Y box

Adds a border around the plot on sides where there is no axis plotted.

Grid lines

Adds grid lines to the plot.

Scrollbar (Review only)

Adds a scroll bar to the plot.

Axis labels and ticks

Axis label:

  • Select this check box to put the axis units label and ticks and units on the plot.
  • Clear this check box to remove the axis units label and ticks and units from the plot.

Auto ticks

This check box is available when the corresponding axis label is selected. Guidelines:

  • Select this check box to allow the software to determine the number of major and minor ticks.
  • Clear this check box to manually set the number of major and minor ticks and the increments using the Major Ticks, Minor Ticks, and Tick Increment fields.

Major ticks

This field is active when the Auto Ticks option is disabled. Use the up arrow to increase the number of ticks and the down arrow to decrease it. The default for the X and Top axes is 10. The default for the Y and Right axes is 5. The range is 0 to 32.

Note: This field is disabled if you enter a tick increment value.

Minor ticks

This field is active when the Auto Ticks option is disabled. Use the up arrow to increase the number of ticks and the down arrow to decrease it. The default for all axes is 4. The range is 0 to 32.

Tick increment

This field is active when the Auto Ticks option is turned off. The value in this field is used to determine the spacing between major ticks. Use the up arrow to increase the increment, and the down arrow to decrease it.

Range: 0.0001 to 1.798e+308

Major and minor tick sizes

The software uses values in these fields to determine the tick length. Use the up arrow in each field to increase the tick size and the down arrow to decrease the tick size. Guidelines:

  • Major Tick Size – Default is 60. Range is 1 to 100.
  • Minor Tick Size – Default is 30. Range is 1 to 100.
