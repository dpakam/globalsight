Define Template Variables page 1 of 2

Use this wizard page to enter information in fields that are identified in the Sample Set method wizard template method as changing from day to day. These fields can include the number of samples and starting well position. You can also define the injector plate your instrument uses. Open Access uses the information you enter to create the sample set method.

Number of sample vials

Specifies the number of samples in the run and the number of lines in the sample set method.

Vial starting position in tray

Specifies the starting vial number or the starting tray designation.

Define Plates

Opens the Define Plates window. Use this window to select one or more default plates.
