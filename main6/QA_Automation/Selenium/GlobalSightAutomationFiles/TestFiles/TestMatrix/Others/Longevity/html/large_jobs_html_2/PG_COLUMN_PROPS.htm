Column Properties dialog box fields

Use this dialog box to modify attributes of a column specified in the Table tab. To modify attributes of a table column, select the corresponding column from the Table Template, then click Column Properties to access the Column Properties dialog box (or double-click the table column). After redefining column attributes, click OK to return to the Table tab.

Original name

Unedited, original name of the selected table column (in the Table Template).

Data type

Type of field included in the table (Double Precision Float, String, Integer, Date, Enumerated, Characters, or Boolean Value).

Field type

Type of database field for the selected table column (for example, the %Amount column is a Peak database field).

Field name

Allows you to label the information in the report with something other than the original field name. The column width in the table is determined by the number of characters in the Field Name if the number of characters exceeds the specified Data Width (see below). For example, a column containing the Channel field is about seven characters wide.

Data precision

Number of digits reported to the right of the decimal. If the reported precision is less than the precision of the stored result, the reported result is rounded. If the reported field width or precision affects the validity of the result, the width is ignored and the entire result is printed.

Note: The software displays a maximum of 14 significant digits for real numbers. If the sum of the number of digits to the left of the decimal place plus the precision is greater that 14, the number will be displayed in exponential notation, whether or not the Use Exponential Notation check box is selected.

Use exponential notation

Exponential notation for fields in reports.

Alignment

Position of the data in the table column within the report.

Choices: Left, Center, Right

Default: Left

Summary functions to apply

Available table column summary functions, which enable you to summarize columns in a table to generate statistical data for a specific field. You can apply more than one function to a table column. To select a summary function, double-click (or drag) the function in the list, and a column is added to the Summary Function table.

Summary functions:

  • None – No function performed
  • Min – Minimum value
  • Max – Maximum value
  • Sum – Sum of all values
  • Count – Total number of rows in the table with data in the specific column
  • Mean – Mean average of all selected values
  • %RSD – Percent relative standard deviation
  • Std. Dev. – Standard deviation around the mean

Note: Verify that your minimum value is not greater than your maximum value. In some cases you can enter a minimum value that is greater than a maximum value without receiving a prompt warning you of the error.

Summary function table

Summary functions used for the specified table column. Summary functions are positioned in the order in which you selected them from the Summary Functions list. Summary Function columns:

  • Summary Function – Selected summary function. When a summary function is listed in the table, this cell becomes a list, from which you can select the summary function.
  • Min Limit – Minimum limit to report for the specified summary function.
  • Max Limit – Maximum limit to report for the specified summary function.
  • Precision – Number of digits reported to the right of the decimal. If the reported precision is less than the precision of the stored result, the reported result is rounded. If the reported field width or precision affects the validity of the result, the width is ignored and the entire result is printed.

Revert (Review main window only)

Resets changes currently made to the table column to the last-saved changes. These last-saved changes appear in the Column Properties dialog box. To apply the last-saved changes to the table, click Apply.

Previous column and next column

Displays data in the Column Properties dialog box from the next or previous column in the selected table.
