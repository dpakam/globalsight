General Tab (Project)

To modify all project properties, log in to Empower software using an account with the Alter Projects privilege. The Alter Projects privilege includes all rights associated with the Change Project Owner, Change Project Quota, Alter Custom Fields, Create Custom Field, and Delete Custom Field privileges.

If you do not have the Alter Projects privilege, you can modify specific project properties related to these privileges: Change Project Owner, Change Project Quota, Alter Project Type, Alter Custom Fields, Create Custom Field, and/or Delete Custom Field privileges.

To prevent users from modifying project properties, remove all above privileges.

General tab properties:

Property

Description

Name

Name of the selected project.

Owner

Owner of the project who initially created the project. If you have the Change Owner access privilege, you can change the owner of a project. To change the owner of a project, click the Owner list box, select a new owner from the list, then click OK to enable the change.

Project parent

Choose whether to add a project as a sub-project of an existing project. You can designate the project as a parent project or as a sub-project of the parent. Click Browse to change the hierarchy of the project.

Tip: If you have sub-projects nested in a deep hierarchy, the full project path may appear truncated. Click Browse to view the full hierarchy of the project.

Validation Template Project

Change or remove the validation template project that is associated with a validation working project. To do so, select a new validation template project or select the blank line from the list.

Enabled options

Change the project type for the selected project.

Project types are typically Standard projects. Method Validation Working or Method Validation Template project types are available only when you have the Method Validation Option installed.

To change the project type, you must be the project owner or have either the Alter Project Type or Alter Any Project privilege.

Data files

Current size of directory

Amount of disk space occupied by the raw data files for the selected project.

Database tablespace

Quota

Amount of tablespace (in megabytes) allocated to the selected project. If you have the Change Quota privilege, you can increase or decrease the amount of tablespace allocated to the selected project. To increase or decrease the amount of tablespace, click the appropriate control. You cannot decrease the tablespace below the current size of the project.

Free

Amount of unused tablespace quota (in megabytes) available for the selected project.

Tip: A minimum of 10MB is required to perform data processing.

If you have a project with sub-projects, the Copy to Sub-Projects button appears.

To copy project information to sub-projects:

  1. To copy general project settings from a project to all of the project's sub-projects, click Copy to Sub-Projects.

Tip: A message appears if you do not have access to one or more projects in the hierarchy.

  1. Click OK to apply the changes to the selected project.
