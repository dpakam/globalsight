Colors tab

Use this tab to define the format of lines, marks, and/or points used in a table, in a plot displayed in Review (or Compare), the sample plate type graphic created in Configuration Manager, or when an associated report group is displayed in a report. Color can be defined for items such as chromatogram plots, contour plots, text, calibration curves, real-time plot in Run Samples, lines in a table, GPC distribution plots, and so on.

Individual objects you can modify. The number of objects available varies with the individual item (table or plot).

To modify the color and line style of an object:

  1. Click the option button of the object to modify.
  2. Use the parameters in the Attributes section to specify the line style, thickness, and color (see below).

Note: You must individually edit each object that requires formatting. You can, however, format each object differently from each other.

Attributes

Line style

Line style to use for the object.

Choices: Solid, Dashed, Dotted, Dash Dot, Dash Dot Dot

Default: Solid

Thickness

Line weight.

Entries: 0 to 1000.

Default: 0 (thinnest line of 1 pixel).

Note: Waters recommends that you specify a line thickness of 0 when using line styles other than Solid. Line thicknesses vary with monitor and printer resolution, which can result in the various line style selections being displayed as Solid. A selection of 0 ensures that the line styles appear properly.

Line color

Color of the line. Click Line Color to access the Color dialog box. You can use the colors supported by the Windows operating system, which includes basic and custom colors.

Background

Background for the object, which can be transparent, transparent when printing, or a user-specified color. You can specify a color for a plot or graphic background, table heading, or plot axis.

To choose a color as a background, clear the Transparent check box, then click Bkgnd Color, Axis Color, or Heading Color to access the Color dialog box. You can use the colors supported by the Windows operating system, which includes the basic and custom colors.

If printing an active plot or table, you can specify Transparent When Printing from the Colors tab. This selection instructs the software to print the item on a white background even though a colored background might be specified for viewing.
