Flow tab (600)

Use this tab to define operating parameters that control pump pressure limits, flow, and mode.

High limit

System pressure above which the system shuts down. The pressure-limit range depends on pump type and head volume. If the system exceeds the limit for 1.2 seconds or more, the system shuts down. Empower software checks that you do not enter a value that is out of range for the pump type and head volume installed in the 600 System.

Default: 4000.0 psi

Note: In determining the high-pressure limit for your system, remember the pressure limitations of your column, injector, and detector. See the appropriate manuals.

Low limit

System pressure below which the system shuts down. The pressure-limit range depends on pump type and head volume. If the system falls below the limit for 60 seconds or more, the system shuts down. Empower software checks that you do not enter a value that is out of range for the pump type and head volume installed in the 600 System.

Default: 0.0 psi

Pump mode

System operating modes:

  • Isocratic (displays Flow and %A–%D text boxes)
  • Gradient (displays the Gradient table)

Flow

Percentage of solvent flow from each reservoir. The total of all solvents in each gradient segment must equal 100%. The system calculates individual flow rates when you enter the percentage value for each solvent. Empower automatically fills in the solvent column in the next column to the right so that the total percentage of the current row yields 100%. For examples of how the autofill feature works, see the related topic link.
