Bath A or B Components Page

Use this page to enter component parameters for Bath A and/or Bath B. This page appears only when you select a dissolution type in the Select Sample Set Method Type page of the Sample Set Method wizard.

If both baths are specified in the Select Sample Set Method Type page, enter all Bath A parameters, followed by all Bath B parameters, including offsets.

Method set

Select an existing method set from the list.

To create a new method set:

  1. From the Pro Interface, click Create New.

Result: The Select Instrument Method page of the New Method Set wizard appears.

  1. Create the method set. Upon completion, you are returned to the Bath A or B Components page.

Dissolution report method

Select the final report method for the dissolution test from the list.

Dissolution components table:

Column

Description
Row number Sequential number assigned by the software.
Name

Select the component name (from the list) included in your dissolution test. The name is from the previously created processing method. When you create a processing method for your sample set, you specify the components you want to test. A processing method generates a peak for each named component.
Claimed Amount

Enter the amount (in milligrams) of the component that the manufacturer claims is in the tested dosage form (one tablet). Entries: 0.000000 to 100000000.000000. Typical entries: 0.0 to 1000.00.

Rule: The Amounts page of the Component Editor must have comparable units (mg/mL) or (mcg/mL). In the Component Editor, enter standard concentration amounts.
Q Factor Minutes Select the length of time (in minutes) to illustrate the percent dissolved. You can select multiple Q factors. Entries: 0.0 to 999999.9. Typical entries: 5.0 to 600.00. The list shows the draw times you entered in the previous page (Samples) Transfer Times table.
Q Factor %

Enter the percent of component expected to dissolve in the time allotted for the dissolution test (for example, 28% dissolution in 30 minutes). Entries: 0.00 to 100.00.
