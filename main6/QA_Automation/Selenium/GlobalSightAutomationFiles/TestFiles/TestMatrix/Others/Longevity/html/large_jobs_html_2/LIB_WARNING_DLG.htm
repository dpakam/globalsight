Library Name Already In Use dialog box

Use this dialog box to specify a new name for a library you are copying to a database that contains a library with the same name. You can instruct the software to merge the spectra from the library you are pasting (dropping) with the spectra of the existing library in the database or another library of your choice in the database.

Create a new library named

In the target database, creates a new library with the name you specify. Click OK to create the new library.

Merge spectra with existing library

Merges any unique spectra in the library you are pasting (dropping) with the spectra in the existing library in the database. For each spectrum in the source library that already exists in the target library, a message box indicates that the duplicate spectra were not added to the library.

Merge spectra with the following library

Merges any unique spectra in the library you are pasting (dropping) with the currently selected library (if any) or a library you select from the list. For each spectrum in the source library that already exists in the target library, a message box indicates that the duplicate spectra were not added to the library.
