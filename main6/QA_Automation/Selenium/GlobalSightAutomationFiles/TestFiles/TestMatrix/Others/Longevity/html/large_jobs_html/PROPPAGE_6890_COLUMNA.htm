Column tab (6890)

Use this tab to set the column 1 or column 2 mode and any optional ramped pressure or flow programs.

Column information

The Length, Diameter, Film Thickness, and Carrier Gas Type fields are populated initially with information read from the 6890 GC system. If you create an instrument method when this information is not available, the Length, Diameter, and Film Thickness fields default to 0, the Description and Serial Number fields are empty, and the Gas Type field value defaults to Unknown.

When the instrument method runs, the Length, Diameter, Film Thickness, Description, and Gas Type fields are sent to the 6890 GC System, overwriting the information in the instrument.

Length (m)

Length of the column in meters.

Entries: 0 and 1.00 to 200.00

Description

Description of the column up to 18 alphanumeric characters.

Diameter (mm)

Column diameter in microns.

Entries: 0 and 50 to 750

Serial number

Serial number of the column.

Film thickness

Film thickness in microns.

Entries: 0 and 0.1 to 50.0

Rule: This value must be at least equal to half of the diameter value.

Gas type

Type of gas.

Choices: Nitrogen, Hydrogen, Helium, Argon / Methane, Oxygen, Air, Argon, and Unknown.

Mode

Column mode in which carrier gas is to be applied and monitored during a run:

  • Constant Pressure – Maintains a constant gauge pressure at the head of the column during a run. If column resistance changes, the mass flow rate is adjusted to maintain a constant pressure. This is the default.
  • Constant Flow – Maintains a constant mass flow rate of carrier gas in the column throughout the run. If column resistance changes, the column head pressure is adjusted to maintain a constant flow. This mode can significantly shorten run times during a ramped temperature program.
  • Ramped Pressure – Increases the column head gauge pressure according to how you program the Ramp table. When you select this parameter, the Ramp table is activated. By entering values in the table, you can program up to three ramped pressure increases during a run. Refer to Ramp table, below for more information.
  • Ramped Flow – Increases the mass flow rate in the column according to how you program the Ramp table. When you select this parameter, the Ramp table is activated. By entering values in the table, you can program up to three ramped flow increases during a run. Refer to Ramp Table below for more information.

Initial pressure/flow

Initial flow rate or pressure value, which depends on your Mode selection. If you selected Constant Flow or Ramped Flow, the units are mL/min. If you selected Constant Pressure or Ramped Pressure, the units are psi.

Default: 0.0.

Initial time

Length of time (from the start of the run) that the initial ramped flow or pressure value is maintained. If you select Ramped Flow or Ramped Pressure mode and enter Rate, Final (pressure/flow), and Hold Time values for row 1 in the Ramp table, flow or pressure is maintained for the length of time you set for Initial Time, then ramps up using the Rate and Final Pressure/Flow values you selected for row 1 in the Ramp table. If you select Constant Flow or Constant Pressure mode, Initial Time is disabled.

Default: 0.00

Ramp table

Allows you to program up to three ramped flow or pressure changes of carrier gas during a run. The table is active only when you select Ramped Flow or Ramped Pressure mode. In each row of the table, define:

  • Rate – Rate of changes in mL/min for flow, or psi/min for pressure.
  • Final (pressure/flow) – Pressure or flow that you want the ramp to reach.
  • Hold Time – Length of time during which you want to hold the final pressure or flow value. When this time expires, the software goes on to the next table row (unless this is the last row in the table).
  • Total Time – Total time as calculated by Empower software. Total Time displayed in the first row is derived from the difference between the Initial Pressure/Flow (at the start of the ramp) and the Final Pressure/Flow (at the end of the ramp) divided by the Rate, plus Hold Time plus the Initial Time. Total Time displayed for subsequent rows is derived from the difference between the Final Pressure/Flow from the previous row and the Final Pressure/Flow for the current row divided by the Rate, plus Hold Time, plus the Total Time from the previous row.

When the run begins, flow or pressure is maintained at the Initial flow or pressure value for the length of time you entered for Initial Time. At the end of that time, flow or pressure ramps up to the Final Pressure/Flow value at the Rate you set in row 1. Final Pressure/Flow is maintained for the length of time you entered for Hold Time. You can enter values in up to three table rows. The ramp process continues for each row in the table.

Note: If a flow or pressure program ends before the analytical run ends, the flow or pressure remains at the final value.
