GPC Calibration Point Table tab

Use this tab to define characteristics of a GPC Individual Points table printed in a report, or displayed in Review or Compare.

Use this tab to define characteristics of a GPC Individual Points table displayed in Review or Compare.

Available fields list

Fields in the database to include in the GPC Individual Points table. Click an item to view fields, then double-click (or drag) the desired field in the Available Fields tree. The field is added to the Table Template as a column.

GPC Calibration Point fields:

Field

Usage

Retention Time (min)

Retention time or adjusted retention time of each standard point (default).

Elution Volume (mL)

Elution volume of each standard point.

Mol Wt (Daltons)

Molecular weight for each standard point.

Log Mol Wt

Log (base 10) of the molecular weight.

Intrinsic Visc (dl/g)

Intrinsic viscosity of each standard as calculated from Mark-Houwink K and alpha constants and molecular weight (GPC universal calibration).

Point K (dl/g)

Mark-Houwink K constant of the standard point (GPC universal calibration).

Point alpha

Mark-Houwink alpha constant of the standard point (GPC universal calibration).

Hydrodynamic Volume (dl)

Hydrodynamic volume. Hydrodynamic volume is the product of the molecular weight and the intrinsic viscosity (Universal calibration).

Log (MolWt[h])

Log (base 10) of the hydrodynamic volume (universal calibration only).

Calculated Weight (Daltons)

Molecular weight at the retention time or elution volume for the calibration point as determined by the calibration curve.

% Residual

Percent residual between the molecular weight and the calculated weight (Mol Wt – Cal Wt /Cal Wt x 100%).

Ignore [h]

Indicates that the software ignored the point in the calculation of the Viscosity Law plot (universal calibration).

Standard Type

Type of standard (narrow, broad, or cumulative match) used to generate the calibration point.

Component

Name of the component for which you want a distribution calculated.

Relative Weight

Relative weight used by calibration point in calculating the calibration curve. You enter relative weights during sample loading.

Manual

When the result and channel Ids are blank, indicates that the point calibration data (retention time, elution time, molecular weight, relative weight, or viscosity fields) have been modified or manually entered. When the Ids are not blank, indicates that the result that produced this point was processed with a method different from the processing method currently associated with the calibration because the calibration curve was copied for use with a newer version of the processing method.

Ignore

Indicates that the software ignored the point in the calculation of the calibration curve.

Result Id

Identification number of the result that produced the point.

Table title

Title above the table. The table title can have up to 250 alphanumeric characters without restrictions. If multiple lines are required, press Ctrl+Enter to force the title onto multiple lines (reporting only).

Note: When you specify a table title, ensure that it is not wider than the potential table width. If the title is longer than the generated table, the title is truncated. Avoid this problem by shortening it or forcing the title onto multiple lines.

Alignment

Position of the title above the table. Choices: Left Alignment, Center Alignment, Right Alignment (reporting only). Default: Center Alignment.

Table template

Headings and order of the table columns as moved from the Available Fields list. To move a column, drag it to a different location in the Table Template. To delete a column from the Table Template, right-click the column name, then select Delete Column (or highlight the columns and press Delete). You cannot leave blanks between columns (reporting only).

Note: You can also specify sorting order in the individual columns of the Table Template.

Column properties

Accesses the Column Properties dialog box to modify column properties specified in the Table tab. To modify properties, select the corresponding column in the Table Template, then click Column Properties. After redefining column properties, click OK to return to the Table tab (reporting only).

Print blank columns

Prints table columns that do not contain values (reporting only).

Default: Enabled.
