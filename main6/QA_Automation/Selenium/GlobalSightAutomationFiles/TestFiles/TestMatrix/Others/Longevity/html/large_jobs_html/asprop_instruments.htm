Instruments tab

Use this tab to view details about instruments connected to the selected Empower Node and enter serial numbers, service dates, and comments.

Table column headings

Note: Information in these table columns is not stored in the database. It resides in the Empower Node local registry. To access this information, the Empower Node must be running.

Note: Information in these table columns is not stored in the database. It resides in the Node local registry. To access this information, the Node must be running.

Type

Instrument type, for example, W2690, W2487, or W996. This field is not editable.

Instrument type, for example, W2489 or 2707. This field is not editable.

Address

IEEE-488 address or serial port number. This field is not editable.

OK?

Indicates if the instrument is powered on and working properly. This field is not editable.

Serial number

Instrument serial number. This field is editable.

Tip: If you changed from IEEE-488 to Ethernet control, these components are renamed to include the module's serial number (2695, 2695D, 2795, 2796, PCM2, TCM2, W2414, 1525/1515, 1500CH). For example, W2690/5@15 is now W2690/5@123456 where 123456 equals the module's serial number.

Last serviced

Enter the date when the instrument was last serviced in the standard short date format for your region.

Exception: Do not change the hour or the time zone.

Next service due

Enter the date of the alarm point for the next service message in the standard short date format for your region.

Exception: Do not change the hour or the time zone.

PROM version

Instrument PROM version, or, in the case of the Waters 996 PDA, the revision of the buslace.hex file. This version can differ from the firmware version of the instrument. This field is not editable.

Note: If you are operation a 600E (firmware version 3.5), it will be detected as version 3.0 in Empower. Contact your service representative or worldwide technical support for further assistance.

Comments

Comments to include. This field is editable.

Details

Information about the instrument configuration, for example, whether a heater/cooler option is installed. This field is not editable.

Controls

Scan instruments

Scans for instruments connected to the Waters busLAC/E card. If an instrument is connected to the Empower Node but does not appear in the list, click Scan BusLAC/E so that the server recognizes the instrument.

Remove instrument

Removes the selected instrument from the list of instruments available to the currently selected Empower Node.

Removes the selected instrument from the list of instruments available to the currently selected Node.
