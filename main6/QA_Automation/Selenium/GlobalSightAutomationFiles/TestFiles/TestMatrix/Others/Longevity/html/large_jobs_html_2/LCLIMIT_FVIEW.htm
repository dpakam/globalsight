Limits tab

Use this tab to define the range of allowable values for each peak that is to have System Suitability testing. The System Suitability limits you set are used to determine limits shown as minimum and maximum values in summary charts, to flag result fields in summary tables, and to fault out-of-range values in printed reports and in the Peaks table in Review.

Tab section

Description

Flag values outside limits check box

Flags any values outside the limits set in the Suitability Limits table. Flagged results (in a faulted color and font) appear only in tabular data in printed reports and in Review.

Default: Enabled

Suitability Components table

The components that require System Suitability field limits using the parameters in the Suitability Components table (at the top of the Limits tab), which lists all components defined in the Components table (on the Components tab). Selecting a component enables the Suitability Limits table for editing limits.

Column

Description

Name

Component as listed in the Component table (or Slicing table if using a GPC processing method). This field is read-only.

Calculate suit results check box

Calculates System Suitability results for the component.

Default: Enabled

Flag outside limits check box

Flags values outside the limits set in the Suitability Limits table. Flagged (faulted) results appear only in tabular data in printed reports and in Review (Peaks table in Review Main window and in the Results window).

Default: Enabled

Suitability Limits table

Component field limits in the Suitability Limits table (at the bottom of the Limits tab). You can choose to calculate System Suitability results and flag values outside limits for each component. The selected component in the Suitability Components table is the component whose limits are defined in the Suitability Limits table. Click a row in the Suitability Components table to access the columns in the Suitability Limits table.

Column

Description

Field name

Peak field to which System Suitability limits are to be applied in the Peaks table.

Target

Target value for the peak field, which provides a visual reference point on a summary plot for the value you want to achieve. Entries: Greater than or equal to the Lower Error and Warning Limits and less than or equal to the Upper Error and Warning Limits.

Default: Blank

Error %

Percentage of error used for faulting peak values. This field becomes active when you enter a value in the Target field. The range is 0 to 100, with a precision of 2.

Default: Blank.

When you enter an Error % value or change the Target value when there is an Error % value, the Lower Error Limit (LCL) and Upper Error Limit (UCL) are calculated using these formulas:

  • LCL = Target – (Target * (Error %) / 100)
  • UCL = Target + (Target * (Error %) / 100)

Tip: If you enter an Error % value and modify either the LCL or the UCL, the Error % value is deleted.

Lower error limit (LCL)

 Entries: –999999999.999 to 9999999999.999

Default: Blank

If you enter an Error % or change the Target value when there is an Error % value, the LCL is calculated using this formula:

LCL = Target – (Target * (Error %) / 100)

Upper error limit (UCL)

Upper Error Limit (UCL) used for faulting peak field values greater than this maximum value and for displaying them in a summary plot (as a control chart).

Value range: –999999999.999 to 9999999999.999

Default: Blank

If you enter an Error % or change the Target value when there is an Error % value, the UCL is calculated using this formula:

UCL = Target + (Target * (Error %) / 100)

Warning %

Percentage of warning used for faulting peak values. This field becomes active when you enter a value in the Target field. The range is 0 to 100, with a precision of 2.

Default: Blank.

When you enter a Warning % or change the Target value when there is a Warning % value, the Lower Warning Limit (LWL) and Upper Warning Limit (UWL) are calculated using these formulas:

LWL = Target – (Target * (Warning %) / 100)

UWL = Target + (Target * (Warning %) / 100)

Note: If you enter a value in this field and modify either the LWL or the UWL, the Error % field will be cleared.

Lower warning limit

Lower Warning Limit value for display in a summary plot (as a control chart).

Value range: –999999999.999 to 9999999999.999

Default: Blank

If you enter a Warning % or change the Target value when there is a Warning % value, the LWL is calculated using this formula:

LWL = Target – (Target * (Warning %) / 100)

Note: Set the Lower Warning Limit within the Lower and Upper Error Limits.

Upper warning limit

Upper Warning Limit value for display in a summary plot (as a control chart).

Value range: –999999999.999 to 9999999999.999

Default: Blank

If you enter an Warning % or change the Target value when there is a Warning % value, the UWL is calculated using this formula:

UWL = Target + (Target * (Warning %) / 100)

Note: Set the Upper Warning Limit within the Lower and Upper Error Limits.

Ignore blank values

When specifying a Lower Error Limit, select or clear the check box to specify whether to fault the corresponding peak field when its value is blank:

  • Selected - Does not mark the field as faulted.
  • Cleared - Marks the field as faulted.

Copying limits to other components

To copy limits to the other components in the processing method, use either command in Review (from the Options menu). Modify the copied limits as necessary.

  • Copy All Limits – Copies every limit associated with a component in the Limits tab to the remaining components in the processing method. If limits were set previously for the other components, this command replaces those limits.
  • Copy Selected Limits – Copies selected limits associated with a component in the Limits tab to the remaining components in the processing method. If limits were set previously for the other components, this command adds the selected limits from the component to the already specified limits. If a limit already exists for a component, it is replaced with the selected limit.
