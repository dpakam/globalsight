Fonts tab

Use this tab to define the format of table text, a plot in Review (or Compare), the real-time plot in Run Samples, the sample plate type graphic created in Configuration Manager, or an associated report group displayed in a report.

Use this tab to define the format of table text, a plot in Review (or Compare), the real-time plot in Run Samples, or an associated report group displayed in a report.

Font to edit

Individual text items you can modify (for example, the Faulted Data type). The number of text items available for font formatting varies with the individual plot, report group, or table.

To format the font of a text item, click the option button of the text item to format. Use the parameters in the Characteristics section of the Fonts tab to define the font family, size, style, and color (see below).

Rule: You must individually edit each text item that requires formatting. You can, however, format each text item differently from each other.

Characteristics

Format of the font in the text item selected in the Font to Edit section. Choices include font family, style (Bold, Italic, Underline, Strikeout), point size, and color. Font families and sizes depend on your computer’s fonts.

To change the text color, click Text Color to access the Color dialog box. You can use the colors supported by the Windows operating system, which includes basic and custom colors.

Font sample

Displays how the text appears with selected characteristics.
