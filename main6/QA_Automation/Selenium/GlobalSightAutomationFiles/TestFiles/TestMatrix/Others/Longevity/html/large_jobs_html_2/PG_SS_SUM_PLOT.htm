Sum Plot tab

Use this tab to specify the format and plot to use in the System Suitability Summary plot printed in a report.

Fields list

Database fields available to include in the Summary plot as the y-axis. Double-click a field in the list to add it to the plot.

Format

Plot type(s) to use in the Summary plot. Multiple plots in the same Summary Plot report group are supported.

Choices: Bar Plot, Line Plot, Scatter Plot.

Fit parameters

Plot fit

Type of line fit to apply to the Summary plot. Select the Plot Fit check box to enable this selection:

  • Linear Fit – Determines the line equation for linear fit.
  • Mean – Produces a straight line showing calculated means.

Exponential notation

Designates if the equation for the fit line will be expressed on the report in exponential notation or as a real value. Choices:

  • Select this check box to view the equation in exponential notation (Default).
  • Clear this check box to view the equation as a real value. In this case, the curve has three decimal places, and the R^2 value has six decimal places.

Labels/limits

Displays labels and limit values in the Summary plot. Choices (one or more):

  • Label Points w/Legend Fields – Plot legend label (as created in the Legend tab) on each point in the Summary plot.
  • Upper Error – Upper Error (UCL) minimum value in the Summary plot (as specified in the System Suitability table of the processing method). You enter the error label for the Upper Error minimum value in the text box (up to 250 alphanumeric characters).
  • Lower Error – Lower Error (LCL) minimum value in the Summary plot (as specified in the System Suitability table of the processing method). You enter the error label for the Lower Error minimum value in the text box (up to 250 alphanumeric characters).
  • Upper Warning – Upper Warning maximum value in the Summary plot (as specified in the System Suitability table of the processing method). You enter the warning label for the Upper Warning maximum value in the text box (up to 250 alphanumeric characters).
  • Lower Warning – Lower Warning minimum value in the Summary plot (as specified in the System Suitability table of the processing method). You enter the warning label for the Lower Warning minimum value in the text box (up to 250 alphanumeric characters).
  • Target – Control limit value in the Summary plot (as specified in the System Suitability table of the processing method). You enter the label for the Target value in the text box (up to 250 alphanumeric characters).
