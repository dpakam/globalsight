Line tab

Use this tab to define the attributes and position of the Line drawing object.

Line attributes

Line style

Line style.

Choices: Solid, Dashed, Dotted, Dash Dot, Dash Dot Dot

Default: Solid

Thickness

Line weight.

Entries: 0 to 1000

Default: 0 (thinnest line of 1 pixel)

Note: Waters recommends that you specify a line thickness of 0 using line styles other than Solid. Line thicknesses vary with monitor and printer resolution, which can result in the various line style selections appearing Solid. A selection of 0 ensures that the line styles appear properly.

Line color

Line color. Click Line Color to access the Color dialog box. You can use the colors supported by the Windows operating system, which includes basic and custom colors.

Line background

Background of the line (transparent or a color).

To choose a background color, clear the Transparent check box, and then click Line Color to access the Color dialog box. You can use the colors supported by the Windows operating system, which includes basic and custom colors.

Position

Orientation of the line in the report.

Choices: Horizontal, Vertical, Forward Diagonal, Rear Diagonal

Default: Horizontal
