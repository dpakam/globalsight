Structures tab (properties of structure plots)

Use this tab to modify how structures appear in Review or in a report. Click Apply while making changes to see how they appear in the plot.

To modify structure appearances within Review or a report:

  1. In the Chemical Structures section, select the structures to display. You can also choose to display the peak plot.
  2. In the Labels section, select the structure information labels to add to the structures.
  3. In the Label Position section, click the location for all structure information labels, upper-left or upper right.
  4. Click OK.
