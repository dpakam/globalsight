Create Channel Set dialog box

Use this dialog box to select the type of channel set you want, which can be one of the following types:

  • Create GPCV Channel Set – Select when the GPCV option is enabled in the project, the instrument method contains a GPCV instrument, and you want to process data from concentration and viscometer channels.
  • Create LS Channel Set – Select when the GPC and GPC/LS options are enabled in the project, the instrument method contains both GPC and LS instruments, and you want to process data from concentration and light scattering channels (dual detection).
  • Create GPCV-LS Channel Set – Select when the GPC/LS and GPCV options are enabled in the project, the instrument method contains both LS and GPCV instruments, and you want to process data from concentration, viscometer, and light scattering channels (triple detection).

After you select the channel set, Click OK to define the channel set.
