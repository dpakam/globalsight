Events Tab (6890)

You can create a maximum of 25 events in the Events table.

Requirement: You must specify a Time, Event, parameter, and a set point value.

To program multiple events in the Events table:

  1. Click Enable Events.
  2. In the Events table, enter the required information.

Events table columns:

Column

Description

Time

Specify when the event will occur by entering the time in minutes after inject start.

Event

Specify one of the following events:

Signal 1 - Data from channel 1 source.

Signal 2 - Data from channel 2 source.

Valve - If valves are configured in the Valves tab, specify a valve number from 1 to 8.

Aux 3 to 5 - If auxiliary are configured in the Auxiliary tab, specify pressure.

Front or Back Det - If you have configured a NPD Nitrogen Phosphorus Detector or Thermal Conductivity Detector (TCD), specify either Front Det or Back Det.

Parameter

Specify an action for each event that you selected.

Example: If valves were defined, specify a valve number from 1 to 8. If an NPD is installed, select H2 flow. If a TCD detector is installed, select polarity.

Option

For some parameters, you can specify a non-numeric value.

Example: If you selected the Source parameter for the Signal 1 event, you can select a new data source for Channel 1. For example, you can change the data source from the front detector to back detector.

Value

For some parameters, you specify a numeric value.

Example:  If you select the Pressure parameter for the Aux 3 event, you specify a new pressure value.

Comment

You can enter comments that appear in the instrument method.
