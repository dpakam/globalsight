New View Filter wizard - Sorting Order page

Use this page to set the sorting order (optional) for each Column Name in the view filter. The sorting order determines the order in which data is sorted within each column in the view.

  • Empty (No order) – Specifies that data within the view table column not be sorted when the view filter is applied. This is the default selection and is designated by an empty cell.
  • Hide – Specifies that data within the view table column not be sorted and hides the column when you apply the view filter. Any conditions specified within a hidden column are applied to the view table even though the column is hidden.
  • Ascend – Sorts information, such as sample names, alphabetically from A to Z, numerically from smallest to largest, and by date.
  • Ascend/Hide – Sorts information just as with Ascend, but the column is hidden in the view table. Although hidden, the column is used in the sorting process.
  • Descend – Sorts information, such as sample names, alphabetically from Z to A, numerically from largest to smallest, and by date.
  • Descend/Hide – Sorts information just as with Descend, but the column is hidden in the view table. Although hidden, the column is used in the sorting process.

After making your selections, click Next.
