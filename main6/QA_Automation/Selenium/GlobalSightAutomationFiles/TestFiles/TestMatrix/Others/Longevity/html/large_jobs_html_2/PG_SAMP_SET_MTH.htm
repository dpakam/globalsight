Sample Set Method tab

Use this tab to identify what sample set method information to include in the report. By default, all items are included in the Sample Set Method report group, except Differences from Previous Version information.

Information to include in the report

You can include the following report information:

  • Method Information – Contents of the associated sample set method.
  • Sample Set Tables – Contents of the Sample Set table (defined from Run Samples or the Sample Set Method Editor).
  • Amount Tables – Contents of the Component Loading table.
  • MW Tables – Contents of the Weight Loading table (GPC/V option).
  • K Alpha Tables – Contents of the K-Alpha table (GPC/V option).
  • Moments Information – Contents of the Moments table (GPC/V option).
  • Dissolution Information – Dissolution parameter values specified in the sample set method. Parameter values are printed for both dissolution baths.
  • Instrument Parameters – Sample set line level parameters for ZQ Detectors (2000/4000 models).

Method revision information

Note: To report on this information, Full Audit Trail must be enabled for the project.

You can include the following method revision information:

  • Revision History Details – Displays a description of the change(s) made in the Revision Description field whenever you modified the sample set method (if Audit Trail Support is activated).
  • Revision Summary Information – Adds a table to the report that includes this information about the method: name, type, comments, date, modified user, locked status, method ID, and method version.
  • Differences from Previous Version – When reporting raw data, you can obtain a report of the method differences between the method version used to collect the data and the method version before the reported method version. When reporting using results, you can obtain a report of the method differences between the method version used to collect or process the data and the method version before it.

Select this check box to activate these option buttons and indicate the type of differences to include in the report:

  • Print Differences Only – Includes a table in the report that only includes information for fields that differ between the two methods.
  • Print Entire Methods – Include a table that lists the entire contents of each method with differences highlighted, allowing you to see what fields changed between the two versions.
