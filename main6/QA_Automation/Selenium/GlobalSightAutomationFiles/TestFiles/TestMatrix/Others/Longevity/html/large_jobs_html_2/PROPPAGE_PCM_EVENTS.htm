Events tab (PCM/15XX)

Use this tab to program up to 11 timed external event outputs on the PCM.

Events overview

A PCM event generates a relay output on the PCM that initiates a process on another device at a specified time. This output could turn on a column heater, control operation of a fraction collector or chart recorder, or activate a column switching valve.

You can program up to 11 ev-0\]ent outputs in the Events table. Define the time that each event is to occur, the event types (Relay or Event), the event function (On, Off, Pulse, Repeat Pulse), and the interval of pulsed events.

Note: Event outputs do not provide power to run the external device. Use an appropriate power supply to run the device.

Enable events

Enables or disables events in the Events table.

Choices: Enabled or Disabled

Default: Disabled

Events table columns

Program operating parameters in these columns to define output events for an instrument connected to the Event or Relay terminals on the rear of the PCM.

Time

Specifies the time that the event function occurs. Event rows are sorted by time.

Event

Event output terminal. Choose from the two Relay outputs and four Event outputs to program up to 11 total event outputs. You can program the same output more than once. Choices:

  • Relay 1 and Relay 2
  • Event 1, Event 2, Event 3, and Event 4

Note: To maintain similarity with the SIM, the output connections on the PCM are divided into two sets of terminal connections. Connections are labeled Event 1 through 4 and Relay 1 and 2. All six connections are relay connections that provide contact closures (not TTL outputs).

Function

Type of function:

  • On – Turns on a contact closure (Relay or Event type). With this function, the switch provides a Low position (low resistance) output that remains on until an Off function is sent.
  • Off – Turns off the contact closure (Relay or Event type). With this function, the switch moves to the High position (high resistance), and the contact closure is broken.
  • Pulse – Transmits one On/Off pulse. With this function, the switch moves to the Low position then to the High position. The contact closure (low resistance) is maintained for the pulse width specified by the Interval parameter. The default pulse width is 0.02 min.
  • Rep Pulse – Transmits multiple On/Off pulses. The switch alternates between the On and Off positions for the pulse width specified by the Interval parameter. Pulses continue until an Off function is sent.

Interval

Duration (in minutes) of a pulse event. You can access this parameter only if you selected Pulse or Rep Pulse as the function. The minimum pulse width is 0.02 minutes On and 0.02 minutes Off.

Comments

Description of the programmed event. The description can be up to 256 characters in length.
