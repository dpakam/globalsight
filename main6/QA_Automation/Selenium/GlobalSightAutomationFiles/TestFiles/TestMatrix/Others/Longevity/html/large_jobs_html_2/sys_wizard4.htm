New Chromatographic System wizard – Name Selection page

Use this page to name the new chromatographic system and enter any optional comments.

System Name

Identifies the chromatographic system in the Empower database and in Configuration Manager. Up to 30 alphanumeric characters are allowed for the system name. Use a meaningful system name in case other users have access to the system.

Identifies the chromatographic system in the database. Up to 30 alphanumeric characters are allowed for the system name. Use a meaningful system name in case other users have access to the system.

Empower Node Name

Name of the Empower Node to which the selected system is connected. This field is not editable.

Breeze 2 Node Name

Name of the Node to which the selected system is connected. This field is not editable.

Online

If all configured system instruments are available (not configured in another online system), this brings the new system online.

Default Comments

Select a suitable default string as a comment.

System Comment

Displays optional remarks (up to 250 characters). Comments that describe the chromatographic system, its components, and its intended use can help others if they have access to the system.

Password

Enter your user password.
