Rotation tab

Use this tab to specify rotation of a plot when printed in a report. When rotation is enabled, all plot components also rotate (legend, x-axis, y-axis, and so on). You can rotate any plot report group in your report.

Rotation choices:

  • No Rotation – No change in orientation occurs for the plot.
  • 90 Degrees Orientation – Reorients the plot 90° counterclockwise.
  • 180 Degrees Orientation – Reorients the plot 180° counterclockwise.
  • 270 Degrees Orientation – Reorients the plot 270° counterclockwise.
