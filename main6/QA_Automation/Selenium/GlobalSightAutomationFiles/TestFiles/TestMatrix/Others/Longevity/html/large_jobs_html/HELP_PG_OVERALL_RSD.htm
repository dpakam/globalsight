Overall %RSD tab

Use this tab to summarize selected fields displayed in an overall %RSD plot printed in a report. The set of summary fields and the title are user definable.

Plot title

Title of the Overall %RSD plot when you print the report. The plot title can have up to 250 alphanumeric characters and has no text restrictions. If multiple lines are required, press Ctrl+Enter to force the plot title onto multiple lines.

Alignment

Position of the title above the %RSD plot.

Choices: Left Alignment, Center Alignment, Right Alignment

Default: Center Alignment

Fields list

Fields available to include in the report. Double-click a field in the list to add the field to the plot.

Table

Identifies the specifics of selected fields and provides the means to modify field specifications. Fields are positioned in the same order in which they are selected from the Fields list. Columns are:

  • Original Name – Unedited, original name of the selected field.
  • Field Name – Modified name of the selected field. If desired, change the field name in the associated table cell (up to 250 alphanumeric characters).
  • Width – Number of spaces allotted for the field. Allowable entries depend on the selected field.
  • Precision – Number of digits reported to the right of the decimal. Allowable entries depend on the selected field.
  • Exponential Notation – Exponential notation for the field.
  • Field Type – Type of field included in the legend (Double Precision Float, String, Integer, Date, Enumerated, or Boolean Value).
