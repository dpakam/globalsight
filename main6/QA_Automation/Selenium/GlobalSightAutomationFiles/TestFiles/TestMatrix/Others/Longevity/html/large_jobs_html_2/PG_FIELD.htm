Field tab

Use this tab to define characteristics of the specified Field report group when the result is reported.

Original name

Name of the selected field report group.

Field type

Database table that the selected field report group comes from.

Data type

Data type for the selected field report group.

Label text

Text to appear in the Field report group (up to 250 alphanumeric characters without restrictions). If multiple lines are required, press Ctrl+Enter to force the label onto multiple lines. The format of the text line is defined using the Fonts tab.

If the text line overlaps the Field report group (as sized in the Report Page), a section of the text line is removed from the group.

Label position

Where the label is printed in relation to the result.

Choices: Left of Data, Right of Data, Above Data, Below Data

Default: Left of Data

Data width

Number of spaces allotted for the reported result. Allowable entries depend on the selected field. This parameter applies to the result only. It does not affect the number of spaces allotted for the label.

Data precision

Number of digits reported to the right of the decimal. If the reported precision is less than the precision of the stored result, the reported result is rounded. If the reported field width/precision affects the validity of the result, the width is ignored and the entire result is printed.

Show units

Units for fields in reports.

Use exponential notation

Exponential notation for fields in reports.

Extend group to fit text

Select to extend the group downward to include all text in the group box on the report by wrapping the output when there is a word break. This check box is clear by default, which results in the output for the first field being limited to the group boundaries.

Note: If this check box is selected for a summary report and the field has multiple values, all values print on the report. Many values can make a report difficult to read.

Alignment

Position of the result in the report.

Choices: Left, Center, Right

Default: Left

Summary function

Function type

Summarizes the result in a report group and generates statistical data:

  • None – No function performed (Default)
  • Min – Minimum value
  • Max – Maximum value
  • Sum – Sum of all values
  • Count – Total number of rows in the table with data in the specific column
  • Mean – Mean average of all selected values
  • %RSD – Percent relative standard deviation
  • Std. Dev. – Standard deviation around the mean

Function name

Labels the information with something other than the Function Type name (up to 250 alphanumeric characters without restrictions).
