Backup Project wizard – Comment Entry

Use this page to verify your choice of projects to be backed up.

Use this page to verify your choice of projects to be backed up and to enter comments in the Backup Comments text box. A comment entry is required (up to 250 alphanumeric characters).

If you want to change the projects you selected for backing up, click Cancel to return to the Projects view of Configuration Manager, then reselect the projects. Restart the Backup wizard and follow the on-screen instructions.
