Enter Revision Description dialog box

This dialog box appears when you modify a method or method set and have the Audit Trail Support feature activated.

Method

Name of the processing method. You cannot edit this field.

Revision

Revision number of the processing method. You cannot edit this field.

Default revision description

List of default descriptions (if previously created).

Revision description

Comments about the revisions you made to the processing method. When you finish, click Finish to save the revisions.

View a history of these comments in the Audit Trail dialog box.
