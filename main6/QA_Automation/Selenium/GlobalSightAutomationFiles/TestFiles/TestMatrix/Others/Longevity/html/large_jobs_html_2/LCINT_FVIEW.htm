Integration tab

Use this tab to define integration parameters (top area) and events (bottom area) of your processing method. The parameters affect the integration of the chromatogram displayed in the Review Main window, as well as all chromatograms processed using the processing method.

The top area of the tab displays parameters that define peak detection and integration for the entire chromatogram.

Integration algorithm

Select either ApexTrack or Traditional. The parameters vary for each integration algorithm.

Note: If Integration Algorithm is unavailable, the processing method uses the Traditional algorithm by default.

Peak detection/integration parameters:

Traditional peak detection/integration parameters:

Parameter

Description

Peak width

Peak width used for calculating the data bunching factor applied to the raw data during peak detection. During peak detection, the integer product of Peak Width and the sampling rate (at which the data was acquired) divided by 15 determines the number of points bunched together to produce a single point in the chromatogram.

Threshold

Liftoff and touchdown values (minimum rate of change of the signal) for peak detection. The software calculates slope changes across three bunched data points and compares this change against the Threshold value.

Minimum area

Minimum peak area (in mV·sec) required for a peak to be reported. Any peak with less than the minimum area requirement is not included in the peak results.

Value range: 0 to 1.000e+090 mV·sec

Default: 0 mV·sec

Set the minimum area using the Minimum Area field.

Note: Selecting Minimum Area displays a reading of 95% of the total area of the selected peak. The software uses this percentage of the value to report peaks of this size. If the area value equaled 100% of the selected peak, the peak might be excluded when processing a different chromatogram (with the same peak).

Minimum height

Minimum peak height (in microvolts) required for a peak to be reported. Peaks below the minimum height requirement are excluded from the peak results.

Value range: 0 to 1.000e+090 mV

Default: 0 mV.

Set the minimum height using the Minimum Height field.

Note: Selecting Minimum Height displays a reading of 95% of the total height of the selected peak. The software uses this percentage of the value to report peaks of this height. If the height value equaled 100% of the selected peak, the peak might be excluded when processing a different chromatogram (with the same peak).

ApexTrack peak detection/integration parameters:

Parameter

Description

Start (min)

Start detection/integration time.

Default: Blank (which means start of data)

Range: If a numerical value is entered, the range is 0 minutes (min) to 655 min. If both values are not blank, Start must be less than End

Peak width

Peak Width @ 5% Height. Sets the filter width.

Default: Blank (which invokes the Auto-Peak Width algorithm). If a numerical value is entered, it must be greater than 0 seconds (sec).

Range: 0.01 to 9999.99 sec

Detection threshold

Peak Detection Threshold. Sets the peak-to-peak baseline noise in response units scaled to the same units as peak height (microvolts).

Default: Blank (which invokes the AutoThreshold feature)

Range: 0.000 to 1.000e+090 microvolts (µV). This is the only field displayed in exponential notation.

Liftoff %

A multiplier used to determine peak start.

Range: A numerical value from 0 to 100% is required

Default: 0.000%

Touchdown %

A multiplier used to determine peak end.

Range: A numerical value from 0 to 100% is required

Default: 0.500%

Minimum area

Minimum peak area (in microvolts per second) required for a peak to be reported. Any peak with less than the minimum area requirement is not included in the peak results.

Entries: 0 to 1.000e+090 mV·sec

Default: 0 mV·sec

Set the minimum area using the Minimum Area text field.

Minimum height

Minimum peak height (in microvolts) required for a peak to be reported. Peaks below the minimum height requirement are excluded from the peak results.

Entries: 0 to 1.000e+090 mV

Default: 0 mV

Set the minimum height using the Minimum Height text field.

Integration event table

The bottom of the tab includes the Integration Event table, where you set time-specific detection/integration events for the selected chromatogram. The events override the individual global detection/integration parameters set in the top of the Integration tab or in the Review Main window. Apply events to a single peak or the entire chromatogram.
