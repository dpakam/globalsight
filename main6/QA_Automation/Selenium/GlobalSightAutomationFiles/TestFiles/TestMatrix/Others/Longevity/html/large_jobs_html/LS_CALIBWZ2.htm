LS Calibration wizard – Detector and Solvent page

This page displays parameters that describe the light scattering detector, the solvent, and the solvent’s refractive index. Solvent Name is copied from the solvent name entered during sample loading in the Components table or Alter Sample for the current channel in Review. If you selected Use existing calibration from the Initial LS Calibration page, this page contains the refractive index from the chosen calibration. If you selected Create a new calibration and there is an existing calibration for the solvent, Empower copies the refractive index from the latest calibration for the solvent.

Note: You must enter a value for the Solvent Refractive Index before proceeding to the next page. The range of values for the Solvent Refractive Index is 1.000 to 10.000.

Your selections determine which page appears after you click Next:

  • If the LS instrument method parameters specify that there is only one LS channel, the next page is the Calibration Standard page.
  • If there are multiple LS channels and you selected Perform Light Scattering Normalization from the LS Calibration Procedures page, the next page is the Normalization Standard page.
  • If there are multiple light scattering channels and you did not select Perform Light Scattering Normalization, the next page is the Normalization Constants page.
