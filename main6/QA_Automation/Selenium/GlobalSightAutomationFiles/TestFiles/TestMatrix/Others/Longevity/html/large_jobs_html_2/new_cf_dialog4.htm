New Custom Field wizard – Numeric Parameters page

Use this page to specify the number of characters to print for the field width and the precision (number of characters to the right of the decimal point) of the numeric entry. You can also specify minimum and maximum values for keyboard-entered parameter values.

Numeric parameter

Description

Width

Maximum number of alphanumeric characters that can be printed in a field. If applicable, include the decimal point and sign.

Entries: 1 to 10 for integer, 1 to 15 for real.

Default: For an Integer data type, 8; for a Real data type, 12.

Minimum Value

For keyboard-entered integers and Real data types, the minimum value for the field.

Entries: –9999999999.999 to 10000000000.000

Default: –99999999.999

Rule: For a Keyboard data source, the software does not accept entries outside the defined minimum and maximum values. When a calculation is performed, these vales are ignored.

Minimum Value

For keyboard-entered integers and Real data types, the maximum value for the field.

Entries: –9999999999.999 to 10000000000.000

Default: +100000000.000
