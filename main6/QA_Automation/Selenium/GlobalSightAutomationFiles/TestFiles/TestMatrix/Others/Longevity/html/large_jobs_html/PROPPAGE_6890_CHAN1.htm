Channel tab (6890)

Use this tab to define data channel 1 or data channel 2 parameters for the 6890 GC Systems.

Note: Define all Channel tab parameters transmitting analog data through a busSAT/IN or busSAT/IN 2 Module. If you enabled the Ch 1 Enable or Ch 2 Enable parameter to transmit digital data through the RS-232 port, then define the Units and Source parameters (see below).

Channel name

Software-designated name for data channel 1 or 2, which appears in the Active Channels table in the Instrument Method Editor.

Description

Optional description for data channel 1 or 2 of the 6890 GC System, up to 250 alphanumeric characters. The description appears in the Active Channels table in the Instrument Method Editor and in the Real-Time Plot pane in Run Samples.

Source

GC output signal to be monitored by Empower software. Choices:

  • Front Detector - Signal from the front detector (default).

Note: When configured for Dual Tower operation, specify the detector (Front or Back), which is connected to the Front Inlet by the column, as the source for Channel 1.

  • Back Detector - Signal from the back detector

Note: When configured for Dual Tower operation, specify the detector (Front or Back), which is connected to the Back Inlet by the column, as the source for Channel 2.

  • Front–Back - Difference signal between two detectors of the same type (front detector signal minus back detector signal)
  • Back–Front - Difference signal between two detectors of the same type (back detector signal minus front detector signal)
  • Comp1 - Signal from column compensation 1
  • Comp2 - Signal from column compensation 2
  • Front–Comp1 - Difference signal between the front detector and column compensation 1
  • Back–Comp1 - Difference signal between the back detector and column compensation 1
  • Front–Comp2 - Difference signal between the front detector and column compensation 2
  • Back–Comp2 - Difference signal between the back detector and column compensation 2
  • Test Plot - Test signal (a stored chromatogram) for diagnostic use
  • Oven Temperature - Current oven temperature in °C
  • Column 1 Flow - Current column 1 flow in mL/min
  • Column 2 Flow - Current column 2 flow in mL/min
  • Column 1 Pressure - Current column 1 pressure in psi
  • Column 2 Pressure - Current column 2 pressure in psi

Sampling rate

Sampling rate used to collect GC data for channel 1 or 2 when transmitting data through the RS-232 output.

Choices: 0.1, 0.2, 0.5, 1.0, 2.0, 5.0, 10, 20, 50, 100, or 200 points per second

Default: 0.1 points per second

Notes:

  • Select a Sampling Rate of 200 points per second for single-channel acquisition only.
  • Set the sampling rate for GC data collected through the analog channels in the busSAT/IN or the busSAT/IN 2 Module tabs, depending on which module is connected to the GC.

Range

Selects and adjusts a portion of the full dynamic range of the output analog signal. The selected portion is adjusted so that the highest possible value of the portion does not exceed the maximum voltage of the given output signal (+1 mV or +1 V output).

Default: 0

Range and Attenuation Value are used to keep peaks of interest on scale when using the analog data (busSAT/IN). See the Agilent 6890 Series Gas Chromatograph Operating Manual for details on selecting range and setting signal attenuation.

Note: To minimize error, set Range to the lowest possible value, ensuring peaks of interest do not exceed 1 V.

Fast peaks

Enables the recording of fast peaks when using FID or NPD detectors to send data to an integrator. To use this feature, ensure that your integrator or recorder operates at approximately 15 Hz, allowing for minimum peak widths of 0.004 minutes (8 Hz bandwidth).

Choices: Enabled or Disabled

Default: Disabled

Data sensitivity

Number of signal data decimal places to be transmitted from the 6890 GC to the Empower database.

Choices: LOW (one decimal place), MEDIUM (three decimal places), or HIGH (five decimal places)

Default: MEDIUM

Note: Higher sensitivity provides better definition of small features in the chromatogram, however the size of large features may be limited.

Zero parameters

Enable

Enables the zeroing feature that corrects an elevated baseline. Zeroing affects only analog signals. Enable this check box before the Offset parameter takes effect (see below).

Choices: Enabled or Disabled

Default: Disabled

Offset

Value to be subtracted from the detector signal to compensate for an undesired, constant background signal (typically caused by the detector, column bleed, or contaminants in the gas supply). The result is an increase in dynamic signal range.

Default: 0

See the HP 6890 Series Gas Chromatograph Operating Manual for information on zeroing signal output.

Attenuation parameters

Note: Attenuation must be physically allowed or disallowed using the 6890 front panel by pressing Signal 1 (or Signal 2) and selecting Attenuation On or Off. Empower software does not overwrite this value.

Enable

Enables the attenuation feature. Enable this check box before the Attenuation parameter takes effect (see below).

Choices: Enabled or Disabled

Default: Disabled

Attenuation

Selects and adjusts a portion of the ranged +1 mV output signal to ensure that the signal does not exceed 1 mV (see Range above). Each step to a higher Attenuation value reduces the output signal level by a factor of 2 (half the previous level).

Default: 0
