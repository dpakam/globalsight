Rect tab

Use this tab to define line and fill attributes of the Rectangle drawing object.

Line attributes

Line style

Line style to use for the Rectangle drawing object.

Choices: Solid, Dashed, Dotted, Dash Dot, Dash Dot Dot

Default: Solid

Thickness

Line weight of the Rectangle drawing object.

Entries: 0 to 1000

Default: 0 (thinnest line of 1 pixel)

Note: Waters recommends that you specify a line thickness of 0 when using line styles other than Solid. Line thicknesses vary with monitor and printer resolution, which can result in the various line style selections appearing Solid. A selection of 0 ensures that the line styles appear properly.

Line color

Color of the lines used to create the Rectangle drawing object. Click Line Color to access the Color dialog box. You can use the colors supported by the Windows operating system, which includes basic and custom colors.

Fill attributes

Transparent

Transparent background for the Rectangle drawing object. When you select this attribute, objects behind the Rectangle drawing object are visible.

Solid

Solid fill pattern for the Rectangle drawing object. When you select this attribute, objects behind the Rectangle drawing object are not visible. You can choose a solid fill color for the rectangle with the Fill Color option.

Note: Clear the Transparent check box to access the Solid option.

Hatch

Hatch fill pattern for the Rectangle drawing object.

Choices: Horizontal, Vertical, Forward Diagonal, Back Diagonal, Cross, Diagonal Cross

Default: Horizontal

You can choose a fill color for the hatch fill pattern in the rectangle with the Fill Color option.

Note: Clear the Transparent check box to access the Hatch option.

Fill color

Color of the Rectangle drawing object (for the Solid or Hatch fill pattern). Click Fill Color to access the Color dialog box. You can use the colors supported by the Windows operating system, which includes basic and custom colors.
