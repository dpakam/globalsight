Info tab

Use this tab to define the label used in the Special Information report group.

Label text

Text, up to 250 alphanumeric characters, that appears in the Special Information report group. To force the label text onto multiple lines, press Ctrl+Enter. You can format the text line using the Fonts tab.

If the text line overlaps the Special Information report group (as sized in the Report Page), a section of the text line is not printed.

Information to print

General information to include in the Special Information report group. Choices: Current Date, Current Time, User Name, User Type, Project Name, Report Method Name, Page Number, Empower Version, Database Service, Project Information, Project Custom Fields, Registered User, Registered Company, Support Plan Number, Report Method ID, and Report Job Number.

You can include only one category of information for each Special Information report group.

Note: For Registered User, Registered Company, Support Plan Number, Report Method ID, and Report Job Number, Empower retrieves the information from the registry of the local machine. In a Client/Server environment, this information comes from a client or Lace³², not the server.

Label position

Where the label prints in relation to the general information.

Choices: Left of Data, Right of Data, Above Data, Below Data, Do Not Show

Default: Left of Data

Alignment

Position of the general information selection in the Special Information report group.

Choices: Left, Center, Right

Default: Left
