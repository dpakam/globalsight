Summary Plot X-Axis tab

Use this tab to specify the field to use as the x-axis in the System Suitability Summary plot when printed in a report.

X-Axis data type

Type of data to display as the x-axis in System Suitability Summary plots:

  • Select X-Axis Value From List at Right – Enables selection of a specific database field as the x-axis in the Summary plot. Choose from the Fields List (see below).
  • Use Run Number As X-Axis – Specifies the use of the data from the selected runs in the Project window as the x-axis in the Summary plot (which might be from one or more selected results, injections, channels, and so on).

Fields list

Database fields available as the x-axis in the Summary plot. Double-click a field to add it to the plot.
