Configure Node Information for Software Qualification

You can specify computer configuration information that you want included in your Software Qualification Reports.

To configure node information for software qualification:

  1. In the Configuration tree in Configuration Manager, click Nodes.
  2. In the Nodes table, select the node.
  3. In the row of the selected node, right-click, and then click Properties.
  4. On the IQ Info tab in the Node properties dialog box, enter information in the fields either by typing or selecting a value from the list that appears, and then click OK.
