Save Current Method dialog box

Use this dialog box to save a new method in the database or rename an existing one.

Method list

All existing methods in the database.

Name

Up to 30 alphanumeric characters are allowed (including spaces) for the name of a new method or an existing one. Do not use special text characters, such as asterisks (*).

Default comments

If Restricted comments are required in a full audit trail project (System Policies > New Project > Project Object (Method), you must select a previously created comment (default string). The comment that you select appears in the Project Audit Trail.

Comments

If Unrestricted comments are required in a full audit trail project (System Policies > New Project > Project Object (Method), you can type a comment, or select a previously created one, to describe the new method.  There are no text restrictions. The comment that you enter appears in the Project Audit Trail.

Method Comments

Type comments to describe the newly created method. The information that you type in this field appears in the Method Comments field of the Method Properties dialog box, the Method tab of the Project window, and on reports.
