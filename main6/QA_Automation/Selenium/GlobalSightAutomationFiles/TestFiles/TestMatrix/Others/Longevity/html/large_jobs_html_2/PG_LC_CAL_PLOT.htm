LC Calibration tab

Use this tab to define characteristics of an LC calibration curve in the Calibration Curves window (in Review), in Compare, or in a report.

Use this tab to define characteristics of an LC calibration curve in the Calibration Curves window (in Review) or in Compare.

Note: Do not include calibration plots in report methods that you will use for standards in Run and Report mode. The software updates calibration curves for standards during processing. Therefore, standard calibration plots printed during processing might include an incorrect number of points. Calibration plots for unknowns are not updated during processing. To print calibration curves, add the Report function after all standards in the sample set or at the end of the sample set.

Data characteristics

Determines what calibration curve data point information is included in the plot:

  • All Points – All points used to create the calibration curve.
  • Averaged Points – Averaged calibration data points on the Calibration Curve plot.
  • Error Range – Standard deviation for averaged data points on the calibration curve.
  • Ignored Points – Points that were specified to be ignored when calculating the curve.
