General tab (Empower Node)

Use this tab to view the Empower Node name and properties, and view or enter comments about the server.

Use this tab to view the Breeze 2 Node name and properties, and view or enter comments about the server.

Empower Node name

Name assigned to the Empower Node when it was created.

Type of Empower Node, for example, LAC/E³².

Breeze 2 Node name

Name assigned to the Breeze 2 Node when it was created.

Type of Breeze 2 Node, for example, Breeze 2.

BusLAC/E status

Status of the Waters busLAC/E card:

  • Booted OK – The busLAC/E card is present and the busLAC/E driver is loaded.
  • BusLAC/E driver not loaded – The busLAC/E driver did not load properly. Access the Windows Control Panel dialog box to start the driver. See your system administrator for assistance.
  • No BusLAC/E.hex file – The busLAC/E.hex file in the \Empower\InstrumentServer folder is missing or cannot be read. See your system administrator for assistance.
  • No BusLAC/E.hex file – The busLAC/E.hex file in the \Breeze 2\InstrumentServer folder is missing or cannot be read. See your system administrator for assistance.
  • Board not present – The busLAC/E card does not appear to be installed in your Breeze workstation.
  • Board not present – The busLAC/E card does not appear to be installed in your Empower Personal workstation.

BusLAC/E type

Type of busLAC/E card (PCI) used in the Empower Node:

  • PCI, PCI Multi-System
  • Unknown

Type of busLAC/E card (PCI) used in the Breeze 2 Node:

  • PCI, PCI Multi-System
  • Unknown

Serial card status

Status of the Waters serial card:

  • Booted OK – The serial card is present, or at least a COM3 port is configured.
  • Serial card not present – The serial card does not appear to be installed in your Breeze workstation.
  • Serial card not present – The serial card does not appear to be installed in your Empower Personal workstation.

Time Zone

The time zone for this Empower Node. Whenever data is acquired, processed, or reported, the time stamp will include the time zone based on the location of clients, workstations, LACE/Es, or servers (Empower Node).

The time zone for this Breeze 2 Node. Whenever data is acquired, processed, or reported, the time stamp will include the time zone based on the location of clients, workstations, LACE/Es, or servers (Breeze 2 Node).

Comment

Allows for entry of or displays comments about the Empower Node.

Allows for entry of or displays comments about the Breeze 2 Node.
