Define Template Variables page 2 of 2

The Define Template Variables page 2 displays sample information.

Sample information table

Displays an editable table containing samples set lines, vial numbers, and all template specified custom field columns. Typical sample information is the sample name, dilution, and sample weight. For users with a ZQ Mass Detector, the table contains the sample name and a column for each mass of interest (determined by your system administrator).
