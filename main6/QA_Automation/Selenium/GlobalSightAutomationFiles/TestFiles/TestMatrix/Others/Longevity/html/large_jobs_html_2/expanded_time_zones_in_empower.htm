Expanded time zones in Empower

You select the time zone you want to use when you log in to Empower. If the time zone you want to use is not in the main (custom) list, select a time zone from the expanded (Oracle) list of zones.

You can also edit the time zone lists to move zones from the Oracle list to the custom list or from the custom list to the Oracle list.

To edit time zones in Empower:

Configuration Manager > View> Customize Time Zone list.

In the Customize Time Zone list dialog box, select the time zone you want to move from one list to the other, and then click the appropriate arrow button to move it.
