Amplification tab

Use this tab for centroid MS spectra, not continuum data. Use this tab to specify amplifiers to apply to the plot for the specified region, including start and end region values and the multiplier.

Amplification table

Enter these values in the table (you cannot overlap ranges in the table):

  • Start m/z – 0.00 to 1.79e+308
  • End m/z – 0.00 to 1.79e+308
  • Multiplier – 0.0001 to 1.7976e+308

Each amplifier is applied to the plot for the region specified by Start m/z and End m/z. If only a portion of the specified region is in the plot, the multiplier will be applied to the region of the plot that contains data.

Once the amplification is complete, markers are placed at the start and end region values for each amplification region, and the multiplier value is listed next to the start line.
