New Sample Set Method wizard – Summary page

Use this page to view a summary of the sample set method or sample set method template you are creating.

Sample Set Method summary

Displays a summary of information about your sample set method, including:

  • The type of bracketing you selected
  • The location of your standard vials
  • The run mode
  • The maximum number of samples in a group
  • The vial starting position in the tray
  • The number of requested standard injections
  • The number of requested sample injections
  • An estimated overall run time for the sample set method

Display Component Editor

The Display Component Editor box is checked by default. This box is useful if you want to display the Component Editor when you have completed creating the sample set method. The Component Editor allows you to enter component data (name, amount, units, molecular weights, and so on) for your samples within your sample set method.

Options button

Displays the Options page, where you can add rows with special functions (such as Clear Calibrate, Equilibrate, Purge Injector, and so on) to your sample set method.

If you are creating or editing a sample set method, click Finish to complete the sample set method and load it into the Samples table. Edit information in the table as required, then click (Save) to save your new sample set method.

If you are creating or editing a sample set method template, a button labeled Next appears instead of the Finish button.

Click Next to display the Define User Prompts page, where you select the user prompts that you want to be displayed when a user runs the sample set method template.
