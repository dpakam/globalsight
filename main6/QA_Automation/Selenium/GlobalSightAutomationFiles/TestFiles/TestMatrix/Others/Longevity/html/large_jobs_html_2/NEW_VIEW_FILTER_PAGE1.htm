New View Filter wizard - Field Selection page

Use this first page to select the view filter columns that make up a view filter.

Field names

Lists field names that can be selected as columns in a view filter. On this page, field names are contained within field name categories. To display the field names within a category, double-click the category. To select a field name as a view filter column, double-click the name (or click to select the name, then click the button with the right-facing arrows).

Displayed columns

Displays field names added as columns to the view filter. Field names that appear from top to bottom in this list appear from left to right in the view filter. To remove a field name from the list, click to select the name, then click the button with the left-facing arrows.

After making your selections, click Next.
