Restore Project wizard – Restore Display page

Use this page to view Oracle messages as the project is imported. When the import finishes, click Finish to complete the restoration.

When restoring multiple projects at the same time, the software completes the process and generates a log file which is located in the Empower\Logs directory. Errors encountered while restoring up multiple projects do not interrupt the restore process. The restore process is halted only when it encounters errors that cause the project to be deleted or cause the restore to fail.
