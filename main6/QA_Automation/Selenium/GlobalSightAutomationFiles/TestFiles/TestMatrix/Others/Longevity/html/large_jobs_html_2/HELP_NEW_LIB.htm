Create a New Library dialog box

Use this dialog box to create a new spectral library in Spectrum Review.

Library list

All spectral libraries in the connected database. Libraries are global to all projects in the software.

Name

Name of the library you select from the list or type in the text box. Library names are case sensitive.

Comments

Allows up to 250 alphanumeric characters for comments. For the benefit of other software users, add comments about the type of spectra in the library.

Create button

Creates the library whose name you specified.
