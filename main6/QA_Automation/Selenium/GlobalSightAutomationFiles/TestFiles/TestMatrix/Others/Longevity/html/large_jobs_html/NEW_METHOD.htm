New Processing Method dialog box

Use this dialog box to create a new processing method, to select the processing type for the new method, and to initiate the Processing Method Wizard (when in Review).

Processing type

Type of processing method to create: LC, GC, IC, GPC, PDA, MS, CIA, or CE.

Type of processing method to create: LC or GPC.

Note: You can select the GPC processing type only if you purchase and install this option.

Integration algorithm

Select the integration algorithm you want the new processing method to use.

Choices: ApexTrack or Traditional.

The default algorithm is determined when the project is created.

Tip: If Integration Algorithm is not available, ApexTrack is not enabled and the processing method uses the Traditional algorithm.

Use Processing Method wizard (Review only)

Initiates the Processing Method Wizard, where you create a new processing method (based on a selected and required 2D chromatogram).

Include a Cross Channel Internal Standard

Tip: This selection is unavailable under the following circumstances:

  • If the processing method type is GPC.
  • If the Use Processing Method Wizard check box is cleared.

When this check box is clicked, the wizard makes the following assumptions:

  • The current 2D channel in Review is the channel with the main component peak(s), not the channel with the internal standard peak.
  • There is only one internal standard component.
