Method Set wizard – Select Instrument Method page

Use this page to select the instrument method (defined for the chromatographic system) to include in your method set. The instrument method is used for acquiring data in Run Samples, performing instrument shutdown, creating a GPCV/LS channel set from an RI concentration channel, a viscometer channel, and LS channels, or clearing a calibration during background processing when the Channel table is empty.

Use this page to select the instrument method (defined for the chromatographic system) to include in your method set. The instrument method is used for acquiring data in Run Samples.

Note: To create a method set for imported data, select (Imported Data) from the list of instrument methods. To create a method set that does not require an instrument method, select (No Acquisition).

If your method set requires an instrument method that is not listed, click Create New to open the Instrument Method Editor, where you create a new instrument method. Save the instrument method after completing it, then select File > Exit to return to the Select Instrument Method page.

After making your selection, click Next.
