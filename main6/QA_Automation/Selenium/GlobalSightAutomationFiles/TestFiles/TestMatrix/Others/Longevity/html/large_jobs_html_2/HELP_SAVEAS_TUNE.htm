Save Current Tune dialog box

Use this dialog box to save the results of the last Easy Tune run or the current Manual Tune parameter values.

Tune list

All tunes in the database.

Name

Up to 32 alphanumeric characters are allowed (including spaces) for the name of a new tune or an existing one. Do not use special text characters, such as asterisks (*).

Save

Saves the results of the last Easy Tune run or the current Manual Tune parameters in the database using the name you specified in the name text box.
