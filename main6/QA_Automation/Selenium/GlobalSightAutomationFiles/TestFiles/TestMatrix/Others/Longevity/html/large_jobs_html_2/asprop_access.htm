Access tab (Empower Node)

Access this tab by selecting an Empower Node, then right-clicking > Properties. Use this tab to control user access to the selected Empower Node (named in title bar). You use nodes primarily in an Empower Enterprise client/server configuration.

Share instruments with other network users

Specifies that other network users (determined by the access option you select; see below) can use systems connected to the selected Empower Node for data acquisition.

Allow access

Owner only

Specifies that the owner of the Empower Node and users with Administrator privileges can access the selected Empower Node.

Owner and groups

Specifies that the owner of the Empower Node, group members as specified in the Allow Access to Groups list, and administrators can access the selected Empower Node. Group access allows a user to access and modify an Empower Node.

Owner, group, and world

Specifies that all users in the database can access and modify the selected Empower Node.

Owner

Specifies the owner of the Empower Node. This field is enabled if you are the current owner of the selected Empower Node or if you have Administrator privileges.

Exception: If the Empower Node was imported as part of a server upgrade, the user name will display in the Owner column of the Empower Node table. The user type will not display in the Owner column.

Password protect access

Password required

Specifies that a user needs a password to access the Empower Node across a database.

Password

Specifies a password of up to 30 characters to password-protect the selected Empower Node.

Confirm password

Specifies reentry of the password to confirm the password choice.

Allow access to groups

Select the user groups that you want to give access to the selected Empower Node. If selected, access privileges of these user groups override access privileges of the selected world user type. If you select Owner, Group and World, groups that were not selected in the Allow Access to Groups list are assigned the World User Type Access.
