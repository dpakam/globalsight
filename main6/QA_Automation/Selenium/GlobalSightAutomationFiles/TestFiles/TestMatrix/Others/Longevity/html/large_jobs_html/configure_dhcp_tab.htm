Configure DHCP Tab

Use this tab to manually enter IP addresses and specify or change other DHCP settings.

Click the Configure DHCP button to display the Waters DHCP Server Configuration screen.

File

Provides a way to exit the Waters DHCP Server Configuration window.

Server Menu

The Server menu provides access to the following utilities:

  • Configuration Wizard
  • MAC Address Configuration
  • Errors…

Configuration Wizard

Use this wizard to set an IP address for an instrument network card.

MAC Address Configuration

The DHCP server supplies IP addresses to Waters MAC addresses. Use this configuration utility to add, remove, or restore default MAC addresses.

The DHCP server supplies IP addresses to Waters, National Instruments, Hewlett-Packard, and Agilent MAC addresses. Use this configuration utility to add, remove, or restore default MAC addresses.

Errors…

Use this wizard to view a list of errors (if any errors have been found). Use the Clear Errors button to remove the errors from the table.

Waters DHCP Server Configuration screen

The IP Address, MAC Address, Type, and Name fields are blank until IP addresses are assigned by the Waters DHCP Server or entered manually. If IP addresses are assigned, the table will be filled in with the applicable information.

To add a new IP address:

  1. Click Add to manually add a new IP address.
  2. In the Add IP Address dialog box, enter the following:
  3. IP address
  4. MAC address (the instrument’s unique hardware number)
  5. Instrument Type (if applicable)
  6. Serial Number or Unique Name (if applicable)
  1. Click OK.
  2. Power cycle the instrument so that it receives the manual IP address.

Note: If you manually add an IP address, it will never be used by another instrument and it will not be removed from the table unless the Remove button is used.

To edit:

  1. Select the IP address you want to modify and then click Edit.
  2. Change the appropriate information, then click OK.

Note: Waters instruments that have IP addresses automatically assigned through the Waters DHCP Server do not require association with an instrument type or a unique name. If you select an automatically assigned address and click Edit, only the Instrument Type and Serial Number/Unique Name fields are accessible for editing.

Remove

Use the Remove button to remove IP addresses from the table that are no longer being used. If an IP address is entered manually, it will never be removed from the table except by using the Remove button.

To remove an IP address from the table, click on the IP address and then click Remove. The IP address is removed from the table and is available for other instruments to use.

Help

Displays help or version information for the Waters DHCP server configuration.
