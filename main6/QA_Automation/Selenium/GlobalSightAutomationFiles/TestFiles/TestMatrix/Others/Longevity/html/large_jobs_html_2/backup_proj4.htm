Backup Project wizard – Start page

Note: If you are using a remote backup device in an Empower Enterprise configuration, you can instruct Empower software to copy the data files, the export file, and the .inf file to one of those network drives. Then, the system administrator can back up the files from that network drive. In this case, you do not need to click the Start Backup Software button.

Use this page to start your backup program and back up directories that appear in the list. You start your backup program from the wizard by clicking Start Backup Software (or the name that you specified for this button when you configured your backup utility in the Backup Options dialog box).
