Structures tab (Properties of structure/library plots)

Use this tab to modify how structures appear in a group. To see how they appear in the plot, click Apply while making changes.

To modify structure appearances within a group:

  1. In the Labels section, select the structure information labels to add to the structures and/or peak plot.
  2. In the Label Position section, click the location for all structure information labels, upper-left or upper right.
  3. Click OK.
