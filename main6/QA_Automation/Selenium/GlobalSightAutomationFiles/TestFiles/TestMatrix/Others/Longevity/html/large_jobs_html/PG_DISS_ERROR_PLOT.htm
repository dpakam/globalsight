Dissolution Error Plot tam

Use this tab to specify bath temperature or apparatus speed versus draw times.

Parameters

You can specify the following parameters:

  • Bath Temperature – Plots the temperature of a bath versus draw time as points on an error plot.
  • Apparatus Speed – Plots the rate of speed in rpms of the apparatus (paddle or basket) versus draw times as an error plot.

Format

Plot type(s) to use in the dissolution plot. You can include multiple plots in the same Dissolution plot report group.

Choices: Line Plot, Scatter Plot.

Limits to plot

You can plot the following limits:

  • Lower Error - Lower temperature target plus one-half degree (temperature) or the lower limit of the apparatus speed minus 4% rpm (apparatus speed).
  • Upper Error – Upper temperature target minus one-half degree (bath temperature) or the upper limit of the apparatus speed plus 4% rpm (apparatus speed).
  • Target – Target temperature (bath temperature) or the apparatus speed target (set point).

Default: All limit lines are on the plot.

Data to plot

You can plot the following data:

  • Actual Value at Transfers – Actual temperature or actual apparatus speed at the transfer time.
  • Running Average between Transfers – Running average temperature or apparatus speed at transfer time.
  • Minimum Value between Transfers – Absolute minimum temperature or minimum apparatus speed with the time it occurred.
  • Maximum Value between Transfers – Absolute maximum temperature or absolute maximum apparatus speed with the time it occurred.
  • All Data Points – All the stored temperature or apparatus speed data points for a dissolution sample set.
