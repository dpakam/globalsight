Bath A or B operation page

Use this page to enter bath operation parameters for Bath A and/or Bath B. This page appears only when you select a dissolution type in the Select Sample Set Method Type page of the Sample Set Method wizard.

If you specify Baths A and B, enter all Bath A parameters, followed by all Bath B parameters, including offsets.

Bath temperature

Specifies the temperature (°C) of the dissolution bath for a Hanson SR8-Plus bath. For all other baths, use this field for documentation.

Entries: 30.0 to 40.0.

Default: 37.0.

Apparatus speed

Specifies the apparatus (paddle or basket) speed (in revolutions per minute) for a Hanson SR8-Plus bath. For all other bath types, use this field for documentation.

Entries: 25.0 to 200.0 rpm.

Default: 75.0.

Out of USP tolerance alarm

Sounds an alarm when USP tolerances are exceeded. This option is available only when the Hanson SR8-Plus bath is configured with the 2690D/2695D.

Automatic printout

Prints a report on the Hanson SR8-Plus local printer. Enter the time interval (in minutes) for the report to print in every n min field. This option is available only when the Hanson SR8-Plus bath is configured with the 2690D/2695D.

Entries: 1.0 to 5999.0.

Typical entries: 1.00 to 30.00.
