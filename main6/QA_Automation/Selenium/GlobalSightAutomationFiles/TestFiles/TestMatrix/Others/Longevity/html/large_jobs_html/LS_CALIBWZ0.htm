LS Calibration wizard – LS Calibration Procedures page

Use this page to accomplish the following tasks:

  • Perform light scattering normalization
  • Perform light scattering calibration
  • Perform RI calibration

Three check boxes are shown, enabled, and selected based on the contents of the instrument method associated with the current channel in Review. If the instrument method parameters list only one light scattering channel, the Perform Light Scattering Normalization check box is hidden, disabled, and cleared. If the instrument method parameters specify that the concentration channel is not from an RI detector, the Perform RI Calibration check box is disabled and cleared.
