Wells tab

Use this tab to specify characteristics of occupied wells in the plate graphic representation that appears in the New Plate Type/Plate Type Properties dialog box in Configuration Manager and Run Samples.

Use this tab to specify characteristics of occupied wells in the plate graphic representation that appears in the New Plate Type/Plate Type Properties dialog box in Run Samples.

Well to edit

Occupied wells

Plate wells (in the graphical representation) on the left side of the Plate Type Properties dialog box. This identifies the well contents, such as a standard or a sample. You can configure line properties and fill properties of occupied wells. Empty wells appear disabled and are not affected by the line and fill properties of the occupied wells.

Line attributes

Line style

Line style to use for the borders of the occupied wells.

Choices: Solid, Dashed, Dotted, Dash Dot, Dash Dot Dot

Default: Solid

Thickness

Line weight of the borders of the occupied wells.

Entries: 0 to 1000.

Default: 0 (thinnest line of 1 pixel).

Note: Waters recommends that you specify a line thickness of 0 when you use line styles other than Solid. Line thicknesses vary with monitor and printer resolutions, which can result in the various line style selections being displayed as Solid. A selection of 0 ensures that the line styles appear properly.

Line color

Color of the lines used to create the outline of the occupied wells in the plate graphic. Click Line Color to access the Color dialog box. You can use all colors supported by the Windows operating system, which includes basic and custom colors.

Fill attributes

Transparent

Specifies a transparent background for the occupied wells.

Solid

Note: Clear the Transparent check box to access the Solid option button.

Solid fill pattern for the occupied wells. You can choose a solid fill color for the occupied wells by clicking the Fill Color button (see below).

Hatch

Note: Clear the Transparent check box to access the Hatch option button.

Fill pattern for the occupied wells in the plate graphic.

Choices: Horizontal, Vertical, Forward Diagonal, Back Diagonal, Cross, Diagonal Cross

Default: Forward Diagonal

Fill color

Note: To use a fill color for the occupied wells, select Solid for the Fill Attributes parameter.

Color of the occupied wells (for the Solid fill pattern). Click Fill Color to access the Color dialog box. You can use all colors supported by the Windows operating system, which includes basic and custom colors.

Point size

Type size for the well identifiers (labels) of the occupied wells. The software displays the labels inside the wells if the labels fit within the boundaries of the wells. If the well labels do not appear inside the wells, decrease the point size.
