General tab (2690/2695 or 2690D/2695D)

Use this tab to define general system, chart out signal, and sample parameters for the 2690/2695 or the 2690D/2695D separations module.

2690/2695 considerations

Values in some fields could change depending on the type of column switching valve you have connected to the 2690/2695 Module.

Note: If the 2690/2695 Module is under Empower control, the AutoStartPLUS feature will not function. This feature is a standalone function only.

Bubble detect

Specifies that the system look for air bubbles in the flow path (and respond by displaying an error message and performing a bubble recovery procedure).

Choices: Enabled or Disabled

Default: Enabled

Syringe draw rate (µL/sec)

Draw rate of the syringe to accommodate viscous samples. The rate for each selection adjusts to the size of the installed syringe.

Selections: Fast, Normal, and Slow

Default: Normal

Pre column volume (µL)

Amount of mobile phase to be delivered by the solvent management system (at the start of the gradient) before making the injection.

Default: 0.0 µL

Depth of needle (mm)

Depth of the needle tip from the bottom of the vial to accommodate for sedimented samples or nonstandard vials.

Default: 0.0 mm

Chart out

Type of analog signal transmitted from the Chart Out terminals on the rear of the 2690/2695 separations module:

  • Flow Rate
  • System Pressure
  • %A (Default), %B, %C, or %D (each is a separate selection)
  • Column Temperature
  • Sample Temperature
  • Loop Pressure
  • Primary Pressure
  • Vacuum Level

Column position

Solvent flow direction. Solvent flow can be directed to a different column or to a restrictor bypass to a mass spectrometer. The No Change selection maintains the current position. Choices:

  • No Change (default).
  • Position 1 to 3, or 1 to 6 (available only when an optional 3- or 6-column selection valve is installed).
  • Automatic (available only when an optional column regeneration valve is installed) – Switches column position, allowing for LC operations using one column while the second column is undergoing regeneration by a second solvent delivery system.

Equilibration time (min)

When a column Position change occurs at the start of a run, specifies the time in minutes that the 2690 Module runs at initial conditions before performing an injection.

Default: 0.00 min
