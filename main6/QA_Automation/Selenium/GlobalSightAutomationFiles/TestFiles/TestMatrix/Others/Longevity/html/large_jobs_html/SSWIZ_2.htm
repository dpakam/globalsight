New Sample Set Method wizard – Select Sample Set Method Type page

Note: Certain options on this page may be grayed out depending on the enabled options in Project.

Use this page to select the type of sample set method (or a sample set method template) you want to create. Make one of the following selections:

  • LC or PDA/MS
  • LC or GPC
  • GPC or GPC/V
  • GC

Note: If the Dual Tower status box is enabled next to the GC selection (indicating that the system is connected to a dual tower GC), specify the tower (Front or Back) on which the first group of injections is to occur. After you finish the wizard for the selected tower (and load its corresponding sample set method into the Samples table), you rerun the wizard to specify the remaining tower (and to append its corresponding sample set method to the Samples table).

If you are operating a GC system you can select RF Internal Standards.

GC syringe info (button)

Click the GC Syringe Info button to access the GC Syringe Size Parameters dialog box where you can specify volume parameters to check the injection volume of syringe A (front injector) and syringe B (back injector).

Define the plates to be used with this sample set method / Define Plates (button)

For systems configured for plate operation, clicking the Define Plates button displays the Define Plates for Sample Set Method dialog box, where you can:

  • Select the pre-defined plate types from which injections are to be made
  • Select the position of the plates within the autosampler
  • Activate a range of available vial/well locations (from which injections can be made) for each selected plate
  • Specify the Plate Sequencing Mode (injection order) for the vial/well locations
  • Optionally create a new plate type

Select a dissolution type

You can select one of the following dissolution types:

  • No Dissolution – Specifies that dissolution parameters not be associated with the sample set method.
  • Dissolution (Bath A) – Specifies control of dissolution samples transferred from a single bath designated Bath A.
  • Dissolution (Bath B) – Specifies control of dissolution samples transferred from a single bath designated Bath B.
  • Dissolution (Baths A and B) – Specifies control of dissolution samples transferred from two baths designated Bath A and Bath B.

Note: If a dissolution type is not selected, the 2690D Installed option below does not appear.

2690D/2695D installed

Specifies control of the following dissolution parameters:

  • Sample transfer
  • Bath temperature and speed
  • Injection
  • Processing

When you select 2690D/2695D Installed, the Empower software controls:

  • All the above parameters when the Waters Transfer Module and Hanson SR8-Plus baths are installed.
  • Only sample transfer, injection, and processing when the Waters Transfer Module (but not a Hanson SR8-Plus bath) is installed. Bath parameters are used for documentation only.

When you do not select 2690D Installed, the Empower software controls only sample injection (using a Waters autoinjector) and processing. You must load the correct samples in the correct vials. Bath and transfer parameters are used for documentation only.

6 Syringes/8 syringes (2690D only)

Specifies the number of needles/syringes (6 or 8) in the 2690D/Waters Transfer Module. After you make your selection, click Next to advance to the next wizard page. The wizard pages that appear vary according to the selections you make on this page.
