Vial tab

Use this tab to include this sample components information:

  • Vial Information – Specific information for the associated vial, such as vial name, vial number, and component amounts.
  • Amount Tables – Contents of the Component Loading table.
  • Sample History – Change history associated with the sample (performed in the Alter Sample window). Each line in the printed Sample History text represents a change made to the original sample.
  • MW Tables – Contents of the Weight Loading table (GPC/V option).
  • K Alpha Tables – Contents of the K-Alpha table (GPC/V option).
  • Moments Information – Contents of the Moments table (GPC/V option).
  • MW Tables – Contents of the Weight Loading table (GPC option).
  • K Alpha Tables – Contents of the K-Alpha table (GPC option).
  • Moments Information – Contents of the Moments table (GPC option).

By default, these items are included in the Sample Components report group.
