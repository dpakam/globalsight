Structures tab (Properties of PDA/MS report groups)

Use this tab to include or remove a chemical structure in a plot. To see how they appear in the report, click Apply while making changes.

To include a structure:

  1. Select the Draw structure in plot check box.
  2. Click the option button to select the structure to include.

Note: Only one structure can be displayed in a plot.

  1. Click OK.

To remove a structure:

  1. Clear the Draw structure in plot check box.
  2. Click OK.

Note: Some PDA and MS report groups do not accept chemical structures.
