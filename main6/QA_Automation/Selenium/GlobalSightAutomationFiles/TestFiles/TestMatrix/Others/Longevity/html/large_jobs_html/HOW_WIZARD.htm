Use this page to instruct the software how to proceed with creating or modifying a LC processing method in Review. Choices:

  • Create New Processing Method – Use the wizard to create a new processing method and start a new calibration (which activates the New Processing Method dialog box where you select the processing type for the new processing method).
  • Edit an Existing Processing Method – Use the wizard to edit the current processing method in Review, clear any calibration curves associated with that method, and start a new calibration for the modified method.
