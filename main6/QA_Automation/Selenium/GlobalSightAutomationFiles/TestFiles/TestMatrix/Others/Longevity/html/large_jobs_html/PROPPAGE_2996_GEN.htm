General tab (2996 PDA)

Use this tab to define data acquisition and instrument control parameters for the 2996 PDA Detector.

Data collection

Select either option for data collection and enter the data acquisition parameters for that type of collection:

  • 2D Data Collection – Accesses a table where you can specify the method channel lambda and method channel bandwidth for up to eight discrete channels.
  • 3D Data Collection – Collects 3D data.

Data acquisition – 2D

Method channel lambda

Click the cell in the table to enter the lambda for the channel. Range: 190.0 nm to 800.0 nm.

Default: 190 nm

Method channel bandwidth

Click the cell in the table to enter the bandwidth for the channel.

Range: 1.2 nm to 600 nm

Default: 1.2 nm

The allowable bandwidth depends on the Method Channel Lambda (wavelength) value. The value of both [Lambda – (Bandwidth/2)] and [Lambda + (Bandwidth/2)] must fall in the detector’s operating range of 189.4 to 800.6 nm. For instance, a 10 nm bandwidth is invalid for a Method Channel Lambda of 190 nm because [Lambda – (Bandwidth/2)] yields a value of 185 nm, which is below the operating range of the detector.

Sampling rate

Acquisition rate in spectra per second. For reproducible integration, acquire at least 12 to 15 data points across a peak.

Choices: 10, 5, 2, 1, 0.5, 0.25, 0.1, 0.05, and 0.017 data points/sec

Default: 1 data points/sec

Data acquisition – 3D

Start wavelength

Starting wavelength of each acquired spectrum. This value applies to all spectra collected during the run.

Range with Resolution set to 1.2: 190.0 nm to 800.0 nm

Range at all other Resolution settings: 190.0 + (Resolution/2) to 800.0 – (Resolution/2)

Default: 210.0 nm

End wavelength

Ending wavelength of each acquired spectrum. This value applies to all spectra collected during the run.

Range with Resolution set to 1.2: Start Wavelength to 800.0 nm

Range at all other Resolution settings: Start Wavelength + Resolution to 800.0 nm – Resolution/2

Default: 400.0 nm

Sampling rate

Acquisition rate in spectra per second. For reproducible integration, acquire at least 12 to 15 spectra across a peak. For peak purity you need at least 12 data points.

Choices: 10, 5, 2, 1, 0.5, 0.25, 0.1, 0.05, and 0.017 spectra/sec

Default: 1 spectra/sec

Note: To calculate peak purity, acquire at least 12 spectra across each peak.

Resolution

Number of diodes that are averaged together as a single spectral data point. To differentiate closely related spectra and obtain greater spectral resolution, use a small resolution number. Be aware, however, that a small resolution value generates more data points and therefore requires more disk space than a large resolution value. Find a resolution value just small enough to identify spectral features.

Default: 1.2 nm

Data delay:

  • Select this check box to enable a 6 second delay before acquiring data.
  • Clear this check box to begin acquiring data immediately (600 ms delay).

Instrument control

Auto exposure

Enables the 2996 PDA Detector optics to calculate the optimum exposure time needed to recharge the diodes based on the lamp energy, the lamp spectrum, and the selected wavelength range and mobile phase absorbances.

Choices: Enabled or Disabled

Default: Enabled

Note: Enable Auto Exposure for most routine analyses.

Exposure time

Length of time in milliseconds (ms) that the photodiodes are exposed to light before they are read. This parameter is dimmed if Auto Exposure is enabled.

Default: 15.00 ms

Note: Not all selections are available. The detector will attempt to use the value, but will adjust to the next, closest available number.

Digital filter:

  • Clear this check box to set the filter response to 0 for noise angle determination based on total system noise.
  • Select this check box to enable the digital filter and specify a value in the Filter Response field. You cannot specify a value of 0 in the Filter Response field. To set the filter response to 0, clear this check box.

Filter response

If the digital filter is enabled, enter the response time for filtering acquired data. The filter is an enhanced rolling average filter applied to absorbance data from the PDA detector before the data is sent to the Empower workstation. The filter reduces high-frequency noise across the entire wavelength range specified for the acquisition. High values decrease peak response.

Entries: 0.1 to 3.0

Interpolate 656 nm

Instructs the detector to ignore the signal from the photodiode at 656 nm and interpolate a value from the adjacent diodes. This prevents oversaturation of the detector at 656 nm (Balmer line for deuterium).

When you disable this parameter, the detector reports the signal from the photodiode at 656 nm. Disable this parameter only if you are working with compounds that absorb in the 656-nm range.

Choices: Enabled or Disabled

Default: Enabled

Note: If this parameter is disabled, the deuterium lamp high emission line at 656 nm may cause spectral artifacts and autoexposure errors.

Analog outputs

Analog channel 1 enable/Analog channel 2 enable

Enables Analog Channel 1 and/or Analog Channel 2. After enabling a channel, specify the operating parameters for that channel in its corresponding Chan 1 or Chan 2 tab.

Choices: Enabled or Disabled

Default: Ch 1 and Ch 2, Disabled

Event settings

Events enable

Enables the events in the Events table.

Choices: Enabled or Disabled

Default: Disabled
