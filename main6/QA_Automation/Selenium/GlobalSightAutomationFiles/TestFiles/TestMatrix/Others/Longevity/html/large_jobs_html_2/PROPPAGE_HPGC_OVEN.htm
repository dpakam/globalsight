Oven tab (5890/5890S)

Use this tab to define operating parameters for the oven in the 5890 GC System.

Note: The 5890 GC System measures run time by the length of the oven time. It will not accept data after the oven time expires.

Cryogenic enable

Opens the optional cryogenic valve.

Choices: Enabled or Disabled

Default: Disabled

Maximum temperature

Maximum oven operating temperature. Set Initial Temperature to a value no greater than the Maximum Temperature value. Enable Oven Enable for this setting to take effect.

Default: 70 °C

Initial temperature

Initial oven temperature at the start of the run. This temperature value is retained for the time defined in the Initial Time parameter. Once the final Hold Time value expires at the end of the run, the oven returns to the Initial Temperature value.

Default: 70 °C

Equilibrium time

Time required for the oven temperature to equilibrate when the oven temperature is modified. Equilibrium time begins when the actual oven temperature comes within 1 °C of the Initial Temperature setting.

Default: 0 min

Initial time

Length of time that the oven remains at the Initial Temp after the run starts.

Default: 1.00 min

Temperature ramp enable

Enables the Temperature Ramp table.

Choices: Enabled or Disabled

Default: Disabled

Temperature ramp table columns

Rate

Rate at which the oven is heated or cooled to Final Temp in °C/min.

Final temp

Temperature the oven reaches at the end of the heating or cooling ramp. Final Temp is maintained for the Hold Time duration (see below).

Hold time

Length of time that the oven maintains the Final Temp before starting the next ramp or terminating the run.

Total time

Total time as calculated by Empower software.

Total Time displayed in the first row is derived from the difference between Initial Temperature (at the start of the ramp) and Final Temp (at the end of the ramp) divided by Rate, plus Hold Time plus Initial Time.

Total Time displayed for subsequent rows is derived from the difference between Final Temp from the previous row and Final Temp for the current row divided by Rate, plus Hold Time plus Total Time from the previous row.
