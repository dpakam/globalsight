Method Set wizard - Define LS Channel Set page

Use this page to select channels to use as the LS channel set.

Note: If the instrument method you select for this method set includes parameters for an LS instrument, parameters on this page are loaded from the instrument method. Edit the parameters as required.

To define the LS channel set:

  1. Select the concentration channel from the Concentration Channel list.
  2. Edit or enter the channel information in the table and select the channels to process.
  3. If appropriate, select the Concentration Channel Connected to RI Detector check box.
  4. After creating an LS channel set, click Next.
