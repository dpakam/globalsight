Method Set wizard - Name Derived Channel Set Page or GPCV/LS Channel Set page

Use this page to confirm your selection of a PDA or MS derived channel, GPCV, LS, or GPCV-LS channel set, and to name it. Review the derived channel or GPCV, LS, or GPCV-LS channel set summary, then enter an appropriate name.

After naming the channel, click Next.
