User Types/Interface Tab

Use this tab to define the default user types and interfaces for a user and the ones he or she can access.

Requirement: You must be a System or Group Administrator or have the Alter Users privilege to modify a user’s default and allowed interfaces and user types.

To select the default and allowed user types and interfaces:

  1. Control the user's access rights and software privileges by selecting user types that the user can choose from.
  2. Select the default user type for this user based on their tasks and responsibilities. The user type that you select determines the privileges assigned to the selected user.

Requirement: You must select at least one user type for a new user, however, you can select multiple user types that the user can choose from. For example you might select both the analyst and administrator user to a scientist who has both analytical and administrative duties.

  • Administrator – Provides unlimited access to Empower software. This user type cannot be modified.
  • Analyst – Provides an intermediate level of privileges.
  • Chemist – Provides more privileges than an Analyst, but fewer than an Administrator.
  • Guest – Provides the user with read-only access to Empower software.
  • Method Validation Administrator - Provides the user with all the chemist privileges, all the method validation-related privileges and several additional privileges, including create/delete custom fields, create/delete default strings, specify report methods for signoff, signoff results 1, and signoff results 2.
  • Method Validation Chemist - Provides the user with all the Chemist privileges and the Save Validation Studies privilege.
  • Any custom user types.
  1. Select the default user interface that appears when a user logs in to Empower software:
  • System Default – The interface specified in System Policies, User Account Policies page as the global default user interface.
  • QuickStart – Streamlined version of Empower software that is easy to use.
  • Pro – Provides access to all features of Empower software.
  • OpenAccess – Designed for routine analysis environments with access to all features in Empower software.
  1. Select the interfaces that the user has access to and select the default interface. If you do not choose any interfaces, the user will log in to the interface specified in the System Policies, User Account Policies page as the global default user interface.
  2. Click Apply. Changes to the user account take effect the next time the user logs in to Empower software.
