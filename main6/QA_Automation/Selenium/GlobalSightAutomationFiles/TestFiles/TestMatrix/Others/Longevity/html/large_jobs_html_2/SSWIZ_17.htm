New Sample Set Method template – User Prompts page

Use this page to define the total number of samples you plan to run, set the starting tray or carousel position for the first sample, and enter values for custom fields to identify your samples. This page appears when you choose to create a sample set method using a previously created sample set method template.

Number of sample vials

Specifies the total number of sample vials in the tray or carousel.

Vial starting position in tray

Specifies the tray position number that contains the first sample vial for the sample set. Valid entries: 1 to 120 for the Waters 2690 Separations Module, 1 to 96 for Waters 717/717plus Autosamplers, 1 to 100 for the HP 5890 and 6890 GC autosamplers, and 1 to 999 for other non-Waters autosamplers.

Specifies the tray position number that contains the first sample vial for the sample set. Valid entry: 1 to 96 for Waters 717/717plus Autosamplers

Define the plates to be used with this sample set method / Define Plates (button)

For systems configured for plate operation, clicking the Define Plates button displays the Define Plates for Sample Set Method dialog box, where you can:

  • Select the pre-defined plate types from which injections are to be made
  • Select the position of the plates within the autosampler
  • Activate the range of available vial/well locations (from which injections are to be made) for each selected plate
  • Specify the Plate Sequencing Mode (injection order) for the vial/well locations
  • Optionally create a new plate type

Note: The plate definitions you set are not saved as part of the sample set method template. They are used only to establish a valid plate starting position (when using the template to create a sample set method).

Identify today’s samples

Specifies or modifies values for each available custom field. Enter or modify the custom field value(s) in the Value column of the table, as required.

Note: The fields on this page appear only if they were specified for user entry when the sample set method template was created.

After making your entries, click Finish to complete the sample set method and load the sample set method into the Samples table.
