Backup Project wizard – Select Destination page

Use this page to specify a destination for the backup files:

  • Backup to current project directory – Specifies that the directory where the raw data files currently reside will be the destination path for the backed up files.
  • Alternative path text box – Allows you to enter a destination for the project backup files. Clicking this option button enables a Browse button to select the folder to which you want to back up the project. If you specify a directory other than the default, enter only the root folder in the text box provided (for example: c:\backup). The software creates a new subfolder based on the project name.

When you complete this page, click Next to start the project export process.
