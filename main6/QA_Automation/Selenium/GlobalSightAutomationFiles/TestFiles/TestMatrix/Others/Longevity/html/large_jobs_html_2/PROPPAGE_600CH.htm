Channel tab (600)

Use this tab to specify the Waters 600 Controller parameter that is to be transmitted over the IEEE-488 interface to Empower software. The signal is monitored and collected as data by Empower software.

You can specify one of these parameters:

  • Temperature
  • Pressure (Default)
  • Flow
  • %A or %B or %C or %D

Channel enabled

Activates the 600 channel. Channel Enabled must be selected to monitor the selected parameter.

Channel name

Software-designated name for the channel, which appears in the Active Channels table in the Instrument Method Editor.

Description

Optional description for the channel, up to 250 alphanumeric characters. The description appears in the Active Channels table in the Instrument Method Editor and in the Real-Time Plot pane in Run Samples.

Parameter to monitor

Parameter (see above) to transmit over the IEEE-488 interface as a data channel and monitored by Empower software. During a run, you can view the selected parameter (temperature, pressure, flow, and %A, %B, %C, or %D) from Run Samples. From the Channel view in the Project window, you can view the parameter as data in the Channel column.
