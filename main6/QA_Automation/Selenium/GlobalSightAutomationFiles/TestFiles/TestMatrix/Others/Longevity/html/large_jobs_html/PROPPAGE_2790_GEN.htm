General tab (2790/2795)

Use this tab to define system, column, injector, autosampler, wash, and rapid equilibration parameters for the 2790/2795 separations module.

Note: Values in some fields could change depending on the type of column switching valve you have connected to the 2790/2795 Module.

Pre column volume (µL)

Specifies the volume of mobile phase delivered to the system after the gradient starts, but before an injection is made.

Use this parameter when you use a method from a system that has a delay volume smaller than that of the 2790/2795 separations module. You can also use this parameter to reduce the delay volume of the 2790/2795 separations module for narrow or microbore columns or for high-throughput separations, because the sample is held in the sample loop (if desired) until the gradient front reaches the sample loop.

Default: 0.0 µL

Bubble detect

Specifies that the system look for air bubbles in the flow path and respond by displaying an error message and by performing a bubble recovery procedure.

Choices: Enabled or Disabled

Default: Enabled

Injection parameters

Injection mode

One of two modes:

  • Sequential – Sample aspiration and loop fill occur at the start of each injection cycle, after completion of the previous injection.
  • Parallel – Sample aspiration and loop fill occur concurrently with other separation method functions for higher throughput.

Loop offline (min)

When Parallel Injection mode is selected, specifies the time after an injection when the next sample is preloaded into the sample loop. Preloading the sample into the loop can cause a small decrease in injection-to-injection cycle time.

The value of 0.00 ensures that the next sample will not be preloaded into the sample loop. Values greater than 0.00 and less than or equal to the method run time will cause the next sample to be preloaded into the sample loop at the given time. However, the time entered should not be less than the time needed for the current sample to leave the sample loop (based on the size of the sample loop and the current flow rate).

In Sequential mode, there is no "preload" of sample, but the loop can be brought offline to achieve a decrease in delay volume during the separation.

Pre-sample air gap (µL)

Volume of air aspirated into the holding loop (before sample aspiration) to minimize sample diffusion.

Default: 3.0 mL

Loop option

The way sample is placed in the sample loop:

  • Full loop – Does not allow you to control the injection volume in the Sample Set Editor. Full-loop injections provide the best peak area reproducibility.
  • Partial loop – Allows full control of the injection volume in the Sample Set Editor or Single Inject page in Run Samples, including the option to revert to Full-loop operation.

Attention: After saving an instrument method with a full-loop injection mode and a designated number of loop overfills, you cannot make partial-loop injections in Run Samples using that instrument method. Similarly, an instrument method with a partial-loop injection mode cannot be used for full-loop injections in Run Samples. Waters recommends naming instrument methods to designate them as either full- or partial-loop methods.

Full-loop overfill

Excess volume (expressed as a multiple of the sample loop volume) of sample drawn when you select the Full-loop option (see above), or when you override the Partial-loop option in the Sample Set Editor.

Default: 4.0x

Post-sample air gap (µL)

Volume of air aspirated into the holding loop (after sample aspiration) to minimize sample diffusion.

Default: 3.0 mL

Column parameters

Position

Solvent flow direction. Solvent flow can be directed to a different column or to a restrictor bypass to a mass spectrometer. The No Change selection maintains the current position. Choices:

  • No Change (default).
  • Position 1 to 3, or 1 to 6 (available only when an optional 3- or 6-column selection valve is installed).
  • Automatic (available only when an optional column regeneration valve is installed) – Switches column position, allowing for LC operations using one column while the second column is undergoing regeneration by a second solvent delivery system.

Equilibration time (min)

When a column Position change occurs at the start of a run, specifies the time in minutes that the 2790 Module runs at initial conditions before performing an injection.

Default: 0.00 min

Rapid equilibration parameters

These parameters allow for rapid equilibration by switching the column offline, purging the system at a high flow rate, then loading the next sample during column reequilibration.

Path

Direction for rapid equilibration flow. Flow can be redirected to a different column or to waste. No Change maintains the current position. Choices:

  • Off (default)
  • Injector Waste (liquid flows through the inject valve to waste)
  • No Change
  • Position 1 to 3, or 1 to 6 (available only when an optional 3- or 6-column selection valve is installed)

Duration (mins)

Rapid equilibration flow time for the system.

Default: 0.2 min

This parameter is disabled if the Path parameter is set to Off.

Flow (mL/min)

System flow rate for rapid equilibration.

Default: 5.0 mL/min

This parameter is disabled if the Path parameter is set to Off.

Column re-equil time (mins)

Time after system equilibration to reequilibrate the column.

Default: 0.0 min

This parameter is disabled if the Path parameter is set to Off.

Autosampler parameters

Seek well bottom

Specifies that the needle locate the well bottom at the well position defined for the first injection. This location is stored for subsequent injections from the same plate. When the carrier advances to the next plate, a new well-bottom measurement occurs.

Choices: Enabled or Disabled

Default: Disabled

Check plate height

Specifies that the needle locate the plate corner nearest the injector port (before the first injection). The plate height value is recorded and checked against an acceptable range for the plate type.

Choices: Enabled or Disabled

Default: Disabled

Autosampler wash parameters

Frequency

When an inject port and needle wash are to occur. The inject port and interior and exterior of the sample needle are washed with wash solvent, then replaced with degassed purge solvent (using the volume specified in the Purge Solvent Volume parameter, below). Choices:

  • Never
  • Every Injection (default)
  • Every Well

Wash sequence

Select one of these wash sequences:

  • Wash Purge – Performs a needle wash then a purge before the injection. This is the default.
  • Purge Wash Purge – Performs a purge, a needle wash, and another purge before the injection.

Wash cycles

Number of inject port/needle wash cycles to be performed.

Default: 1

Inj port wash time (sec)

Number of seconds to wash the inject port.

Default: 6 sec

Needle wash time (sec)

Number of seconds to wash the needle interior and exterior.

Default: 15 sec

Purge cycles

Number of times that the syringe is refreshed with degassed purge solvent (using the volume specified in Purge Solvent Volume, below).

Default: 1

Purge solvent volume (µL)

Volume needed to replace wash solvent with degassed purge solvent (when performing a needle/inject port wash), or to replace old solvent, clear bubbles, or prime the syringe with degassed purge solvent (when performing a refresh syringe). Click for details on using a Wash Needle or Refresh Syringe function in a sample set method.

Default: 600.0 µL
