Archive and Remove Records dialog box

Use this dialog box to define how you want to save the audit trail or project/sample archive records you are about to remove - you can save them as binary or text files. If you choose to save them as text files, you will not be able to restore them as offline records.

To archive and remove the system audit trail or project/sample archive records:

  1. Select the type of file to create when archiving and removing the records: Binary or Text.
  2. Select the date to archive and remove all records created before the specified date.
  3. Click OK to access the Save As dialog box.
  4. Select the folder for the archived files. Empower software supplies a file name for your archived files that includes today's date. You can change this file name, but you cannot change the file extension.
  5. Click Save. If you choose to save the archived records as text files, a message informs you that you should archive the text file information to a secure location.
  6. Click OK.
