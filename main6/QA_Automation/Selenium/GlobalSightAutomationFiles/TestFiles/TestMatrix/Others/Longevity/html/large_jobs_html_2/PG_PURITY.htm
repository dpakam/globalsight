Purity tab

Use this tab to define characteristics of a Purity plot displayed in Review or printed in a report.

Annotate peak apex/max impurity

Vertical line that indicates the time of the maximum impurity spectrum in the component Peak Purity plot. Displays multiple labeled vertical lines, depending on the number of Purity passes you specified.

Default: Disabled

Keep zoom level (Review only)

Keeps the plot zoom level the same when you view the results of different Purity passes.

Default: Disabled

Number of purity passes (Report Publisher only)

Total number of individual peak apex or maximum impurity lines to display (1 to 4 purity passes). If the Annotate Peak Apex/Max Impurity parameter is enabled (see above), the apex is labeled in each plot. Each purity pass analyzes peak spectra using the apex spectrum and maximum impurity spectra from all previous passes (for example, purity pass number 3 uses the apex spectra and maximum impurity spectra from passes 2 and 1).

Note: The number of purity passes you choose to report should correspond to the number of purity passes you specified in the associated PDA processing method, although this is not a requirement. If this number does not match, a limited number of purity passes might be displayed in your report.
