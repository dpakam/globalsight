Peak Plot tab

When using an internal standard, use this tab to select the type of plot to display on the report.

Overall peak with internal standard (1 peak/plot)

Displays one peak per plot with the internal standard peak overlaid.

Each plot will have the peak from the chromatogram with its internal standard peak, whether the internal standard is from the same channel or a different one. If you select this option and three peaks in the chromatogram have an internal standard, then three plots appear in the report.

You cannot obtain a plot for the peak if your data meets the following criteria:

  • A peak is not quantitated using an internal standard
  • The processing method used to obtain the result cannot be loaded
  • The internal standard chromatogram cannot be loaded

Overlay chromatogram with all internal standards

Displays the entire chromatogram with all associated internal standard chromatograms overlaid.

You cannot obtain a plot if your data meets the following criteria:

  • A chromatogram does not contain any peaks that use an internal standard
  • The processing method used to obtain the result cannot be loaded
  • None of the internal standard chromatograms can be loaded

Missing internal standard peak

When you are missing the internal standard peak, it may be due to one of the following reasons:

  • If the internal standard peak chromatogram cannot be loaded, the peak appears as a missing peak from the same channel as the main peak.
  • If the internal standard is missing from the channel it is from, it appears as a missing peak from that channel.
