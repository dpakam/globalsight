Load an Existing Library dialog box

Use this dialog box to open a spectral library, load its contents into the Spectral table, and display the library spectra in the Spectral plot.

You must load a library before you remove or view spectra from the library.

Library list

All spectral libraries in the connected database. Libraries are global to all projects in database.

Name

Name of the library you select from the list or type in the text box. Library names are case sensitive.

Comments

Any comments entered when the library was created.

Load button

Loads the spectral library you selected from the list into the Spectral table.
