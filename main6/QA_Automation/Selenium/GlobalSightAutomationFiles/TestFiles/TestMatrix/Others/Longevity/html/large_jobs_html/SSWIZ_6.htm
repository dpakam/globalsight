New Sample Set Method wizard – Enter Optional Information page

Use this page to enter optional standards/sample information. Even if you choose not to enter or modify any of the parameters in this page, they appear as columns in the Samples table listed with their default values. The Enter Optional Information page contains the parameters described below.

Sample weight

Displays the factor used for adjusting the calculated component amount to compensate for differences in the prepared solutions used for calibration and analysis. The default value of 1.000 provides no factoring.

During calibration, the software multiplies the calculated amount of each standard component by the Sample Weight value.

During quantitation, the software divides the amount read from the calibration curve (of the standard components) by the Sample Weight value to calculate amounts for each unknown sample.

Dilution

Displays the factor used for adjusting the calculated component amount to compensate for differences in the prepared solutions used for calibration and analysis. The default value of 1.000 provides no factoring.

During calibration, the software divides the calculated amount of each standard component by the Dilution value.

During quantitation, the software multiplies the amount read from the calibration curve (of the standard components) by the Dilution value to calculate amounts for each unknown sample.

When you finish entering information into the table, click Next to advance to the next wizard page.

Data start

Specifies the time (in minutes) after injection start time that data collection is to begin. Data Start allows you to delay the start of acquisition to control the size of the datafile, and is useful for eliminating eluent fronts from the file. Refer to the Data Start/Next Injection Delay Sequence below.

Notes:

  • If your laboratory operates under regulatory guidelines, verify with the applicable agency that you can use this parameter.
  • If you are creating a sample set method for dual tower GC operation, you must select the same value for Data Start for both the Front and Back tower to allow for simultaneous injections. To create a dual tower sample set method, you use the wizard twice, once for each tower.

Next injection delay

Specifies the delay time in minutes that is added to the end of data acquisition and before the start of setup for the next injection. Next Injection Delay allows you to wash away undesired peaks and stabilize the conditions of a gradient run before the next injection.

Note: Using the Data Start and Next Injection Delay options does not affect timing of programmed gradient changes or timed events.

If you are creating a sample set method for dual tower GC operation, you must select the same value for Next Injection Delay for both the Front and Back tower to allow for simultaneous injections. To create a dual tower sample set method, you use the wizard twice, once for each tower.

After making your selections, click Next to advance to the next wizard page.
