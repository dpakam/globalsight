Alter Sample window

From the Alter Sample window you can modify acquired sample sets, channels, or injections. You can correct errors made while loading samples, change sample identifiers, and enter component amounts after acquisition.

Altered sample and component information is applied to all categories of information. The changes you make in Alter Sample are reflected in the Injection view, Channels view, and Results view. In Results view, only subsequently created results will reflect the change made in Alter Sample.

Parts of the window

The Alter Sample window includes the following parts:

  • Menu Bar – Displays menus for performing the commands used in the Alter Sample Window.
  • Toolbar – Displays shortcut buttons that provide quick access to the primary commands used in the Alter Sample window.
  • Samples table – Lists all identifiers for each sample within a sample set.
  • Apply Table Preferences list – Lists the table preferences available in the project. Use table preferences to select the fields that appear in the samples table. Select the table preference you want from the Apply Table Preferences toolbar, or click File > Apply Table Preferences. If the Apply Table Preferences toolbar is not visible, click View > Toolbars > Apply Table Preferences.
