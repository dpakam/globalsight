New Sample Set Method template – Name Template page

Use this page to name the sample set method template and to enter any comments about the template.

What would you like to name this template?

Specifies the name of the sample set method template. Up to 30 alphanumeric characters are allowed for the name (including spaces). When you finish creating the sample set method template, the template is automatically saved to the database using this name. Sample set method templates are listed by name in the Project window Methods view tab.

Default comments

Displays a method descriptor previously created in the Default Strings dialog box in Configuration Manager. If you are saving changes to an existing method you may choose a method revision descriptor from the list.

Comments

Allows you to include information related to the creation of the sample set method template (for example, information that describes the changes you made in this version of the template). Comments can have up to 250 characters (including spaces). You can also select the default string. Comments are optional when the Full Audit Trail feature is enabled. They must be entered if Method Changes is selected when you create the project. After making your entries, click Finish to complete the sample set method template and save it to the database.
