User information dialog box

You can change information about the licensed user who installed the software, company who purchased the software licenses, and software support plan number. You may want to change this information when the designated licensed user changes positions within the company or leaves the company, when the company name changes, or if you upgrade your service plan.

Licensed user name

Name of the licensed user for the software license.

Company names

Name of the company that purchased the software license.

Software Support ID#

The software support number (if applicable) issued by Waters Corporation.
