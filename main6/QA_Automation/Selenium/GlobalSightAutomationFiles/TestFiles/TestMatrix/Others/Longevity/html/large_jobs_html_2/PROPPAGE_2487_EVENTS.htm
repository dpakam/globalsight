Events tab (W2487

Use this tab to define threshold and timed events for the Waters 2487 Dual l Absorbance Detector.

Threshold events

Specify the channel 1 and/or channel 2 threshold parameter settings in this section of the property page. The software compares the value you specify for Threshold with the detector analog output. If the analog output value exceeds the Threshold value, Switch 1 or 2 is set to the programmed state defined by the Event and Function parameters (see below). The analog output reflects the Output Mode parameter you select in the Analog A and/or Analog B tabs, and is displayed to the right of the Channel label (for example, Channel 1: Absorbance A Ch1).

Enable

Enables threshold parameters for the channel.

Choices: Enabled or Disabled

Default: Disabled

Threshold

Threshold value above which Switch 1 or Switch 2 outputs one of the following signal types: a single-pulse contact closure (whose duration is defined by the Pulse parameter), a train of repeated-pulse contact closures (whose duration is defined by the Repeat Pulse parameter), a constant Off (contact open) state, or a constant On (contact closed) state. Once the value becomes less than or equal to the threshold value, the output returns to its Off default state.

Default: 1.0000

Event

2487 Detector switch (Switch 1 or 2) to use as a contact closure if the Threshold parameter value is exceeded.

Choices: Switch 1 or Switch 2

Default: Switch 1

Function

Type of contact closure generated by Switch 1 or 2.

Choices: Off (contact open state), On (contact closed state), Pulse (defined by the Pulse parameter), or Rep Pulse (defined by the Repeat Pulse parameter)

Default: Off

Pulse periods

Specify the repeat and pulse period times in this section of the property page. You use the specified pulse periods when programming the Timed Events table and/or when setting Threshold Events.

Repeat pulse

Duration of each pulse period in a train of rectangular pulses (contact closures) to be output through Switch 1 or Switch 2 on the detector. The train of pulses can be output as a timed event (by programming the Timed Events table), or as a threshold event (by selecting this parameter in the Threshold Events section).

Default: 1.0 sec

Pulse

Duration of one pulse (contact closure) to be output through Switch 1 or Switch 2 on the detector. The pulse can be output as a timed event (by programming the Timed Events table), or as a threshold event (by selecting this parameter in the Threshold Events section).

Default: 1.0 sec

Timed events table

Note: To perform a detector auto zero, program an auto zero at the beginning of the injection in this table.

Enter values in the these table columns to program up to eight timed parameter changes during a run. Each parameter can be changed more than once.

Time

When the programmed event occurs.

Event

Event that is to occur at the specified time. The Value and Function cells in the row are activated or deactivated according to the Event selection you make. For example, selecting Lamp activates the Function cell (where you choose + or –), and deactivates the Value cell.

Choices: Wavelength, Filter, AUFS, Chart Mark, Chart Polarity, Auto Zero, Lamp, Switch 1, Switch 2, Threshold

Channel

Detector channel (A or B). If you want an event to occur for both channels simultaneously, program the events in two consecutive rows using the same Time value.

Value

Value for Wavelength, Filter, AUFS, and Event types. Range Varies according to selected Event type.

Function

Function for Chart Polarity, Lamp, Switch 1, Switch 2, and Threshold Event types. The choices vary according to the selected Event type. For example, for a Chart Polarity event, choices are + or –. For a Switch 1 event, choices are On, Off, Pulse, and Rep Pulse.

Note: Ensure the time you set for a Lamp Off function is greater than the run time. An instrument failure results when a Lamp Off function occurs during an active run.

Comments

Description of the programmed event. The description can be up to 256 characters in length.
