CIA/CE options tab

Use this tab to enter the CIA/CE instrument parameter values used to calculate the mobility and deposition time of a peak. Both parameters define the capillary properties in the Waters CIA Analyzer or Waters Quanta 4000E CE instrument.

Parameter

Description

Total length of capillary (cm)

Overall length of the capillary in the Waters CIA Analyzer or Waters Quanta 4000E CE instrument.

Entry: 20 to 200 cm.

Capillary length to detector (cm)

Capillary length from the sample entry end to the detector flow cell.

Entry: 20 to 200 cm
