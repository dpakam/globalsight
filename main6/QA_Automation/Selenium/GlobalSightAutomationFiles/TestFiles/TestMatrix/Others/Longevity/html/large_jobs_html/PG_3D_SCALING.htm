Use this tab to customize scaling for the x-axis, y-axis, and z-axis of the PDA 3D plot. Choices:

  • X-start – Time after injection (in minutes) at which to start the x-axis of the 3D plot.
  • X-end – Time after injection (in minutes) at which to stop the x-axis of the 3D plot.
  • Y-start – Low value (in detector units) for scaling the y-axis of the 3D plot.
  • Y-end – High value (in detector units) for scaling the y-axis of the 3D plot.
  • Z-start – Low value (in wavelength units for the PDA detector or in m/z for the MS detector) for scaling the z-axis of the plot.
  • Z-start – Low value (in wavelength units for the PDA detector) for scaling the z-axis of the plot.
  • Z-end – High value (in wavelength units for the PDA detector or in m/z for the MS detector) for scaling the z-axis of the plot.
  • Z-end – High value (in wavelength units for the PDA detector) for scaling the z-axis of the plot.
  • Get Values from Plot (Review Only) – Loads values for X-start, X-end, Y-start, Y-end, Z-start, and Z-end from the data in the 3D plot.
