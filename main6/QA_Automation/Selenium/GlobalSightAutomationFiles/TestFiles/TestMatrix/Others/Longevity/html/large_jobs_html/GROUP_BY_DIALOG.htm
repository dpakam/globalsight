Group By Condition dialog box

Use this dialog box to search for information based on the maximum or minimum condition of a specific view filter column. For example, to search for the latest result generated from a channel, create a Results view filter that uses the maximum value in the Result ID column.

MAX/MIN

In the upper field of this dialog box, choose whether the search is to be based on the MAX or MIN condition of the displayed view filter column. In the lower field, select how search items are to be grouped. In this example, the search items are grouped by Channel ID.
