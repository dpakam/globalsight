ISIS Login dialog box

Use this dialog box to log in to an ISIS database (if you have access to ISIS/Base software).

If you have ISIS/Base software installed locally or on the network, and it is properly registered on your computer, leave the ISIS computer text box empty. Otherwise, you can access standalone ISIS software, installed and registered on some network computer, using DCOM (distributed COM) connection. You must configure DCOM on both your local computer and network computer To configure DCOM on your computer. If DCOM is properly configured, fill in the ISIS computer name text box.
