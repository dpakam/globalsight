Saving, modifying, or deleting changes

Empower automatically tracks system-level activities using the audit trail; you cannot turn the audit trail off. The system objects that Empower tracks are identified in the System Audit Trail Policies tab of the System Policies dialog box. Your System Administrator can require you to enter a default comment, a free form comment, and/or a password every time one of these objects is created, modified, or deleted.

To enter a comment or password for an audit trailed object:

  1. Depending upon the settings either in the System Audit Trail tab of the System Policies dialog box or during project creation, you may need to do one of the following:
  • enter a default comment by selecting one from the list. These comments are default strings that are typically created by your System Administrator.
  • enter a free form comment to explain why you made a change to the audit trailed object.
  • enter your Empower password.
  1. Click OK to save your changes.
