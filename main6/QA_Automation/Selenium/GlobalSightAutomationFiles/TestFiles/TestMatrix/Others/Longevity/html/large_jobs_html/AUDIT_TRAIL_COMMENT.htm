Audit Trail Comment dialog box

If this is a Full Audit Trail project, enter comments when a system object (project, custom field, plate type, etc.) is created, renamed, modified, copied, or deleted.

Comment fields:

Field

Usage

Name

Name of the system object. This is blank when you create an object.

Action

Action performed (created, renamed, modified, copied, or deleted).

Default Comments

Select pre-defined comments to describe why the system object was changed.

Comments

Enter free-form comments to described why the system object was changed.

Note: You must either select default comments or enter free-form comments to describe why the system object was changed. Blank comments are not allowed.

OK

Applies all comments to all system objects acted upon. For example, if you delete four custom fields, click this button to save the same comment for all audit trail records.
