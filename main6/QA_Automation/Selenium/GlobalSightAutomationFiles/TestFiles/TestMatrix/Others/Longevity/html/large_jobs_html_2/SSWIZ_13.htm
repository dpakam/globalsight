New Sample Set Method wizard – Describe Standard Injections page

Use this page of the Sample Set Method Wizard to describe your narrow standard injections for a GPC or GPCV sample set. You define the number of narrow or broad standard vials, the number of injections from each vial, the injection volume, the run time, the method set to be used, and so on.

Inject narrow standards

Specifies the use of narrow standards in your GPC or GPCV sample set. Empower software uses narrow integration parameters to detect and integrate the peak(s) in the chromatogram, uses viscosity narrow integration parameters to integrate the viscometer channel data (GPCV only), incorporates data from the integration results and sample information into a calibration curve, and creates a viscosity law plot for the calibration curve (GPCV only). When you enable this check box, the remaining fields and controls in this page are activated.

Standard information section

Use this section to define information about your standards injections.

Number of narrow standards

Specifies the number of narrow standard vials for the sample set. Valid entries: 1 to 99.

Number of broad standards

Specifies the number of broad standard vials for the sample set. Valid entries: 1 to 99.

Number of injections per vial

Specifies the number of injections from each standard vial. Valid entries: 1 to 99.

Injection volume

Specifies the volume in microliters of the injected sample, valid entries are 0.0 to 29999.9. Note the following considerations:

  • For Waters autosamplers, small injector loops may omit the maximum volume. Ensure hardware configuration is compatible with your desired injector size.
  • For Agilent 1100 autosamplers, the maximum injection volume is the seat capillary volume, plus the syringe size. Ensure that the hardware configuration is specified correctly in your method.

Specifies the volume in microliters of the injected sample, valid entries are 0.0 to 29999.9.

Tip: For Waters autosamplers, small injector loops may omit the maximum volume. Ensure hardware configuration is compatible with your desired injector size.

Run time

Specifies the length of time in minutes to collect data for the standard.

Valid entries: 0.01 to 655.90 minutes

k

Specifies the optional Mark-Houwink constant, k, where Log k is the intercept of the viscosity law plot. Enter a value in this field if you specified Universal Calibration as a calibration technique when you created your processing method. Valid entries are 0.000000000 to 2.000000000. Leave this field set to the default of 1.000000000 if you specified Relative Calibration as a calibration technique in your processing method.

Tip: If you decide to run multiple narrow standards in a vial, make sure that you enter the same k value for each standard in a sample vial.

Alpha

Specifies the optional Mark-Houwink constant, alpha, which is the slope of the viscosity law plot. Enter a value in this field if you specified Universal Calibration as a calibration technique when you created your processing method. Valid entries are -1.000000 to 2.000000. Leave this field set to the default of 0.000000 if you specified Relative Calibration as a calibration technique in your processing method.

Options button

Displays the Enter Optional Information page, where you can specify a Sample Weight factor, a Dilution factor, a Data Start time and/or a Next Injection Delay time.

Method set section

Use this section to define or create the method set used to acquire, process, and/or report data for your standards.

Select

Specifies the method set used to acquire, process, and/or report data for the standards. Select the appropriate method set from the drop-down list. If you are using the Pro interface, click Create New if you need to create a new method set.

Specifies the method set used to acquire, process, and/or report data for the standards. Select the appropriate method set from the drop-down list.

Create New button (Pro interface only)

Displays the first page of the New Method Set Wizard, where Empower software interactively leads you through the steps to create a new method set.

After making your selections, click Next to advance to the next wizard page.
