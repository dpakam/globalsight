Inlet tab (6890)

Use this tab to specify gas control parameters for the front or back inlet of the 6890 GC system.

Inlet type

Type of front inlet used in the 6890 GC system. Choices:

  • PP EPC (Purged Packed, EPC)
  • COC EPC (Cool on-Column, EPC)
  • S/SL EPC (Split/Splitless, EPC)
  • PTV (Programmed Temperature Vaporization, EPC)
  • PP (Purged Packed, non-EPC)
  • COC (Cool on-Column, non-EPC)
  • S/SL (Split/Splitless, non-EPC)
  • Unknown (used for other inlet types)
  • None
  • Volatiles Interface
  • CIS4 (Gerstel PTV inlet with auxiliary temperature control)

Default: None. (You can also set Inlet type in the General tab.)

Note: For the PTV, you can set only the front inlet.

Mode

Inlet mode for CIS4, PTV, S/SL EPC, and Volatiles Interface types. The modes vary according to your Inlet Type selection. Choices:

  • Split – Typical for major component analysis; available choice for CIS4, PTV, S/SL EPC, and Volatiles Interface types.
  • Splitless – Typical for trace analysis; available choice for CIS4, PTV, S/SL EPC, and Volatiles Interface types.
  • Pulsed Split – Typical for major component analysis, but allows for large injection volumes; available choice for CIS4, PTV, and S/SL EPC inlet types.
  • Pulsed Splitless – Typical for trace analysis, but allows for large injection volumes; available choice for CIS4, PTV, and S/SL EPC inlet types.
  • Solvent Vent – Typical for large or multiple injections of low concentration sample where evaporated solvent is swept out of the vent; available choice for CIS4 and PTV inlet types.

Temperature parameters

Enable

Enables the inlet zone temperature control. Enable this check box to set an Initial Temp or use the temperature ramp table.

Choices: Enabled or Disabled

Default: Disabled

Initial temp

Inlet zone temperature control for the injector. When the inlet temperature ramp table is disabled, the Initial Temp value is retained for the length of run. When the inlet temperature ramp table is enabled, the Initial Temp value is retained for the length of time defined in the Initial Time parameter. Once the Total Time value expires (in the inlet temperature ramp table) at the end of the run, the inlet returns to the Initial Temp value.

Default: 50 °C

Attention: To avoid a possibly unstable inlet temperature, set the inlet Initial Temp to a value at least 6 °C higher or lower than the Oven Initial Temperature.

Initial time

Length of time (from the start of the run) during which the initial ramped temperature value is to be maintained. This parameter is active only when Enable Ramps (see below) is selected. When you enter Rate, Final (temperature), and Hold Time values for row 1 in the inlet temperature table, temperature is maintained for the length of time you set for Initial Time, then ramps using the Rate and Final values you selected for row 1 in the table. After the Total Time value expires at the end of the run, the inlet returns to the Initial Temp value.

Default: 0.00 min

Track oven

Specifies that the inlet automatically maintain a temperature 3 °C above the oven temperature throughout the run. This check box is active only when a COC or COC EPC inlet type is selected (and the Enable check box is selected). Enable Ramps is disabled when Track Oven is enabled.

Choices: Enabled or Disabled

Default: Disabled

Enable ramps

Enables the inlet temperature ramp table. This check box is active only when a COC, COC EPC, or PTV inlet type is selected. Track Oven is disabled when Enable Ramps is selected.

Choices: Enabled or Disabled

Default: Disabled

Inlet temperature ramp table

Allows you to program up to three inlet temperature ramps during a run. The table is active only when you select Enable Ramps. In each row of the table, define:

  • Rate – Rate at which the inlet is to be heated or cooled to Final Temp (in °C/min).

Default: 0.0 °C/min

  • Final (temperature) – Temperature that the inlet is to reach at the end of the heating or cooling ramp. Final temperature is maintained for the Hold Time duration (see below).
  • Hold Time – Time in minutes that the inlet temperature is to be held at the current Final temperature before starting the next ramp or terminating the run.
  • Total Time – Total time as calculated by Empower software. Total Time displayed in the first row is derived from the difference between the Initial Temp (at the start of the ramp) and the Final temperature (at the end of the ramp) divided by the Rate, plus Hold Time plus the Initial Time. Total Time displayed for subsequent rows is derived from the difference between the Final temperature from the previous row and the Final temperature for the current row divided by the Rate, plus Hold Time, plus the Total Time from the previous row.

When the run begins, inlet temperature is maintained at the Initial Temp for the length of time you entered for Initial Time. At the end of that time, inlet temperature ramps to the Final (temperature) value at the Rate you set in row 1 in the Inlet Temperature Ramp table. Final temperature is maintained for the length of time you entered for Hold Time. The ramp process continues for each row in the table.

Note: If an inlet temperature program ends before an oven temperature program ends, the inlet temperature remains at the Final value until the oven program and run end.

Purge parameters

Use these parameters to open the purge valve at a specified time after injection and to sweep out the split vent and any vapors that remain in the liner.

On time

Time after injection when the purge begins.

Default: 0.00 min

Note: If you are using Gas Saver, set Gas Saver On Time to a value greater than the Purge On Time value.

Flow

Purge flow rate in mL/min.

Default: 0.0 mL/min

Note: Configure your column before setting this parameter. See the Agilent 6890 Series Gas Chromatograph Operating Manual.

Sampling end time

Time in minutes after injection that sampling is to end (for Volatiles Interface types in Split mode).

Default: Blank

Note: This value must be less than the Purge On Time and Gas Saver On Time.

Gas saver parameters

Enable

Enables the gas saver feature for the inlet. Use this feature to reduce the flow of carrier gas into the inlet and out of the split vent after an injection. Gas saver does not alter column head pressure or flow through the column.

Default: Disabled

Note: See the HP 6890 Series Gas Chromatograph Operating Manual for details.

On time

Time when the gas saver flow is to begin. In Split mode, set the time after injection time; in Pulsed Split mode, set the time after pulse time; in Splitless or Pulsed Splitless mode, set the time after Purge On Time.

Default: 0.00 min

Flow

Gas saver flow rate.

Default: 15.0 mL/min

Note: Set Flow to a value at least 15 mL/min greater than the maximum column flow.

Split/splitless parameters

Use these parameters to further define your inlet when using S/SL EPC, PTV, CIS4, and Volatiles Interface types in Split or Pulsed Split mode. See the Agilent 6890 Series Gas Chromatograph Operating Manual for more information.

Note: Specify either Split Ratio or Split Flow. When you specify one parameter, Empower software calculates the second parameter.

Split ratio

Ratio of split flow to column flow when Split or Pulsed Split operating mode is selected.

Default: 0.1

When you enter a value for Split Flow, Split Ratio is calculated according to this formula:

Split Ratio = Split Flow/Column Flow

Split flow

Flow from the split/purge vent (when Split or Pulsed Split operating mode is selected).

Default: 0.1

When you enter a value for Split Ratio, Split Flow is calculated according to this formula:

Split Flow = Split Ratio × Column Flow

Notes:

  • This field is disabled if the column mode in the Column tab is set to Constant Pressure or Ramped Pressure.
  • Empower 4.0 does not use column length, diameter, film thickness, and gas type in this equation.

Pulse time

Time during which the Pulse Pressure is to be maintained. When this time elapses, pressure returns to the Initial Pressure/Flow value set in the appropriate Column tab. Default: 0.00.

Pulse pressure

Desired inlet pressure at the beginning of the run. Pressure rises to this value at the start of the run and remains constant until the Pulse Time elapses, at which time pressure returns to the Initial Pressure/Flow value specified in the appropriate Column tab.

Default: 0.0

Vent parameters

Use these parameters to specify vent parameters when a PTV inlet type is set to Solvent Vent mode. See the HP 6890 Series Gas Chromatograph Operating Manual, Volume 2, Inlets for details.

End time

Time, after the start of the run, when solvent venting ends.

Default: 0.00 min

Flow

Flow of carrier gas out of the split vent during venting. High flows sweep the liner more quickly and speed solvent elimination. For most columns, a 100 mL/min flow rate eliminates solvent at an acceptable rate and puts minimal material on the column.

Default: 0.1 mL/min

Pressure

Inlet pressure during venting. Decreasing pressure increases the speed of solvent elimination, while decreasing the amount of carrier gas and solvent vapor that enters the column during the venting period.

Default: 0.0 psi
