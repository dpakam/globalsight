Delete an Existing Library dialog box

Use this dialog box to delete a spectral library. When you delete a library in software, you remove it from the database to which your Empower workstation is connected.

Attention: Libraries are global to all projects in the software. Before deleting a library, ensure that no other users need the library.

Library list

All spectral libraries in the connected database. Libraries are global to all projects in the database.

Name

Name of the library you select from the list or type in the text box. Library names are case sensitive.

Comments

Any comments entered when the library was created.

Delete

Displays a message box prompting you to delete the selected library. Clicking Yes deletes the selected library from the database to which your workstation is connected. If the library contains spectra, an additional message box appears. Clicking Yes deletes the library.
