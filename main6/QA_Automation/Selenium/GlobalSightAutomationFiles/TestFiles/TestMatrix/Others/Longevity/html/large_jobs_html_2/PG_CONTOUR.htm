Contour tab

Use this tab to define characteristics of the PDA or MS Contour plot displayed in Review or printed in a report.

Use this tab to define characteristics of the PDA displayed in Review.

Levels

Number

Number of absorbance levels (PDA data) or intensity levels (MS data) into which the PDA or MS data set is divided for contour display.

Number of absorbance levels (PDA data) into which the PDA data set is divided for contour display.

Range for Linear distribution: 2 to 200

Range for Exponential distribution: 2 to 15

Default: Linear Distribution, 10 levels

Linear distribution/exponential distribution

Algorithm for assigning contours.

Choices: Linear, Exponential.

Default: Linear.

Tip: To accentuate low-level features, use an exponential contour distribution and increase the number of contours.

Plot characteristics (Report Publisher only)

Draw contour/3D legend

Display of the plot legend to the right of the Contour plot.

Graphical legend/textual legend

How the legend is to appear in the Contour plot.

Choices: Graphical, Textual.

Default: Graphical.

Peak separation (MS centroid data only)

For MS data, this value is the size of the bins of the mass axis. A bin is a scan wide and a mass high. Click the up arrow to increase the size of the bin. Click the down arrow to increase the size of the bin.

Contours

Low contour

Absorbance value (PDA data) or intensity value (MS data) of the lowest contour line. If you altered the contour setting, select the AutoScale check box to reset the contour bands to the default settings.

Absorbance value (PDA data) of the lowest contour line. If you altered the contour setting, select the AutoScale check box to reset the contour bands to the default settings.

Note: The Low Contour text box is disabled when the AutoScale check box is selected.

AutoScale

Scales the Contour plot to the lowest contour line in the plot. When you enable this check box, you cannot manually enter a low contour setting.

Default: Enabled.

High contour

High contour

Absorbance value (PDA data) or intensity value (MS data) of the highest contour line. If you altered the contour setting, select the AutoScale check box to reset the contour bands to the default settings.

Absorbance value (PDA data) of the highest contour line. If you altered the contour setting, select the AutoScale check box to reset the contour bands to the default settings.

Note: The High Contour text box is disabled when the AutoScale check box is selected.

AutoScale

Scales the Contour plot to the highest contour line in the plot. When you enable this check box, you cannot manually enter a high contour setting.

Default: Enabled.
