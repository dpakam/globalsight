Flow tab (2690/2695, 2690D/2695D, and 2790/2795)

Use this tab to define operating parameters that control system pressure limits, operating mode, flow acceleration, flow rate, and solvent compositions for isocratic mode. For gradient mode, program additional parameters in the Gradient table.

High limit

High-pressure limit for the system. If the system pressure exceeds this limit within 5 seconds, the separations module displays a pressure alert message, Empower displays an instrument failure message, and flow is stopped.

Default: 4000 psi

Low limit

Low-pressure limit for the system. During an injection only, if system pressure falls below this limit, the separations module displays a pressure alert message within 5 to 10 seconds, Empower displays an instrument failure message, and flow is stopped.

Default: 0 psi

Attention: The system will not flag a low-pressure fault during Initial Conditions in Direct Control mode or during an Equilibration step. However, if an injection occurs after an Equilibration step and the pressure falls below the low-pressure limit, the system will detect the fault immediately.

Pump mode

System operating modes:

  • Isocratic (displays Total Flow and %A–%D text boxes)
  • Gradient (displays the Gradient table)

Accelerate to 10.0 mL/Min In

Time in minutes for the solvent delivery system to reach the maximum system flow rate of 10 mL/min. This limits the rate of change of the flow rate and protects the column from potentially damaging changes in pressure.

Default: 2.00 min

Note: The Accelerate to 10 mL/Min selection overrides any time-based changes to flow rate defined in the Gradient table.

Flow

Percentage of solvent flow from each reservoir. The total of all solvents in each gradient segment must equal 100%. The system calculates individual flow rates when you enter the percentage value for each solvent. Empower automatically fills in the solvent column in the next column to the right so that the total percentage of the current row yields 100%. For examples of how the autofill feature works, see the related topic link.
