Open Existing View Filter dialog box

Use this dialog box to select a view filter to use in the View Filter Editor or to delete a view filter from the list.

Names

Existing view filters in the database.

Name

Name of the view filter selected from the View Filter list or an existing view filter name.

Public and private

Type of view filter. A private filter can be edited only by the user who created it. A public filter can be edited by other users.
