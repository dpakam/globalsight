Conformation Plot tab

Use this tab to define characteristics of the Conformation plot displayed in Review or in Compare, or printed in a report.

Axis data

Specifies data to include in axes of the Conformation plot.

X-Axis data:

  • Slice Log MW – Log MW data displayed along the x-axis of the Distribution plot. Default selection for X-Axis Data.

Note: When the Fitting Model field in the Slicing tab is set to Calibration, there are three additional x-axis choices in the Conformation plot:

  • Slice EV – Elution volume (in mL) of each slice adjusted for the flow rate entered in the Calibration tab of the GPC Processing Method window. Elution Volume equals Adjusted Retention Time if you did not enter a flow rate (assumes a flow rate of 1.0 mL/min).
  • Slice Adj RT – Retention time of each slice adjusted for variations in retention time as observed relative to a reference peak.
  • Slice RT – Retention time of each slice.

Left-Axis Data, Right-Axis Data

Data displayed along the left and/or right axes of the Conformation plot.

Left-Axis and Right-Axis Data:

Data type

Usage

Blank

No data is displayed.

Slice MW

Plots the molecular weight of the polymer at each slice.

Slice Log MW

Plots the log (base 10) of the molecular weight of each slice.

Slice RT

Plots the retention time of each slice.

Slice Adj RT

Plots the retention time of each slice adjusted for variations in retention time as observed relative to a reference peak.

Slice EV

Plots the elution volume (in mL) of each slice adjusted for the flow rate you entered in the Calibration tab of the Processing Method window. Elution Volume equals Adjusted Retention Time if you did not enter a flow rate (assumes a flow rate of 1.0 mL/min).

Slice Area

Plots the area of each slice, which is the height of the integrated peak at the reported retention time multiplied by the duration of the slice.

Slice Height

Plots the height of each slice of the integrated peak at the reported retention time.

dwt/d(log M)

Plots the weight-normalized area of the slice. The absolute value of the area of the slice is divided by the slope of the calibration curve at the adjusted retention time or elution volume of the slice. The ratios of area to slope for all slices are summed, then each ratio is divided by the sum. Default selection for Plot5 Data.

Cumulative %

Plots the amount (as percent area) of the polymer accumulated up to and including the slice.

Slice Fitted [h] (Universal Calibration)

Plots the log (base 10) of the intrinsic viscosity from the fitted viscosity curve (if a GPCV peak). It is calculated from the molecular weight, K, and alpha values when this is a GPC peak.

Plots the log (base 10) of the intrinsic viscosity from the fitted viscosity curve (if a GPCV peak). It is calculated from the molecular weight, K, and alpha values when this is a GPC peak.

Slice Fitted Rg (GPC/V-LS Option)

Plots the log fitted radius of gyration calculated for the slice by fitting the LS data according to the Slicing tab field Rg Fit parameter. Default selection for Left X-Axis Data.

Slice Std HV (Universal Calibration)

Plots the log (base 10) of the hydrodynamic volume at each slice. Hydrodynamic volume is the product of intrinsic viscosity [h] and molecular weight.

Number Density

Plots the normalized distribution based on the number of molecules in a slice.

Slice g' (GPCV or GPC/V-LS Option)

Plots the branching index at each slice.

Slice g' LCB (GPCV or GPC/V-LS Option)

Plots the long-chain branching index at each slice.

Slice g' SCB (GPCV or GPC/V-LS Option)

Plots the short-chain branching index at each slice.

Slice g (GPCV or GPC/V-LS Option)

Plots the Stockmayer-Fixman branching parameter at each slice (assuming the Random intrinsic viscosity fit parameter was specified in the Slicing tab of the Processing Method window). Default selection for Right-Axis Data.

Slice g LCB (GPCV or GPC/V-LS Option)

Plots the Stockmayer-Fixman long-chain branching parameter at each slice. When the Branching Model is set to Star, this field is blank. When the Fitting Model is set to Calibration, Slice g LCB is calculated only when you enter a value for alpha LCB(LS) or Linear % of Data Region.

Slice g SCB (GPCV or GPC/V-LS Option)

Plots the Stockmayer-Fixman short-chain branching parameter at each slice.

Slice d (GPCV or GPC/V-LS Option)

Plots the branching density, or the number of branches per dalton, at each slice (assuming the Random or Star branching parameter was specified in the Slicing tab of the Processing Method window).

Slice f (GPCV or GPC/V-LS Option)

Plots the number of branches per molecule in a star-branched polymer (assuming the Star branching parameter was specified in the Slicing tab of the Processing Method window).

Slice Conc (GPCV or GPC/V-LS Option)

Plots the concentration of the slice based on the area of the slice and the concentration calculated for the peak. Used in viscometry calculations.

Slice K (GPCV Random Processing or GPC/V-LS Option)

Plots the Mark-Houwink constant, K, where Log K represents the instantaneous intercept of the Viscosity Law plot.

Slice alpha (GPCV Random Processing or GPC/V-LS Option)

Plots the Mark-Houwink constant, alpha, which represents the instantaneous slope of the Viscosity Law plot.

Slice Observed [h] (GPCV or GPC/V-LS Option)

Plots the log of the observed intrinsic viscosity of the polymer that elutes at each slice.

Slice Linear [h] (GPCV or GPC/V-LS Option)

Plots the log linear intrinsic viscosity (calculated from the molecular weight, K, and alpha) and used with the fitted (branched) viscosity to calculate g' for the slice.

Slice Unbranched [h] (GPCV or GPC/V-LS Option)

Plots the log intrinsic viscosity (calculated from the molecular weight, Linear K, and Linear alpha) and used with Linear [h] to calculate g' SCB for the slice.

Slice K(LS) (GPC/V-LS Option)

Plots Slice K(LS), where log Slice K(LS) is the intercept of the line tangent to the curve that produced the Slice Fitted Rg values.

Slice alpha(LS) (GPC/V-LS Option)

Plots the slope of the line tangent to the curve that produced the Slice Fitted Rg values.

Slice Observed MW (GPC/V-LS Option)

Plots the molecular weight calculated for the slice by extrapolating the light scattering data to the zero-angle.

Slice Observed Rg (GPC/V-LS Option)

Plots the log observed radius of gyration calculated for the slice by extrapolating the LS data to the zero-angle using the Rg Fit specified in the Slicing tab in the Processing Method window. Use this for comparison with the theoretical linear and fitted curve. Default selection for Plot3 Data.

Slice Linear Rg (GPC/V-LS Option)

Plots the log linear radius of gyration calculated from the Slice Fitted Rg value by fitting a line to the low molecular weight end of the Fitted Rg when the Fitting Model is set to Structural and the Rg is set to one of the higher-order polynomial fits or either of the two step fits. Default selection for Plot4 Data.

Slice Unbranched Rg (GPC/V-LS Option)

Plots the log unbranched radius of gyration of the slice calculated only if you entered unbranched values of K(LS) and alpha(LS).

Slice lambda (GPCV or GPC/V-LS Option)

Plots lambda, the probability for long-chain branching factor, for the slice.

Slice epsilon (GPC/V-LS Option)

Plots the overall shape factor, epsilon, calculated for each slice.

Slice Specific Viscosity

Plots the height of the viscometer peak at the slice retention time.

Relative Viscosity Signal

Plots the signal, in mV, from the derived viscosity channel.

Slice R0 (GPC/V-LS Option)

Plots the Rayleigh ratio at scattering angle 0.

Slice R1, R2, R3 (GPC/V-LS Option)

Plots the slice height of the first to third LS peaks.

Slice Observed HV (GPCV or GPC/V-LS Option)

Plots the log observed hydrodynamic volume.

Slice Fitted HV (GPC/V-LS Option)

Plots the log fitted hydrodynamic volume calculated for the peak according to the Fitting Model, Intrinsic Viscosity Fit, and Rg Fit parameters.

Slice P (GPCV or GPC/V-LS Option)

Plots the local polydispersity of the slice.

Plot3 Data, Plot4, Plot5 Data

Displays a third, fourth, and fifth data plot in the Conformation plot. See above for a complete listing of plot choices.

Additional plots

Displays up to three additional data plots in the Viscosity Law plot (Plot6 to Plot8). See the X-Axis Data parameter for a complete listing of plot choices. If desired, you can select Blank as a data plot choice (no data is displayed).

Plot data

Specifies data to include and plotting direction for the log molecular weight on the x-axis:

  • High to Low – Molecular weights decrease from left to right (default).
  • Low to High – Molecular weights increase from left to right.
  • MW Averages – Molecular weight average labels (Mn, Mw, Mv, Mz, and Mz+1) on the plot.
  • MP - Peak molecular weight (at the peak apex) labels on the Distribution plot.
  • MW Markers – Mol Wt Marker 1 and Mol Wt Marker 2, which indicate molecular weights you entered in the Slicing tab of the Processing Method window.
