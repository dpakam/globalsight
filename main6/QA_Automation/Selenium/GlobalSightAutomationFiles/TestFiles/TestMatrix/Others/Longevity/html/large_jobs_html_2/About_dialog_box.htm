About dialog box

This dialog box contains important information about the software version and release date:

  • The version of Breeze software and the copyright dates of the released software.
  • The version of Empower software and the copyright dates of the released software.
  • The name of the licensed user who is logged into the software.
  • The name of the company who purchased and installed the software.
  • If purchased, the software support ID.  
  • The software build number.
  • The date the software was initially released.
  • The amount of physical memory used and the amount of paging file space (virtual memory) that is available for use.
  • The version of the Oracle database and the database ID.
  • All the installed service packs and feature releases.
  • All the installed Empower options.

Tip: To see the expiration date associated with Qualification licenses, you must open the Verify Files report or click Show Licenses in the Waters Licensing wizard.
