Table Font Preferences Tab

Use this tab to define the format of text in Empower tables on screen and in print. You can select the font, font size, color, and font features. Modifications made here affect all tables in Empower software. However, you can override these table font preferences in individual Empower applications (for example, Project window, Review, or Report Publisher).

Font to Edit

Displays the individual table components whose font(s) can be modified.

To format the font of a table component, click the option button of the component you want to format. Use the parameters in the Characteristics section of the tab to define the font family, size, style, and color (see below).

You must individually edit each table component whose font(s) can be modified. You can, however, format each component differently from each other.

Characteristics

Defines the format of the font in the selected table component. Available choices: Bold, Italic, Underline, Strikeout, Name (font), Size (point), and Text Color. Available font families and sizes depend on which fonts are installed in your computer.

To change the table component color, click Text Color to access the Color dialog box. You can use all colors supported by the Windows operating system, which includes basic and custom colors.

Default: Black

Sample Text

Displays the text as it will appear with selected font characteristics.
