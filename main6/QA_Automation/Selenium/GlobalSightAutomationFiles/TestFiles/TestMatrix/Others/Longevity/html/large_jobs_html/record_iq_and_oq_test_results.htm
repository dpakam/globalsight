Record IQ and OQ Test Results

After you run qualification tests on workstation computers, work group computers, client computers, or LAC/E32 modules, you can record the test results electronically for each computer (Empower node). The information is saved in the database and available for you to view in the Configuration Manager Node view, and available to generate within a report using one of the Software Qualification report methods in Report Publisher for subsequent electronic sign-off.

To record IQ, OQ, or OQ MVM test results:

  1. In the Empower Configuration tree, in Configuration Manager, click Node.
  2. In the Node table, select the node for which to record qualification test results, right-click, and then click Properties.
  3. On the IQ/OQ tab in the Node Properties dialog box, select the status for the IQ, OQ, OQ MVM, or MP (maintenance procedure) that you performed.
  4. Enter the date that you performed the qualification test, and any comments that you want to store with the test results.
  5. Click OK.
