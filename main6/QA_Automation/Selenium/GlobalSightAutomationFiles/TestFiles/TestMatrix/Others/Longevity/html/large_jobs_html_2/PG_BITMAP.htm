Graphic tab

Use this tab to select a bitmap (.bmp) or enhanced metafile (.emf) graphic to include in a report.

Preview

Screen representation of the selected bitmap or enhanced metafile. Only one graphic can appear at a time in the Preview section of the Graphic tab.

Load graphic

Activates the Open dialog box, where you search folders and files for the desired (.bmp) or enhanced metafile (.emf) graphic. Double-click folders and subfolders until you open the subfolder that contains the desired graphic. When you find the graphic file, select it, then click Open. The selected bitmap or enhanced metafile appears in the Preview section of the Graphic tab.

Graphic information

Displays information about the graphic:

  • Graphic Size – Height and width (in pixels).
  • Number of Colors – Number of colors (16, 256, 32K or 64K, and so on).
  • File Format – Type (bitmap or enhanced metafile).

Note: Pay attention to the size of the graphic you want to include in your report method. Ensure that the bitmap is less than 1 MB. Each report method has a size limitation of 1 MB.

Paste

Pastes a copied bitmap from the clipboard into the Preview section of the Graphic tab.
