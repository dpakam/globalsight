New Sample Set Method wizard – Initial page

Use this page to create a sample set method starting with or without a previously created sample set method template. The first time you create a sample set method, Untitled will appear at the top of the page. When you edit a sample set method, the name of the method will appear in the title bar of this page.

  • If you launched the New Sample Set Method Wizard from Run Samples, the Describe Runtime Options page is part of the wizard.
  • If you launched the New Sample Set Method Wizard from Project, the Describe Runtime Options page is not part of the wizard.

Create Sample Set Method using wizard

Specifies that you manually create a new sample set method by defining sample and standard sequencing, specifying a method set, identifying standards and samples, describing how samples are injected, and so on.

Use an existing Sample Set Method template

Specifies that you create a new sample set method by using an existing sample set method template.

Use the Sample Set Method editor instead of wizard

Select this option if you want to manually create a sample set method by using the Sample Set Method Editor.

Note: If you choose to skip the wizard, you will not be able to perform dissolution testing. Dissolution calculations are derived using a sample set method.
