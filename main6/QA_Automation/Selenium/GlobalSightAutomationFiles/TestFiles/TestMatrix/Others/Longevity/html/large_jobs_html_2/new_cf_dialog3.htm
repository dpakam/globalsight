New Custom Field wizard – Formula Entry page

This page appears only if you selected Calculated as the data source in the Source Selection page. Use this page with any field type to create a custom field that performs a calculation automatically.

Field =

Workspace in which you build a custom formula from the Fields boxes and mathematical functions available in the Operations boxes, respectively. You can type entries or double-click a Field or Operation.

Rule: If you type a Field or Operative entry, it must match the case and spelling of those in the list.

Fields

Lists available fields supported by the various field types. The contents of the fields list depend on the Field Type specified in the Data and Type Selection page, as well as the selected Empower software options (for example, System Suitability). For each field type, use only those fields that are supported by the field type, as indicated in this table.

Field formula usage:

Field Type

Supported Fields in Formula

Sample

Sample identifiers (sample weight, sample set custom fields, dilution, transfer time, vessel, and vial ID).

Result

Sample set identifier, sample identifiers, sample set custom fields, system field, injection fields, chromatogram fields, previously created custom database results, result fields, result set fields, component level, and peak fields.

Peak

Sample set identifier, sample set custom fields, component level, sample fields, system field, injection fields, chromatogram fields, previously created custom database results, result fields, all peak fields, and distribution fields when a summary type function is used.

Distribution

Sample set identifier, sample identifiers, component level, system field, injection fields, chromatogram fields, previously created custom database results, result fields, result set fields, GPC/V peak fields, and distribution fields.

Operations

Lists reserved functions (and operators) supported by the Formula Entry dialog box. Each function must be followed by a matched set of parentheses ( ) that contains a variable and/or expression. This table summarizes the use of operators and functions when you develop a formula.
