Plate tab

Use this tab to view or modify the parameter values used to define the physical properties of a sample plate. When you create a sample set method in Run Samples, you select and define sample plates based on plate types you created in Configuration Manager.

Tip:From the Plate Types table in Configuration Manager, you can also export a plate type to a text file, and then import it into Empower. This is useful when you want to reuse a plate type definition on other Empower systems that run the same type of autosampler.

Plate type

Name

A unique name (up to 30 alphanumeric characters) for the plate type.

Format

Physical layout of the plate. Choices:

  • XY – A configurable generic sample plate used in the Waters 2700 Autosampler
  • W2790/5 – A configurable sample plate used in the Waters 2790/2795 Separations Module
  • W2690/5 – Waters 2690/2695 sample plate
  • W48 – Waters 48-vial carousel
  • W96 – Waters 96-vial carousel
  • HP100 – Hewlett Packard 100-vial GC carousel

Default: W2790/5

Tip: To create a plate type for the W2707 autosampler, import the plate type definition file "W2707_Plate_Type.txt" from the <Drive:>\Empower\Scripts directory.

Plate dimensions

X (XY format only)

Physical size of the plate in millimeters along the x-axis.

Entries: 1.00 to 1000.00

Default: 128.00

Y (XY format only)

Physical size of the plate in millimeters along the y-axis.

Entries: 1.00 to 1000.00

Default: 86.00

Height (XY and W2790/5 formats only)

Physical size of the plate in millimeters along the z-axis.

Entries: 1.00 to 1000.00

Default: 43.90

Well dimensions (XY and W2790/5 formats only)

Top Left Well Location, X

Distance in millimeters from the left edge of the plate to the center of the top left well.

Entries: 0.10 to 100.00

Default: 14.10

Top Left Well Location, Y

Distance in millimeters from the top edge of the plate to the center of the top left well.

Entries: 0.10 to 100.00

Default: 11.30

Well Size, Diameter

Diameter in millimeters of each well in the plate.

Entries: 1.00 to 1000.00

Default: 7.00

Well Size, Depth

Depth in millimeters of each well in the plate relative to the top surface of the plate.

Entries: 1.00 to 1000.00

Default: 23.20
