Enable ApexTrack integration

Default Algorithm

Specifies the default integration algorithm used in a processing method: ApexTrack or Traditional

Default Touchdown % to 0.0

To copy project information to sub-projects:

If you have a project with sub-projects, the Copy to Sub-Projects button appears.

  1. To copy processing settings from a project to all of the project's sub-projects, click Copy to Sub-Projects.

Tip: A message appears when you do not have access to one or more projects in the heirarchy.

  1. Click OK to apply changes to the selected project.
