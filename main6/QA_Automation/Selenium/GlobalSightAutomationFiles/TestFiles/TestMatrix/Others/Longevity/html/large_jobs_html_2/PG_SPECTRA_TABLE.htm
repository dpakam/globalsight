Spectral Table tab

Use this tab to define the presentation of spectra (PDA or MS) when printed in a report.

Table title

Title displayed above the table. The table title can have up to 250 alphanumeric characters without restrictions. If multiple lines are required, press Ctrl+Enter to force the title onto multiple lines.

Note: When you specify a table title, ensure it is not wider than the potential table width. If the title is longer than the generated table, the title is truncated. Avoid this problem by shortening the title or forcing it onto multiple lines.

Alignment

Position of the title above the table.

Choices: Left Alignment, Center Alignment, Right Alignment

Default: Center Alignment

Spectral range

Minimum and maximum values (in nm or m/z) of the spectra to include in the Spectra table. All spectral points outside this range are not included in the report.

Entries: Data dependent

Default: Blank

Spectral characteristics

Spline (PDA)

Turns cubic splining on and off to enhance the spectral display (PDA). When a spectrum is splined, it reappears as a spectrum that extends from its ideal low wavelength to its ideal high wavelength with a resolution of 0.2 nm, regardless of its actual wavelengths and resolution. The interpolated points are calculated by performing a cubic spline operation. A low-resolution spectrum that was splined can produce a more accurate wavelength of maximum absorbance.

Baseline correct (PDA and MS)

Compares spectra with or without baseline correction (PDA and MS). You can perform baseline correction only if the spectra are within peak limits.

Note: MS continuum data never has negative intensities, even after performing baseline subtraction.

Derivative

Number of times the derivative algorithm is applied to the spectra.

Choices: None, First, Second, Third, or Fourth

Default: None

Smoothing

Controls the level of smoothing by specifying the length of the Savitsky-Golay filter applied to the spectra for both smoothing and derivatives. Use the lowest smoothing number possible to avoid over distortion of the spectra.

Choices: None, and odd values (except 1 and 3) None to 31

Default: None

Smoothing is not performed if the number of points in the spectrum is less than the smoothing number plus 5.

Note: If Smoothing is set to None, setting Derivative to First, Second, Third, Fourth applies a smoothing number of 5.

MS noise threshold

Removes noise from spectral data before a library search. The threshold value is applied as a percentage of the Base Peak intensity. Mass intensity values in the sample spectrum that are less than the Threshold value are set to 0. During a library search, the PBM algorithm ignores the ion masses with intensity values of 0. You can apply the Threshold parameter to any spectrum in Spectrum Review, then view the result in the Spectral plot or in your report.

Entries: 0.000 to 100.000

Default: 0.100

Note: The Threshold parameter does not permanently affect the original spectral data.
