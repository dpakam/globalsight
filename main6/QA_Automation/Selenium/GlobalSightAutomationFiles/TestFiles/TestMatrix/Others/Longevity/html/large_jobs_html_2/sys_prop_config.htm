Configuration Tab (Chromatographic System)

Use this tab to reconfigure the selected system by adding or removing instruments. You add an instrument to the selected system by dragging the instrument from the Available Components list to the System Configuration list. (You can also double-click the instrument in the Available Components list.) To remove an instrument from the selected system, drag the instrument from the System Configuration list to the Available Instruments list.

Note: You cannot drag the last instrument from an existing system. To free up the last instrument on a system, you must delete the system.
