Annotation Object Properties - Text tab

Use this dialog box to change properties of the annotation text in Report Preview. Access this dialog box, selecting the Text tool from the drawing objects list, or right-clicking existing annotation text and selecting Properties.

Text

Type the text you want to add to the report, up to 250 characters. Press Ctrl + Enter to start a new line.

Left alignment

Left aligns the text.

Center alignment

Center aligns the text.

Right alignment

Right aligns the text.

Characteristics

When you change a characteristic, it is applied to all annotation text. For example, if you select the Bold characteristic, all annotation text becomes bold. You cannot bold an individual word.

Bold

Bolds the text.

Italic

Italicizes the text.

Underline

Underlines the text.

Strikeout

Draws a line through the text.

Name

Select the font for the text from this list. Available fonts are based on the Windows-installed fonts.

Size

Select the size for the text from the list.

Range: 6 to 72.

Note: Using a larger text size can distort the view of the text in the Preview window of the Annotation Object Properties dialog box, as well as on the Preview page.

Text color

Click this button to access the Color dialog box and change the color of the text. You can use all colors supported by the Windows operating system, which includes basic and custom colors.

Preview

As you type text in the Text field, it appears in a preview field at the end of the dialog box. As you change characteristics of the text, the changes appear in this field.
