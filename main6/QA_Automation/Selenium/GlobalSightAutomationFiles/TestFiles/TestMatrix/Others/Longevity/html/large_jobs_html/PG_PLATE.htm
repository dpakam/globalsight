Plate Properties tab

Use this tab to configure drawing characteristics and well annotations of the plate graphic representation that appears in the New Plate Type/Plate Type Properties dialog box in Configuration Manager and Run Samples.

Use this tab to configure drawing characteristics and well annotations of the plate graphic representation that appears in the New Plate Type/Plate Type Properties dialog box in Run Samples.

Plate characteristics

Show border

Plate border, which graphically outlines the relative size and shape of the plate. Configure the border line style, thickness, color, and background in the Colors tab.

Well annotations

Well identifiers

Textual designations of the wells in a sample plate. The software displays the well identifiers inside the wells if they can fit within the boundaries of the well. This depends on font type and size. Configure the well identifier font type, size, color, and style in the Fonts tab.
