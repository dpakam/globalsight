New Sample Set Method wizard – Options page

Use this page to add rows with special functions (such as Clear Calibrate, Equilibrate, Purge Injector, and so on) to your sample set.

This page of the wizard displays the following tabs (depending on which kind of bracketing you select): Beginning of Sample Set, End of Sample Set, End of Sample Group, and End of Standard Group. Each tab displays a table with fields that correspond to the fields in the Samples table. You define sample set bracketing and other functions by entering the appropriate Function (Clear Calibrate, for example), Label References, and other column information in rows within the tables.

To define sample set bracketing and other sample set functions in the Options page:

  1. Determine where you need to insert rows in the sample set. For example, if you need to insert an Equilibrate or Condition Column step, do so in the table in the Beginning of Sample Set tab. To set up your sample set with multiple sample groups and a sliding bracket, the software inserts a Quantitate step in the table in the End of Sample Group tab and Calibrate and Clear Calibrate steps in the table in the End of Standard Group tab.
  2. Click the appropriate tab to access its table. In the first row, click the Function cell and select the appropriate function from the drop-down list. Enter and/or select information in other cells within the row, as required.

Note: Certain Function selections cause some cells in the row to gray out (prohibiting data entry).

  1. To use the Label and Label Reference feature for bracketing and reporting, in each of these tabs: End of Standard Group, End of Sample Group, or End of Sample Set tab, perform these tasks as required.
  • In the first row, click the Function cell and select the appropriate function from the drop-down list (Calibrate, for example).
  • In the Label Reference cell, type an appropriate label (S01*, for example).

Tip: By default, the Label and Label Reference field is hidden. Right-click then select Show All Columns.

  1. When finished, click OK.
  2. In the Summary page of the New Sample Set Method Wizard, Click Finish to complete the sample set method and load the sample set method into the Samples table.

Alternative: If you are creating a sample set method template, click Next to display the Define User Prompts page.

Tip: You can modify your sample set method, as required, in the Samples table.
