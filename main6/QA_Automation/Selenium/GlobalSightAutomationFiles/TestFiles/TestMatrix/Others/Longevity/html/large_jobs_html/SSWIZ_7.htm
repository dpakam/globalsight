New Sample Set Method wizard – Enter Level Details page

Use this page to further define standard level information by modifying cells in the table. This page appears whenever you select both of the following:

  • Different Standard Levels Are Created with Different Injection Volumes Using an Autoinjector (in the Specify Calibration Options wizard page).
  • A value for Number of Levels in Each Standard Group that is greater than the value you select for Number of Standards in Each Group (in the Describe Standard Bracket wizard page).

The table on this page contains the columns described below.

Vial (or plate/well for systems configured with plates)

Displays the vial numbers for the standards in the group. Note the following:

  • The vial numbers in the column are automatically generated by an Empower algorithm. Modify the numbers according to your requirements.
  • The vial numbers in the column are automatically generated by a Breeze algorithm. Modify the numbers according to your requirements.
  • If you previously selected All Standard Vials Are Located at the Beginning of the Autosampler Tray (in the Specify Calibration Options page), the vial numbers displayed in the column reflect actual vial number positions in the tray. If you previously selected Separate Groups of Standard Vials Are Located Throughout the Autosampler Tray (in the Specify Calibration Options page), the vial numbers displayed in the column reflect the relative position of the standard vials in each group of standards.
  • The Number of Levels in Each Standard Group parameter defines the number of vials displayed in the column. For example, if you select two levels in each group, the Vial column displays two vial numbers, each corresponding to a different injection volume.

Injection volume (µL)

Displays cells where you can enter a different injection volume (in microliters) for each level of standard.

Level

Displays the different standard levels for the sample set. The levels correspond to the value you chose for Number of Levels in Each Standard Group.

Sample weight

Displays the factor used for adjusting the calculated component amount to compensate for differences in the prepared solutions used for calibration and analysis. The default value of 1.000 provides no factoring.

During calibration, the software multiplies the calculated amount of each standard component by the Sample Weight value.

During quantitation, the software divides the amount read from the calibration curve (of the standard components) by the Sample Weight value to calculate amounts for each unknown sample.

Dilution

Displays the factor used for adjusting the calculated component amount to compensate for differences in the prepared solutions used for calibration and analysis. The default value of 1.000 provides no factoring.

During calibration, the software divides the calculated amount of each standard component by the Dilution value.

During quantitation, the software multiplies the amount read from the calibration curve (of the standard components) by the Dilution value to calculate amounts for each unknown sample.

When you finish entering information into the table, click Next to advance to the next wizard page.
