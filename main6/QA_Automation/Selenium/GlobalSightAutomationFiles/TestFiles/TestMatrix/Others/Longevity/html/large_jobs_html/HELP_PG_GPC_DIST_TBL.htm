GPC Distribution Table tab

Use this tab to define characteristics of a GPC Distribution table printed in a report.

Available fields list

Slice fields in the Empower database to include in the GPC Distribution table. Click an item to hide or show the parameter as a column in the Table template.

These fields determine which slice columns of the table to include in the report.

GPC Distribution Table fields:

Field

Usage

Slice MW

Molecular weight of the polymer at each slice. The field is blank when the slice is outside the calculation range (V0 to Vt).

Slice Log MW

Log (base 10) of the molecular weight of each slice. The field is blank when the slice is outside the calculation range (V0 to Vt).

Slice RT

Retention time at each slice.

Slice Adj RT

Retention time of each slice adjusted for variations in retention time as observed relative to a reference peak.

Slice EV

Elution volume (in mL) of each slice adjusted for the flow rate you entered in the Calibration tab of the Processing Method window. Elution Volume is equal to the Adjusted Retention Time if you did not enter a flow rate (assumes a flow rate of 1.0 mL/min).

Slice Area (µV*sec)

Total area of each slice, which is the height of the integrated peak at the reported retention time multiplied by the retention time.

Slice Height (µV)

Height of each slice of the integrated peak at the reported retention time.

dwt/d(log M)

Weight-normalized area of the slice. The absolute value of the area of the slice is divided by the slope of the calibration curve at the adjusted retention time or elution volume of the slice. The ratios of area to slope for all slices are summed, then each ratio is divided by the sum.

Cumulative %

Amount (as percent area) of the polymer accumulated up to and including the slice.

Slice Fitted [h] (Iniversal Calibration)

Log (base 10) of the intrinsic viscosity from the fitted viscosity curve (if a GPCV peak). It is calculated from the molecular weight, K, and alpha values when this is a GPC peak.

Number Density

Normalized distribution based on the number of molecules in a slice.

Outside V0-Vt

Slice is beyond the valid range of the calibration curve.

Slice Observed [h] (GPCV Processing)

Observed intrinsic viscosity of the polymer that elutes at each slice.

Slice Std HV (Universal Calibration)

Hydrodynamic volume at each slice. Hydrodynamic volume is the product of intrinsic viscosity [h] and molecular weight.

Slice g' (GPCV or GPC/V-LS Option)

Branching index at each slice.

Slice g' LCB (GPCV or GPC/V-LS Option)

Long-chain branching index at each slice.

Slice g' SCB (GPCV or GPC/V-LS Option)

Short-chain branching index at each slice.

Slice d (GPCV or GPC/V-LS Option)

Branching density, or the number of branches per dalton, at each slice (assuming the Random or Star branching parameter was specified in the Slicing tab of the Processing Method window).

Slice g (GPCV or GPC/V-LS Option)

Stockmayer-Fixman branching parameter at each slice (assuming the Random intrinsic viscosity fit parameter was specified in the Slicing tab of the Processing Method window).

Slice g LCB (GPC/V-LS Option)

Stockmayer-Fixman long-chain branching parameter at each slice. When the Branching Model is set to Star, this field is blank. When the Fitting Model is set to Calibration, Slice g LCB is calculated only when you enter a value for alpha LCB(LS) or Linear % of Data Region.

Slice g SCB (GPC/V-LS Option)

Stockmayer-Fixman short-chain branching parameter at each slice.

Slice f (GPCV or GPC/V-LS Option)

Number of branches per molecule in a star-branched polymer (assuming the Star branching parameter was specified in the Slicing tab of the Processing Method window).

Slice Conc (GPCV Option)

Concentration of the slice based on the area of the slice and the concentration calculated for the peak. Used in viscometry calculations.

Slice K (GPCV Random Processing or GPC/V-LS Option)

Mark-Houwink constant, K, where log K represents the instantaneous intercept of the Viscosity Law plot.

Slice alpha (GPCV Random Processing or GPC/V-LS Option)

Mark-Houwink constant, alpha, which represents the instantaneous slope of the Viscosity Law plot.

Slice Linear [h] (GPCV or GPC/V-LS Option)

Linear intrinsic viscosity (calculated from the molecular weight, K, and alpha) and used with the fitted (branched) viscosity to calculate g' for the slice processing.

Slice Unbranched [h] (GPCV or GPC/V-LS Option)

Intrinsic viscosity (calculated from the molecular weight, Linear K, and Linear alpha) and used with Linear [h] to calculate g' SCB for the slice.

Valid g' (GPCV Processing)

Indicates whether the g' value is valid (1 or below).

Slice lambda

Plots lambda, the probability for long-chain branching factor, for the slice.

Slice Specific Viscosity (GPCV Option)

Plots the height of the viscometer peak at the slice retention time.

Slice Relative Viscosity

Plots the signal, in mV, from the derived viscosity channel.

Slice Observed MW (GPC/V-LS Option)

Plots the molecular weight calculated for the slice by extrapolating the LS data to the zero-angle.

Slice R0 (GPC/V-LS Option)

Plots the Rayleigh ratio at scattering angle 0.

Slice Observed Rg (GPC/V-LS Option)

Plots the observed radius of gyration calculated for the slice by extrapolating the LS data to the zero-angle using the Rg Fit specified in the Slicing tab in the Processing Method window. Use this for comparison with the theorectical linear and fitted curve. This is the default selection for Plot3 Data.

Slice Fitted Rg (GPC/V-LS Option)

Plots the fitted radius of gyration calculated for the slice by fitting the LS data according to the Slicing tab field Rg Fit parameter. This is the default selection for Left X-Axis Data.

Slice Linear Rg (GPC/V-LS Option)

Plots the linear radius of gyration calculated from the Slice Fitted Rg value by fitting a line to the low molecular weight end of the Fitted Rg when the Fitting Model is set to Structural and the Rg is set to one of the higher-order polynomial fits or either of two step fits. This is the default selection for Plot4 Data.

Slice R1, R2, R3 (GPC/V-LS Option)

Plots the slice height of the first to third LS peaks.

Slice Observed HV (GPC/V-LS Option)

Plots the observed hydrodynamic volume.

Slice Fitted HV (GPC/V-LS Option)

Plots the fitted hydrodynamic volume calculated for the peak according to the Fitting Model, Intrinsic Viscosity Fit, and Rg Fit parameters.

Slice P (GPC/V-LS Option)

Plots the local polydispersity of the slice.

Slice epsilon (GPC/V-LS Option)

Plots the overall shape factor, epsilon.

Slice K(LS) (GPC/V-LS Option)

Plots Slice K(LS), where log Slice K(LS) is the intercept of the line tangent to the curve that produced the Slice Fitted Rg values.

Slice alpha(LS) (GPC/V-LS Option)

Plots the slope of the line tangent to the curve that produced the Slice Fitted Rg values.

Slice Unbranched Rg (GPC/V-LS Option)

Plots the unbranched radius of gyration of the slice calculated only if you entered unbranched values of K(LS) and alpha(LS).

Table title

Title to display above the table. The table title can have up to 250 alphanumeric characters without restrictions. If multiple lines are required, press Ctrl+Enter to force the title onto multiple lines.

Note: When you specify a table title, ensure that it is not wider than the potential table width. If the title is longer than the generated table, it is truncated. Avoid this problem by shortening the title or forcing it onto multiple lines.

Alignment

Position of the title above the table.

Choices: Left Alignment, Center Alignment, Right Alignment

Default: Center Alignment

Table template

Headings and order of the table columns as they are moved from the Available Fields list. To move a column, drag it to a different location in the Table Template. To delete a column from the Table Template, right-click the column name, then select Delete Column (or highlight the columns and press Delete). You cannot leave blanks between columns.

Note: You can also specify sorting order in the individual columns of the Table Template.

Column properties

Accesses the Column Properties dialog box to modify column properties specified in the Table tab. To modify properties, select the corresponding column in the Table Template, then click Column Properties. After redefining column properties, click OK to return to the Table tab.

Print blank columns

Prints table columns that do not contain values.

Default: Enabled.
