Channel 2 tab (W2487)

Use this tab to define channel 2 parameters for the Waters 2487 Dual l Absorbance Detector.

Wavelength (nm)

Primary wavelength for the channel.

Default: 280 nm

Sampling rate (pt/sec)

Rate used to collect data for the channel. Set a sampling rate to achieve at least 15 data points across the narrowest peak of interest.

Rate used to collect data for the channel. Typically, for LC, IC, and GC, you set a sampling rate to achieve at least 15 data points across the narrowest peak of interest.

Choices (Single Wavelength mode): 1, 2, 5, 10

Default: 1

Choice (Dual Wavelength mode): 1

Data mode

Data parameter to be monitored through the IEEE-488 interface. The selected data is output to the IEEE-488 interface on the rear of the detector (for connection to a workstation). See the Waters 2487 Dual l Absorbance Detector Operator’s Guide for parameter details. Parameter selections vary according to the Wavelength Mode (Single or Dual) selected in the General tab. Choices:

  • Absorbance A (Ch1) (Single and Dual l Modes) – Represents the current absorbance at wavelength 1.
  • Absorbance B (Ch2) (Dual l Mode only) – Represents the current absorbance at wavelength 2.
  • Log Absorbance A (Ch1) (Single and Dual l Modes) – Represents the log (base 10) of absorbance at wavelength 1, allowing small and large peaks to be viewed on one chromatogram.
  • Log Absorbance B (Ch2) (Dual l Mode only) – Represents the log (base 10) of absorbance at wavelength 2, allowing small and large peaks to be viewed on one chromatogram.
  • Sample Energy A (Ch1) (Single and Dual l Modes) – Represents the energy from the sample side of the flow cell (at wavelength 1).
  • Sample Energy B (Ch2) (Dual l Mode only) – Represents the energy from the sample side of the flow cell (at wavelength 2).
  • Reference Energy A (Ch1) (Single and Dual l Modes) – Represents the energy from the reference side of the flow cell (at wavelength 1).
  • Reference Energy B (Ch2) (Dual l Mode only) – Represents the energy from the reference side of the flow cell (at wavelength 2).
  • Maxplot A, B (Ch1, Ch2) (Dual l Mode only) – Represents the maximum absorbance value for each sample component. The higher of the two absorbance values (for each component) is output on the selected channel.
  • Ratioplot A/B (Ch1/Ch2) (Dual l Mode only) – Represents the ratio of absorbances at the two selected wavelengths.
  • Ratioplot B/A (Ch2/Ch1) (Dual l Mode only) – Represents the ratio of absorbances at the two selected wavelengths.
  • Difference Plot A–B (Ch1–Ch2) (Dual l Mode only) – Represents the A–B difference in absorbance at the two selected wavelengths.
  • Difference Plot B–A (Ch2–Ch1) (Dual l Mode only) – Represents the B–A difference in absorbance at the two selected wavelengths.

Note: Monitor a second parameter over the IEEE-488 interface by selecting a Data Mode in the Channel 1 tab. Also, you can monitor up to two additional parameters by selecting an Output Mode in the Analog A and Analog B tabs.

Filter type

Filter type used on analog and IEEE-488 output. The Hamming filter is designed to create the same degree of peak-height degradation as the RC filter for the same response time, but enhances filtering of high-frequency noise. The performance of the selected filter depends on the Time Constant value (see below).

Choices: None, Hamming, RC

Default: Hamming

Time constant (sec)

Noise filter (for analog and IEEE-488 outputs) to achieve the optimum signal-to-noise ratio. With the Hamming filter selected, the default Time Constant setting of 1.0000 is satisfactory for most applications. This parameter is disabled if you select None for Filter Type.

Range with Hamming (in Single Wavelength mode): 0.1 to 5.0 sec

Default: 1.0 sec

Range with Hamming (in Dual Wavelength mode): 1.0 to 50.0 sec

Default: 1.0 sec

Range with RC (in either Wavelength mode): 1.0 to 99.0 sec

Default: 1.0 sec

Ratio minimum (AU)

Minimum absorbance required before the actual ratio is calculated (when Ratioplot A/B or Ratioplot B/A Data Mode is selected). Both absorbance values A and B must be above this value, or 0.0 V is output. When both A and B are above this value, a ratio is calculated and output (on the IEEE-488 interface and/or the analog output). For analog output, the voltage is proportionally scaled to the ratio (according to your Scaling Minimum and Scaling Maximum settings in Analog A and Analog B tabs).

Default: 0.1000 AU

Auto zero parameters

Specify channel 1 auto zero parameters in this section of the property page.

On wavelength changes

Specifies an auto zero each time you change the wavelength. Except for special advanced operations, you should keep this parameter enabled. Disabling it can cause significant changes in measured absorbance if the wavelength is changed.

Choices: Enabled or Disabled

Default: Enabled

On inject

Specifies an auto zero each time the 2487 Detector receives an inject start signal, either by an event input (contact closure), an input through the IEEE-488 interface or the detector front panel.

Choices: Enabled or Disabled

Default: Enabled

From keypad or event input

Specifies an auto zero each time the 2487 Detector receives an auto zero request through the keypad or by an event input (contact closure).

Choices: Enabled or Disabled

Default: Enabled
