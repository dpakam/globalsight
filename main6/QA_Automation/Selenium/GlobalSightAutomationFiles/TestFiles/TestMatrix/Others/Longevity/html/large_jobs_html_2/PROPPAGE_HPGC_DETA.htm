Detector A tab (5890/5890S)

Use this tab to specify the temperature for detector zone A and to define the Detector A operating parameters.

Note: The method will not work unless you select the correct detector type.

Detector type

Type of detector used in the 5890 GC System.

Choices: FID (flame ionization), TCD (thermal conductivity), NPD (nitrogen phosphorous), None, and ECD (electron capture)

Default: None

Polarity

Polarity of the Thermal Conductivity Detector (TCD) output signal. Set this parameter to negative (–) for components giving negative peaks.

Choices: + or –

Default: +

Requirements:

  • Set Detector Type to TCD.
  • Set Polarity Switch Time to some value (see below).

Polarity switch time

When to change the polarity of the TCD output signal during a run (see Polarity above). Polarity returns to its original state at the end of the run.

Default: 0.0 min

Temperature enable

Enables the temperature set point parameter for Detector A. Enable this check box before Temperature Set Point takes effect (see below).

Choices: Enabled or Disabled

Default: Disabled

Temperature setpoint

Detector A set point value for the 5890 GC System. Enable Temperature Enable for this parameter to take effect.

Default: 50 °C
