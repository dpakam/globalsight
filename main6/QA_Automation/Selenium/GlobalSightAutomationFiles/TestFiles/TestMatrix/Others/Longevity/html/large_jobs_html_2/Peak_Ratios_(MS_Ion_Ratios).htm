Peak Ratios (MS Ion Ratios)

In the processing method, use the items on this tab to specify information used to calculate cross-channel peak ratios with acceptance tolerance checking on injection data. You can use ion ratio processing to calculate ion ratio peak values for LC, PDA, GC, IC, CE, CIA and MS results.

Peak Ratios tab items:

Processing method item

Use to specify this information

Primary Peak table (Primary MS Ion table)

One or more existing components, and the primary channel by name or by description. You also can specify whether to determine the peak ratio by area or height.

Secondary Peak table (Secondary MS Ion table)

For the component you select in the Primary Peak table, can contain as many as four secondary peaks.

Peak Response field

Whether to determine the peak (ion) ratio by peak area or peak height.
