LS Instrument Properties tab

Use this tab to specify parameters for creating an LS channel set used with a GPC or GPCV channel set. Although the LS detector is not an instrument that physically exists in a system, this "virtual" instrument allows you to define instrument method parameters to use to create an LS channel set. After specifying parameters here (and parameters for other instruments in your GPC/V-LS system), save them as an instrument method. If you later define an LS channel set in the Method Set wizard, the LS parameters you set here transfer to corresponding fields in the Define LS Channel Set wizard page.

Notes:

  • The LS instrument is available only if the Light Scattering and either GPC or GPCV options are installed on your system.
  • These LS parameters are used only for processing LS data, not acquiring LS data.
  • You can add the LS instrument to the instrument method using the Add Instrument command or tool in the Instrument Method Editor. If the instrument method contains the LS instrument, click the LS instrument icon to display this tab.

Concentration detector parameters

Channel

Concentration channel for your system. Select the channel from the list. You can select a PDA channel for this field.

Flow rate (ml/min)

Flow rate used to calculate retention times. This is the flow rate you must enter in your processing method when the x-axis of the calibration is set to elution volume. This parameter is used only for processing purposes, not defining the flow rate for data acquisition.

Default: None

Note: The value entered here is used for reference. It is transferred only to the Flow Rate field in your processing method when you create a new processing method. To create a new processing method, select Process > Calibrate Light Scattering System or Process > GPC-LS Data or Process > GPCV-LS Data or Process > GPCV Data in Review. Otherwise, you must enter the correct flow rate value in your processing method. Click for details on defining the flow rate in a GPC/V/LS processing method.

Connected to RI detector

Specifies that the concentration channel is connected to an RI detector.

RI calibration constant

Calibration constant for the RI detector in units of 1/µV. This field is active only if Connected to RI Detector is enabled. Default: None.

LS detector parameters

Name

Identifies the type of LS detector for your system. Select the name from the list.

Interdetector time (min)

Time offset for data points in the LS channel.

Default: 0.000 min

Detector type

Specifies the LS detector manufacturer.

Laser wavelength

Specifies the wavelength, in nm, of the laser used in the LS detector.

Glass refractive index

Enabled when you select Wyatt or Unknown from the Detector Type field. Specifies the refractive index of the glass used in the flow cell of the LS detector.

Description

Optional description for the LS instrument parameters that define the LS channel set.
