LS Instrument Properties Channels tab

Use this tab to specify the channel name and geometric angle for each LS channel in the channel set.

Note: The LS instrument is available only if the Light Scattering and either GPC or GPCV options are installed on your system.

Channel

Specifies the channel for each LS angle.

Geometric angle (degree)

Displays the geometric angle entered or calculated using the Calibrate Light Scattering System wizard.

Description

Enter an optional description of each LS channel.
