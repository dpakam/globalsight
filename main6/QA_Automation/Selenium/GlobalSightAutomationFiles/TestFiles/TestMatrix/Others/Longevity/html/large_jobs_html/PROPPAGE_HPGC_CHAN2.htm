Channel 2 tab (5890/5890S)

Use this tab to define data channel 2 parameters for the 5890 GC System.

Note: Define all Channel 2 tab parameters when transmitting analog data through a busSAT/IN or SerialSAT/IN 2 Module. If you enabled the Ch 2 Enable parameter to transmit digital data through the RS-232 port, then define the Units and Source parameters (see below).

Channel name

Software-designated name for data channel 2, which appears in the Active Channels table in the Instrument Method Editor.

Description

Optional description for data channel 2 on the 5890 GC System, up to 250 alphanumeric characters. The description appears in the Active Channels table in the Instrument Method Editor and in the Real-Time Plot pane in Run Samples.

Units

Units that appear on the y-axis of the chromatogram, for example, mV or AU.

Default: mV

Range

Adjusts a portion of the full dynamic range of the output signal. The selected portion is adjusted so that the highest possible value of the portion does not exceed the maximum voltage of the given output signal (+1-mV or +1 V output).

Range and Attenuation Value are used to keep peaks of interest on scale. See the HP 5890 Series II Gas Chromatograph Operating Manual for details on selecting range and setting signal attenuation.

Entries: 0 to 13

Default: 0

Note: To minimize error, set Range to the lowest possible value, ensuring peaks of interest do not exceed 1 V.

Source

GC output signal to be monitored by Empower software. Choices are:

  • None - No output signal
  • Detector A - 1-V signal from detector A

Note:When configured for Dual Tower operation, specify Detector (A or B), which is connected to Inlet A by the column, as the source for Channel 1.

  • Detector B - 1-V signal from detector B

Note: When configured for Dual Tower operation, specify the Detector (A or B), which is connected to Inlet B by the column, as the Source for Channel 2.

  • A–B - Difference signal between two detectors of the same type (detector A signal minus detector B signal)
  • B–A - Difference signal between two detectors of the same type (detector B signal minus detector A signal)
  • A–Comp1 - Difference signal between detector A and column compensation 1
  • A–Comp2 - Difference signal between detector A and column compensation 2
  • B–Comp1 - Difference signal between detector B and column compensation 1
  • B–Comp2 - Difference signal between detector B and column compensation 2
  • Comp1 - Signal from column compensation 1
  • Comp2 - Signal from column compensation 2
  • Test Plot - Test signal (a stored chromatogram) for use in verifying operation

Zero enable

Enables the zeroing feature. Enable this check box before Zero Value parameter takes effect (see below).

Choices: Enabled or Disabled

Default: Disabled

Zero value

Value that is subtracted from the detector signal to compensate for an undesired, constant background signal. The result is an increase in dynamic signal range.

Default: 0.0

See the HP 5890 Series II Gas Chromatograph Operating Manual for information on zeroing signal output.

Attenuation enable

Enables the attenuation feature. Enable this check box before Attenuation Value takes effect (see below).

Choices: Enabled or Disabled

Default: Disabled

Attenuation value

Selects and adjusts a portion of the ranged +1-mV output signal to ensure that the signal does not exceed 1 mV (see Range above). Each step to a higher Attenuation Value reduces the output signal level by a factor of 2 (half the previous level).

Entries: 0 to 10

Default: 0
