PDA Library Search tab

Use this tab to specify parameter values used during Library Matching.

Library match

These parameters specify how a library search is conducted.

Derivative

Number of times the software applies the derivative algorithm to a spectrum before it performs Library Matching.

Choices: None, First, Second, Third, or Fourth

Default: None

Note: Set the Smoothing parameter if you intend to use both smoothing and derivatives as part of the Library Matching criteria in the processing method. Derivatives require a smoothing number (set by the Smoothing parameter). If you select a Derivative value of First, Second, Third, or Fourth, the PDA software selects a default Smoothing value of 5.

Smoothing

Length of the Savitsky-Golay filter applied to the spectra for both smoothing and derivatives. If you are using smoothing or derivatives, specify a smoothing number. Choices: If the Derivative is None, values are None and odd values from 5 to 31 with a default of None. If the Derivative is First, Second, Third, or Fourth, values are odd numbers from 5 to 31 with a default of 5.

Note: For background information about Savitsky-Golay filtering, refer to Abraham A. Savitsky and M.J.E. Golay, "Smoothing and Differentiation of Data by Least-Squares Procedures," Analytical Chemistry, Vol. 36: 1964, p. 1627.

Search threshold (degrees)

Threshold for the reported Match Angle. Empower software does not report any matches with a Match Angle greater than the Search Threshold value you choose.

Entries: 0.00 to 90.00

Default: 10.0

Search depth

Number of matches reported for a library search. For example, with a search depth of 3, the software performs all possible searches, then selects and reports the best 3 matches. Results are reported only when the calculated Match Angle is less than the Search Threshold.

Entries: 1 to 3

Default: 3

Threshold criteria

Estimates the noise and/or solvent contribution to the Match Angle of a spectrum during Library Matching, and flags match results that exceed this threshold. For details about using the Threshold Criteria parameter during Library Matching, click. Choices:

  • Noise + Solvent – Adds the Solvent Angle to the Noise Angle. You must fill in the Solvent Angle field.
  • Noise – Reports and uses only the Noise Angle, which is calculated based on total system performance, ignoring solvent contribution.
  • Solvent – Reports and uses only the Solvent Angle. You must enter a value in the Solvent Angle field.

Solvent angle (degrees)

Compensates for solvent contribution to Spectral Contrast angles. If a solvent angle is in the Threshold Criteria, enter a value in the Solvent Angle field. Solvent Angle allows you to estimate the solvent-induced spectral changes.

Entries: 0.00 to 90.00

Default: 1.00

Search libraries

Spectral libraries used for Library Matching. Use libraries acquired at the same wavelength range and resolution as the spectra you are trying to match.

Retention time presearch

Limits the search to library spectra with retention times similar to the acquired spectrum, thus reducing search time.

RT window (%)

Specifies a retention time window when you use Retention Time Presearch.

Entries: 0.0 to 100.0% of the actual retention time of the acquired spectrum.

Default: ±10.0% of the retention time.

Wavelength presearch

Limits the search to library spectra with wavelength maxima within the specified wavelength window.

Window (nm)

Specifies a wavelength window when you use Wavelength Presearch.

Entries: 0.0 to 100.0 nm

Default: 5.0 nm
