Legend tab

Use this tab to customize the display of the plot legend in your plot when printed. For Result Custom Fields reports, the Legend tab information becomes part of the table title instead of a legend in a plot.

Use this tab to customize the display of the plot legend in your plot when printed.

Plot title

A legend title directly above the plot. The plot title can have up to 250 alphanumeric characters with no text restrictions. If multiple lines are required, press Ctrl+Enter to force the title onto multiple lines.

Text alignment

Where the legend prints in relation to the plot.

Choices: Left Alignment, Center Alignment, Right Alignment

Default: Center Alignment

Fields list

Fields available for inclusion in the plot legend when it is printed. Double-click (or drag) a field in the list to add it to the Legend table.

Exception: Some plot legend fields are not populated with data. However, these fields are reportable from Report Publisher. For example, the values for sample set fields appear in Preview but are not populated when printing from Review.

Table

Specifics of fields selected for the plot legend. You can modify the field specifications. Fields are positioned in the order in which they are selected from the Legend Fields list. Columns are:

  • Original Name – Displays the unedited, original name of the selected field.
  • Field Name – Displays the modified name of the selected field. If necessary, change the field name in the associated table cell.
  • Width – Number of spaces allotted for the field. Allowable entries depend on the selected field.
  • Precision – Number of digits reported to the right of the decimal. Allowable entries depend on the selected field.
  • Exponential Notation – Displays exponential notation for the field.
  • Field Type – Identifies the type of field included in the legend (double-precision float, string, integer, date, enumerated, or Boolean value).
  • Summary Function Name – Labels the information with a name other than the Summary Function Type name.
  • Summary Function Type – Summarizes the field and generates statistical data. Choices (from the table cell list) are:
  • None – No function performed
  • Min – Minimum value
  • Max – Maximum value
  • Sum – Sum of all values
  • Count – Total number of rows in the table
  • Mean – Mean average of all selected values
  • %RSD – Percent relative standard deviation
  • Std. Dev. – Standard deviation around the mean
