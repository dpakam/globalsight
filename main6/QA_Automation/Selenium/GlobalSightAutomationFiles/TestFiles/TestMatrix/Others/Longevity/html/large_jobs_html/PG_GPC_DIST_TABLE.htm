Slices tab

Use this tab to define the number of slices to appear in the Distribution table printed in a report.

Slice information

Number of slices to appear in the Distribution table according to these parameters:

  • Show All Slices – All slices in the Distribution table (default).
  • Show 100 Slices – 100 slices by cumulative % in the Distribution table
  • Slice By Retention Time – Slices by retention time in the Distribution table.
  • Slice By MW Marker – Slices calculated for the molecular weight markers in the Distribution table.

Note: If there are no molecular weight markers, there will not be any slices.

  • Show Moments Slices – Slices calculated for the molecular weight markers (Mn, Mw, Mv, Mz, Mz+1, and MP) in the Distribution table.

Number of slices

Number to use to slice the distribution by retention time. This parameter is available when Slice by Retention Time is selected.

Entries: 1 to 10000

Default: 100
