Dissolution Plot tab

Use this tab to specify the format of the Dissolution plot printed in a report. The x-axis of the Dissolution plot is always transfer time.

Y-axis

Y-axis data units of the Dissolution plot:

  • Amount Released – Amount of component that is dissolved.
  • Percent Released – Percent of the claimed amount of component that is dissolved.

Format

Plot type(s) to use in the dissolution plot. You can include multiple plots in a Dissolution plot report group.

Choices: Scatter Plot, Line Plot.

Average All Vessels – Average value of all vessels per transfer time. If you choose this option, multiple injections per vessel are averaged for a specific transfer time. This value is then averaged with the other vessels for the same transfer time.

Plot Min/Max Points – If you are averaging all vessel transfer times, select this check box to plot the minimum and maximum points for each transfer time.

Include in plot

Dissolution information to include in the report:

  • Infinity Point – Infinity point in the Dissolution plot (assuming that the Infinity test is enabled for the associated sample set method). The Infinity point appears along the x-axis of the Dissolution plot as the last data point. If the Infinity point is displayed in a Line plot, it appears as a square at the end but not connected to the remainder of the Line plot. Empower software supports one Infinity point for each bath.
  • Q Factor Point – Percent of component that is expected to dissolve after a specified time (for example 70% at 50 minutes). The Q Factor Point appears as an ellipse on the plot and is labeled with the percentage or the amount (depending on the Y-Axis setting). If Amount Released is specified, the Q Factor Point (which is originally entered in the sample set method as a percent) is converted by the software to an amount. The software displays one Q Factor Point for each bath (for each component tested).
  • Claimed Amount – Nominal amount of component in the dosage form claimed by the manufacturer’s label, for example, 50 mg. The Claimed Amount appears in the Dissolution plot legend. The software displays one Claimed Amount for each bath (for each component tested).
  • Line to Origin – If Line Plot is selected as the format for the Dissolution plot, you can select this field to draw a line from the origin to the first transfer time.
