General tab (717 and 717plus autosampler)

Use this tab to define operating parameters for the heater/cooler in the Waters 717 and 717plus autosamplers.

Temperature enable

Enables use of the temperature value in Set Point to set the temperature of the heater/cooler.

Choices: Enabled or Disabled

Default: Disabled

Set point

Temperature set point of the optional heater/cooler. This parameter takes effect if Temperature Enable is enabled.

Default: 25.0 °C
