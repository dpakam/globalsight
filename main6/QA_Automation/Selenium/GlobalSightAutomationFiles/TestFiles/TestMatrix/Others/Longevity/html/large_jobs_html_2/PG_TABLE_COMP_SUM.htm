Component Summary tab

Use this tab to summarize peak field table columns to include in a Table report group. Peak field summaries are printed as the last rows in the generated table. You can only select one peak field to summarize. If you select an unknown peak to be reported, only the unknown peak with the lowest retention time is reported.

Use component summary

Enables component summary capability.

Notes:

  • When you make a selection in the Component Summary tab, the software informs you that specifying a component summary removes all peak-related fields from the report table.
  • You might want to also use the Order By tab to control the order in which the table rows print and how the rows are grouped.

Field to summarize

Entire list of peak fields available for the component result summary. You can choose one peak field to summarize.

Original name

Unedited, original name of the selected peak field.

Data Type

Type of field included in the table (Double Precision Float, String, Integer, Date, Enumerated, Characters, or Boolean Value).

Data precision

Number of digits reported to the right of the decimal. If the reported precision is less than the precision of the stored component summary result, the reported result is rounded. If the reported field width or precision affects the validity of the result, the width is ignored and the entire result is printed.

Use exponential notation

Exponential notation for peak fields in reports.

Summary functions to apply

Available summary functions, which enable you to summarize columns in a table to generate statistical data for a specific peak field. You can apply more than one function to a table column. To select a summary function, double-click (or drag) the function in the list, and the column is added to the Summary Function table.

Summary functions:

  • None – No function performed
  • Min – Minimum value
  • Max – Maximum value
  • Sum – Sum of all values
  • Count – Total number of rows in the table with data in the specific column
  • Mean – Mean average of all selected values
  • %RSD – Percent relative standard deviation
  • Std. Dev. – Standard deviation around the mean

Summary function table

Summary functions used for the table column. Summary functions are positioned in the order in which you selected them from the Summary Functions list. Columns are:

  • Summary Function – Selected summary function. When a summary function is listed in the table, this cell becomes a list from which you can select the specified summary function.
  • Min Limit – Minimum limit to report for the specified summary function.
  • Max Limit – Maximum limit to report for the specified summary function.
  • Precision – Number of digits reported to the right of the decimal. If the reported precision is less than the precision of the stored result, the reported result is rounded. If the reported column precision affects the validity of the result, the entire result is printed.
