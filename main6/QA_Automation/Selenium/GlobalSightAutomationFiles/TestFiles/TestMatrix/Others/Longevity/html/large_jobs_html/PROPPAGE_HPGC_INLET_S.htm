Inlets tab (5890S)

Use this page to set injector A and B inlet zone temperatures, and to set the on/off times for the purge valves in the 5890 GC System.

Inlet A parameters

Inlet enable

Enables or disables temperature inlet zone A.

Choices: Enabled or Disabled

Default: Disabled

Temperature

Inlet zone temperatures for injector A. Enable the corresponding Enable check box before the value takes effect.

Default: 50 °C

Purge enable

Enables purge valve A. Enabling the purge valve purges the inlet of solvent vapor. Disabling the purge valve turns the purge valve off.

Choices: Enabled or Disabled

Default: Disabled

On time

Specifies the time after injection at which the purge begins. The Purge Enable parameter must be selected for this parameter to take effect.

Default: 0.00 min

Off time

Time after injection when the purge stops. Select Purge Enable for this parameter to take effect.

Default: 0.00 min

Inlet B parameters

Inlet enable

Enables or disables temperature inlet zone B.

Choices: Enabled or Disabled

Default: Disabled

Temperature

Inlet zone temperatures for injector B. Enable the corresponding Enable check box before the value takes effect.

Default: 50 °C

Purge enable

Enables purge valve B. Enabling the purge valve purges the inlet of solvent vapor. Disabling the purge valve turns the purge valve off.

Choices: Enabled or Disabled

Default: Disabled

On time

Time after injection when the purge begins. Select Purge Enable for this parameter to take effect.

Default: 0.00 min

Off time

Time after injection when the purge stops. Select Purge Enable for this parameter to take effect.

Default: 0.00 min

Note: You can display the purge time settings on the front panel of the 5890S. If you cannot determine the purge value settings, see the HP5890 Series II Gas Chromatograph Operating Manual for details about viewing these values.
