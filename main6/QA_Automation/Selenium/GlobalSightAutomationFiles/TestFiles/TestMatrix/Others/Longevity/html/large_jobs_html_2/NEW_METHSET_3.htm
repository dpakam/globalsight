Method Set wizard – Default Methods page

Use this page to select the default processing method, Individual and Summary By Injection report methods, and/or the export method to include in your method set. Selected methods are applied to all channels in the Method Set view. If you leave any default method selections blank (that is, if you select No Processing, No Reporting, and/or No Exporting from the list), that function is not performed by your method set.

Use this page to select the default processing method, report method, and/or the export method to include in your method set. Selected methods are applied to all channels in the Method Set view. If you leave any default method selections blank (that is, if you select No Processing, No Reporting, and/or No Exporting from the list), that function is not performed by your method set.

Note: If creating a PDA or MS derived channel or a GPCV/LS channel set, choose an appropriate processing method from the Processing Method list.

To modify the contents of the processing method, click Edit to access the Processing Method window. Modify the processing method, save changes, then exit this window to return to the Method Set Editor.

After making your selection, click Next.
