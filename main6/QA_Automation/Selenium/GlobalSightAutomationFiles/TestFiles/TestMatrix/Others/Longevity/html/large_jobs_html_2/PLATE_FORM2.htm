Rows and Columns tab

Use this tab to specify the row and column dimensions and offsets of an XY- or W2790/5-format plate type.

Row and column dimensions

Rows, Number

Number of rows of wells in the sample plate.

Entries: 1 to 1000

Default: 8

Rows, Spacing

The center-to-center spacing in millimeters between the wells in adjacent rows in a sample plate.

Entries: 0.10 to 100.00

Default: 9.00

Columns, Number

Number of columns of wells in the sample plate.

Entries: 1 to 1000

Default: 12

Columns, Spacing

The center-to-center spacing in millimeters between the wells in adjacent columns in a sample plate.

Entries: 0.10 to 100.00

Default: 9.00

Row and column offsets (XY format only)

Row Offset

Subset of rows in the sample plate that have an offset in millimeters applied to them. Choices:

  • None – No offset is applied to any rows in the plate.
  • Odd – An offset, whose value you enter in the Offset text box, is applied to only odd-numbered rows in the plate.
  • Even – An offset, whose value you enter in the Offset text box, is applied to only even-numbered rows in the plate.

Default: None.

Row Offset Value

Value of the offset in millimeters applied to the odd- or even-numbered rows in a sample plate.

Entries: 0.00 to 100.00

Default: 0.00

Column Offset

Subset of columns in the sample plate that have an offset in millimeters applied to them. Choices:

  • None – No offset is applied to any columns in the plate.
  • Odd – An offset, whose value you enter in the Offset text box, is applied to only odd-numbered columns in the plate.
  • Even – An offset, whose value you enter in the Offset text box, is applied to only even-numbered columns in the plate.

Default: None

Column Offset Value

Value of the offset in millimeters applied to the odd- or even-numbered columns in a sample plate.

Entries: 0.00 to 100.00

Default: 0.00
