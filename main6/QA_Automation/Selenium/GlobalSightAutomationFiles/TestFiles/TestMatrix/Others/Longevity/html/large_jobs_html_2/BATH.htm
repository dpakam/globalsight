Bath A or B setup page

Use this page to enter setup parameters for Bath A and/or B. This page appears only when you select a dissolution type in the Select Sample Set Method Type page of the Sample Set Method wizard.

Range of vessels

Vessel numbers or vessel ranges to be processed from Bath A or B during the dissolution test. Separate the vessel ranges with commas, for example, 1–6, or 1–8, or 1,3,5–8.

Entries: With the 2690D/2695D, 1–6 or 1–8; without the 2690D/2695D, 1–99.

Dissolution media name

Dissolution media used in the dissolution test, for example, 0.1N hydrochloric acid.

Initial volume

Initial volume (in milliliters) of dissolution media in each vessel.

Entries: 0.0 to 99999.99.

Typical: 0.0 to 9999.99.

Default: 900.00.

WTM syringe speed

Waters Transfer Module syringe speed: normal or slow with 2690D/2695D only.

Post sample wash volume

The volume (in 2.5-mL increments) of post sample wash. It occurs after sample transfer and back flush. The post sample wash draws from the C line and rinses the D or waste line.

Note: Do not use Recirculate Media with this option.

Entries: 0.0 to 1000.00 for 2690D/2695D.
