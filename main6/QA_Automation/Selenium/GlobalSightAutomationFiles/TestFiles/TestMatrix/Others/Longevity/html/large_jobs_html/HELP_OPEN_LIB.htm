Open an Existing Library dialog box

Use this dialog box to open a spectral library in Spectrum Review. Opening a library makes the library available for use in Spectrum Review, but does not display the library spectra in the Spectral plot. To view the library spectra in the Spectral plot, use the Load Library command.

You must open a library before you can match spectra against or add spectra to the library.

Library list

All spectral libraries in the connected database. Libraries are global to all projects in the database.

Name

Name of the library you select from the list or type in the text box. Library names are case sensitive.

Comments

Any comments entered when the library was created.

Open button

Opens the spectral library you selected from the list.
