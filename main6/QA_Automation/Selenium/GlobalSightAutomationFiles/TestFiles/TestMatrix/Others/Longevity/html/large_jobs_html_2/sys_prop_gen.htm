General Tab - Chromatographic System Properties

Use this tab to view or modify the following fields.

Empower Node Name

Name of the Empower Node connected to the selected system. This field is not editable.

Breeze 2 Node Name

Name of the Breeze 2 Node connected to the selected system. This field is not editable.

Online

Controls whether a chromatographic system is active and available for data acquisition.

System Name

Name of the selected system. You can rename the selected system by typing a new name in the text box, then clicking OK.

System Comment

Displays or allows for entry of optional remarks. Comments describing the chromatographic system, its components, and its intended use may helpful others if you are sharing the system. You can enter up to 250 alphanumeric characters.
