Structure Viewer dialog box (Chemical Structures option)

This dialog box displays a graphic of a loaded chemical structure. You can do the following things:

  • Load a chemical structure from a *.mol file or an existing library.
  • Dissociate a structure from a component.
  • Store a structure in a chemical structure library.
  • Create a new library.

If you open this dialog box from the View Filter Editor in the Project window, you can search for exact structure matches.

Use the Advanced Chemical Structures option to do these tasks:

  • Start ChemSketch from the Structure Viewer dialog box.
  • Use ChemSketch to create or edit structures, and later view them in the Structure Viewer dialog box.
  • When you open the Structure Viewer dialog box from the Peaks View Filter Editor, you can search for any structure that contains a specific portion of a molecule (substructure search).

Note: Search features apply only when you open the Structure Viewer dialog box from the View Filter Editor.
