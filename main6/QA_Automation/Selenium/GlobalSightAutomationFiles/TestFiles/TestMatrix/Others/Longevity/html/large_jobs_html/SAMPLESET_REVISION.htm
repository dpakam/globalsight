Enter Revision Description dialog box - Method or Sample

Appears whenever you modify a method, method set, template, sample data, or sample set data, and have the Audit Trail Support feature activated, requiring comments for methods.

Either select Default String or type a description of the changes made in the Revision Description field, then click OK. The revision description is saved to the database.

You can view a history of changes made to samples (and the revision description comments you entered) in the Sample History dialog box and changes made to Sample Set Methods in the Sample Set Method History box.

Note: You can also view changes made to a processing method, instrument method, and so on in a report. From Report Publisher, turn on the Revision History Details check box in the appropriate method report group.
