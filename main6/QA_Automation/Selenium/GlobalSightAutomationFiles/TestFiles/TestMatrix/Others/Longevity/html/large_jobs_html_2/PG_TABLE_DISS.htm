Dissolution Summary Table tab

Use this tab to summarize (in the Dissolution Table report group) the dissolution transfer time data for a given peak. Columns appear as transfer time and table rows appear as amount dissolved or percent dissolved. Table columns can be summarized as needed using available summary functions.

Field to summarize

Dissolution data field to summarize:

  • Amount Dissolved – Displays on the y-axis of the Dissolution Plot the amount of component that is dissolved.
  • Percent Dissolved – Displays on the y-axis of the Dissolution Plot the percent of the claimed amount of component (amount nominally in the dosage form) that is dissolved.

Data precision

Number of digits reported to the right of the decimal. If the reported precision is less than the precision of the stored component summary result, the reported result is rounded. If the reported field width or precision affects the validity of the result, the width is ignored and the entire result is printed.

Use exponential notation

Exponential notation for dissolution peak fields in reports.

Summary functions to apply

Available summary functions. These functions let you summarize columns in a table to generate statistical data for a specific dissolution peak field. You can apply multiple functions to a table column. To select a summary function, double-click (or drag) the function in the list, and the column is added to the Summary Function table:

  • None – No function performed
  • Min – Minimum value
  • Max – Maximum value
  • Sum – Sum of all values
  • Count – Total number of table rows with data in the specific column
  • Mean – Mean average of all selected values
  • %RSD – Percent relative standard deviation
  • Std. Dev. – Standard deviation around the mean

Summary function table

Summary functions used for the table column. Summary functions are positioned in the order in which you select them from the Summary Functions list. Summary Function columns are:

  • Summary Function – Selected summary function. When a summary function is listed in the table, the table cell provides a list from which you can select the specified summary function you want.
  • Min Limit – Minimum limit to report for the specified summary function.
  • Max Limit – Maximum limit to report for the specified summary function.
  • Precision – Number of digits reported to the right of the decimal. If the reported precision is less than the precision of the stored result, the reported result is rounded. If the reported column width or precision affects the validity of the result, the width is ignored and the entire result is printed.
