Auxiliary Tab (6890)

Use this page to specify the parameters associated with the optional thermal and pressure auxiliary controls.

Tip: For channel selections, Aux1 and Aux2 are the thermal auxiliary channels and Aux3 through Aux5 are the pressure channels.

Channel Select

Select an auxiliary channel. You can select one at a time.

Channel Options:

Option

Description

Thermal Channel Function

Choices: Valve Box, Not installed, or Unknown

Default: Valve Box

Pressure Channel Source (Gas)

Choices: Air, Ar, ArCH4, H2, He, N2, O2, Unknown

Default: Unknown

Ramp Data:

Channel

Description

Initial Time

Length of time (from the start of the run) that the initial ramped temperature or pressure value is maintained.

Temperature or pressure is maintained for the length of time you set for Initial Time. The temperature or pressure then ramps up using the Rate and Final Pressure/Temp values you selected in row 1 of the Ramp table.

Range: 0.00 through 99.99 minutes

Initial Value

Thermal = 0.00 through 200.00 °C

Pressure = 0.00 through 150.00 psi

Default:
Thermal = 50 °C
Pressure = 1.0 psi

Ramp Table

Parameters

Description

Rate

Rate of changes in Deg C/min for temperature, or pressure units/min for pressure.

Final

Specifies the final set point value for that segment of the ramp.

Hold time

Length of time during which you want to hold the final pressure or temperature value. When this time expires, the software goes on to the next table row (unless this is the last row in the table).

Total time

Total time as calculated by Empower software. Total Time displayed in the first row is derived from the difference between the Initial Pressure/Temp (at the start of the ramp) and the Final Pressure/Temp (at the end of the ramp) divided by the Rate, plus Hold Time plus the Initial Time. Total Time displayed for subsequent rows is derived from the difference between the Final Pressure/Temp from the previous row and the Final Pressure/Temp for the current row divided by the Rate, plus Hold Time, plus the Total Time from the previous row.

When the run begins, temperature or pressure is maintained at the Initial value for the length of time you entered for Initial Time. At the end of that time, temperature or pressure ramps up to the Final Pressure/Temp value at the Rate you set in row 1. Final Pressure/Temp is maintained for the length of time you entered for Hold Time. You can enter values in up to three table rows. The ramp process continues for each row in the table.

Tips:

If a temperature or pressure program ends before the analytical run ends, the temperature or pressure remains at the final value.

This table editor design requires an initial entry to set the default values before you begin editing. Press any key to create the default values, then begin editing.
