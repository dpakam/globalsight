Plot Color Preferences Tab

Use this tab to define the format of lines in Empower plots on the screen and in print. Modifications made here affect all plots in Empower software. However, you can override these plot color preferences in individual Empower applications (for example, Review and Report Publisher).

Pen to Edit

Displays the individual line types that can be used in plots.

To modify the color and style of a plot line, click the option button of the line type you want to modify, then specify the line style, thickness, and color (see below) in the Attributes section of the tab.

You must edit individually each plot line type that supports modification. You can, however, format each plot line type differently from each other.

Attributes

Line Style

Specifies the line style to use for the plot line type.

Choices: Solid, Dashed, Dotted, Dash Dot, and Dash Dot Dot

Default: Solid

Thickness

Defines the line weight.

Entries: 0 to 1000

Default: 0

Line Color

Specifies the line color. Click Line Color to access the Color dialog box. You can use all colors supported by the Windows operating system, which includes basic and custom colors.

Default: Black

Background

Defines the background for the object, which can be transparent, transparent when printing (Review or Compare only), or a user-specified color. You can specify a color for the plot background, or plot axis (dependent on the specific report group).

To choose a color as the background, clear the Transparent check box, then click Bkgnd Color or Axis Color to access the Color dialog box. You can use all colors supported by the Windows operating system, which includes basic and custom colors.

If you print an active plot from either Review or Compare, you have the option to specify Transparent When Printing from the Colors tab. This selection instructs Empower software to print the active plot on a white background even though a colored background may be specified for viewing in Review or Compare.
