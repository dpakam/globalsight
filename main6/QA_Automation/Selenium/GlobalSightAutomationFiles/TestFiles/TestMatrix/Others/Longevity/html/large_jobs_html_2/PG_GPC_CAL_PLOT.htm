GPC Calibration tab

Use this tab to define characteristics of a GPC calibration curve displayed in Review or Compare, or in a report.

Use this tab to define characteristics of a GPC calibration curve displayed in Review or Compare.

Information to report

Specifies which calibration curve information to include in the report:

  • All Points – All points used to create the calibration curve or Viscosity Law curve.
  • Ignored Points – Points that were specified to be ignored when calculating the calibration curve or Viscosity Law curve.
