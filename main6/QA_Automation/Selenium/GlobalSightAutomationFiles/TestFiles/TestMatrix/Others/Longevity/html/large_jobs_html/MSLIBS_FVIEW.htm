MS Library Search tab

Use this tab to specify:

  • MS library match parameter values
  • Extraction parameters for MCA spectra

Specifying MS library match parameter values

Search Mode

How the software searches the spectral libraries and compares the unknown spectrum with the library spectra. Click for details on the two search modes.

Choices: Pure or Mixture

Default: Pure

Reference Set

Size of the set of library spectra used during the presearch phase of the library search.

Choices: 1 or 3. A value of 3 increases the number of library spectra selected during the presearch phase of the library search.

Default: 1.

Note: A value of 1 is adequate for most library searches. Increasing the number of library spectra increases the reliability of the search in approximately 7% of unknowns. However, a large Reference Set also slows the speed of the library search.

Search depth

Maximum number of reported library matches you want the MS software to list in the results.

Choices: 1, 2, or 3

Default: 3

For example, with a search depth of 3, the software performs all possible searches, then selects and reports the best three matches.

Mass limit (m/z) start, end

Limits the search to library spectra with masses within the specified range. You use the m/z range, which should include at least the six most important peaks to focus on the spectral features of interest. Range is 2 to 4000.

Notes: If you specify a value outside the range of the acquired data, Empower software uses the limits of the acquired data.

The End Mass value must be greater than the Start Mass value.

Search threshold (%)

The PBM Fit values that Empower MS software reports. The software discards matching spectra with a PBM Fit value less than the Search Threshold value.

Entries: 0.000 to 100.000

Default: 50.000

Noise threshold (%)

Removes noise from spectral data before a library search. The threshold value is applied as a percentage of the Base Peak intensity. Mass intensity values in the sample spectrum less than the Threshold value are set to zero. During a library search, the PBM algorithm ignores ion masses with intensity values of zero. A blank field indicates no noise threshold.

Entries: 0.000 to 100.000.

Default: 0. 000.

Note: The Threshold value does not affect the original spectral data. You can change the Noise Threshold value and perform additional searches on the same data.

Spectra to average

Number of spectra to be averaged to produce the extraction. When spectra are extracted, they can be averaged over a series of scans to reduce the noise level in the spectra. For example, a value of 1 indicates no averaging. A value of 3 averages the extracted spectrum with the spectrum on each side of the extracted spectrum.

Entries: 1 to 30000

Default: 1

Combine method

Indicate how to combine a spectrum:

  • Average – Averages the intensities of the spectrum.
  • Sum – Adds the intensities of the spectrum.

Noise baseline (min)

A value that specifies in minutes how Empower software calculates a baseline spectrum:

  • Null (Default) – When this field is clear, the peak start and ends define the baseline spectrum.
  • Value – When you enter a value, Empower software uses the spectra collected during the times specified in this field to calculate the baseline spectrum. When you enter a range, Empower software averages all spectra in the specified time ranges to produce a noise spectrum, which is subtracted from any extracted spectra.

Entering values: The format for entering minutes is T1:T2, T3:T4,… . For example, to specify a noise spectrum of 1 to 3 minutes and 5 to 7 minutes, enter the following: 1.0:3.0, 5.0:7.0.

Multiplier

A value by which to multiply the Noise Baseline.

Default: 1.0

Libraries

The library or libraries you want to search, including all spectral libraries in the current project. The list contains user-created project libraries and any optional commercial library (for example, Wiley for the MS software option) installed in the system.

When you specify the spectral libraries:

  • Verify that both the user-created libraries and the unknown spectra span the mass range that contains all the characteristic ions of the analytes.
  • Be aware of the effect of the mobile phase on the unknown spectra.
  • Be aware that the best matches are based on library spectra acquired from your own standards.

Peak separation (da)

Binning value to apply to the baseline-corrected spectrum. Empower software bins the extracted and combined spectrum at the resolution in this field.

Wiley duplicate spectra

Increases the likelihood of matching a library spectrum acquired under similar lab conditions if you are using the Wiley Library. For many compounds, the Wiley Registry contains multiple spectra acquired under a variety of conditions. However, the library search takes longer.

Default: Disabled

Retention time presearch

Limits the search to spectra with retention times a specified range, thus reducing search time. Search spectra with retention times similar to your acquired spectrum.

Default: Disabled

Note: This applies only to user-created libraries because the Wiley Registry spectra have no associated retention time information.

RT window (%)

Range of acceptable retention times as a percentage of the actual retention time of the target spectrum. If you enable Retention Time Presearch, enter a retention time (RT) window value.

Entries: 0.0 to 100.0

Default: 10.0

Specifying Extraction Parameters for MCA Spectra

Specify a section of an MCA spectrum for review and adjust its appearance with these parameters:

  • Start Mass Limit – Mass value that marks the beginning of the MCA spectrum section you want to review. The range should span at least six significant peaks.
  • End Mass Limit – Mass value that marks the end of the MCA spectrum section you want to review. The range should span at least six significant peaks.
  • Noise Threshold –Level of noise you allow in the section of interest.
  • Peak Separation – A spectral mass binning value—a bin is a scan wide and a mass high—and the number of spectral points the software combines to constitute the spectrum.
