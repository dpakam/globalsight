Custom Fields Tab (Project)

Use this tab to view existing custom fields and their parameter values.

  • To modify all project properties, log in to Empower software using an account with the Alter Projects privilege. If you do not have this privilege, you can modify specific project properties using the Change Project Owner, Change Project Quota, Alter Custom Fields, Create Custom Field, and/or Delete Custom Field privileges.
  • The Alter Projects privilege includes all rights associated with the Change Project Owner, Change Project Quota, Alter Custom Fields, Create Custom Field, and Delete Custom Field privileges. To prevent users from modifying project properties, remove all above privileges.

Table Columns

Column

Description

Name

Name of the Custom Field that appears in the table column heading.

Type

Data type (Integer, Real, Text, Date, Bool, or Enum) for the custom field.

Field Type

Field type of the selected custom field.

Width

Maximum number of characters that can be printed or displayed.

Precision

Number of characters that can be printed to the right of the decimal point for real numbers.

Minimum

Minimum value for the custom field for keyboard-entered fields with an integer or real data type.

Maximum

Maximum value for the custom field for keyboard-entered field with an integer or real data type.

Default

Default value specified when the custom field was created or modified.

Source

Origin (Keyboard, External, or Calculated) of the data for the field.

Required

Indicates that users must enter the value for the field before an acquisition run is started.

Formula

Custom formula created for a software-calculated field.

CField Id

Unique number that the software assigns to a custom field.

All or Nothing

Indicates whether the All or Nothing flag is enabled. If selected, the field will not generate a value on subtractions or additions unless all values are present.

Search Order

When an intersample label is specified, the order in which results are searched.

Use As

A Boolean or Enumeration type custom field can be used as an index, text, or value.
