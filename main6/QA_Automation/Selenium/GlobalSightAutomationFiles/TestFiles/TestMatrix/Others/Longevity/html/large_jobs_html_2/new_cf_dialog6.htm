New Custom Field wizard – Name Entry page

Use this page to name the new custom field and create the custom field in the current project or in multiple projects.

Field

Description

Field Name

Name for the new custom field (up to 29 characters). You can use any alphanumeric characters, but the name cannot begin with a numeric character. The only symbol character that you can use is underline (_).

Rule: Certain words in the Empower database are reserved exclusively for use by Empower software. Do not use these reserved words to name a custom field.

Create this field in

Project:

Specifies that the custom field be created only in the project that you selected before you activated the New Custom Field wizard.

Multiple Projects:

Specifies that the custom field be created in more than one project. When you select this option and click Finish, the Custom Field: Select Projects dialog box appears with a list of projects to which you have access. You select the projects in which you want Empower software to create the custom field, then click OK. The custom field is saved to all the projects that you select.

Tip: Hold down the Ctrl key before selecting other projects (the name of the project that you select before you activated the New Custom Field wizard is highlighted automatically).
