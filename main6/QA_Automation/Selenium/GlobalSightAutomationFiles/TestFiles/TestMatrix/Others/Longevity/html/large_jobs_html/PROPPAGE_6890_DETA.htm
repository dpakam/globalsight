Detector tab (6890)

Use this tab to specify control parameters for the front or back detector of the 6890 GC system.

Settings

Detector type

Type of detectors used in the 6890 GC system.

Choices: Unknown (used for other detector types), None, FID (flame ionization), TCD (thermal conductivity), ECD (electron capture), NPD (nitrogen phosphorous), FPD (flame photometric), and µECD (micro electron capture)

Default: None

Flame enable

Enables autoignition of the flame when an FID or FPD detector type is selected.

Choices: Enabled or Disabled

Default: Disabled

Note: Enabling the flame activates temperature and gas requirements. A temperature of >150 °C must be set before you can save the instrument method.

Lit offset

Threshold you can set to initiate igniting a FID or FPD detector. If the detector goes below this value, the flame is out and the detector will try to relight itself.

Target output

Enter the target output for the NPD detector.

Entries: 10 to 99

Default: 30

Filament enable

Enables the filament on a TCD detector.

Default: Disabled

Note: When the filament is enabled, reference gas flow is required and the temperature must be >150 °C.

Temperature settings

Enable

Enables the temperature set point parameter for the detector. Enable this check box before the Set Point parameter takes effect (see below).

Choices: Enabled or Disabled

Default: Disabled

Setpoint

Specifies the detector temperature set point value for the 6890 GC system. Enable the Enable parameter for this parameter to take effect.

Default: 150 °C

Polarity settings

Neg polarity

Turns the negative polarity for the TCD detector on or off.

Default: Off

Note: If you select On, the processing method must allow negative peaks.

Switch time

When to change the polarity of the TCD output signal during a run. Negative polarity returns to its original state at the end of the run.

Default: 0.0

Pneumatics

Oxidizer flow

Flow rate for the oxidizer (air) gas.

Default: 1.0 mL/min

Fuel flow

Flow rate for the detector fuel (carrier) gas.

Default: 1.0 mL/min

Reference flow

Reference gas to the TCD detector.

Default: 1.0 mL/min

Flow mode

Makeup gas flow mode. Select Constant Makeup to provide a constant flow of makeup gas to the detector. Select Constant Makeup + Column to provide a variable flow of makeup gas to the detector (gas flow varies as column flow increases or decreases).

Default: Constant Makeup

Makeup/combo flow

Flow rate of the makeup gas to the detector. If you select Constant Makeup for Flow Mode, this value specifies the constant makeup gas flow rate. If you select Constant Makeup + Column for Flow Mode, this value specifies the constant combined (column + makeup) flow rate to the detector.

Default: 1.0 mL/min
