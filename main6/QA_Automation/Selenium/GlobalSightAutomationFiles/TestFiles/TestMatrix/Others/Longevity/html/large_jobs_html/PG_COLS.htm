Columns tab

Use this tab to define the format of the table columns in a specific table in the software.

Hidden columns

Individual table column titles contained in the associated table. Selecting the check box to the left of a table column title hides the column. Clearing the check box shows the column.

Note: To make all hidden columns within the table reappear, right-click any column of the table, then select Show All.

Column sizing

Width of the columns within the table:

  • Automatic – Changes the width of the table column to fit the length of the longest entry in that column.
  • Manual – Column sizing remains fixed at the set size. If a table column entry is longer than the column, the text in the column is clipped.

Fixed column

Selects a specific column in the associated table that is to always remain visible, along with all columns to the left of the selected column as you scroll through the table. You also have the option not to specify a column (choice of blank) so all columns move as you scroll through the table. To make a selection, click the Fixed Scroll Column list, then select the appropriate column name (or choose the blank selection if you do not want a fixed column).

Hide all

Selects all column names in the Hidden Columns list. When you click OK, all columns in the selected table disappear.

Show all

Clears all column names in the Hidden Columns list. When you click OK, all available columns for the selected table appear in the table.
