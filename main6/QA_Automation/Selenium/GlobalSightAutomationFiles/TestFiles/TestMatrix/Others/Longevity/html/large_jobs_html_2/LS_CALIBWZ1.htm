LS Calibration wizard

Access the LS Calibration wizard in Review by selecting Process > Calibrate Light Scattering System. This menu selection is enabled when the LS option is activated and there is a current channel in Review.

Note: If you do not have a processing method open, Empower launches the Processing Method wizard. Step through the wizard to establish the processing parameters for all channels. When reviewing subsequent channels, click Back twice to open the Integration – Peak Detection page to edit the peak width. When you finish the Processing Method wizard, Empower launches the LS Calibration wizard.

On the Initial LS Calibration page, click either:

  • Create a new calibration – Creates a new calibration based on your selections.
  • Update existing calibration – Modifies an existing LS calibration and saves it as the current calibration for that solvent-detector combination. This option is disabled if the LS calibration table (in the Curves tab of the Project window) is empty.

The LS calibration table on the Initial LS Calibration page displays all existing LS calibrations that have the same instrument method, system, and solvent as the current channel in Review.

Click Next to go to the LS Calibration Procedures page.

Depending on your selections, subsequent pages can include:

  • Detector and Solvent page
  • Normalization Standard page
  • Normalization Constants page
  • Calibration Standard page
  • Calibration Constants page
