Access tab (chromatographic system)

Use this tab to control how users access the selected chromatographic system.

Share System with Other Network Users

Specifies that other network users (determined by the access option you select; see below) can use the system.

Allow Access

Owner Only

Specifies that the owner of the chromatographic system and users with Administrator privileges can access the selected chromatographic system.

Owner and Group(s)

Specifies that the owner of the chromatographic system, group members as specified in the Allow Access to Groups list, and administrators can access the selected system. Group access only allows a user to operate a chromatographic system.

Note: To modify a chromatographic system, you need access to the chromatographic system and its Empower Node.

Owner, Group, and World

Specifies that all users in the database can access the selected chromatographic system.

Password Required

Specifies that users need a password to access the selected chromatographic system across databases.

Password

Specifies a password (up to 30 alphanumeric characters) to limit the use of the system to authorized users.

Confirm Password

Requires reentry of the password to confirm it was spelled correctly.

Owner

Specifies that the user has access to the selected chromatographic system. The field is enabled if you are the current owner of the selected chromatographic system or if you have Administrator privileges.

Allow Access to Group(s)

Select the user group(s) that you want to give access to the selected chromatographic system. If selected, access privileges of these user groups override access privileges of the selected world user type. If you select Owner, Group, and World, groups that were not selected in the Allow Access to Groups list are assigned the World User Type Access.
