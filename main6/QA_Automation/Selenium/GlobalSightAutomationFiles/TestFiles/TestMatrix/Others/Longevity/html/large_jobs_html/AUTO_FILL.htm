Auto Fill dialog box

Use the Auto Fill dialog box to perform the following tasks:

  • Insert vial numbers and related information into the Samples table. You specify whether the information is for standards or samples.
  • Modify information for existing vials (standards and/or samples).
  • Generate sample names with automatic prefix and suffix incrementation.

Tip: For details on using Auto Fill to modify information in the Samples table, click.

Note: The Auto Fill dialog box displays various parameters, depending on the column (in the Samples table) that you right-clicked.

Selected table column field

Specifies the information that you want to auto fill into the current column in the Samples table.

Incrementing prefix

If you select the SampleName, Label, Label Reference, Level, or Auto Additions column, this field adds and increments a prefix to the sample name. Do not mix numbers and letters in the same prefix.

Incrementing suffix

If you select the SampleName, Label, Label Reference, Level, or Auto Additions column, this field adds and increments a suffix to the sample name. Do not mix numbers and letters in the same suffix.

Which rows to fill

Specifies three options for filling rows in the table.

Vials

Specifies the range of rows (defined by vial number) in the selected column that is to be automatically filled with the specified column information.

All rows

Specifies that all rows in the selected column in the Samples table are to be automatically filled with the selected column information.

Selected rows

Specifies that only the rows you selected in the Samples table prior to activating the Auto Fill dialog box are to be automatically filled with the selected column information.

Fill these row types

Specifies the type of row to be auto filled.

Inject standards

Specifies that only rows that contain an Inject Standards function be automatically filled with the specified information.

Inject samples

Specifies that only rows that contain an Inject Samples function be automatically filled with the specified information.

Non-injection rows

Specifies that only rows that contain a noninjection function be automatically filled with the specified information.
