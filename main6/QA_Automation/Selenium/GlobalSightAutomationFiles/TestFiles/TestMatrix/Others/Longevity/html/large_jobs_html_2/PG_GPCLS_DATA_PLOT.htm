LS Data Plot tab

Use this tab to define characteristics of the LS Data plot displayed in Review or in Compare, or printed in a report.

Axis data

Specifies data to include in axes of the LS Data plot.

X-Axis Data:

  • Slice EV – Plots the Elution Volume (in mL) of each slice adjusted for the flow rate entered in the Calibration tab of the GPC Processing Method window. Elution Volume is equal to Adjusted Retention Time if you did not enter a flow rate (assumes a flow rate of 1.0 mL/min). Default selection for X-Axis Data.
  • Slice RT – Plots the retention time of each slice.

Left-Axis Data, Right-Axis Data

Data displayed along the left and/or right axes of the LS Data plot:

  • Blank – No data is displayed.
  • Slice MW – Plots the molecular weight of the polymer at each slice. Default selection for Plot4 Data.
  • Slice Log MW – Plots the log (base 10) of the molecular weight of each slice.
  • Slice RT – Plots the retention time of each slice.
  • Slice Adj RT – Plots the retention time of each slice adjusted for variations in retention time as observed relative to a reference peak.
  • Slice EV – Plots the Elution Volume (in mL) of each slice adjusted for the flow rate you entered in the Calibration tab of the Processing Method window. Elution Volume is equal to Adjusted Retention Time if you did not enter a flow rate (assumes a flow rate of 1.0 mL/min).
  • Slice Area – Plots the area of each slice, which is the height of the integrated peak at the reported retention time multiplied by the duration of the slice.
  • Slice Height – Plots the height of each slice of the integrated peak at the reported retention time.
  • Slice Conc (GPCV or GPC/V-LS Option) – Plots the concentration of the slice based on the area of the slice and the concentration calculated for the peak. Used in viscometry calculations (GPCV or GPC/V-LS processing only).
  • Slice Observed MW (GPC/V-LS Option) – Plots the molecular weight calculated for the slice by extrapolating the light scattering data to the zero-angle. Default selection for Left-Axis Data.
  • Slice Observed Rg (GPC/V-LS Option) – Plots the log observed radius of gyration calculated for the slice by extrapolating the LS data to the zero-angle using the Rg Fit specified in the Slicing tab in the Processing Method window. Use this for comparison with the theoretical linear and fitted curve. Default selection for Right-Axis Data.
  • Slice Fitted Rg (GPC/V-LS Option) – Plots the log fitted radius of gyration calculated for the slice by fitting the LS data according to the Slicing tab field Rg Fit parameter. Default selection for Plot5 Data.
  • Slice R0 (GPC/V-LS Option) – Plots the Rayleigh ratio at scattering angle 0. Default selection for Plot3 Data.
  • Slice R1, R2, R3 (GPC/V-LS Option) – Plots the slice height of the first to third LS peaks.
  • Plot3 Data Data – Displays a third data plot in the LS Data plot. See above for a complete listing of plot choices.

Additional plots

Displays up to five additional data plots in the Viscosity Law plot (Plot4 to Plot8). See the X-Axis Data parameter for a complete listing of plot choices. If desired, you can select Blank as a data plot choice (no data is displayed).
