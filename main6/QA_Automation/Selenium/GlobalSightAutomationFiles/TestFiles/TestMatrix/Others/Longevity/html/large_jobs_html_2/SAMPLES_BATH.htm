Bath A or B Samples page

Use this page to enter sample parameters for Bath A and/or Bath B. This page appears only when you select a dissolution type in the Select Sample Set Method Type page of the Sample Set Method wizard.

First vial position

The starting vial position for a range of vials to be filled. For example, for 8 syringes and 5 sample transfers, you need 40 vial positions. If you specify 9 as the first vial position, vials 9 to 48 are filled.

Entries: with the 2690D/2695D, 1–120; without the 2690D/2695D, 1–999

Default: 1

2690D/2695D Considerations

The position of the first sample vial must be divisible by 6 or 8 (depending on the number of syringes in the Waters Transfer Module) plus 1. For example, in a 6-syringe system, you can specify #13 as the first sample vial position (12 + 1), but not #11.

Sample transfers cannot occur between carousels. The 2690D/2695D allows up to three for an 8-syringe system and four transfers per carousel for a 6-syringe system. However, it does not allow the transfer of four samples to one carousel and four samples to another carousel from the same sample transfer.

Transfer times

Click a row in the Transfer Times table to specify the time (in minutes) to transfer each sample.

Note: The maximum number of vial positions in the carousels is 120.

If you have a 2690D/2695D and run at least one dissolution test, you cannot edit the transfer times. If you do not have a 2690D/2695D and run at least one dissolution test, you can edit the transfers times from the Alter Samples menu.

Entries: 1.00 to 59999.99 with no more than 20 entries in ascending order and no duplicate times.

Minimum entry: 1 at 1:00

Dissolution test duration

Specifies the total length of time (in minutes) to run the dissolution test.

Entries: 1.00 to 59999.00 min

Default: 60.00

Perform infinity test

Runs an infinity test after the timed transfers in a dissolution test finishes. An infinity test spins the apparatus (if applicable, paddles or basket) at 200 rpm (for a Hanson bath v. 2.1 or lower) or 250 rpm (for a Hanson bath that is v. 2.1 or higher) for Bath A or B. Enter the length of time (in minutes) you want to perform the infinity test. The high rate of speed allows the dosage form to completely dissolve in the media before another aliquot of sample is transferred to vials.

Entries in the time field: 2.00 to 599.00 min

Typical entries: 2.00 to 20.00

Default: Disabled

Collect all samples before any LC (2690D/2695D only)

Transfers all aliquots in the sample set to vials before any samples or standards are injected.

Offsets bath A or B (2690D/2695D only)

If you selected Dissolution (Baths A and B) in the Select Sample Set Method Type page, Bath A and B Offset appear on the Bath B Samples page.
