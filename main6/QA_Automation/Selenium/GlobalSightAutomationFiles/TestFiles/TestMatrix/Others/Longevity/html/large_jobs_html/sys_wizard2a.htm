New Chromatographic System wizard – System Selection page

Use this page to add instruments to a new chromatographic system.

To add instruments:

  1. Select an instrument from the Available Instruments list and drag it to the New System Instruments list. You can use instruments from the Unused Components and/or an existing system. If the instrument is not in the Available Instruments list, click Cancel, and check the properties of the acquisition server.
  2. Click Next.

Note: You cannot drag the last instrument from an existing system. To free up the last instrument on a system, delete the system.
