Axial Dispersion Tab (GPC)

Use this tab to specify axial dispersion correction parameters. Because the shape of the molecular weight distribution calculated for a broad peak depends on the peak shape, axial dispersion (band broadening) can cause inaccurate results (molecular weight distribution, intrinsic viscosity, and branching) to be calculated for a broad peak.

Tips:

  • Axial dispersion parameters correct the axial dispersion (band broadening) of the data that occurs during a chromatographic separation, especially when large-particle (20-mm) columns separate the GPC samples.
  • When axial dispersion is enabled for GPC/V-LS data, the same correction is performed on all the GPC channels collected in the injection. The corrected peaks are used to calculate the GPC/V-LS results.

Parameters

Description

Axial dispersion correction

Type of axial dispersion correction you want to apply to the data:

Type

Description

None

Axial dispersion correction is not applied to the data.

Iterative Deconvolution

Applies a small series of corrections to the data that terminate at a point of diminishing returns. Use iterative deconvolution for any peak shape, including multi-modal. This correction type causes some increase in baseline noise.

Peak Compression

Calculates an axial dispersion shape and uses it to compute a new chromatogram by stretching the time axis, which has the effect of narrowing the peak. This correction type is ideal for symmetrical (Gaussian-shaped) peaks. If the baseline is noisy, peak compression correction is preferred over iterative deconvolution correction. Do not use peak compression correction for multi-modal peak shapes or highly tailed peaks.

Deconvolution shape

Sigma

Values for the variance (width) constant, sigma, used to calculate a normalized Gaussian curve, which is the shape of the axial dispersion function.

Entries: 0.001 to 10.000 (mL).

Noise interval (Iterative deconvolution only)

Start (min), End (min)

Segment of the chromatogram baseline used to determine the convergence of the algorithm. Convergence depends on minimizing root mean square (rms) deviation between the current trial solution and the correct solution. Enter values for Start (min) and End (min).

Rule: If you select Iterative Deconvolution as the Axial Dispersion Correction type, the time span between Start and End must be between 0.5 and 2.0 minutes.

Noise reduction (Iterative deconvolution only)

Smoothing

Degree of smoothing used during the iterative deconvolution type of axial dispersion correction.

Choices: None and 5 to 101 points

Default: None
