Chromatogram tab

Use this tab to define characteristics of a chromatogram when displayed in the Chromatogram plot in Review (or Compare) or in a report.

Use this tab to define characteristics of a chromatogram when displayed in the Chromatogram plot in Review (or Compare).

Chromatogram tick marks

Enable tick marks

Tick marks in the Chromatogram plot.

Label tick marks

Specifies if the tick marks are to be labeled directly in the Chromatogram plot. Tick mark labels are positioned between the marks on the Chromatogram plot to indicate the range associated with the tick marks.

Parameters

Note: Select Enable Tick Marks to define a tick mark time window.

Defines the time window in which to display tick marks within the chromatogram:

  • Start Time – Time after injection at which to start the display of tick marks in the Chromatogram plot.

Entries: 0.0 to 655.90 minutes

Default: Blank

  • End Time – Time after injection at which to stop display of tick marks in the Chromatogram plot.

Entries: 0.0 to 655.90 minutes

Default: Blank

  • Increment – Time interval (in seconds) at which to display tick marks in the time window in the Chromatogram plot.

Entries: 0.0 to 39354.00-second increments

Default: Blank

Chromatogram Fraction Regions

Display Fraction Regions

Controls the overall display of fraction regions.

Enable Fraction Shading

Draw a gray shaded background in the fraction regions for the Overlay Stacked or Non-overlay chromatograms.

Draw Fraction Lines

Draw a vertical line at the start and end of fraction regions using the "Tick Marks/Fractions" pen. If a fraction spans multiple vessels, it can have several fraction regions drawn, which are actually part of the same fraction. In this case, the start line of the first region and the end line of the last region associated with the fraction, will be drawn using the line style defined by the "Tick Marks/Fractions" pen, all fraction lines between them will be dotted lines so that you can easily see how all these lines comprise one fraction.

Label Fractions

Draw a label in the fraction region using the "Tick Marks/Fractions" font.
