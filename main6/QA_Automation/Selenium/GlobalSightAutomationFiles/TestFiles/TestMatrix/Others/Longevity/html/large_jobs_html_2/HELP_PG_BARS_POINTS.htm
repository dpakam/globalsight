Bars/Points tab

Use this tab to define characteristics of the System Suitability Summary plot when printed in a report.

Bars/Points to edit

Type of data to display in the System Suitability Summary plot.

Choices: Normal Data, Faulted Data.

Line attributes

Line style

Line style to use for the plot lines.

Choices: Solid, Dashed, Dotted, Dash Dot, Dash Dot Dot.

Default: Solid.

Thickness

Line weight of the plot lines.

Entries: 0 to 1000.

Default: 0 (thinnest line of 1 pixel).

Note: Waters recommends that you specify a line thickness of 0 when using line styles other than Solid. Line thicknesses vary with monitor and printer resolution, which can result in the various line style selections being displayed as Solid. A selection of 0 ensures that the line styles appear properly.

Line color

Color of the lines used to create the plot. Click Line Color to access the Color dialog box. You can use all colors supported by the Windows operating system, which includes basic and custom colors.

Fill attributes

Transparent

Transparent background for the plot lines. When this property is selected, objects behind the System Suitability bar plot line are visible.

Solid

Solid fill pattern for the plot lines. When this property is selected, objects behind the plot line are not visible. You can choose a solid fill color for the rectangle by clicking the Fill Color button (see below).

Note: You must clear the Transparent check box to access the Solid option button.

Hatch

Hatch fill pattern for the plot lines. Choices: Horizontal, Vertical, Forward Diagonal, Back Diagonal, Cross, Diagonal Cross. Default: Forward Diagonal.

You can choose a fill color for the hatch fill pattern in the bar plot by clicking the Fill Color button (see below).

Note: Clear the Transparent check box to access the Hatch option button.

Fill color

Color of the plot lines (for the Solid or Hatch fill pattern). Click Fill Color to access the Color dialog box. You can use all colors supported by the Windows operating system, which includes basic and custom colors.

Point size

Point size used to display the plot.

Entries: 0 to 10000.

Default: 100.
