New View Filter wizard - Save View Filter page

Use this page to name and save your view filter.

To name and save your view filter:

  1. Enter a name for the view filter in the Name field.
  2. Click Public if you want other users to access the filter. A new filter is private by default (designated by a ! prefix).
  3. Click Finish to save the view filter and load it into the table in the View Filter Editor.
