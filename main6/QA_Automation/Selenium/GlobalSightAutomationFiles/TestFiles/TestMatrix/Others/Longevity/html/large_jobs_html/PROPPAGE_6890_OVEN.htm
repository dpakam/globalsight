Oven tab (6890)

Use this tab to define operating parameters for the oven in the 6890 GC system.

6890 considerations

Keep these considerations in mind:

  • The 6890 GC system measure run time by the length of the oven time. It will not accept data after the oven time expires.
  • Although you can physically set times to 999.99 on the 6890 instrument, when the instrument is under Empower control, the maximum time entry is 650.00.

Oven enable

Turns on the oven. Enable this parameter before any oven temperature settings take effect.

Choices: Enabled or Disabled

Default: Enabled

Cryo enable

Enables use of cryogenic cooling.

Choices: Enabled or Disabled

Default: Disabled

Note: Set the cryo type through the instrument front panel.

Temperature ramp enable

Enables use of the Temperature Ramp table.

Choices: Enabled or Disabled

Default: Disabled

Maximum temperature

Maximum oven operating temperature. Set Initial Temperature to a value no greater than the Maximum Temperature value. Enable Oven Enable for this setting to take effect.

Default: 70.0 °C

Equilibrium time

Time required for the oven temperature to equilibrate whenever the temperature is modified from one run to the next. Equilibrium time begins when the actual oven temperature comes within 1 °C of the Initial Temperature setting.

Default: 0.00 min

Initial temperature

Initial oven temperature at the start of the run. This temperature value is retained for the length of time defined in the Initial Time parameter. Once the Total Time value expires at the end of the run, the oven returns to the Initial Temperature value.

Default: 70 °C

Initial time

Length of time that the oven remains at the Initial Temperature once the run has started.

Default: 60.00 min

Temperature ramp table

Allows you to program up to six oven temperature ramps during a run. In each row of the table, define:

  • Rate – Rate at which the oven is heated or cooled to Final (temperature) in °C/min.

Default: 0.0 °C/min

  • Final (temperature) – Temperature the oven reaches at the end of the heating or cooling ramp. Final (temperature) is maintained for the Hold Time duration (see below).
  • Hold Time – Time in minutes that the oven temperature is held at the current Final Temperature before starting the next ramp or terminating the run.
  • Total Time – Total time as calculated by Empower software. Total Time displayed in the first row is derived from the difference between the Initial Temperature (at the start of the ramp) and the Final temperature (at the end of the ramp) divided by the Rate, plus Hold Time plus the Initial Time. Total Time displayed for subsequent rows is derived from the difference between the Final temperature from the previous row and the Final temperature for the current row divided by the Rate, plus Hold Time plus the Total Time from the previous row.

Note: Empower software only collects data from the 6890 during the time the oven temperature is programmed. Guidelines:

  • For non-ramped methods, this is the initial time.
  • For ramped methods, this is the total time.
