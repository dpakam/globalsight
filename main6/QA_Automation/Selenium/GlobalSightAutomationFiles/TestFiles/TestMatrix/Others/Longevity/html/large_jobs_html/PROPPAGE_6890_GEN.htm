General tab (6890)

Use this tab to define detector and inlet types, and to set the enable states of the injectors, columns, detectors, and channels in the 6890 GC system.

Note: When creating an instrument method for Dual Tower operation, you must enable both injectors/inlets, columns, detectors, and channels.

Configuration parameters

Inlet type (front and back)

Type of inlet used in the 6890 GC system.

Choices: None, CIS4 (Gerstel PTV inlet with auxiliary temperature control), COC (Cool on-column, non-EPC), COC EPC (Cool on-column, EPC), PP (Purged packed, non-EPC), PP EPC (Purged packed, EPC), PTV (Programmed Temperature Vaporization, EPC), S/SL (Split/Splitless, non-EPC), S/SL EPC (Split/Splitless, EPC), Volatiles Interface, and Unknown (used for other inlet types). You can also set Inlet type in the Front and Back Inlet tabs.

Note: For the PTV, you can set only the front inlet.

Default: None

Lit offset

Threshold you can set to initiate igniting a FID or FPD detector. If the detector goes below this value, the flame is out and the detector will try to relight itself.

Default: 2.0 (pA). If two attempts to re-ignite fail, the GC performs a safe shutdown and shuts off the gas flow to the detector.

Detector type (front and back)

Type of detectors used in the 6890 GC system.

Choices: Unknown (used for other detector types), None, FID (flame ionization), TCD (thermal conductivity), ECD (electron capture), NPD (nitrogen phosphorous), FPD (flame photometric), and µECD (micro electron capture). You can also select detector types in the Front and Back Detector tabs.

Default: None

Enable parameters

Injector (front and back)

Enables the injector (front and/or back). After enabling an injector, define its parameters in the corresponding Front and/or Back Injector tabs.

Choices: Enabled or Disabled

Default: Disabled

Note: If you enable an injector, you must enable the corresponding inlet.

Inlet (front and back)

Enables the inlet (front and/or back). After enabling an inlet, define its parameters in the corresponding Front and/or Back Inlet tabs.

Choices: Enabled or Disabled

Default: Disabled

Column (column 1 and column 2)

Enables column 1 and column 2. After enabling a column, define its pneumatic parameters in the corresponding Column tab.

Choices: Enabled or Disabled

Default: Disabled

Note: Use the GC front panel to verify that the Columns are programmed for connection to the correct detectors. They should be connected as you want the Empower instrument method to be.

Detector (front and back)

Enables the front detector and/or back detector in the 6890 GC system. After enabling a detector, enter its parameters in the corresponding Detector tab.

Choices: Enabled or Disabled

Default: Disabled

Note: Enable the front and/or back detector before specifying detector parameters in the corresponding detector tab. If you specify detector parameters in the Detector tab without first enabling the detector, the parameter settings do not take effect.

Channel (1 and 2)

Specifies that the channels transmit digital data through the 6890 GC system RS-232 port to the Empower workstation. After enabling a digital channel, define the Source and Units parameters in the appropriate channel tab.

Choices: Enabled or Disabled

Default: Enabled

Notes:

  • If you are injecting on inlet 1, channel 1 should be set to the detector to which inlet 1 (column 1) is linked. The same rule applies to channel 2 and inlet 2.
  • You can simultaneously transmit digital data (RS-232 port on the 6890 GC system) and analog data (Signal 1 and Signal 2 terminals on the 6890 GC system) to an Empower workstation. You need not enable the analog outputs; they are always active. The analog outputs must be converted to digital signals by an A/D converter (typically a busSAT/IN or SerialSAT/IN 2 Module) before they are transmitted to the Empower workstation. See the Empower System Installation and Configuration Guide for information on connecting your 6890 GC system to an Empower workstation through a busSAT/IN or busSAT/IN 2 Module.
