Configuring DHCP settings

The Waters DHCP Server automatically assigns IP addresses to Ethernet instruments that are attached to an instrument network card in the acquisition server. When the Waters DHCP Server assigns an IP address to an instrument, it is given that IP address for 30 days (its lease time).  If the instrument is powered on, the lease time will be automatically renewed and it will continue to use the same IP address (even if the instrument is off for short periods, it will still use the same IP address).  If the instrument is powered off for a long period of time (for example, until the lease time expires) then the IP address is available to be reused if another instrument requests an IP address.

Tip: Most of these activities take place without user intervention; however, some tools will need to be accessed occasionally.  

To access the Waters DHCP Server tools:

  1. Access Configuration Manager in the Acquisition Servers view.
  2. Click Manage > Systems.
  3. In the Manage System dialog box, select a chromatographic system and then click Node Properties.
  4. Right-click the row to select the appropriate acquisition server in the View table, and then click Properties.
  5. In the Acquisition Server Properties dialog box, open the Configure DHCP tab.
  6. In the Breeze 2 Node Properties dialog box, open the Configure DHCP tab.
  7. Click the Configure DHCP button.

Rule: If you have the system policy “Require User Comments on Changes to System Objects” enabled, you must enter a comment after clicking the Configure DHCP button.  A modified acquisition server audit message is created, and a called DHCP entry is entered in the Details column in the System Audit Trail (in addition to the user-entered reason).

  1. From the Server menu, do one of the following:
  • Click Server > Configuration Wizard … to select the instrument network card and set its IP address.
  • Click Server > MAC Address Configuration … to add, remove, or restore the default MAC addresses.
  • Click Server > Errors … to view a list of errors or to clear errors (if any errors have been found).

Configuring Network Connections and IP Addresses

The Configuration Wizard sets an IP address for an instrument network card.  

To set the IP address for an instrument network card:

  1. Select  Server > Configuration Wizard.
  2. Select an instrument network card and click Next.
  3. Click Default (192.168.0.1), then click Next.

Tips:

  1. The IP address will be assigned to the instrument network card. If the default and alternate IP addresses are already assigned to another network card, contact Waters Technical Service for custom settings.
  2. The subnet mask of 255.255.255.0, which is not shown in the wizard, is automatically set.
  3. A second IP address of 64.1.1.1 is assigned to the instrument network card in order to be available for use with Waters mass detectors.
  4. If you select Custom to assign the IP address, the first IP address assigned to an Ethernet instrument will be incremented by 1.  For example, if a custom IP address of 192.168.1.99 is assigned to the instrument network card then the first IP address assigned by the Waters DHCP Server is 192.168.1.100.
  1. Click Finish.
  2. Click Yes. The acquisition server reboots. If the IP address for an acquisition server is being set from a remote client, the acquisition server properties window must be closed. Click OK on the message that appears.

Tip: If you click No in the Waters DHCP Server Configuration dialog box, a message appears to remind you to reboot the computer.

  1. After you set the IP address and the acquisition server has rebooted, restart the Configuration wizard to verify the IP address information.

Tip: The second IP address is for Waters mass detectors and is automatically assigned, regardless of whether or not the acquisition server controls a Waters mass detector.

  1. Click Cancel to exit the wizard.

Configuring MAC Addresses

The DHCP server supplies IP addresses for Waters MAC addresses. Use the MAC Address Configuration utility to add, remove, or restore the default MAC addresses.

The DHCP server supplies IP addresses for Waters, National Instruments, Hewlett-Packard, and Agilent MAC addresses. Use the MAC Address Configuration utility to add, remove, or restore the default MAC addresses.

To add:

  1. Click Add to add a MAC address that is not included in the MAC address list.
  2. Enter the first six digits of the MAC address.
  3. Enter a company name (if applicable).
  4. Click OK.

Result: The MAC address is added to the list.

To remove:

  1. Select the MAC address you want to remove and click Remove.
  2. Click Yes to confirm deletion.
  3. Click OK.

Result: The MAC address is removed from the list.

Restore Defaults

Click Restore Defaults to return to the default addresses.

Errors …

Use the Errors list to view a list of errors (if any errors have been found). Use the Clear Errors button to remove the errors from the table.

Changing Information in the Waters DHCP Server Configuration Table

The IP Address, MAC Address, Type, and Name fields are blank until IP addresses are assigned by the Waters DHCP Server or entered manually. If IP addresses are assigned, the table will be filled in with the applicable information.

To add:

  1. Click Add to manually add a new IP address.
  2. In the Add IP Address dialog box, enter the following:
  3. IP address
  4. MAC address (the instrument’s unique hardware number)
  5. Instrument Type (if applicable)
  6. Serial Number or Unique Name (if applicable)
  1. Click OK.
  2. Power cycle the instrument so that it receives the manual IP address.

Note: If you manually add an IP address, it will never be used by another instrument and it will not be removed from the table unless the Remove button is used.

To edit:

  1. Select the IP address you want to modify and click Edit.
  2. Change the appropriate information and click OK.

Note: Waters instruments that have IP addresses automatically assigned through the Waters DHCP server do not require association with an instrument type or a unique name. If you select an automatically assigned address and click Edit, only the Instrument Type and Serial Number/Unique Name fields are accessible for editing.

Remove

Use the Remove button to remove IP addresses from the table that are no longer being used. If an IP address is entered manually, it will never be removed from the table except by using the Remove button.

To remove an IP address from the table, click on the IP address and then click Remove. The IP address is removed from the table and is available for other instruments to use.
