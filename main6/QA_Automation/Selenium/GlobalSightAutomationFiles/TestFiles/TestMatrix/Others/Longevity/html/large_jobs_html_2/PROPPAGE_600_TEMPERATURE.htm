Temperature tab (600)

Use this tab to specify temperature controls for the optional internal column heater installed in the slide-out drawer of a Waters 600E, 616, 625 LC, or 626.

Attention: To run an instrument method with a different column temperature than the instrument is currently using, equilibrate the system at this new temperature. Do this by running a suitably long equilibration or condition column function ahead of any acquisition with the new method.

Temperature set

Required operating temperature of the optional internal column heater. This value must be less than or equal to the Temp High Limit setting.

Range: 0 to 99 °C (0 = Off)

Minimum setting: 5 °C over ambient

Maximum for RCSS heater: 64 °C

Maximum for stainless steel heater: 99 °C

Default: 0.0 °C

Temp high limit

Highest operating temperature of the optional internal column heater. This setting must be greater than or equal to the Temperature Set value.

Range: 0 to 99 °C (0 = Off)

Minimum setting: 5 °C over ambient

Maximum for RCSS heater: 64 °C

Maximum for stainless steel heater: 99 °C

Default: 25.0 °C

Note: Exceeding the Temp High Limit for more than 1.2 seconds causes a system shutdown.
