Sample History dialog box

Automatically displays the modifications made to a sample (in the Alter Sample window). Besides listing the sample modifications, the Sample History dialog box lists the name of the user who made the modifications, the time and date that they were made, and any revision comments that the user entered.

Note: The Full Audit Trail Support feature must be turned on (for the project) to automatically log sample changes in the Sample History dialog box.

You access this dialog box from the Alter Sample window by right-clicking and selecting View Sample History from the context menu.

To print a copy of the information in the Sample History dialog box, right-click and select Print from the context menu.
