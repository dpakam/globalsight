LS Calibration wizard - Calibration Standard page

Use this page to specify how to perform the calibration. The table lists narrow standards that have the same instrument method name, system, and solvent as the current channel in Review. If the table is empty, the Calibrate with narrow standard option button is disabled. If the table has rows and you select Calibrate with narrow standard, you must select at least one injection from the table before proceeding to the next page. If you select multiple standards, calibration constants from the different standards are averaged.

Once you click Next the wizard starts the calibration calculation if you selected Calibrate with narrow standard.

Note: Since the results of LS processing depend strongly on the model assumption for large molecules, the wizard recommends a mono-dispersed standard that is relatively small.
