Select projects (Copy to project)

Use this dialog box to select projects to which you want to copy information or on which you want to perform an action. All projects (except the project that you are copying from) appear in the list. You access this dialog box by selecting Database > Copy to Project from the Project window after selecting information from one of the Project window views, or by selecting various menu commands when Global Projects is selected in the Project window Multi-Project mode (Pro interface only).

When you select projects from the list and click OK, the information you selected from the Project window is copied to the selected projects.

When you select projects from the list and click OK, the information you selected from the Find Data window is copied to the selected projects.
