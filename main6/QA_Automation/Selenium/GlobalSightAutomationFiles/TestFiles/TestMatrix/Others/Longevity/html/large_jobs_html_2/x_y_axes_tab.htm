X/Y Axes tab

Use the X/Y Axes tab options in a Plot Properties dialog box to define the data that you want to view in a plot in Validation Review or in Report Publisher. Here is the original text. Plot properties vary for each plot and data type.

X/Y Axes Plot Properties:

Axis

Plot options

x-axis

Select one of these options to define the data you want to plot on the x-axis:

  • Select - Plots the data based on the field that you select from the list that appears.
  • Run Number (default) - Plot the injections by run number on the x-axis.

y-axis

Defines the data you want to plot on the y-axis based on the field that you select from the list that appears.
