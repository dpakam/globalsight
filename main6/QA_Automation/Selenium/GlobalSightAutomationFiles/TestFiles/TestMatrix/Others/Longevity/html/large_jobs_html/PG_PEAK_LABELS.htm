Peak Labels tab

Use this tab to label peaks in a processed chromatogram when displayed in the Chromatogram plot in Review or in a report.

Use this tab to label peaks in a processed chromatogram when displayed in the Chromatogram plot in Review.

Peak annotation

Determines which chromatogram data attributes appear:

  • Peak Start/End – Markers at peak start and ends.
  • Peak Maxima – Marker above each peak at the position of the peak maximum.
  • Peak Baseline – Baseline beneath each peak.
  • Annotate Unknown Peaks – Label above unknown peaks.
  • Annotate Missing Peaks – Label above missing peaks.
  • Peak Name – Name of each peak.
  • Component Name – Component name above each peak (or within the peak) based on Peak Name Placement (see below).
  • 1st Library Match (PDA/MS Only) – Spectrum name that provided the closest library match during processing. If no library match was performed, the component name is used to label the peak.
  • Use Numeric Annotations - Annotate with numeric markers.
  • Retention Time – Retention time above each peak in the Chromatogram plot.
  • Scan Number (MS) – Scan number above each peak in the Chromatogram plot.

Peak label placement

Specifies the location of the peak label:

  • Auto Fit To Peak – Peak label above each peak (or within the peak if there is insufficient space above the peak).
  • Above Peak Apex – Peak label directly above the peak apex.
  • Inside Peak – Peak label starting at the peak apex and moving toward the baseline.
  • At Peak Start – Peak label directly above the peak start marker.
  • At Peak End – Peak label directly above the peak end marker.
  • Draw Arrow to Apex – Used with the At Peak Start and At Peak End selections. Draws a line (with an arrowhead) from the peak label to the peak apex.

Peak markers

Peak start, peak end, peak drop

Symbols used for the peak start, peak end, and peak drop line markers in the Chromatogram plot. Selections: triangle, diamond, x, or ½.

Default: Peak Start and Peak End (triangle), Peak Drop (diamond)

Marker size

Line weight for the four chromatogram markers used in the plot.

Entries: 0 to 1000

Default: 100
