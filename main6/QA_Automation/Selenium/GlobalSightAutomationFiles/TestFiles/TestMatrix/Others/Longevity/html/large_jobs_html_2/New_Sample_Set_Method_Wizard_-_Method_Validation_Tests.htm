New Sample Set Method Wizard - Method Validation Tests

Use this page to select the validation tests that you want to turn on, off, or hide in the sample set method that you are creating.

For each validation test, select one of the following options:

  • To hide the column in the Sample Set Method Editor, click Hidden.
  • To turn on this validation test by default for all rows in the sample set method, click On.
  • To turn off this validation test by default for all rows, but display it as a column in the Sample Set Method Editor, click Off.

Tip: When you modify the sample set method in the Sample Set Method Editor, you can modify these selections by adding or removing columns for the validation tests in the Samples table.
