Edit GPCV Channel Set Definition dialog box

Use this dialog box to combine data from concentration (RI) and one or two viscometer channels, and to save the multiple channels as a channel set. A channel set allows you to process multiple channels of an injection for one GPCV result.

Name of GPCV channel set

Name for the GPCV channel set in the text box.

Concentration channel

Concentration (RI) channel (selected from the list).

Viscometry channel

Derived viscometer channel (selected from the list). Click for details on creating a derived viscometer channel.

Concentration channel connected to RI detector

Specifies the selected concentration channel is connected to an RI detector.
