Legend (Result Custom Fields) tab

Use this tab to display information about the custom fields in the report.

Fields list

Fields available to describe the Custom Fields. Double-click (or drag) a field in the list to add the field to the table.

Table

Specifics of the fields selected for the Custom Fields, and allows you to modify the field specifications. Fields are positioned in the order in which they are selected from the Fields list. Columns are:

  • Original Name – Unedited, original name of the selected field.
  • Field Name – Modified name of the selected field. If necessary, change the field name in the associated table cell.
  • Width – Number of spaces allotted for the field. Allowable entries depend on the selected field.
  • Precision – Number of digits reported to the right of the decimal. Allowable entries depend on the selected field.
  • Exponential Notation – Exponential notation for the field.
  • Field Type – Type of field included in the legend (Characters, String, Date, Double Precision Float, Integer, Enumerated, or Boolean Value).
