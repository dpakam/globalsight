Running Empower Applications dialog box

When logging out of an Empower Pro session with one or more applications still active, this dialog box prompts you to choose one of these two options, prior to logging out:

Option

Resulting action

Leave them running

Logs out of the Empower session, allowing all active applications to continue running.

Leave them running, but lock them

Logs out of the Empower session, allows the active applications to continue running in locked state so that no other user can access them. The word Locked appears in the application title bar.

To release a locked application, the user who locked the application must log into Empower.

Exception: If the system policy "Don’t Allow Applications to Stay Running After Logging Out" is enabled, the option "Leave them running" is not available. The option "Leave them running, but LOCK them" is automatically selected. You cannot clear this selection.

Important: If you have Administrator or other high-level Empower access privileges, and you exit Empower leaving applications running without locking them, a user with limited privileges, such as a user whose type is Guest, can log into Empower and modify these unlocked, active applications.
