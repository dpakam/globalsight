LC Calibration Point Table tab

Use this tab to define characteristics of an LC Calibration Curve Point table when it is included in a report.

Available fields list

Fields in the Empower database to include in the LC Calibration Point table. Click an item to view available fields, then double-click (or drag) the desired field in the Available Fields tree. The field is added to the Table Template as a column.

LC Calibration Point fields:

Field

Usage

Level

Level used by Empower software when the software averaged by level.

X Value

Calculated concentration or amount of the component.

Response

Detector response generated by the component.

Calc. Value

Component concentration or amount corresponding to the point on the calibration curve at the measured response for that calibration point.

% Deviation

Difference between the calculated and actual point x values expressed as a percentage of the x value.

Manual

Whether the point’s calibration data were modified, manually entered by the user, or if the calibration curve point was produced by an older version of the processing method.

Ignore

Indicates if the software ignored a component data point when calculating the calibration curve.

Result Id

Result identifier generated by the Oracle database while processing the result that produced the data point.

Channel Id

Chromatogram identifier generated by the Oracle database when the chromatogram that was processed to produce the result was acquired.

Table title

Title directly above the table. The table title can have up to 250 alphanumeric characters without restrictions. If multiple lines are required, press Ctrl+Enter to force the title onto multiple lines.

Note: When you specify a table title, ensure that it is not wider than the potential table width. If the title is longer than the generated table, the title is truncated. Avoid this problem by shortening the title or forcing it onto multiple lines.

Alignment

Position of the title above the table.

Choices: Left Alignment, Center Alignment, Right Alignment

Default: Center Alignment

Table template

Headings and order of the table columns as they are moved from the Available Fields list. To move a column, drag it to a different location in the Table Template. To delete a column from the Table Template, right-click the column name, then select Delete Column (or highlight the columns and press Delete). You cannot leave blanks between columns.

Note: You can also specify sorting order in individual columns of the Table Template.

Column properties

Accesses the Column Properties dialog box to modify attributes of a specified column in the Table tab. To modify column attributes, select the corresponding column in the Table Template, then click Column Properties. After redefining column attributes, click OK to return to the Table tab.

Print blank columns

Prints table columns that do not contain values.

Default: Enabled
