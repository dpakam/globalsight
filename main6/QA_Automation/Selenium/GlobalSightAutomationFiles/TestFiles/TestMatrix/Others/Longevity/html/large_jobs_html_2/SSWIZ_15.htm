Inject narrow samples/Inject broad samples/Inject no samples

Defines your samples. Note the following:

  • When you select Inject Narrow Samples, Empower software: uses narrow integration parameters to detect and integrate the peak(s) in the chromatogram, uses viscosity narrow integration parameters to integrate the viscometer channel data (GPCV only), quantitates the integration results (calculates peak molecular weight) using a calibration curve, and calculates the intrinsic viscosity of the peak (GPCV only).
  • When you select Inject Narrow Samples, Breeze software: uses narrow integration parameters to detect and integrate the peak(s) in the chromatogram, and quantitates the integration results (calculates peak molecular weight) using a calibration curve.
  • When you select Inject Broad Samples, Empower software: uses broad integration parameters to detect and integrate the peak(s) in the chromatogram, uses viscosity broad integration parameters to integrate the viscometer channel data (GPCV only), quantitates the integration results (calculates peak molecular weight averages and slice weights) using a calibration curve, and calculates the intrinsic viscosity of the distribution and peak and the branching index g' (GPCV only).
  • When you select Inject Broad Samples, Breeze software: uses broad integration parameters to detect and integrate the peak(s) in the chromatogram and quantitates the integration results (calculates peak molecular weight averages and slice weights) using a calibration curve.
  • When you select Inject No Samples, your sample set method includes only Inject Narrow Standard and/or Inject Broad Standard steps. When you make this selection, all parameters in this page of the wizard are grayed out.

Sample information section

Use this section to define information about your narrow or broad sample injections.

Number of samples

Specifies the number of narrow or broad sample vials for the sample set.

Valid entries: 1 to 999

The software determines the maximum number of samples you can enter in this field according to:

  • The number of standard vials you entered in the Describe Narrow Standard Injections and/or Describe Broad Standard Injections wizard pages.
  • The capacity of your autoinjector or autosampler tray or carousel.

Number of injections per vial

Specifies the number of injections from each sample vial.

Valid entries: 1 to 99

Injection volume

Specifies the volume in microliters of the injected sample, valid entries are 0.0 to 29999.9. Note the following considerations:

  • For Waters autosamplers, small injector loops may omit the maximum volume. Ensure hardware configuration is compatible with your desired injector size.
  • For Agilent 1100 autosamplers, the maximum injection volume is the seat capillary volume, plus the syringe size. Ensure that the hardware configuration is specified correctly in your method.

Specifies the volume in microliters of the injected sample, valid entries are 0.0 to 29999.9.

Tip: Small injector loops may omit the maximum volume. Ensure hardware configuration is compatible with your desired injector size.

Run time

Specifies the length of time in minutes to collect data for the standard.

Valid entries: 0.01 to 650.00 minutes

Maximum # of samples per group

Specifies the maximum number of samples in a sample group. This field appears only if you chose to inject standards before or around sample groups (in the Select Location of Standards page). Empower software determines the maximum number you can enter in this field according to your Number of Sample selection and the capacity of your autoinjector or autosampler tray or carousel. The value you enter for Number of Samples is divided by the value for Maximum # of Samples Per Group to determine the number of sample groups for your sample set.

k

Specifies the optional Mark-Houwink constant, k, where Log k is the intercept of the viscosity law plot. Valid entries are 0.000000001 to 2.000000000. Keep in mind the following:

  • Enter a value in this field if you specified Universal Calibration as a calibration technique when you created your processing method.
  • Make sure you enter the same alpha value that you entered for narrow and/or broad standards.
  • Leave this field set to the default of 1.000000000 if you specified Relative Calibration as a calibration technique in your processing method.

Alpha

Specifies the optional Mark-Houwink constant, alpha, which is the slope of the viscosity law plot. Valid entries are 0.000000 to 1.000000. Keep in mind the following:

  • Enter a value in this field if you specified Universal Calibration as a calibration technique when you created your processing method.
  • Make sure you enter the same alpha value that you entered for narrow and/or broad standards.
  • Leave this field set to the default of 0.000000 if you specified Relative Calibration as a calibration technique in your processing method.

Options button

Displays the Enter Optional Information page, where you can specify a Sample Weight factor, a Dilution factor, a Data Start time, and/or a Next Injection Delay time.

Method set section

Use this section to define or create the method set used to acquire, process, and/or report data for your samples.

Select

Specifies the method set used to acquire, process, and/or report data for the samples. Select the appropriate method set from the drop-down list. If you are using the Pro interface, click Create New if you need to create a new method set.

Specifies the method set used to acquire, process, and/or report data for the samples. Select the appropriate method set from the drop-down list.

Create New button (Pro interface only)

Displays the first page of the New Method Set Wizard, where Empower software interactively leads you through the steps to create a new method set.

After making your selections, click Next to advance to the next wizard page.
