Scaling tab

Use this tab to customize scaling of plots for display in Review, Compare, or Report Publisher. You can also adjust the scaling of the Run Samples real-time plot.

Use this tab to customize scaling of plots for display in Review or Compare. You can also adjust the scaling of the Run Samples real-time plot.

Note: You can also graphically scale a plot in Review by drawing a zoom box around a section of the displayed plot.

Scaling type (chromatogram plot only)

Type of scaling to use for displaying a chromatogram plot in Review, Compare or Report Publisher:

  • Autoscale – Scales the chromatogram to the highest data point. The data does not have to constitute a detected peak.
  • Highest Peak – Scales the chromatogram to the absorbance of the highest integrated peak in the region between the X-Start and X-End time points (see below).
  • 2nd Highest Peak – Scales the chromatogram to the absorbance of the second highest integrated peak in the region between the X-Start and X-End time points (see below).
  • Scale to Baseline – Scales the chromatogram to the lowest/highest peak start and lowest/highest peak end in the region between the X-Start and X-End time points (see below).
  • % Highest Peak – Scales the chromatogram to a user-specified height percentage of the highest integrated peak in the region between the X-Start and X-End time points (see below).

Entries: 0.1 to 100.00

Default: 100

Scaling type (calibration curve only)

Type of scaling to use for displaying a calibration curve plot:

  • Scale to Point – Scales the calibration curve to the highest data point in the curve.
  • Scale to Curve – Scales the calibration curve to the top of the curve.

Normalization

Specifies that the display of data along the x-axis or y-axis is to be 100% of the full data range. Normalization is valid typically with chromatograms, calibration curves, spectra, and GPC distributions. The x-axis and/or y-axis tick marks, labels, and scale do not appear in the plot when the data is normalized.

Normalization is valid typically with chromatograms, calibration curves, spectra, and LS and GPC distributions. The x-axis and/or y-axis tick marks, labels, and scale do not appear in the plot when the data is normalized.

Note: Use the choices in this section to normalize MS peaks and cross channel internal standard peaks that appear in the Internal Standard Peak Plot report. These peaks are shown at the proper retention time by default.

Normalization choices:

  • Normalize X – Displays the plot at 100% of the plot’s x-axis (which, depending on the plot data, could be time, amount, and so on), thereby normalizing the x-axis of all data appearing in the plot.
  • Normalize X – Displays the plot at 100% of the plot’s x-axis (which, depending on the plot data, could be time, amount, concentration, and so on), thereby normalizing the x-axis of all data appearing in the plot.

Note: For overlaid chromatograms with a normalized x-axis, if the first points and last points are not the same the chromatograms will have different x-axes and the same retention time on two chromatograms appears at different points along the x-axis.

  • Normalize Y – Displays the plot at 100% of the highest data point on the y-axis (which, depending on the plot data, could be voltage, absorbance units, response, and so on), thereby normalizing the y-axis to all data appearing in the plot. Normalize Y also enables use with these selections to further define normalization:
  • Min/Max – Scales the plot to the largest data point on the y-axis.
  • Use Full X Range – Used with the Min/Max selection to specify the use of a full time window for the x-axis when normalizing the y-axis. This selection is enabled only when you select the Min/Max option.
  • At X Value – Scales the plot to a specified x-axis value when normalizing the y-axis.

Note: The Normalization selections disable associated scaling selections in the Scaling section. For example, enabling Normalize Y disables the Y-Start and Y-End scaling options. If you enter values in the X-Start and X-End scaling options, Y-axis normalization occurs only over the specified x-axis range.

Scaling

Parameters to scale plots. You can specify ranges for one or more scaling parameters, allowing you to mix manual scaling and autoscaling in reports.

Chromatogram plot or PDA/MS contour plot scaling settings:

  • X-Start – Time after injection at which to start the x-axis of the plot.
  • X-End – Time after injection at which to stop the x-axis of the plot.
  • Y-Start – Low value (in detector units) for scaling the y-axis.
  • Y-End – High value (in detector units) for scaling the y-axis.

Chromatogram plot or PDA contour plot scaling settings:

  • X-Start – Time after injection at which to start the x-axis of the plot.
  • X-End – Time after injection at which to stop the x-axis of the plot.
  • Y-Start – Low value (in detector units) for scaling the y-axis.
  • Y-End – High value (in detector units) for scaling the y-axis.

LC calibration curve scaling settings:

  • X-Start – Component amount at which to start the x-axis of the plot.
  • X-Start – Component amount or concentration at which to start the x-axis of the plot.
  • X-End – Component amount at which to end the x-axis of the plot.
  • X-End – Component amount or concentration at which to end the x-axis of the plot.
  • Y-Start – Component response at which to start the y-axis of the plot.
  • Y-End – Component response at which to end the y-axis of the plot.

GPC/V calibration curve scaling settings:

  • X-Start – Elution volume or retention time at which to start the x-axis of the plot.
  • X-End – Elution volume or retention time at which to end the x-axis of the plot.
  • Y-Start – Log MW[h] or Log Hydrodynamic Volume at which to start the y-axis of the plot.
  • Y-End – Log MW[h] or Log Hydrodynamic Volume at which to end the y-axis of the plot.

PDA/MS spectral plot and library match plot scaling settings:

  • X-Start – Low value (in wavelength) of the x-axis.
  • X-End – High value (in wavelength) of the x-axis.
  • Y-Start – Low value (in absorbance units or intensity) of the y-axis.
  • Y-End – High value (in absorbance units or intensity) of the y-axis.

PDA spectral plot scaling settings:

  • X-Start – Low value (in wavelength) of the x-axis.
  • X-End – High value (in wavelength) of the x-axis.
  • Y-Start – Low value (in absorbance units or intensity) of the y-axis.
  • Y-End – High value (in absorbance units or intensity) of the y-axis.

GPC distribution plot scaling settings:

  • X-Start – Low value for the x-axis.
  • X-End – High value for the x-axis.
  • Y-Start – Low value for the left y-axis.
  • Y-End – High value for the left y-axis.

Dissolution plot scaling settings:

  • X-Start – Low transfer time value for the x-axis.
  • X-End – High transfer time value for the x-axis.
  • Y-Start – Low value for the y-axis (which can be Amount Dissolved or Percent Dissolved).
  • Y-End – High value for the y-axis (which can be Amount Dissolved or Percent Dissolved).

Get values from plot (Review and Compare)

Reads the x- and y-axis values from the active plot in Review (or Compare) or from another plot that you want to use to scale the other overlaid plots using the values as the X-Start, X-End, Y-Start, and Y-End scaling parameters.

Note: If you normalize the x- or y-axis of the plot, associated plot values are not used when you select Get Values From Plot and the button is disabled.
