Define plates for Sample Set

This dialog box appears when you click Define Plates from the Sample Set Method wizard, or when you select the Edit Plates tool or Plates command from the Edit menu in Run Samples, the Sample Set Method Editor, or the Alter Sample window.

Use this dialog box to do these things:

  • Select the pre-defined plate types from which injections are to be made
  • Select the position of the plates within the autosampler
  • Activate the vial/well locations (from which injections are to be made) for each selected plate
  • Specify the Plate Sequencing Mode (injection order) for the vial/well locations
  • Insert or append active vial/well locations into the Samples table
  • Optionally create a new plate type

2790 layout check box

Displays the Waters 2790 plate layout in the dialog box (the default setting). In this layout, the left section of the dialog box displays the four-position 2790 plate carrier. When deselected, the layout changes to a tabular view, typically used for autosamplers or systems other than the Waters 2790.

Create New Plate Type button

Displays the New Plate Type dialog box, where you can create a new plate for your system (typically created from Configuration Manager).

Clear Plates button

When you create a sample set method using the Sample Set Method wizard, this button clears the dialog box of any selected plates, allowing you to reselect plates, plate positions, and vial/well locations.

When you manually create or modify a sample set method, this button displays a warning that continuing with the clear removes all injection rows from the Samples table. This button also clears the Define Plates for Sample Set Method dialog box of any selected plates, allowing you to reselect plates, plate positions, and vial/well locations.

Plate Sequencing Mode buttons

Specifies the injection order for the selected vials in the displayed plate. This injection order of selected vials is transferred to Samples table rows when you click Insert or Append. Sequencing selections are:

  • (Horizontal Discontinuous) – Injections occur from left to right across each row, starting at the top of the plate.
  • (Horizontal Serpentine) – Injections occur from left to right until the end of the row, then right to left starting at the next row.
  • (Vertical Discontinuous) – Injections occur from top to bottom, starting at the left of the plate.
  • (Vertical Serpentine) – Injections occur from top to bottom until the end of the column, then bottom to top starting at the next column.

Note: The Plate Sequencing Mode buttons have no effect on how the vials are labeled (referenced). You set vial reference in the Referencing tab when creating a plate type. Plate sequencing affects only the order of the vials once inserted or appended into the Samples table. The Samples table defines the vial injection order.

Select plate types and plate positions in this section of the dialog box. You can create a new plate type from this dialog box by clicking the Create New Plate Type button. Plate position refers to the position of the plate within the autosampler.

Plate/Vial section

For each selected plate, select the vial/well locations from which injections are to be made. Selected locations appear in color.

To select:

  • Individual vial/well, click the vial/well
  • Single group of vial/wells, drag the selection box around the desired group
  • Multiple groups of vial/wells, press Ctrl and drag the selection box around each desired group
  • All wells, right-click, then select Activate All Wells from the context menu

To deselect:

  • Individual vial/well, click the vial/well
  • Group of vial/wells, press Shift and drag the selection box around the desired group
  • All wells, right-click, then select Clear All Activate Wells (see above) from the context menu

Selection differences

Vial/well selection differs according to whether you accessed the Define Plates for Sample Set Method dialog box from the Sample Set Method wizard or from the Samples table.

Vial/well selection differs according to whether you accessed the Define Plates for Sample Set Method dialog box from the Sample Set Method wizard, from the Samples table, from the Run Samples Only mode in Open Access, or from the Run Samples Using a Template mode in Open Access.

When you access this dialog box from the wizard, the vial/wells that you select represent a range of locations available for injection. Later, in a wizard page, you specify the actual vial/well locations that are to be injected. When you access this dialog box from the Samples table, the vial/wells that you select are the actual locations from which injections are made.

When you access this dialog box from the wizard or from the Run Samples Using a Template mode in Open Access, the vial/wells that you select represent a range of locations available for injection. Later, in a wizard page, you specify the actual vial/well locations that are to be injected. When you access this dialog box from the Samples table or from the Run Samples Only mode in Open Access, the vial/wells that you select are the actual locations from which injections are made.

Inject pull-down menu, Insert button, Append button

When you make a selection from the inject pull-down menu (Inject Standards, for example), then click the Insert or Append button, the active well/vial locations are inserted or appended into the Samples table (with the Inject function appearing in the Function column).

Note: These controls are not available when this dialog box is accessed from the Sample Set Method wizard.

Note: These controls are not available when this dialog box is accessed from the Sample Set Method wizard or Open Access.

To insert or append active well/vial locations:

  • Insert or append into an empty Samples table, click Insert or Append.
  • Append to existing rows at the end of the Samples table, click Append.
  • Insert within existing rows of the Samples table, close this dialog box. In the Samples table, click the row number immediately below where you want to insert active vial/wells. Select Plates from the Edit menu to reopen this dialog box. If necessary, select the vial/well locations and an inject function, then click Insert.
