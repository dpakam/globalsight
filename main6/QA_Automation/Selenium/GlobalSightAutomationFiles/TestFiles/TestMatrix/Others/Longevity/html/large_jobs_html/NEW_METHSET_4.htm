Method Set wizard – Default Methods page

Use this page to specify the type of PDA or MS derived channel or GPCV/LS channel set to create for your application. Choices:

  • Define PDA Derived Channel – Creates a Single Wavelength, Max Plot, Total Plot, or Ratio Plot derived channel.
  • Define MS Derived Channel – Creates a Single Mass, Max Plot, Total Plot, Ratio Plot, or Total Ion Chromatogram (TIC) derived channel.
  • Define GPCV/LS Channel Set – Creates a GPCV/LS channel set, which is a combination of data acquired from the concentration channel, up to two viscometer channels, and LS channels used in a method set to process GPCV/LS data.

After making your selection, click Next.
