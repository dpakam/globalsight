Smoothing and Time Offset Tab

You can apply smoothing and/or add a time offset to data. Smoothing decreases noise by filtering data points. The smoothing types include Savitsky-Golay and mean. Savitsky-Golay smoothing averages data points, whereas mean smoothing calculates the mathematical mean.

Use a time offset to adjust the time axis of the data in the channel relative to the time axes of the other channels in the injection.

To apply smoothing and/or a time offset:

  1. Click the Smoothing/Offset tab.
  2. Enter a value for time offset (in minutes) and select the smoothing type and level for the concentration detector.
  • If you are processing GPCV data, you can define the time offset and smoothing parameters for the concentration and viscometry detectors.
  • If you are processing LS data, you can define the time offset and smoothing parameters for the light scattering detector.
  • If you are processing GPCVLS data, you can define the time offset and smoothing parameters for the concentration, viscometry, and light scattering detector.

Tip: If you did not specify a time offset and set the smoothing parameters to none, select the supersede checkbox to override the time offset and smoothing parameters specified the channel's derived channel. The data is adjusted using the time offset and smoothing parameters in the processing method (no time offset and no smoothing).

  1. Click File > Save to save the processing method.
