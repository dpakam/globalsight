Referencing tab

Use this tab to configure how and in what order the software labels and references the wells in a plate.

Origin (XY format only)

Location of the first well to be labeled in the referencing scheme.

Choices: Top Left, Bottom Left, Top Right, and Bottom Right

Default: Top Left

Scheme (XY and W2790/5 formats only)

Referencing

How the software labels the wells.

Choices: XY (coordinates) and Sequential

Default: XY

Horizontal

For the XY labeling scheme, the software labels the horizontal (X) component of each well with either a letter or a number.

Choices: ABC or 123

Default: 123

Vertical

For the XY labeling scheme, the software labels the vertical (Y) component of each well with either a letter or a number.

Choices: ABC or 123

Default: ABC

Sequential Continuous

For this labeling scheme, the first well in the next row or column is always adjacent to the last well of the previous row or column; otherwise the labeling starts at the beginning of the next row or column after reaching the end of the previous row or column.

Default: Deselected

Horizontal First Priority

For the XY labeling scheme, the first component of the well label refers to the x-axis position of a well within a row (the column the well is in), using coordinates in the format X,Y (selected); otherwise it refers to the y-axis position of a well within a column (the row the well is in), using coordinates in the format Y,X (deselected).

For the Sequential labeling scheme, the wells in a row are labeled before wells in the next row are labeled. When deselected, specifies that the wells in a column are labeled before wells in the next column are labeled.

Default: Deselected

Terminology

Plate

This term refers to a platform of this type. Enter up to 30 alphanumeric characters.

Default: Plate

Note: You can specify other terms, for example, rack, tray, or carousel.

Well

This term refers to sample containers in a plate of this type. Enter up to 30 alphanumeric characters.

Default: Well

Note: You can specify other terms, for example, tube, bottle, or vial.
