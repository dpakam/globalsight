General tab (5890 or 5890S)

5890 consideration

Use this tab to enable the channel enable, detector enable, and purge enable 5890 GC System functions for use with Empower software to define the sampling rate for digital data transmission, and to set the on/off times for the purge valves.

5890S considerations

Use this tab to enable 5890S channels and detectors for use with Empower software to define the sampling rate for digital data transmission.

Dual tower mode

To run a dual tower mode, enable both channels and both detectors. If you are using a:

  • 7673S Autosampler, you must enable both towers.
  • 7673 Autosampler, you can make dual manual injections.

Ch 1 enable/Ch 2 enable

Specifies that the channel transmit digital data through the 5890 GC System RS-232 port to Empower software. The optional RS-232 board must be installed in the 5890 GC System before digital data can be transmitted. After enabling a digital channel, define the Source and Units parameters in the Channel 1 or Channel 2 tab.

Choices: Enabled or Disabled

Default: Enabled for Ch 1 and Ch 2

Note: You can choose to simultaneously transmit digital data (RS-232 port on the 5890) and analog data (Signal 1 and Signal 2 terminals on the 5890) to the Empower workstation. You need not enable the analog outputs; they are always active. The analog outputs must be converted to digital signals by an A/D converter (typically a busSAT/IN or busSAT/IN 2 Module) before transmission to the Empower workstation. See the Empower System Installation and Configuration Guide for information on connecting your 5890 to the Empower workstation through a busSAT/IN Module.

Detector A enable/detector B enable

Enables Detector A or Detector B in the 5890 GC System. After enabling Detector A or B, enter detector parameters in the corresponding Detector tab.

Choices: Enabled or Disabled

Default: Disabled for Detectors A and B

Note: Enable Detector A or Detector B before specifying detector parameters in the Detector A or Detector B tab. If you specify detector parameters in the Detector tab without first enabling the detector, the parameter settings do not take effect.

Sampling rate

Sampling rate used to collect GC data for channels 1 and 2 when transmitting data through the RS-232 output. See the Ch 1 Enable and Ch 2 Enable parameters.

Choices: 1.25, 2.5, 5.0, 10.0 and 20.0 points per second

Default: 1.25 points per second

Note: Set the sampling rate for GC data collected through the analog channels in the busSAT/IN Module, or the busSAT/IN Module (SAT/IN 2) tabs, depending on which module is connected to the GC.

Purge A enable/purge B enable (5890 only)

Enables purge valve A and/or B. Disabling a purge valve turns the purge valve off. Enabling a purge valve purges the inlet of solvent vapor.

Choices: Enabled or Disabled

Default: Disabled

Purge A on time/purge B on time (5890 only)

Sets the time when the purge begins (0.00 time = injection time). Select the corresponding Purge Enable parameter for this parameter to take effect.

Default: 0.00 min

Purge A off time/purge B off time (5890 only)

Sets the time when the purge stops (0.00 time = injection time). Select the corresponding Purge Enable parameter for this parameter to take effect.

Default: 0.00 min

Note: You can display the purge time settings on the front panel of the 5890S. If you cannot determine the purge value settings, see the Agilent 5890 Series II Gas Chromatograph Operating Manual for details about viewing these values.
