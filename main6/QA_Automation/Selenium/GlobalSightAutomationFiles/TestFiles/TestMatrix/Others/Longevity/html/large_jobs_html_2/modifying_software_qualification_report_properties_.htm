Modifying Software Qualification Report Properties

You can modify the properties for the Software Qualification Report group in a qualification report method. For example, you can specify whether to include information in a report for one node or all nodes that you qualified, and whether to produce a summary report or a detailed report.

Requirement: Activate your Empower SystemsQT for Software license option.

To modify the Software Qualification Report Properties:

  1. In Browse Project, select a qualification project from the project list, and then click OK.
  2. In the Project window, click the Methods tab.
  3. On the Methods tab, select the report method containing the Qualification Report Group whose properties you want to modify.
  4. With the method selected, right-click, and then click Preview/Publisher.
  5. In the Report Method, on the Software Qualification Report group, right-click, and then click Software Qualification Report Properties.

Software Qualification Report Properties:

Info tab property

Information

Nodes included: This node only or All Nodes

Specify whether to report information for the current node or all nodes. All nodes are those computers to which you have access.

Node information

Choose whether to include node information in the report.

Verify files information

Specify whether to include verify files information in the report, and whether you want a summary report or detailed report (includes all file checksum details).

IQ Status

Reports IQ status information specified in the Configuration Manager IQ/OQ Node properties.

IQ node information

Reports information specified in the Configuration Manager Node properties Node Info Tab.

OQ Status

Reports OQ status information specified in the Configuration Manager IQ/OQ Node properties.

Tip: If your system includes the Method Validation Manager option, you can report OQ MVM status.
